홈 화면 > 프로젝트 선택 > ***관리*** > ***에이전트 설정***

에이전트를 재기동하거나 삭제하는 방법을 알아봅니다.

## 에이전트 재기동

![에이전트 재기동](/img/mysql-rds-agent-settings-restart.png)

에이전트를 재기동하려면 에이전트 설정 옵션을 변경하고 화면 오른쪽 위에 ***적용*** 버튼을 선택하세요.


## 에이전트 삭제

![에이전트 삭제](/img/mysql-rds-agent-remove.png)

***에이전트 목록***에서 에이전트 항목 오른쪽에 ![삭제 아이콘](/img/ico-trash.svg) 버튼을 선택하세요. 확인 메시지 창이 나타나면 ***확인*** 버튼을 선택하세요. 선택한 에이전트를 목록에서 삭제합니다.
