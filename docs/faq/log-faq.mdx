와탭 로그 모니터링 서비스 사용자들이 자주 묻는 질문을 확인해 보세요.

<div class ='faq'>

### 로그 복원

|Q. 로그를 수집한 후 복구할 수 있나요?|
|---|

> A. 현재 로그 수집은 보안 용도가 아닌 분석 용도로 제공합니다. SaaS형의 경우 로그 복원 기능을 제공하지 않습니다. 
>:::note
>설치형의 경우 로그 원문 복원 기능을 제공합니다.
>:::

<br/>

### 타사 로그 솔루션 연동

|Q. 다른 로그 솔루션과 연동할 수 있나요?|
|---|

>A. 현재 타사 로그 솔루션과의 연동 기능은 제공하지 않습니다.

<br/>

### 에이전트 로그 전송 주기

|Q. 에이전트는 로그를 언제 전송하나요?|
|---|

>A. 에이전트의 로그 버퍼 용량(64 KB)에 도달하거나 전송 주기(2초)에 도달한 경우 로그를 전송합니다.

<br/>

### 로그 백업 및 복구

|Q. 로그를 백업하고 복구할 수 있나요?|
|---|

>A. 수집된 로그는 일자 단위로 저장됩니다. 과거 파일을 백업해 두고 해당 폴더에 다시 복구할 수 있습니다. 네트워크 전송 시 압축해 전송하며, 수집 서버에서 수신 시 압축을 해제해 저장합니다.

<br/>

### 데이터 삭제 주기

|Q. 데이터 삭제 주기는 어떻게 되나요?|
|---|

>A. 시간 단위로 삭제합니다.
