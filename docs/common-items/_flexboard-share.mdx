Flex 보드를 다른 계정의 사용자와 공유하거나 재사용할 수 있습니다.

## 구성원과 Flex 보드 공유하기

사용자가 생성한 Flex 보드를 다른 사용자와 공유해 동일한 대시보드를 볼 수 있습니다. 이를 통해 업무를 공유할 수 있습니다. 공유한 대시보드를 수정한 경우 공유 받은 모든 사용자는 동일한 대시보드를 확인할 수 있습니다. 이 기능은 홈 화면 > ***Flex 보드*** 메뉴에서만 이용할 수 있습니다.

1. ***Flex 보드*** 메뉴의 ***대시보드 목록***에서 공유하려는 대시보드의 ![공유 아이콘](/img/ico-flexboard-share.svg) 버튼을 선택하세요.

  ![Flex 보드 공유](/img/flexboard-share-01.png)

1. ***대시보드 공유하기*** 창이 나타나면 ***+ 계정*** 버튼을 선택하세요.

  ![Flex 보드 공유](/img/flexboard-share-02.png)

1. 대시보드를 공유할 사용자 계정을 선택하세요.

  ![Flex 보드 공유](/img/flexboard-share-06.png)

    * 공유 대상을 추가하려면 ***+ 계정*** 버튼을 선택하세요. 사용자 계정을 추가로 선택할 수 있습니다.
    * 대시보드를 공유할 수 있는 사용자 계정은 선택한 프로젝트에 소속된 사용자 계정입니다.
    * 대시보드의 모든 프로젝트에 대한 **수정 권한**이 없는 경우 `읽기 전용` 태그가 나타납니다.

1. ***대시보드 공유하기*** 버튼을 선택하세요. ***읽기 전용*** 체크박스를 선택하면 공유 받은 사용자는 대시보드를 수정할 수 없습니다.

  ![Flex 보드 공유](/img/flexboard-share-03.png)

대시보드를 공유한 항목은 ***대시보드 목록***에서 `공유` 태그가 표시된 것을 확인할 수 있습니다. 공유 태그에 마우스를 오버하면 대시보드를 공유한 구성원에 대한 정보를 확인할 수 있습니다.

| 대시보드를 공유 받은 경우 | 대시보드를 공유한 경우(대시보드 소유자)
| -- | -- |
| ![Flex 보드 공유](/img/flexboard-share-04.png) | ![Flex 보드 공유](/img/flexboard-share-05.png) |

:::note 

**대시보드 공유 조건**

* 대시보드 공유 기능은 홈 화면 > ***Flex 보드*** 메뉴에서만 이용할 수 있습니다.

* 대시보드를 공유하려는 멤버는 대시보드의 모든 프로젝트에 대한 **수정 권한**을 가지고 있어야 합니다. 

* 모든 프로젝트에 대한 **수정 권한**이 없는 상태에서 대시보드를 공유하면 ***읽기 전용*** 모드로 공유합니다. 권한에 대한 자세한 내용은 [다음 문서](../project/project-structure)를 참조하세요.

* 공유 받은 대시보드를 다른 사용자에게 다시 공유할 경우 해당 대시보드의 속성은 ***읽기 전용*** 모드가 아니어야 합니다.

* 공유 조건에 해당하는 경우에만 대시보드 공유 버튼이 노출됩니다.

  ![Flex 보드 공유](/img/flexboard-sharable-list.png)

:::

:::note

* 대시보드 소유자가 대시보드를 삭제하는 경우 대시보드를 공유한 모든 사용자 계정에서도 해당 대시보드를 삭제합니다.
* 대시보드를 공유 받은 사용자가 대시보드를 삭제하면, 해당 사용자의 ***대시보드 목록***에서만 삭제합니다. 원본 대시보드는 그대로 유지합니다.

:::
