## 사용 설정 안내

***파일 로그 알림*** 메뉴를 이용하려면 XOS 에이전트에 다음 옵션을 추가하세요. 키워드를 설정하거나 제외해 로그 발생 내역을 조회할 수 있습니다.

```ini title='xos.conf'
file={fullpath} # 로그 파일 경로 및 파일명
filekey={알림 발생 키워드}
fileexcl={알림 제외 키워드}
```

:::note

* 로그 파일에 대한 읽기 권한이 필요합니다.
* 여러 개의 파일을 설정하려면 `file1`, `filekey1`, `fileexcl1` / `file2`, `filekey2`, `fileexcl2`와 같은 형식으로 옵션을 추가하세요.
* 한 개의 파일에서 여러 개의 키워드를 설정하려면 `filekey=test1|test2`와 같은 형식으로 수직선(|)을 구분자로 이용하세요.
* `filekey`와 `fileexcl`을 같이 적용할 경우 AND 조건에 해당합니다. 예를 들어, `filekey=error`, `fileexcl=row`로 옵션을 추가했다면 로그 파일에 error, row를 포함하는 문자열은 ***파일 로그 알림***에서 표시되지 않습니다.

:::
