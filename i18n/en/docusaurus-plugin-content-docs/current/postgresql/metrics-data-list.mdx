Database metrics are a comprehensive collection of database workloads and resource usages. The WhaTap monitoring service continuously collects metric data to identify bottlenecks in the database and provides data to help identify the root cause of the performance issue.

For more information about other metrics, see [the link](https://www.postgresql.org/docs/).

<div class='db-metric'>

| Metric                   | Description                                                                                                                                                                                                   |
| ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `active_sessions`        | Number of sessions running queries                                                                                                                                                                            |
| `blk_read_time`          | Time in milliseconds spent reading data file blocks from the backend of that database, if enabled by `track_io_timing`. Otherwise, the value`0`.                                                              |
| `blk_write_tiem`         | Time in milliseconds spent writing data file blocks from the backend of that database, if enabled by `track_io_timing`. Otherwise, the value`0`.                                                              |
| `blks_hit`               | Number of cases where the disk block was already found in the buffer cache and did not require read (this only includes hits from the PostgreSQL buffer cache, not the operating system's file system cache). |
| `blks_hit_ratio`         | Buffer cache's hit rate<br/>sum(blks_hit)\*100/sum(`blks_hit` + `blks_read`)                                                                                                                                  |
| `blks_read`              | Number of disk blocks read from the database                                                                                                                                                                  |
| `buffers_checkpoint`     | Number of buffers written during checkpoint (`check_point`)                                                                                                                                                   |
| `check_point`            | `check_point` indicates a time point at which all data files have been physically written among the sequential transaction logs. `check_point` occurred count                                                 |
| `checkpoints_write_time` | Total time (in milliseconds) spent for the checkpoint processing portion where files are written to disk.                                                                                                     |
| `checkpoints_tiemed`     | Number of scheduled checkpoints performed                                                                                                                                                                     |
| `commit_count`           | Number of committed transactions                                                                                                                                                                              |
| `conflicts`              | Number of queries aborted due to crashes during recovery on the database.                                                                                                                                     |
| `connect_error`          | Server connection failure count                                                                                                                                                                               |
| `deadlock_count`         | Number of deadlocks detected in the database.                                                                                                                                                                 |
| `dml_count`              | DML (Data Manipulation Language) statement execution count.                                                                                                                                                   |
| `fatal_count`            | Number of database sessions that ended by fatal errors.                                                                                                                                                       |
| `index_hit_ratio`        | Index rate in cache <br/>(sum(`idx_blks_hit`) - sum(`idx_blks_read`)) / sum(`idx_blks_hit`) \* 100                                                                                                            |
| `lock_wait_sessions`     | Number of waits to acquire the current row lock                                                                                                                                                               |
| `long_running_sessions`  | Number of queries running indefinitely on the current database.                                                                                                                                               |
| `max_connections`        | **Too ManyConnection Error** may occur if the maximum number of concurrent client connections exceeds or the connection count is insufficient.                                                                |
| `numbackends`            | Number of backends connected to the database, or `nULL` for shared objects                                                                                                                                    |
| `rollback_count`         | Number of transactions rolled back in the database.                                                                                                                                                           |
| `table_hit_ratio`        | Ratio of table blks hit (cache hit) to blks read (disk read count)                                                                                                                                            |
| `temp_bytes`             | Total amount of data written to temporary files by queries in the database                                                                                                                                    |
| `temp_files`             | Number of temporary files created by queries on the database.                                                                                                                                                 |
| `total_runtime`          | Sum of execution times of the sessions running queries                                                                                                                                                        |
| `total_sessions`         | Number of all sessions                                                                                                                                                                                        |
| `tup_deleted`            | Number of rows deleted by the queries to the database                                                                                                                                                         |
| `tup_fetched`            | Number of rows fetched by the queries to the database                                                                                                                                                         |
| `tup_inserted`           | Number of rows added by the queries to the database                                                                                                                                                           |
| `tup_returned`           | Number of rows returned by the queries to the database                                                                                                                                                        |
| `tup_updated`            | Number of rows updated by the queries to the database                                                                                                                                                         |
| `uptime`                 | Time elapsed since the server startup                                                                                                                                                                         |
| `vacuum_running_count`   | Number of running vacuum functions                                                                                                                                                                            |
| `wait_sessions`          | Number of queued sessions                                                                                                                                                                                     |
| `warning_count`          | Total number of errors, alerts, and notes                                                                                                                                                                     |
| `cpu(xos)`               | OS metrics collected by the XOS agent                                                                                                                                                                         |
| `cpu_idle(xos)`          | OS metrics collected by the XOS agent                                                                                                                                                                         |
| `cpu_iowait(xos)`        | OS metrics collected by the XOS agent                                                                                                                                                                         |
| `cpu_sys(xos)`           | OS metrics collected by the XOS agent                                                                                                                                                                         |
| `cpu_user(xos)`          | OS metrics collected by the XOS agent                                                                                                                                                                         |
| `mem(xos)`               | OS metrics collected by the XOS agent                                                                                                                                                                         |
| `mem_available(xos)`     | OS metrics collected by the XOS agent                                                                                                                                                                         |
| `mem_buffcache(xos)`     | OS metrics collected by the XOS agent                                                                                                                                                                         |
| `mem_free(xos)`          | OS metrics collected by the XOS agent                                                                                                                                                                         |
| `mem_swapfree(xos)`      | OS metrics collected by the XOS agent                                                                                                                                                                         |
| `mem_swaptotal(xos)`     | OS metrics collected by the XOS agent                                                                                                                                                                         |
| `mem_total(xos)`         | OS metrics collected by the XOS agent                                                                                                                                                                         |
| `mem_used(xos)`          | OS metrics collected by the XOS agent                                                                                                                                                                         |
| `net_recv_bytes(xos)`    | OS metrics collected by the XOS agent                                                                                                                                                                         |
| `net_recv_packets(xos)`  | OS metrics collected by the XOS agent                                                                                                                                                                         |
| `net_send_bytes(xos)`    | OS metrics collected by the XOS agent                                                                                                                                                                         |
| `net_send_packets(xos)`  | OS metrics collected by the XOS agent                                                                                                                                                                         |
