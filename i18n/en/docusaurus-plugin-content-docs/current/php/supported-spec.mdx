## PHP agent support environment

Before installing the WhaTap PHP monitoring agent, check the support environment.

-   **Support versions**

    -   Operates by the OPHP extension module
    -   PHP 5.2 or later, 7.0 or later, 8.0 to 8.2
    -   NTS / ZTS (Zend Thread Safe)

-   **OS**

    -   CentOS/Red Hat 6.x (64 bit) or later
    -   Debian/Ubuntu 12.04 (64 bit) or later
    -   Alpine Linux (64 bit)
    -   FreeBSD 10.x (64 bit) or later

-   **Database**

    -   Oracle (OCI8)
    -   MySQL
    -   MSSQL (mssql, sqlsrv)
    -   PostgreSQL (psql)
    -   Redis (phpredis)
    -   CUBRID
    -   Tibero (odbc)

-   **Support libraries**

    -   PDO
    -   MySQLi
    -   cURL
    -   oci8
    -   mssql
    -   sqlsrv
    -   psql
    -   phpredis
    -   CUBRID
    -   odbc

-   **Application Server**

    -   Apache
    -   PHP-FPM (Nginx)
