-   **whatap.tx_caller_meter_enabled**<span class='type'>Boolean</span><br/>
      Default `false`<br/>
      It creates the topology statistics based on the transaction information.

-   **whatap.sql_dbc_meter_enabled**<span class='type'>Boolean</span><br/>
      Default `false`<br/>
      It creates the topology statistics based on the database connection information.

-   **whatap.httpc_host_meter_enabled**<span class='type'>Boolean</span><br/>
      Default `false`<br/>
      It creates the topology statistics based on the HTTP external calls.
