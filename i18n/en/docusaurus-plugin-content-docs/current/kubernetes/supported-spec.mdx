Before starting Kubernetes Monitoring, check out the following supported specifications:

## Supported platforms

-   EKS (Amazon Elastic Kubernetes Service) 1.16 to 1.23
-   AKS (Azure Kubernetes Service) 1.16 to 1.23
-   GKE (Google Kubernetes Engine) 1.16 to 1.23
-   OKE (Oracle Container Engine for Kubernetes) 1.16 to 1.23
-   Openshift 3.xx/4.xx
-   Cocktail 1.16 to 1.23
-   Accordion 1.16 to 1.23
-   PaaS-TA 1.16 to 1.23
-   Plug-ins: Supports ISTIO and GPU
