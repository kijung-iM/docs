Home > Select Project > **_Analysis_** > **_Stack_**

![Stack](https://img.whatap.io/media/images/whatap_kubernetes_analysis_stack.png)

WhaTap can use the thread stacks collected every 10 seconds (default value) to analyze the performance delay sections in the method level. This feature allows you to check the **_Top stack_**, **_Unique stack_**, and **_Active Stack_** of each container.

:::note

For more information about **_top stack_**, **_unique stack_**, and **_active stack_**, see the following.

-   [Java](../java/analysis-apm)
-   [Python](../python/analysis-apm)

:::
