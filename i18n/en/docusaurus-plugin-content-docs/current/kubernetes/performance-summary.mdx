Home > Select Project > **_Resource_** > **_Performance Summary_**

Through the recent time-based chart, you can see what changes have been made in key metrics of Kubernetes, such as numbers of containers/pods and nodes.

![Performance Summary](https://img.whatap.io/media/images/whatap_kubernetes_resource_performance_summary.png)

On the upper right of the node count widget, if you select ![More icon](/img/ico-more.svg) > ![Shortcut icon](/img/ico-shortcut.svg), **_Resource_** > **_Node List_** appears. 

The widgets for node data are as follows:

-   Recent 30 minutes chart
-   **_Node CPU (%)_**, **_Node Memory (%)_**, etc.

If you select ![More icon](/img/ico-more.svg) > ![Shortcut icon](/img/ico-shortcut.svg) on the upper right of the node CPU widget or memory widget, **_Analysis_** > **_Metrics Chart_** appears. Using this menu, you can see the chart with detailed conditions.

The widgets for container data are as follows:

-   Recent 10 minutes chart
-   **_Container Status_**, **_Container CPU (%)_**, **_Container Memory (%)_**, etc.

The widgets for application data are as follows:

-   Recent 10 minutes chart
-   **_Active TX_**, **_TPS_**, **_Response Time_**, etc.

:::note
The application data is displayed when the target application agent to be installed exists.
:::
