You can check detailed information about the target selected in the **_container map_** menu on the **_Display Detail_** screen.

1.  Select a target (individual, set) in **_Container Map_**. 

2.  On the right of the screen, the **Display Detail** screen appears. 

  ![Container Map - Display Summary](/img/kube-container-map-detail-button.png)

1.  On the upper right, select ![Leader line 1](/img/number-01.png) ![Detailed view icon](/img/ico-detail-mode.svg) to see the details.

The **Display Detail** screen of the container map consists of the following:

![Display Detail screen](/img/container-map-detail-mode-desc.png)

-   ![Leader line 2](/img/number-02.png) Target card (container or pod)

-   ![Leader line 3](/img/number-03.png) Target list

-   ![Leader line 4](/img/number-04.png) Screens for each menu
    -   **_Trace_** > **_Dashboard_**, **_Transaction List_**
    -   **_Metrics_**
    -   **_Log_** > **_Recent Log Messages_**, **_Log Search_**
    -   **_Event_** > **_Kubernetes_**, **_WhaTap_**
