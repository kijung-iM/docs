Home > Select Project > **_Management_** > **_Agent CONFIG._**

Let's learn how to restart or remove the agent.

## Restarting the agent

![Restarting the agent](/img/mysql-rds-agent-settings-restart.png)

To restart the agent, change the agent setting options and then select **_Apply_** on the upper right of the screen.

## Agent deletion

![Agent deletion](/img/mysql-rds-agent-remove.png)

Select ![Delete icon](/img/ico-trash.svg) on the right of the agent in the **_agent list_**. When the confirmation message appears, select **_OK_**. Delete the agent selected from the list.
