**Access log** is a log file that leaves access data of the clients connecting the server. The access log contains: 

-   Remote host IP address (requester)
-   Request time
-   HTTP status code
-   Bytes transmitted excluding the HTTP header
-   Time to process the request (ms)
-   Referrer
