## Viewing the project list

<span class='apitype'>GET</span> https://api.whatap.io/open/api/json/projects

```bash title='command'
$ curl -L -w "\n" -H "x-whatap-token: {{ACCOUNT API TOKEN}}" "https://api.whatap.io/open/api/json/projects"
```

```json title='response'
{
  "data": [ {
        "groupName": "",
        "projectCode": 35,
        "projectName": "APM_TEST2",
        "createTime": "2018-10-19 07:17:03.774",
        "status": "subscribe",
        "productType": "APM",
        "platform": "JAVA",
        "gatewayName": "LOCAL",
        "apiToken": "5**********************R"
    }, {
    }, (...)
  ],
  "accountEmail": "xxx@whatap.io",
  "total": 6
}
```

| Response Field  |      Field Type                 | Description                        |
| --------------- | :-----------------------------: | ---------------------------------- |
| gourpName       | <span class='api'>string</span> | Name of the project's upper group  |
| projectCode     | <span class='api'>number</span> | pcode                              |
| projectName     | <span class='api'>string</span> | 프로젝트 이름                            |
| createTime      | <span class='api'>string</span> | Project creation date              |
| lastUpdatedTime | <span class='api'>string</span> | Project modification date          |
| status          | <span class='api'>string</span> | Project status                     |
| productType     | <span class='api'>string</span> | Application server type            |
| platform        | <span class='api'>string</span> | Application server platform        |
| gatewayName     | <span class='api'>string</span> | Gateway name                       |
| apiToken        | <span class='api'>string</span> | Project API token                  |
| accountEmail    | <span class='api'>string</span> | Email address of the user account  |
| total           | <span class='api'>number</span> | Project count                      |

## Viewing the project information{#project-info}

<span class='apitype'>GET</span> https://api.whatap.io/open/api/json/project

:::note
This function is included in the application monitoring suite.
:::

```bash title='command'
$ curl -L -w "\n" -H "x-whatap-token: {{PROJECT API TOKEN}}" -H "x-whatap-pcode: {{projectCode}}" "https://api.whatap.io/open/api/json/project"
```

```json title='response'
{
  "platform": "JAVA",
  "createTime": "Tue Sep 18 08:34:27 GMT 2018",
  "gatewayName": "Office-OTE",
  "projectCode": 1**1,
  "status": "Subscribe",
  "lastUpdatedTime": "Tue Sep 18 08:34:34 GMT 2018", 
  "name": "SEO_APM",
  "productType": "APM"
}
```

| Response Field  |      Field Type                 | Description                     |
| --------------- | :-----------------------------: | ------------------------------- |
| platform        | <span class='api'>string</span> | Application server platform     |
| createTime      | <span class='api'>string</span> | Project creation date           |
| projectCode     | <span class='api'>number</span> | pcode                           |
| status          | <span class='api'>string</span> | Project status                  |
| lastUpdatedTime | <span class='api'>string</span> | Project modification date       |
| productType     | <span class='api'>string</span> | Application server type         |

## Viewing the project in the group

<span class='apitype'>GET</span> https://api.whatap.io//open/api/json/group/<code>groupKey</code>/projects

This API views meta information of the projects in the group. You need your account's API token, and an example request using CURL is below.

```bash title='command'
$ curl -L -w "\n" -H "x-whatap-token: {{ACCOUNT API TOKEN}}" "https://api.whatap.io/open/api/json/group/{{groupKey}}/projects" 
```

```json title='response'
{
  "data": [
    {
      "projectCode":35,
      "name":"APM_TEST2",
      "status":"Subscribe",
      "productType":"APM",
      "platform":"JAVA",
      "gatewayName":"LOCAL"
    }, {
    }, (...)
  ]
}
```

| Response Field  |              Field Type                  | Description                                                                                             |
| --------------- | :--------------------------------------: | ------------------------------------------------------------------------------------------------------- |
| projectCode     | <span class='api'>number(integer)</span> | pcode                                                                                                   |
| name            |      <span class='api'>string</span>     | Project name                                                                                            |
| status          |      <span class='api'>string</span>     | Project status (`pending`, `trial`, `subscribe`, `close_pending`, `limited`, `closed`, `trial_limited`) |
| productType     |      <span class='api'>string</span>     | Products used in the project                                                                            |
| platform        |      <span class='api'>string</span>     | Platform of the product used in the project                                                             |
| gatewayName     |      <span class='api'>string</span>     | Data server's region                                                                                    |
