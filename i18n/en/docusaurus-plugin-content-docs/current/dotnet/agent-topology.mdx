This option creates the topology map by using the data collected by the agents.

-   **tx_caller_meter_enabled** <span class='type'>Boolean</span><br/>Default `false`<br/>
    It creates the topology statistics based on the transaction information.

-   **sql_dbc_meter_enabled** <span class='type'>Boolean</span><br/>Default `false`<br/>
    It creates the topology statistics based on the database connection information.

-   **httpc_host_meter_enabled** <span class='type'>Boolean</span><br/>Default `false`<br/>
    It represents the HTTPC outbound data from the topology map.
