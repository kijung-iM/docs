:::tip

-   Namespaces are collected as the WhaTap metrics **category**. For the notation format, change the Naver Cloud namespace to **lowercase** and then change `/` to `**_**`.
    > e.g. **_ncloud\__**
-   Metrics are collected as the WhaTap **metrics**. The notation format is `NaverCloudMetric.Stat`.
-   For the collected Naver Cloud metrics, see [the following](https://api.ncloud-docs.com/docs/management-cloudinsight-productinfo).
:::
