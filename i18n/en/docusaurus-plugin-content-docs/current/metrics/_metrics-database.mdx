## Database metrics

The metrics that can be searched in **_Metrics Chart_**.

<!-- 추후 업데이트 필요 관련 지표 추가 및 필드 값으로 교체 -->

### MySQL

<div class='metric'>

| Metric                           | Description                                                                                                                                       |
| -------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------- |
| Innodb Buffer Pool Read Requests | Logical read request count<br/>Logical read + Physical read. Because the value of logical read is large, it is used as a concept of logical read. |
| Innodb Buffer Pool Reads         | Physical read count<br/>Physical read                                                                                                             |
| Threads Connected                | Number of connected sessions                                                                                                                      |
| Threads Running                  | Number of sessions running queries                                                                                                                |
| Questions                        | Number of queries executed                                                                                                                        |
| Com Select                       | Number of executed Select commands                                                                                                                |
| Writes                           | Write command count<br/>Com_insert + update + delete                                                                                              |
| Bytes Received                   | Data bytes received by the DB server                                                                                                              |
| Bytes Sent                       | Data bytes sent by the DB server                                                                                                                  |
| Innodb Row Lock Current Waits    | Lock count                                                                                                                                        |
| Connections                      | DB connection trial count                                                                                                                         |
| Com Commit                       | Number of commits                                                                                                                                 |
| Com Rollback                     | Rollback count                                                                                                                                    |
| Innodb Rows Read                 | Number of processed rows by Select                                                                                                                |
| Innodb Rows Writes               | Number of rows for write<br/>Innodb_rows_inserted + updated + deleted                                                                             |
| Innodb Row Lock Waits            | row lock wait count                                                                                                                               |
| Innodb Row Lock Time             | row lock wait time                                                                                                                                |
| Sort Rows                        | Number of rows sorted                                                                                                                             |

</div>

### PostgreSQL

<div class='metric'>

| Metric                | Description                                                                                                                    |
| --------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| Active Sessions       | Number of sessions running queries                                                                                             |
| Deadlock Count        | Detected deadlock count                                                                                                        |
| Commit Count          | Number of committed transactions                                                                                               |
| Rollback Count        | Number of rolled back transactions                                                                                             |
| DML Count             | DML(insert, update, delete) execution count                                                                                    |
| BLKS Read             | Number of disk blocks read (physical read)                                                                                     |
| BLKS Hit              | Number of disk block cache hits (number of cases not requiring read because disk blocks are in the buffer cache. Logical read) |
| BLKS Hit Ratio        | Ratio of blks hit (cache hit) to blks read (disk read count)                                                                   |
| Check Point           | checkpoint execution count                                                                                                     |
| Total Sessions        | Number of all sessions                                                                                                         |
| Total Runtime         | Sum of execution times of active sessions                                                                                      |
| Long Running Sessions | Number of long-running sessions                                                                                                |
| Lock Wait Sessions    | Number of sessions waiting for a lock                                                                                          |
| Table Hit Ratio       | Ratio of table blks hit (cache hit) to blks read (disk read count)                                                             |
| Index Hit Ratio       | Ratio of index blks hit (cache hit) to blks read (disk read count)                                                             |
| Vacuum Running Count  | vacuum execution count                                                                                                         |

</div>

### Oracle

<div class='metric'>

| Metric                   | Description                                                        |
| ------------------------ | ------------------------------------------------------------------ |
| CPU                      | CPU utilization of the DB                                          |
| Active Sessions          | Number of sessions running queries                                 |
| Total Sessions           | Number of all sessions                                             |
| Lock Wait Sessions       | Number of sessions waiting for a lock                              |
| Session Logical Reads    | Number of data reads in the buffer cache                           |
| Physical Reads           | Number of data reads from the physical disk                        |
| Execute Count            | SQL execution count                                                |
| Opened Cursors Current   | Number of open cursors                                             |
| Parse Count (hard)       | Hard parsing count                                                 |
| Parse Count (total)      | Total parsing count (hard + soft)                                  |
| Enqueue Waits            | Number of enqueue waits                                            |
| Logons Current           | Current logon count                                                |
| User Commits             | Commit execution count                                             |
| CPU used by this session | Sum of CPU time used by the session                                |
| DB Time                  | Sum of time spent for using DB                                     |
| Physical Writes          | Number of data writes to the disk                                  |
| Sorts (memory)           | Rate of times sorted while writing from memory to disk             |
| DB File Sequential Read  | Rate of waiting for writes during the service (Single-block reads) |
| DB File Scattered Read   | Rate of waiting for writes during the service (Multi-block reads)  |
| Wait Sessions            | Number of sessions waiting for an event                            |
| Txn Sessions             | Number of sessions with transactions                               |
| Long Running Sessions    | Number of sessions waiting for a lock                              |

</div>
