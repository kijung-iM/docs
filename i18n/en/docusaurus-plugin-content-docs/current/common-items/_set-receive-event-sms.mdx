### Adding to the SMS alert reception

SMS 알림 수신이 필요한 경우 **_계정 정보 관리_**에서 사용자 전화번호를 설정하세요. 

<ImgLang img='set-event-sms-notification.png' desc='사용자 전화번호' />

1.  Select the profile icon on the upper right of the screen. 

2.  If a pop-up menu appears, select **_Manage Account INFO._**. 

3.  **_사용자 전화번호_** 섹션에서 **_일반 휴대전화_** 버튼을 선택하세요. 

4.  **_전화번호_**에 SMS를 수신할 전화번호를 입력하세요.

5.  **_SMS 인증_** 버튼을 선택하세요.

6.  **_SMS 인증_** 창이 나타나면 사용자의 휴대전화로 전송된 인증 번호를 입력하세요.

7.  **_전화번호 변경_** 버튼을 선택하세요.

:::note

-   등록한 전화번호를 변경하려면 **_번호 변경하기_** 버튼을 선택한 다음 위의 과정을 반복하세요.

-   SMS를 알림으로 수신할 수 있는 전화번호는 **한국 휴대전화 번호**만 등록할 수 있습니다.

:::
