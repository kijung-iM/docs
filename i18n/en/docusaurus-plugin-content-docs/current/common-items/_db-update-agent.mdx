## Agent Update

Home > Select Project > **_Management_** > **_Agent Update_**

![Update Agent](/img/db-update-agent.png)

To update agents individually, select **_Latest version_** on the right of each agent. To update all agents at once, select **_Download to all agents with the latest version_**.

It is automatically downloaded to the server where the agent is located. After downloading, restart the agent. The agent is automatically updated to the latest version.
