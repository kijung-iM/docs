## Column information guide{#column-info}

| Item             | Description                                                                                                                                                                                          |
| ---------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| pid              | Process ID                                                                                                                                                                                           |
| cpuusage         | CPU Utilization                                                                                                                                                                                      |
| cpu(xos)         | CPU utilization collected by the installed XOS                                                                                                                                                       |
| dataname         | Database name                                                                                                                                                                                        |
| application_name | Client application name                                                                                                                                                                              |
| wait_event_type  | Type of the event where the backend is waiting                                                                                                                                                       |
| wait_event       | Waiting event name where the backend is waiting                                                                                                                                                      |
| runtime          | Execution time of the current command (sec)                                                                                                                                                          |
| query            | Statement being executed or the SQL statement that separates literal strings                                                                                                                         |
| query_param      | Literal value separated from `query`                                                                                                                                                                 |
| state            | Current thread status (executing, updating, sending data, etc.)                                                                                                                                      |
| bind_query       | Bind variable used in query                                                                                                                                                                          |
| xact_start       | Time when the current transaction for the process started (null if no transaction is active)                                                                                                         |
| backend_start    | Time when this process started, that is, the time when the client connected to its write-ahead logging (WAL) sender.                                                                                 |
| client_addr      | IP address of the client connected to the WAL sender                                                                                                                                                 |
| client_hostname  | Host name of the connected client (reported by reverse DNS lookup of `client_addr`)                                                                                                                  |
| client_port      | TCP port number that the client uses to communicate with the WAL sender (-1 if using Unix sockets)                                                                                                   |
| query_hash       | Hash value of query (internal WhaTap control value)                                                                                                                                                  |
| query_start      | Time when the active query started (if the state is not active, it is the last query started time)                                                                                                   |
| query_pre        | SQL statement and PLAN saved in the session process using the `PREPARE` command.                                                                                                                     |
| rss(xos)         | Resident Set Size (RSS) that is the number of physical pages associated with the process.                                                                                                            |
| pss(xos)         | Proportional Set Size (PSS) that is USS (Unique Set Size) + (shared page count/number of shared processes). <br/>Process specific memory usage + Percentage of shared memory occupied by one process |
| ioread(xos)      | Actual time spent reading the block (milliseconds)                                                                                                                                                   |
| iowrite(xos)     | Actual time spent writing the block (milliseconds)                                                                                                                                                   |
| holder type      | Type of the holder session                                                                                                                                                                           |
| lock mode        | holder_mode                                                                                                                                                                                          |
| waiter type      | Type of the waiter session                                                                                                                                                                           |
| lock request     | waiter_mode                                                                                                                                                                                          |
