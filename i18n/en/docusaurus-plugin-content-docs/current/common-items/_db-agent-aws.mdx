The description of the agent configuration using the AWS CloudWatch

-   **aws_region** <span class='type'>String</span>

    Set the AWS region.

-   **aws_access_key** <span class='type'>String</span>

    Set the AWS access key.

-   **aws_secret_key** <span class='type'>String</span>

    Set the AWS secret key.

-   **cloud_watch** <span class='type'>String</span>

    Set whether or not to use the Cloud Watch Monitoring. The value is `on` or `off`.

-   **cloud_watch_instance** <span class='type'>String</span>

    Set the instance name of the cloud watch.

-   **cloud_watch_interval** <span class='type'>Second</span>

    Set the interval to get monitoring values by Cloud Watch (unit: second).

-   **cloud_watch_metrics** <span class='type'>String</span>

    Set the monitoring metric name to be imported into the cloud watch. You can set it as follows:

    ```ini
    cloud_watch_metrics=CPUUtilization,FreeableMemory,FreeStorageSpace,NetworkTransmitThroughput,NetworkReceiveThroughput
    ```

-   **aws_arn** <span class='type'>String</span>

    Instead of `aws_access_key` and `aws_secret_key`, you can use the ARN of the IAM ROLE with appropriate roles.
