## Setting alarms by day of the week and time

You can select whether or not to receive notifications by day of the week or time. In the user list of the **_Event Notification Settings Per-User_** section, select ![More icon](/img/ico-more-down.svg) at the utmost left. Select the desired day of the week or select the time to receive alerts. You can set for each notification reception method.
