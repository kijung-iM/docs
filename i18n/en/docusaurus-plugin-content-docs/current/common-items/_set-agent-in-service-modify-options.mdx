### Modifying or deleting options

<ProdImg img='modify-agent.png' desc='옵션 수정' />

Scroll up or down the screen or select an option category on the left to find the option to edit or delete. To go to the top of the screen, select **_Go to Add Options_**.

Select or edit the desired value for the option to change. To delete an option, select ![Delete icon](/img/ico-trash.svg). Select **_Apply_** to apply your changes.

:::note

If you select **_Apply_** without entering options, the options are deleted.

:::
