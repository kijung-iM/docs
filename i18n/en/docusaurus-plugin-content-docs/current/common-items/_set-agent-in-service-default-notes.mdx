:::note

-   If you select **_Object Kind Default_** or **_Project Default_** tab and then apply the options, it is useful to commonly manage agents.

-   The application order of options is as follows: **_Object Kind Default_**, **_Project Default_**, and **_Agent CONFIG._**. 

    -   If the value of option b is 1000 in **_Project Default_** and it is 2000 in **_Object Kind Default_**, the option value set in **_Object Kind Default_**, 2000 takes precedence.
    -   If the value of option b is null in **_Project Default_** and it is 2000 in **_Object Kind Default_**, the option value set in **_Object Kind Default_** takes precedence.

:::
