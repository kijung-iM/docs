You can customize your dashboard by handling the widgets placed on the Flex board.

## Adding a widget

You can add any widget to the created Flex board.

1.  In the **_Board_** list of **_Flex Board_**, select a dashboard to add the widget.

2.  On the upper left of the screen, select **_Edit Mode_**.

3.  Select a desired widget from the **_Widget Template _** list on the left. To add a metrics widget, select **_All Metrics_** and then click a desired widget.

    -   If you have entered **_Flex Board Menu_** on the Home screen and added a widget, click **_+ Select Project_** and then select target projects to collect.
    -   If you have more widgets to add, repeat the same procedure.

After all steps are finished, select **_View Mode_** to check the layout and then select ![Go to list icon](/img/ico-exit.svg) (**_Go to List_**).

## Changing the widget attributes

In the **_board_** list on **_Flex Board_**, select a board to change the widget properties. It goes to **_Edit Mode_**.

### Changing the widget name

On the right of the widget name to change, select ![Edit icon](/img/ico-edit.svg). Set the widget values for **_Title_**, **_Font Size_**, and **_Color_**, and then select **_Save_**.

:::tip

-   If the **_Add to the entire widget_** toggle button is enabled, the font size and color are applied to other widgets.
-   To lock the widget title, select [Configuration icon](/img/ico-invisible.svg) and then click **_Save_**.

:::

### Setting the time

On the upper right of the widget to set the time, select ![Time settin icon](/img/ico-time-set.svg). The **_Time Selector_** window appears.

-   To apply the set time to the dashboard, check **_Use dashboard settings_**. The time setting on the dashboard is located on the upper right.

-   To apply a desired time setting, uncheck **_Use dashboard settings_**. Use ![Left arrow icon](/img/left-arrow.svg) or ![Right arrow icon](/img/right-arrow.svg) to set the desired time.

After setting the time, select ![End icon](/img/ico-close.svg).

:::note
Depending on the data type of the widget, the time setting may not be available.
:::

### Setting the output data options

By setting the output data options of the chart widget, you can select the desired data. On the upper right of the widget, select ![More icon](/img/ico-more.svg).

-   You can filter data by selecting types for project, agent, application, etc.

-   You can change the type of the graph displayed on the chart by selecting individual or merged data.

-   Select ![More icon](/img/ico-more.svg) > ![Configuration icon](/img/ico-setting.svg). You can change the option details through the **_Chart Settings_** window. The options that can be set are as follows.

    -   **_Chart Type_**: This option allows you to display the current aggregated figures on the chart.
    -   **_Y axis Chart Max Value_**: The chart maximum value on the Y axis can be selected with one of **_Auto_** and **_Fix_**.
    -   **_Set Horizontal Line_**: Select **_+ Add_** to display the threshold or range as a horizontal line. You can apply multiple items added. After adding the horizontal line, select **_Apply_** to apply it to the chart.
    -   **_Show Chart Max Value_**: The maximum value of the graph is displayed on the chart.
    -   **_보조 차트_**: 위젯의 주요 차트에 보조 차트를 추가 설정할 수 있습니다. 보조 차트의 종류를 선택한 다음 레이아웃, 크기, 데이터 병합 여부를 선택하세요. 다른 위젯에도 공통 적용하려면 **_전체 위젯에 적용_** 버튼을 선택하세요. 보조 차트는 데이터를 다양한 방법으로 보고 싶을 때 유용합니다.

    After setting all options, on the left of the **_Chart Settings_** window, click ![Close icon](/img/ico-close.svg) or click the outside of the **_Chart Settings_** window.

:::note
Depending on the widget's data type, available options may differ.
:::

### Changing the metrics widget data{#changedata-metric-widget}

You can change the data query conditions of the metrics widget. 

1.  In **_Flex Board_** > **_Edit Mode_**, on the upper right of the widget to change, select ![More icon](/img/ico-more.svg).

  ![Changing the metrics widget data](/img/metric-widget-data.png)

1.  Select ![Edit icon](/img/ico-edit-metric.svg). The data setting edition window appears.

  ![Metrics widget data editing](/img/metric-edit-window.png)

1.  Change the settings referencing the following, and then select ![Apply icon](/img/left-arrow.svg) **_Apply to the Widget_**.

    -   Metrics options that are created by default when creating a widget appear on the tab.
    -   Metrics options are **_Category_**, **_Field_**, and **_Tag_**.
    -   The **_category_** currently restricts the editing function.
    -   In **_Field_** and **_Tag_**, multiple items can be selected. The chart data is grouped using the selected tag options.
    -   Select ![Search icon](/img/ico-btn-search.png) to query the original data.
    -   For mor information about **_Object Merge_** and **_Time Merge_**, see the [Data merge options](#datamerge-option).  

### Data merge options{#datamerge-option}

The metrics widget placed on the **_All Metrics_** list of the Flex board provides the data merge options.

1.  In **_Edit Mode_**, select **_All Metrics_** and then change the **_widget template_** list to the **_All Metrics_** list.

2.  Place the metrics widget on the layout.

3.  메트릭스 위젯 오른쪽 위에 ![더보기 아이콘](/img/ico-more.svg) 버튼을 선택하세요.

4.  If a tag (e.g. agent name) option is selected, the data merge options appear. Depending on the widget size, the metrics option's location may differ.

![Metrics widget - Data merge options](/img/flexboard-datamerge.png)

Data merge provides the following methods: **_object merge_** and **_time merge_**.

-   **_Object Merge_**<br/>
    It is used to group the chart data based on the tag (e.g. agent name) selected on the left. At this time, if the tags match among the data with different fields (e.g. CPU), the corresponding data is merged.

-   **_Time Merge_**<br/>
    It is mainly used if the lookup time is too long. Data with the same fields in the original data are merged at regular time intervals (5 minutes or 1 hour) depending on the time range. This method is called "Time Merge."

-   **AWS CloudWatch 메트릭스**<br/>
    In case of AWS monitoring metrics, the statistical method is added after the field name in the form of suffix. Data merging defaults reference this suffix.

    ![AWS ColudWatch Metrics](/img/blog_12.png)

## Copying a widget

1.  Select ![More icon](/img/ico-more.svg) on the upper right of the widget. 
2.  Select ![Copy icon](/img/ico-copy.svg).
3.  When the **_Copy_** window appears, select **_+ Add Widget_**.
4.  Change the widget name in the list.
5.  Select **_Apply_**.

In **_Edit Mode_**, you can see the copied widget.

## Deleting a widget

1.  Select ![More icon](/img/ico-more.svg) on the upper right of the widget. 
2.  Select ![Delete icon](/img/ico-remove.svg).
3.  If the confirmation message appears, select **_Delete_**. To cancel the deletion, select **_Cancel_**.
