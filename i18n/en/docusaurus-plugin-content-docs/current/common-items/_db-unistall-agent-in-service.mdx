### Deleting on the service screen

To delete the agent from [Whatap Monitoring Service](https://service.whatap.io), go to **_Agent List_**. Before deleting the agent, terminate the agent process on the server where the agent has been installed. 

1.  Select a project to delete the agent from the **_project list_**.

2.  On the left of the screen, select **_Management_** > **_Agent List_**.

3.  Select ![Delete icon](/img/ico-trash.svg) on the utmost right of the agent to delete from **_Agent List_**.

:::note

-   Only inactive agents can be deleted.

    ![](/img/db-disable-agent.png)

-   Depending on the agent version, the delete function may not be supported. Update the agent to the latest version.

:::
