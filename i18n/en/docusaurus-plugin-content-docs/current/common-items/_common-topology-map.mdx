-   **tx_caller_meter_enabled** <span class='type'>Boolean</span><br/>
    Default `false`<br/>
    It creates the topology statistics by using transactions. If the value of the `mtrace_enabled` option is `true`, it works.

-   **sql_dbc_meter_enabled** <span class='type'>Boolean</span><br/>
    Default `false`<br/>It creates the topology statistics based on the database connection information.

-   **httpc_host_meter_enabled** <span class='type'>Boolean</span><br/>
    Default `false`<br/>It represents the HTTPC outbound data from the topology map.

-   **actx_meter_enabled** <span class='type'>Boolean</span><br/>
    Default `false`<br/>It represents the status of active transactions from the topology map.
