### app_proc_counter

* * *

It is the metric of the application process.

-   Collection interval: 5 seconds
-   Statistical data: 5 minutes, 1 hour

#### Tags

| Tag name     | Description                  | Remarks       |
| ------------ | ---------------------------- | ------------- |
| container    | Container name               | Unique value  |
| containerKey | Container's key value        | Unique value  |
| host_ip      | Host IP                      | Unique value  |
| okindName    | Name of the application type | -             |
| oname        | Application name             | Unique value  |
| onodeName    | Name of the application node | -             |
| pid          | Application PID              | -             |
| type         | Application type             | Language name |
