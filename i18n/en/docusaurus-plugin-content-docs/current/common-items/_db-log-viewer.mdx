You can check the log file content of the database server or cloud database server in which the agent has been installed. You can check a single database at a time.

![Log File Viewer](/img/analysis-log-viewer.png)

-   ![Leader line 1](/img/number-01.png) **_Instance_**

    An instance in the project can be selected. Click the instance entry and then select each individual instance.

-   ![Leader line 2](/img/number-02.png) **_Log File List_**

    This is a list of log files that can be checked for each instance. If you have too many files and it is difficult to find them, you can find the desired file by searching.

-   ![Leader line 3](/img/number-03.png) **_Log File Viewer_** 

    You can check the log content at an interval of 2000 bytes. Select ![Previous page icon](/img/left-arrow.svg) or ![Next page icon](/img/right-arrow.svg) and then go to pages at an interval of 2000 bytes. To go to the initial or end page, select ![First page icon](/img/ico-first.svg) or ![Fnal page icon](/img/ico-last.svg).

    To go to the desired location, enter the desired location and then select **_Go_**.
