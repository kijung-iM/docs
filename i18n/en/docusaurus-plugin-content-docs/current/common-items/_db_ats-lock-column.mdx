## Column information guide{#column-info}

| Item            | Description                                                                                                                                     |
| --------------- | ----------------------------------------------------------------------------------------------------------------------------------------------- |
| Instance Name   | Instance Name                                                                                                                                   |
| id              | Connection identifier                                                                                                                           |
| user            | SQL execution account                                                                                                                           |
| host            | Hostname that this thread is connected to.                                                                                                      |
| db              | Default database name. If nothing, it is NULL.                                                                                                  |
| command         | Running command type (Query, Sleep, Daemon, etc.)                                                                                               |
| time            | Execution time of the current command (sec)                                                                                                     |
| state           | Current thread status (executing, updating, sending data, etc.)                                                                                 |
| query           | Running statement. It displays the literal value SQL separated from `Processlist.info`.                                                         |
| query_param     | Literal value separated from `query`                                                                                                            |
| type            | Thread type. FOREGROUND/BACKGROUND, etc.                                                                                                        |
| connection_type | The protocol used to establish a connection: TCP/IP, SSL/TLS                                                                                    |
| query_hash      | Hash value of query (internal WhaTap control value)                                                                                             |
| os_id           | OS thread ID associated with the MySQL thread. It may be NULL when using the thread pool plugin.                                                |
| thread_id       | thread ID                                                                                                                                       |
| name            | Name associated with the server's code. User connection handling includes Thread, sql, one_connection, and such.                                |
| cpu(xos)        | CPU utilization collected by the installed XOS                                                                                                  |
| query_id        | Query ID                                                                                                                                        |
| time_ms         | Execution time (milliseconds)                                                                                                                   |
| stage           | This is the step where the current process is in progress.                                                                                      |
| max_stage       | Maximum stage count (MariaDB)                                                                                                                   |
| progress        | Process progress within the current step (0-100%).                                                                                              |
| memory_used     | Memory (bytes) used in the threads                                                                                                              |
| max_memory_used | Maximum memory (bytes) used in the threads                                                                                                      |
| examined_rows   | The row checked by the thread. It is only updated by UPDATE, DELETE, and similar statements. Values in SELECT and other statements remain at 0. |
| holder type     | Type of the holder session                                                                                                                      |
| lock mode       | holder_mode                                                                                                                                     |
| waiter type     | Type of the waiter session                                                                                                                      |
| lock request    | waiter_mode                                                                                                                                     |
| table           | holder_table/waiter_table                                                                                                                       |
| index           | holder_index/waiter_index                                                                                                                       |
