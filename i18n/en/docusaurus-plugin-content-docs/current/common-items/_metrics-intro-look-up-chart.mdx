## Viewing the metrics chart

Home > Select Project > **_Analysis_** > **_Metrics Chart_**

You can see the metrics data collected from the monitoring targets through a chart.

<ProdImg img='metrics-chart-example-01.png' desc='' />

-   Time to view (**required**)
-   Metric to view (**required**)
-   Monitoring targets to view (**_Optional_**, if nothing is selected, all items are applied)
