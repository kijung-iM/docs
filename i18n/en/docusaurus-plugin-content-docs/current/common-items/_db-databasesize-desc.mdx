## Database size

Home > Select Project > **_STAT/REPORT_** > **_Database Size_**

It aggregates the sizes of tables for each database and provides trend data in a chart. The collection cycle is collected every 3 hours. 

<InDoc product='postgresql'>

<ProdImg img='db-database-size-postgresql.png' desc='데이터베이스 사이즈'/>

</InDoc>

1.  Set the time to view in **_Time_**. You can select a lookup time after selecting the green button.

2.  Select a target to view in **_Instance_**.

3.  Select ![Search icon](/img/ico-btn-search.png).

:::note

You can select a lookup time (up to 3 weeks). Select the clock display area to query a random date and time. If you click the date and time text area, the selectable options for the date and time appear.

:::
