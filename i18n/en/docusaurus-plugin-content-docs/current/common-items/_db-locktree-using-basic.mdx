Set the desired ![number1](/img/number-01.png) date and time, and then select an instance from the ![number2](/img/number-02.png) list. If a lock occurs at the set time, **_lock wait sessions_** and the retrieved data appear in the table at the bottom of the screen.

-   If you click a specific time in the graph chart, you can check the lock information for the selected time in the table at the bottom of the screen. If you select the button in seconds, you can view the lock information in 5-second increments.
-   If you drag a specific time on the graph chart, you can enlarge the selected area and check the lock information.
