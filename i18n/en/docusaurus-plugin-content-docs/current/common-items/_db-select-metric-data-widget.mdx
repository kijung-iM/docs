## Selecting a metric

You can change the metric of the graph chart widget placed on the screen to another one.

1.  Select ![Configuration icon](/img/ico-setting.svg) on the upper right of the widget to change the metrics.

2.  When the **_count_** window appears, select a desired metric.

    ![Count window](/img/db-redis-select-count.png)

3.  Select **_Save_**.

The metric of the selected widget is changed.

:::note

For more information about each metric, see [the following](metrics-data-list). 

:::
