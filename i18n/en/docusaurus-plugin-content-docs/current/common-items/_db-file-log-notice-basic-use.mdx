## Basic screen guide

Home > Select Project > **_Alert_** > **_File Log Alert_**

![File Log Alert](/img/db-file-log-notice.png)

-   **_Time_**: You can select or enter the lookup time. 
-   **_Instance_**: Select an instance to view.
-   **_Log File List_**: You can see a list of log files for your instance.
-   **_Log History_**: You can check the log occurrence time through the set file name and keyword.
