## Filtering the data list

You can filter the data in the table after entering a search term.

1.  Select ![Filtering icon](/img/ico-filter.svg) on the upper right of the table. 

2.  Select a column and a condition, and enter a search term. 

3.  Select **_Save_**.

The saved content appears next to ![Filtering icon](/img/ico-filter.svg). 

:::note

-   You can apply or disable filtering by selecting a saved condition. 
-   Additional filtering conditions can be created and overlapped. 
-   To delete filtering conditions, select ![Delete icon](/img/ico-remove-white.svg).

:::
