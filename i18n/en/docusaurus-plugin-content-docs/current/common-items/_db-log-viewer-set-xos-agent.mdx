### XOS agent configuration

Set the log file path and file name to check in the DB server. To set up multiple log files, use comma (,) as the delimiter.

```ini title='xos.conf'
fileview={fullpath}
# fileview=/home/mysql/t.log,/home/kwlee/t2.log
```

:::note

To set keywords to view the occurrence history, see [the following](file-log-notice).

:::
