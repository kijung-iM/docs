The metrics that can be searched in **_Metrics Chart_**.

<!-- 추후 업데이트 필요 관련 지표 추가 및 필드 값으로 교체 -->

<div class='metric'>

| Metric                | Description                                                                                                                    |
| --------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| Active Sessions       | Number of sessions running queries                                                                                             |
| Deadlock Count        | Detected deadlock count                                                                                                        |
| Commit Count          | Number of committed transactions                                                                                               |
| Rollback Count        | Number of rolled back transactions                                                                                             |
| DML Count             | DML(insert, update, delete) execution count                                                                                    |
| BLKS Read             | Number of disk blocks read (physical read)                                                                                     |
| BLKS Hit              | Number of disk block cache hits (number of cases not requiring read because disk blocks are in the buffer cache. Logical read) |
| BLKS Hit Ratio        | Ratio of blks hit (cache hit) to blks read (disk read count)                                                                   |
| Check Point           | checkpoint execution count                                                                                                     |
| Total Sessions        | Number of all sessions                                                                                                         |
| Total Runtime         | Sum of execution times of active sessions                                                                                      |
| Long Running Sessions | Number of long-running sessions                                                                                                |
| Lock Wait Sessions    | Number of sessions waiting for a lock                                                                                          |
| Table Hit Ratio       | Ratio of table blks hit (cache hit) to blks read (disk read count)                                                             |
| Index Hit Ratio       | Ratio of index blks hit (cache hit) to blks read (disk read count)                                                             |
| Vacuum Running Count  | vacuum execution count                                                                                                         |

</div>
