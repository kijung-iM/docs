## Increasing or decreasing the table size

Home > Select Project > **_STAT/REPORT_** > **_Table Size Increase_**

![Tablespace Size](/img/db-stat-table-size.png)

It collects the usage of table and index on a daily basis and provides overall and each table's usage increase over time. You can check the increase or decrease from 1 day up to the maximum of a month.
