### Basic screen guide

<ProdImg img='db-database-sql-stat.png' desc='SQL 통계' />

<Xclude product='postgresql'>

1.  In the **_Instance_** list, select a target to query from the list. You can select duplicate items. To select all instances, select **_All_**.

2.  Set values for **_Start Date_**, **_Start Time_**, and **_Period_**.

3.  Select ![Search icon](/img/ico-btn-search.png).

</Xclude>

<InDoc product='postgresql'>

1.  In ![number 1](/img/number-01.png) **_Time_**, select the time and date to view the data. You can select a lookup time after selecting the green button.

2.  In ![number 1](/img/number-01.png) **_Instance_**, select a target to view the data. You can select multiple instances. To select all instances, click **_Select All_**.

3.  Select ![Search icon](/img/ico-btn-search.png).

</InDoc>

The top 50 entries appear in the table.

:::note

<Xclude product='postgresql'>

-   You can filter the desired content by entering a SQL parameter value in the **_Filter_** field.

-   The default sort order is **_execute count_**. To change the sort order, select a desired criterion from the **_Sort Order_** list.

-   To download the viewed data as a CSV file, select ![csv Download icon](/img/ic-csv-download-ms.svg).

</Xclude>

<InDoc product='postgresql'>

-   Select the clock display area to query a random date and time. If you click the date and time text area, the selectable options for the date and time appear.

-   테이블 헤더의 각 컬럼을 클릭하면 선택한 컬럼을 기준으로 목록을 정렬할 수 있습니다.

-   **_조회 기준_**에서 선택한 항목 따라 조회 결과를 정렬할 수 있습니다. 원하는 항목을 선택한 다음 ![검색 아이콘](/img/ico-btn-search.png) 버튼을 선택하세요. 

-   조회한 내용을 csv 형식의 파일로 다운받으려면 ![Download icon](/img/ico-download.svg) 버튼을 선택하세요.

</InDoc>

:::
