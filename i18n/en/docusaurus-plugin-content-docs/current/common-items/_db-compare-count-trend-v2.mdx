## Comparison between count trends

홈 화면 > 프로젝트 선택 > **_분석_** > **_카운트 추이 비교_**

<ProdImg img='db-compare-count-trend.png' desc='카운트 추이' />

You can compare the count trends for different dates. 

1.  Select the desired instances from the **_Instance_** area. You can also select different instances.

2.  Set values for **_Reference Date_** and **_Comparison Date_**. 

3.  Select ![Search icon](/img/ico-btn-search.png).
