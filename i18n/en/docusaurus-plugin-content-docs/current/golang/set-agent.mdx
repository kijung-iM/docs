The application server in the Golang environment uses libraries to configure the environment.

```go title='라이브러리 다운로드'
$ go get github.com/whatap/go-api
```

## Initial configuration

-   Start the monitoring module by using the `trace.Init` function at the beginning of the main function.

-   `defer trace.Shutdown()` makes sure that the monitoring ends.

```go title='GO'
import "github.com/whatap/go-api/trace"

func main(){
    trace.Init(nil)
    //It must be executed before closing the app.
    defer trace.Shutdown()
	...
}
```

This can be set in the initial stage of the application by declaring it in the form of `map[string]string`.

```go title='GO'
func Init(m map[string]string)
```

The agent is connected based on UDP 127.0.0.1:6600. If the UDP port 6600 is being used internally, change it to another port.

```go title='GO'
m := make(map[string]string)
m["net_udp_port"] = "6601"

trace.Init(m)
```

Settings can be changed while the application is running by directly modifying the _whatap.conf_ file, or in the Whatap Monitoring Service's Project menu.

When the application ends, all collected data is delivered and the UDP connection is closed.

```go title='GO'
func Shutdown()
```
