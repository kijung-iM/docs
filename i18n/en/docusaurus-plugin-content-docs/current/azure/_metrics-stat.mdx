:::tip

-   Namespaces are collected under the WhaTap metrics **category**. The notation format changes the Amazon Azure Monitor namespace to **lowercase** and then changes "/" to "**\_**."
    > e.g. **_azure\__**
-   Metrics are collected as the WhaTap **metrics**. The notation format is "AzureMonitorMetric.Stat."
-   For the collected Azure Monitor metrics, see [the following](https://learn.microsoft.com/ko-kr/azure/azure-monitor/essentials/metrics-supported).

:::
