## Release details

### Linux monitoring agent

Release date: 2022-12-27

Added to support <Status>New</Status> CentOS Stream 9 / Oracle Linux 9  - <code>systemctl</code>.

### Fixing automatic restart error of the Windows monitoring agent around 02:00

Release date: 2022-12-14

<Status>Fixed</Status> There was an automatic restart of the Windows monitoring agent due to a code error in version 2.2.8. Around 2:00 am, the agent's stop may cause false detection as Agent NoData. If the version 2.2.8 is being used, it is recommended to update the agent.
