Release date: 2022-09-08

## New features

-   Added an option to determine the metering policy by checking the Kakao Cloud environment

-   KIC data transfer in the first 5 minutes after startup, and KIC data transfer function added every 24 hours

-   Generation of the hash64 v2 logic

## Change (update)

-   트랜잭션 에러 알림 발생 시 `status_code`가 있는 경우에만 `status_code` 항목을 표시하도록 수정

-   INFO 모드인 경우 `status_code`가 없는 경우에는 알림 메시지를 전송하지 않도록 수정

-   INFO 모드인 경우 지연된 http 호출 발생시 `status_code` 여부와 상관없이 메시지 전송하도록 수정

## Fixed (bug fixing)

미터링 정보 대상인 `cpu_core` 데이터를 write 권한이 없는 경우에도 수집할 수 있도록 버그 수정
