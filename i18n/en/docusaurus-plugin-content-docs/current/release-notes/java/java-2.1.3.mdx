Release date: 2022-09-08

## New features

-   Added an option to determine the metering policy by checking the Kakao Cloud environment

-   KIC data transfer in the first 5 minutes after startup, and KIC data transfer function added every 24 hours

-   Generation of the hash64 v2 logic

## Change (update)

-   Modified to display `status_code` only if there is `status_code` when a transaction error notification occurs.

-   Modified not to transfer notification messages if there is no `status_code` in INFO mode.

-   Modified to send messages regardless of `status_code` upon delayed HTTP call in INFO mode.

## Fixed (bug fixing)

Fixed the bug to collect the `cpu_core` data (metering target) with no write role.
