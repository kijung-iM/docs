## `공통`

**_Flex 보드_** - 액티브 트랜잭션 이퀄라저 차트 클릭시 액티브 트랜잭션을 볼 수 있는 기능 추가

**_에이전트 설정_** - 옵션 추가로 이동 버튼 추가

**_로그 모니터링_** - 로그 목록 스타일 수정

**_이벤트 조회_** - 메트릭스 알림 등 다른 종류의 알림에서도 처리내역 기능 추가

**_보고서_** - 애플리케이션 사용자정의 보고서에서 불필요한 선택항목 제거

**_Flex 보드_** - 테이블 위젯 필터 미적용 오류 수정

**_메트릭스 조회_** - 메타가 없는 카테고리 선택 시 메트릭스 탐색기 바로가기 링크 미작동 오류 수정

**_로그 탐색기_** - 스크롤을 내리면 데이터 불일치가 발생하는 오류 수정

**_로그 검색기_** - 검색 결과가 없을 때, 이전에 검색된 데이터가 보여지는 오류 수정

**_멤버 목록_** - 멤버 검색 시 검색창에 입력된 값이 남아지는 오류 수정

**_프로젝트 설정_** - 프로젝트 이름 변경 후 다른 메뉴 이동하면 이전 이름으로 보이는 오류 수정

**_프로젝트 목록_** - 조직 선택 시, 프로젝트 카드 목록과 조직의 프로젝트가 일치하지 않는 오류 수정

**_에이전트 설정_** - 에이전트 클릭시 에이전트 목록 트리가 닫히는 오류 수정

## Application

**_트랜잭션 추적_** - 트리뷰 기능 추가 및 보완

**_히트맵_** - 히트맵 x축 이동 버튼 클릭시 시간변경 오류 수정

## Container

**_컨테이너맵_** - 이전 컨테이너 로그를 조회할 수 있는 탭 추가

**_컨테이너맵_** - 발생한 이벤트에 맞는 탭에 포커스가 맞춰지도록 기능 추가

**_컨테이너맵_** - 정상(파란색) 임계치 구역별 3단계 구분 없애고 단일 컬러로 수정

**_컨테이너맵_** - Pod 카테고리에서 최신로그메시지 탭이 보이지 않는 오류 수정

**_노드 상세_** - 컨테이너 목록에 컬럼 및 데이터 변경

**_컨테이너 목록_** - MemWsByRequest 컬럼 데이터 오류 수정
