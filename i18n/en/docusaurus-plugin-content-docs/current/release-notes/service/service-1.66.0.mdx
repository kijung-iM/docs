## `공통`

**_클라우드 모니터링_** - 네이버 클라우드 설치 기능 추가

**_그룹 관리_** - 그룹 간 프로젝트 이동 기능 사용성 개선

**_LogSink_** - 로그 필터링 기능 개선

## Application

**_멀티트랜잭션_** - 브라우저 크기 조절 시 차트 깨짐 현상 수정

**_이벤트 기록_** - 테이블 내 알림 메시지가 길어지는 경우 개행 처리

## Server

**_정비 계획_** - 프로젝트 정비 계획 일간, 주간, 년간 반복 설정 기능 추가 

**_이벤트 설정_** - 프로세스 이벤트 설정 시 PID를 입력하여 프로세스명을 검색할 수 있도록 기능 개선

**_서버 목록_** - 프로세스 그룹화 해제 기능 추가
