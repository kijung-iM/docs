## `공통`

**_Flex Board_**

-   Added the function to export the dashboard data as a PDF file 
-   위젯 간 그룹화 기능 추가 (위젯 템플릿 > 기타 > 위젯 그룹화)
-   Renewed the time inquiry range when navigating menus between favorite dashboards

**_통합 대시보드_** - 트랜잭션 프로파일 연속 조회 불가 건 수정

## Integration

**_인티그레이션 프로젝트_**

-   클라우드 서비스를 비롯한 다양한 모니터링 대상을 단일 프로젝트에서 통합적으로 관리하기 위한 신규 상품 추가
-   Provided the dashboards for each integration item, analysis menus with collected metrics, and event setting functions
-   Integration Items
    -   Amazon CloudWatch
    -   Amazon ECS
    -   Naver Cloud Monitoring
    -   Oracle Cloud Monitor
    -   Azure Monitor

## Application

**_멀티 트랜잭션 추적_** - 프로젝트 목록 계층화 표시 오류 수정

**_트랜잭션 프로파일_** - 다이어그램 폰트 사이즈 조절 문제 수정

-   상세 화면 > 이벤트 Normal, Warning 타입 필터링 기능 추가
-   상세 화면 > 카드 상단 상태별 수량 부분 표시 스타일 개선

**_Out of Memory Killed 컨테이너_** - 조회 시간 범위 선택 옵션 제거, 테이블 정렬 기능 수정
