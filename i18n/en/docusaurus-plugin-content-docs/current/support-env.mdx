## DB Monitoring

### Supported versions by database type

| Environment | Support Environment      |
| ----------- | ------------------------ |
| Oracle      | Oracle 10g or later      |
| Tibero      | Tibero 5.x or later      |
| CUBRID      | CUBRID 9 or later        |
| MS-SQL      | SQL Server 2005 or later |
| MySQL       | MySQL 5.5 or later       |
| MariaDB     | MariaDB 5.5 or later     |
| Altibase    | Altibase 6 or later      |
| Redis       | Redis 6.0.5 or later     |
| MongoDB     | MongoDB 4.2 or later     |
| PostgeSQL   | EBD 9.2 or later         |

:::note

-   With DB Monitoring, a monitoring account with DB metadata (Dictionary, performance_schema) inquiry role is required.

-   The XOS agent can be applied to only the OS environment running on the x86 architecture.

:::
