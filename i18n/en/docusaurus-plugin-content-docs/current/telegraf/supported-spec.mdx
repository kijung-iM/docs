Before installing the WhaTap Telegraf agent, check the support environment. 

-   **OS**
    -   Linux Red Hat 6.x x64 이상
    -   Ubuntu 12.x x64 or later
    -   FreeBSD 10.x x64 or later
-   **Telegraf**
    -   Telegraf 1.3 이상 
        :::note
        socket_writer ouput plugin을 지원하는 버전만 가능합니다.
        :::
