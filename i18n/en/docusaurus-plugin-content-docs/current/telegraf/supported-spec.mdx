Before installing the WhaTap Telegraf agent, check the support environment. 

-   **OS**

    -   Linux Red Hat 6.x x64 or later
    -   Ubuntu 12.x x64 or later
    -   FreeBSD 10.x x64 or later

-   **Telegraf**

    -   Telegraf 1.3 or later 

    :::note

    The version that supports the socket_writer output plugin is allowed.

    :::
