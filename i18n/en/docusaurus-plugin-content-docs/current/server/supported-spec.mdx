## Support environments for each OS

와탭 서버 모니터링 에이전트를 설치하기 전에 지원 환경을 확인하세요. X86 기반에서 동작하는 **Linux**, **FreeBSD**, **Windows** 에 적용할 수 있습니다.

<!-- * Rocky Linux 8 이상 -->

-   **Linux**  
    -   CentOS 5 or later
    -   RedHat Server 5.0 이상
    -   Rocky Linux 8.6 or later
    -   SUSE 10.1 or later
    -   Amazon Linux 모든 버전 
    -   Ubuntu 12.04 or later
    -   Debian 6.0 or later
-   **Unix**  
    -   AIX 6.1 이상
    -   HP-UX 11.23 이상
    -   SunOS 5.8 이상
    -   FreeBSD 10 or later
-   **Windows**    
    -   Windows 2008 R2 이상
    -   Windows 7 SP1 버전 이상

:::note
지원 리스트에 포함되지 않는 환경이라면 support@whatap.io로 문의해 주세요.
:::
