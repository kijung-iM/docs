Home > Select Project > **_Alert_** > **_Event History_**

경고 알림이 발생한 이력을 확인할 수 있습니다. 발생한 모든 이벤트 내역 리스트 조회가 가능합니다. 

![서버 이벤트 기록 sc](/img/event-history-server.png)

![number1](/img/number-01.png) **상단 옵션**

-   **_입력창_**: 입력창에서 서버 이름 혹은 이벤트 이름으로 검색할 수 있습니다.
-   **_진행 중인 이벤트만 보기_**: 이벤트 기록 목록에서 **_진행 중인 이벤트_**만 조회할 수 있습니다.
-   **_마지막 변경 시간_**: 이벤트 기록 목록에 **_마지막 변경 시간 컬럼_**을 추가할 수 있습니다.
-   **_CSV_**: 조회한 이벤트 기록 결과를 csv 파일로 저장할 수 있습니다. **_최대 CSV 라인 수_**를 입력한 다음 **_다운로드_** 버튼을 선택하세요.

![number2](/img/number-02.png) **이벤트 기록 목록**

-   **_위험도_**<br/>
      위험도는 <span class='vslow'>Fatal</span>(위험), <span class='slow'>Warning</span>(경고) 색깔로 표시됩니다.
-   **_이벤트 이름_**<br/>
      이벤트 정책에서 설정한 값을 바탕으로 발생한 이벤트 이름입니다.
-   **_이벤트 발생 시각_**<br/>
      이벤트 발생 시간입니다.
-   **_+ 처리 내역_**<br/>
      이벤트 처리 내역을 추가할 수 있습니다.
-   **_서버_**<br/>
      서버 이름입니다. 
-   **_스냅샷_**<br/>
      발생한 이벤트에 대한 **_CPU_**, **_Memory Used_**, **_디스크_**, **_네트워크_**에 대한 스냅샷 정보입니다. 

### Event Detail Info

이벤트 기록 목록에서 해당 **_이벤트 이름_**을 선택하면 이벤트 발생 전후 5분간의 **_상세_** 정보로 이동합니다.

![서버 이벤트 상세](/img/server-event-detail.png)

-   **_차트 위젯_**<br/>
      알람 발생 전후 5분간의 **_CPU_**, **_Memory Usage_**, **_Disk I/O_**, **_Network_** 차트를 보여줍니다. 이벤트에 해당하는 부분의 **_위험도_**를 색깔로 표시하고 있습니다.
-   **_프로세스_**<br/>
      이벤트에 해당하는 프로세스 정보를 상위로부터 10개를 보여주고 있습니다. 
-   **_처리 내역 상세 정보_**<br/>
      이벤트에 대한 처리 내역 리스트를 보여줍니다. **_처리 내역 추가_** 버튼을 선택하면 내역을 작성하고 저장할 수 있습니다.
