:::tip

-   Namespaces are collected under the WhaTap metrics **category**. The notation format changes the Oracle CloClou namespace to **lowercase** and then changes "/" to "**\_**."

    > e.g. **_oci\__**

-   Metrics are collected as the WhaTap **metrics**. The notation format is "OracleCloudMetric.Stat."

-   For the collected Oracle Cloud metrics, see [the following](https://docs.oracle.com/en-us/iaas/Content/Monitoring/Concepts/monitoringoverview.htm#SupportedServices).

:::
