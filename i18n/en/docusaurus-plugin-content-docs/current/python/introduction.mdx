WhaTap Python Application Monitoring provides the monitoring service for Python-based web application servers. It is the only product in Korea that monitors the Web frameworks, servers, and batches at once.

WhaTap Python Monitoring can be applied to major web frameworks, and it can also track DB queries and external calls to find which calls are being delayed. To trace data for a specific transaction other than HTTP or HTTPS, use the corresponding method.
