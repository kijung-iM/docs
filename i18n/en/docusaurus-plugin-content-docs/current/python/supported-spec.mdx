## Python agent-supported environment

Before installing the WhaTap Python monitoring agent, check the support environment.

-   **OS**

    -   Redhat/CentOS 6.x (64 bit) or later

    -   Debian/Ubuntu 12.04 (64 bit) or later

    -   OSX 10.15 Catalina

-   **Support versions**

    -   WSGI Application (Python 2.7 & Python 3.3 or later)

    -   ASGI Application(Python 3.7 or later)

-   **Support libraries**

    -   External calls: HttpLib, Requests, Urllib

    -   Database : MySQL, SQL Server, PostgreSQL, Redis

    -   Kombu, Pika

-   **Application Server**

    Django, Bottle, CherryPy, Flask, Tornado, Nameko, Celery, FastAPI(0.84.0 or later)
