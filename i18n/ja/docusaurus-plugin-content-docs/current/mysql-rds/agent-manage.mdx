ホーム画面 > プロジェクト選択 > **_管理_** > **_エージェント設定_**

エージェントを再起動または削除する方法について案内します。

## エージェント再起動

![エージェント再起動](/img/mysql-rds-agent-settings-restart.png)

エージェントを再起動するには、エージェント設定オプションを変更し、画面右上の**_適用_**ボタンを選択してください。

## エージェント削除

![エージェント削除](/img/mysql-rds-agent-remove.png)

**_エージェント一覧_**でエージェント項目の右側の![削除アイコン](/img/ico-trash.svg)ボタンを選択してください。確認メッセージウィンドウが表示されたら、**_確認_**ボタンを選択してください。選択したエージェントを一覧から削除します。
