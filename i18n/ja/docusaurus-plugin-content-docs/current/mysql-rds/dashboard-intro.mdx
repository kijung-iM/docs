**_ダッシュボード_**は、リアルタイムで変化するプロジェクトの全体現況をひと目で把握できる機能です。**インスタンス**は、エージェントのインストールを通じて接続したデータベースを意味します。**_インスタンス一覧_**では、プロジェクトに登録した全体インスタンスの現況を把握することができます。**_インスタンスモニタリング_**では、個別インスタンスの性能指標やアクティブセッションをより多様にモニタリングすることができます。**_マルチインスタンスモニタリング_**では、複数のインスタンスの指標とアクティブセッションをひと目で把握することができます。

-   データベースのステータスや主要な性能指標などをひと目で把握できるように構成されています。性能指標は収集周期（5秒）間隔で更新します。
-   現在実行中のアクティブセッション（Query含む）、ロック（Lock）、プロセス情報を確認することができます。基本的な指標はクエリ方式で照会するため、OSやCloud環境などに制限されません。OSプロセス情報と一部のシステムリソース指標を一緒に表示するには、XOSエージェントを追加でインストールする必要があります。
-   リアルタイムモニタリングデータをすべて保存し、特定時点で進行中だった状況を分析できる機能を提供します。

## インスタンス一覧

エージェントをインストールすることで、接続したデータベースサーバーの一覧を確認し、各インスタンスに対する主要な性能指標や状態などを把握できます。性能指標は5秒間隔で更新し、カラム設定を通じて順序を変更したり、希望する指標に変更することができます。詳細については、[次の文書](instance-list)を参考にしてください。

## インスタンスモニタリング

データベースサーバーのコア指標、アクティブセッション、ロックツリーに関する詳細情報をリアルタイムでモニタリングできます。また、トラブルになり得る警告を確認することもできます。収集データは5秒間隔で更新します。詳細については、[次の文書](instance-monitoring)を参考にしてください。
