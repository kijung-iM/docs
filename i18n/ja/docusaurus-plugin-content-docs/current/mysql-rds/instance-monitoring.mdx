## 基本画面案内

<ProdImg img='db-instance-monitoring.png' desc='인스턴스 모니터링' />

### ![](/img/number-01.png)照会時間と対象を選択する

基本的に、過去10分間の収集データをリアルタイムで照会します。**_時間_**と**_インスタンス_**項目で時間と対象を選択すると、収集データが![数字2](/img/number-02.png)グラフチャートと![数字4](/img/number-04.png)の**_アクティブセッション_**テーブルに自動反映されます。

-   過去のデータを照会するには、緑のボタンをクリックし、照会時間を選択してください。過去3週間までのデータを照会できます。 

-   任意の日付と時刻を照会するには、時計表示を選択してください。日付と時刻のテキスト領域をクリックすると、日付と時刻を選択するオプションが表示されます。再びリアルタイムでデータを照会するには![リアルタイムアイコン](/img/ico-play-time.svg)ボタンを選択してください。

-   他のインスタンスのデータを照会するには、**_ インスタンス_** をクリックし、データベースサーバーの名前を選択してください。

### ![](/img/number-02.png)グラフチャートウィジェット領域

![数字3](/img/number-03.png)**_Default_**プリセットで保存した16の指標をグラフチャート形式のウィジェットで表示します。横軸は時間、縦軸は各指標の数値です。各指標の意味を知りたい場合は、名前の横の<span class='mark'>i</span>ボタンを選択するか、[次の文書](metrics-data-list)を参考にしてください。

-   ウィジェットの位置を変更するには、ウィジェットの上部を選択した状態でドラッグしてください。

    :::note

    ウィジェットのサイズは変更できません。

    :::

-   ウィジェットで特定の時間を選択すると、リアルタイム照会モードを停止し、選択した時間に実行したアクティブセッション情報を![数字4](/img/number-04.png)の**_アクティブセッション_**テーブルに表示します。

### ![](/img/number-04.png)アクティブセッション/ロックツリー

リアルタイムで実行中のアクティブセッションとロックツリー情報を照会することができます。ウィジェットで特定の時間を選択すると、リアルタイム照会モードを停止し、選択した時間に実行したセッション情報をテーブルに表示します。テーブルの上には5秒単位でデータを照会できる時間選択ボタンが表示されます。

## 指標を選択する

ウィジェットに指標グラフを追加・交替するには、ウィジェットの右上の![設定アイコン](/img/ico-setting.svg)ボタンを選択してください。**_Metric Select_**ウィンドウが表示されます。設定を完了して**_保存_**ボタンを選択してください。

![指標選択](/img/mysql-rds-merics-select.png)

-   ***プレビュー***：選択した指標に対するグラフをプレビューすることができます。

-   **_タイトル_**：ウィジェットの名前を入力してください。

-   **_おすすめ指標_**：トグルボタンを選択すると、***おすすめ指標***または**_すべての指標_**が確認できます。**_おすすめ指標_**は、データベースモニタリングでよく見る指標を集めておいた一覧です。

-   ![数字5](/img/number-05.png)領域で追加する指標を検索し、その指標の右側の![追加アイコン](/img/ico-add.svg)ボタンを選択してください。選択した指標は、![数字6](/img/number-06.png)領域に追加されます。

-   ![数字7](/img/ic-number-07.svg)領域で指標の推移をグラフで確認し、ウィジェットを追加することもできます。ウィジェットの右上の![追加アイコン](/img/ico-add.svg)ボタンを選択してください。**_Cateogry_**でご希望の分類を選択して指標一覧を要約することができます。

-   追加した指標を別の指標に変更するには、![数字6](/img/number-06.png)領域で変更したい指標を選択してください。以下の指標一覧で、変更したい指標の名前またはウィジェットを選択してください。

-   追加した指標を削除するには、![数字6](/img/number-06.png)領域で削除しようとする項目の![削除アイコン](/img/ico-trash.svg)ボタンを選択してください。指標は、少なくとも1つ以上選択する必要があります。

## プリセット保存

ウィジェットの指標を追加・変更し、レイアウトステータスをプリセットとして保存して読み込みできます。**_Default_**プリセットは修正できません。 

1.  ウィジェットの指標を追加・変更してください。

2.  画面右上の![数字3](/img/number-03.png)**_プリセット保存_**ボタンを選択してください。 

3.  名前を変更した後、**_保存_**ボタンを選択してください。

**_プリセット_**一覧で保存した項目を選択して設定を読み込むことができます。
