WhaTap MySQL RDSモニタリングサービスは、既存のMySQLモニタリング機能を提供するだけでなく、エージェントのインストールから管理までWhaTapで提供します。データベース接続さえ成功すれば、DBエージェント（DBX）をダウンロードしてインストール·設定するなどの面倒な過程なく、簡単にDBモニタリングを開始できます。
