WhaTapのモニタリングサービスに対する迅速な理解と効率的な利用のために学習資料を提供します。 アプリケーションモニタリングやサーバーモニタリング、ブラウザー（Browser）モニタリング製品の活用に関心のあるユーザーを対象とする基本的な案内文書を補完したものです。 

![学習する1.100.0 sc](/img/learning.png)

### アプリケーションモニタリング

#### [アプリケーションダッシュボードを確認する](best-practice-guides/about-apm-dashboard)

WhaTapの**_アプリケーションダッシュボード_**メニューのチャート型ウィジェットに対する分析方法について、次のような手順で案内します。

-   アプリケーション接続のステータスを確認

-   トランザクション分析

-   ユーザーおよびサービス、リソース分析

#### [ヒートマップトランザクションを確認する](best-practice-guides/about-apm-hitmap-class)

WhaTapの**_ヒートマップトランザクション_**メニューを活用してサービスの障害状況を把握する方法について、次のような手順で案内します。

-   パターン分析

-   区間およびスタック分析

-   呼び出し関係分析

-   トランザクションログ分析

### サーバーモニタリング

#### [リソースボードを確認する](best-practice-guides/about-server-dashboard)

WhaTapの**_リソースボード_**メニューのチャート型ウィジェットに対する分析方法について、次のような手順で案内します。

-   サーバモニタリングの主要なメトリック

-   サーバステータスの概要

-   システム運用統計

-   早期アラートおよび通知

### ブラウザー（Browser）モニタリング

#### [ブラウザーモニタリングを活用する](best-practice-guides/using-browser-monitoring)

WhaTapのブラウザーモニタリングを活用する方法について、次のような手順で案内します。

-   ページロード分析

-   AJAXモニタリング・分析

-   ユーザー接続環境分析

-   エラー分析
