와탭 모니터링 서비스 초기 화면에서 프로젝트를 선택한 다음 **_프로젝트 메뉴_** 하위에 **_보고서_**를 선택하세요.

レポートは、個々のプロジェクトに対してサービス利用状況と障害の発生記録を報告する文書です。サービスモニタリング担当者は、レポートを通じて関連部署の担当者と現況を共有します。

モニタリングデータ分析は、サービスの改善方向を定める指標となるため、極めて重要です。しかし、複数のダッシュボードのデータを収集して文書化するのは面倒です。  

WhaTapの**_レポート_**メニューは、**レポート作成業務の自動化**をサポートします。レポートメニューから、週次レポートを作成したり、設定した時間にレポートを共有したり、さまざまな形式で管理することができます。

:::tip

**レポート**をダウンロードするか、印刷またはメール送信の予約を希望する場合は、そのボタンをクリックしてください。ダウンロードはhtml形式で可能です。

:::
