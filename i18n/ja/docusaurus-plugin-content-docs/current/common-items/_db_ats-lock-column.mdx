## カラム情報の案内{#column-info}

| 項目              | 説明                                                                                  |
| --------------- | ----------------------------------------------------------------------------------- |
| Instance Name   | インスタンス名                                                                             |
| id              | connection識別子                                                                       |
| user            | SQL実行アカウント                                                                          |
| host            | このスレッドが接続されたhostname                                                                |
| db              | default database名、 ない場合はNULL                                                        |
| command         | 実行中のコマンドtype（Query/Sleep/Daemonなど）                                                  |
| time            | 現在COMMANDの遂行時間（秒）                                                                   |
| state           | スレッドの現在の状態（executing/updating/sending dataなど）                                       |
| query           | 実行されているstatementです。`Processlist.info`からリテラル値を除外したSQLを表示します。                         |
| query_param     | `query`から分離したリテラル値                                                                  |
| type            | Threadのタイプ。FOREGROUND/BACKGROUNDなど                                                  |
| connection_type | Connectionをする時に使用されるprotocol。TCP/IP, SSL/TLS                                        |
| query_hash      | queryのハッシュ値（WhaTap内部管理値）                                                            |
| os_id           | MySQL threadと接続されたOSのthread Id。Thread pool pluginを使用する場合はNULLである可能性があります。           |
| thread_id       | thread識別子                                                                           |
| name            | サーバーのコードに関連付けられた名前。ユーザConnection処理は、Thread/sql/one_connectionなど                     |
| cpu(xos)        | XOSをインストールした場合、収集されるCPUの使用率                                                         |
| query_id        | Query ID                                                                            |
| time_ms         | 遂行時間（milliseconds）                                                                  |
| stage           | 現在プロセスが進行中の状態です。                                                                    |
| max_stage       | STAGEの最大数（MariaDB）                                                                  |
| progress        | 現在ステップ内のプロセス進行率（0-100%）。                                                            |
| memory_used     | スレッドで使用するメモリ（バイト）です。                                                                |
| max_memory_used | スレッドで使用した最大メモリ（バイト）です。                                                              |
| examined_rows   | スレッドから検索した行です。UPDATE、DELETE及び類似の命令文によってのみアップデートされます。SELECTおよびその他の問い合わせ値は0として維持されます。 |
| holder type     | holderセッションのtype                                                                    |
| lock mode       | holder_mode                                                                         |
| waiter type     | waiterセッションのtype                                                                    |
| lock request    | waiter_mode                                                                         |
| table           | holder_table/waiter_table                                                           |
| index           | holder_index/waiter_index                                                           |
