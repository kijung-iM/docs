### app_proc_counter

---

アプリケーションプロセスの指標です。

-   収集間隔：5秒
-   統計データ：5分、1時間

#### Tags

| タグ名          | 説明            | 備考    |
| ------------ | ------------- | ----- |
| container    | コンテナ名         | ユニーク値 |
| containerKey | コンテナキーの値      | ユニーク値 |
| host_ip      | Host IP       | ユニーク値 |
| okindName    | アプリケーションの種類名  | -     |
| oname        | アプリケーション名     | ユニーク値 |
| onodeName    | アプリケーションNode名 | -     |
| pid          | アプリケーションPID   | -     |
| type         | アプリケーションタイプ   | 言語名   |

| proc_fd              | 整数  | プロセスfdの使用数                     | -  |
