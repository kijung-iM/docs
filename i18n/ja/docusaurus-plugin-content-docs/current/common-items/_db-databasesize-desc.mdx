## データベースのサイズ

ホーム画面 > プロジェクト選択 > **_統計/レポート_** > **_データベースサイズ_**

データベースごとのテーブルのサイズを集計し、推移情報をチャートで提供します。収集サイクルは3時間に1回収集します。 

1.  **_시간_**에서 조회하고 싶은 시간을 설정하세요. 녹색 버튼을 선택해 조회 시간을 선택할 수도 있습니다.

2.  **_인스턴스_**에서 조회하려는 대상을 선택하세요.

3.  ![検索アイコン](/img/ico-btn-search.png)ボタンを選択してください。

:::note

照会時間は、最大3週間まで選択できます。任意の日付と時刻を照会するには、時計の部分を選択してください。日付と時刻のテキスト領域をクリックすると、日付と時刻を選択するオプションが表示されます。

:::
