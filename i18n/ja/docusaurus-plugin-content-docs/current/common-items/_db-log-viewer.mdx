ホーム画面 > プロジェクト選択 > **_分析_** > ***ログファイルビューア***または**_ログファイルビューア（cloud）_**

エージェントをインストールしたデータベースサーバーまたはクラウドデータベースサーバーのログファイルの内容を確認できます。一度に1つのデータベースログを確認できます。

![ログファイルビューアー](/img/analysis-log-viewer.png)

-   ![指示線1](/img/number-01.png)**_インスタンス_**

    プロジェクトに含まれるインスタンスを選択できます。インスタンス項目をクリックして、個々のインスタンスを選択してください。

-   ![指示線2](/img/number-02.png)**_ログファイル一覧_**

    各インスタンスごとに確認できるログファイルの一覧です。ファイルが多くて見つけにくい場合は、検索を通じて目的のファイルを見つけることができます。

-   ![指示線3](/img/number-03.png)**_ログファイルビューア_** 

    2000バイト（byte）単位でログ内容を確認することができます。![前ページアイコン](/img/left-arrow.svg)または![後ページアイコン](/img/right-arrow.svg)ボタンをクリックして2000バイト（byte）単位でページを移動することができます。最初と最後に移動するには![最初のページアイコン](/img/ico-first.svg)または![最後のページアイコン](/img/ico-last.svg)ボタンを選択してください。

    ユーザーが希望する任意の場所へ移動するには、希望する位置を入力し、**_移動_**ボタンを選択してください。
