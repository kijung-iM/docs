### 基本画面案内

<ProdImg img='db-database-sql-stat.png' desc='SQL 통계' />

<Xclude product='postgresql'>

1.  **_インスタンス_**一覧から照会する対象を選択してください。重複選択することができ、すべてのインスタンスを選択するには**_全体_**を選択してください。

2.  **_開始日_**、**_開始時間_**、**_期間_**で照会日付と時間、期間を設定してください。

3.  ![検索アイコン](/img/ico-btn-search.png)ボタンを選択してください。

</Xclude>

<InDoc product='postgresql'>

1.  ![숫자 1](/img/number-01.png) **_시간_**에서 데이터를 조회할 시간과 날짜를 선택하세요. 녹색 버튼을 선택해 조회 시간을 선택할 수도 있습니다.

2.  ![숫자 1](/img/number-01.png) **_인스턴스_**에서 데이터를 조회할 대상을 선택하세요. 여러 개의 인스턴스를 선택할 수 있으며 모든 인스턴스를 선택하려면 **_전체 선택_**을 클릭하세요.

3.  ![検索アイコン](/img/ico-btn-search.png)ボタンを選択してください。

</InDoc>

上位50件までの内容をテーブルに表示します。

:::note

<Xclude product='postgresql'>

-   SQLのパラメータ値を**_フィルター_**テキストボックスに入力し、希望する内容をフィルタリングすることができます。

-   デフォルトのソート順は実行回数です。ソート順を変更するには、 **_ソート順 _**の一覧から希望する基準を選択してください。

-   조회한 내용을 csv 형식의 파일로 다운받으려면 ![csv 다운로드 아이콘](/img/ic-csv-download-ms.svg) 버튼을 선택하세요.

</Xclude>

<InDoc product='postgresql'>

-   임의의 날짜와 시간을 조회하려면 시계 표시 부분을 선택하세요. 날짜와 시간 텍스트 영역을 클릭하면 날짜와 시간을 선택할 수 있는 옵션이 나타납니다.

-   테이블 헤더의 각 컬럼을 클릭하면 선택한 컬럼을 기준으로 목록을 정렬할 수 있습니다.

-   **_조회 기준_**에서 선택한 항목 따라 조회 결과를 정렬할 수 있습니다. 원하는 항목을 선택한 다음 ![검색 아이콘](/img/ico-btn-search.png) 버튼을 선택하세요. 

-   조회한 내용을 csv 형식의 파일로 다운받으려면 ![Download icon](/img/ico-download.svg) 버튼을 선택하세요.

</InDoc>

:::
