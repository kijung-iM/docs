### 基本画面ガイド

<ProdImg img='db-database-sql-stat.png' desc='SQL統計' />

<Xclude product='postgresql'>

1.  **_インスタンス_**一覧から照会する対象を選択してください。重複選択することができ、すべてのインスタンスを選択するには**_全体_**を選択してください。

2.  **_開始日_**、**_開始時間_**、**_期間_**で照会日付と時間、期間を設定してください。

3.  ![検索アイコン](/img/ico-btn-search.png)ボタンを選択してください。

</Xclude>

<InDoc product='postgresql'>

1.  ![数字1](/img/number-01.png) **_時間_**で、データを照会する時間と日付を選択してください。緑のボタンを選択して照会時間を選択することもできます。

2.  ![数字1](/img/number-01.png) **_インスタンス_**でデータを照会する対象を選択してください。複数のインスタンスを選択でき、すべてのインスタンスを選択するには、**_全体選択_**をクリックしてください。

3.  ![検索アイコン](/img/ico-btn-search.png)ボタンを選択してください。

</InDoc>

上位50件までの内容をテーブルに表示します。

:::note

<Xclude product='postgresql'>

-   SQLのパラメータ値を**_フィルター_**テキストボックスに入力し、希望する内容をフィルタリングすることができます。

-   デフォルトのソート順は実行回数です。ソート順を変更するには、 **_ソート順 _**の一覧から希望する基準を選択してください。

-   照会した内容をcsv形式のファイルにダウンロードするには、![csvダウンロードアイコン](/img/ic-csv-download-ms.svg)ボタンを選択してください。

</Xclude>

<InDoc product='postgresql'>

-   任意の日付と時刻を照会するには、時計の部分を選択してください。日付と時刻のテキスト領域をクリックすると、日付と時刻を選択するオプションが表示されます。

-   テーブルヘッダーの各カラムをクリックすると、選択したカラムに基づいて一覧をソートすることができます。

-   **_照会基準_**で選択した項目によって照会結果をソートすることができます。ご希望の項目を選択した後![検索アイコン](/img/ico-btn-search.png)ボタンを選択してください。

-   照会した内容をcsv形式のファイルにダウンロードするには、![Download icon](/img/ico-download.svg)ボタンを選択してください。

</InDoc>

:::
