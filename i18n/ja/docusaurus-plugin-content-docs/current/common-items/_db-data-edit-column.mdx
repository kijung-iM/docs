## テーブルカラムの変更

テーブルのカラム項目を表示または非表示にすることができ、順序を変更できます。

1.  テーブルの右上の![コラムアイコン](/img/ico-column.svg)ボタンを選択してください。 

2.  **_コラムの編集_**ウィンドウが表示されたら、テーブルのカラムに表示する項目を非表示または追加、順序を変更してください。

    -   カラム項目を選択し、[左方向アイコン](/img/left-arrow.svg)または![右方向アイコン](/img/right-arrow.svg)、![追加アイコン](/img/ico-add.svg)または![抜きアイコン](/img/ico-minus.svg)ボタンを選択してください。***表示***または***非表示***へ移動します。
    -   カラム項目を選択した状態でドラッグすると、位置を移動できます。

3.  設定を完了した後、**_保存_**ボタンをクリックしてください。

:::note

設定した内容はブラウザクッキー値で保存され、ページの更新後もステータスを維持します。クッキーの削除またはその他の理由で設定間にエラーが発生した場合は、初期化します。

:::
