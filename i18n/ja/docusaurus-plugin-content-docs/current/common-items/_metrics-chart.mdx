ホーム画面 > プロジェクト選択 > **_分析_** > **_メトリクスチャート_**

**_マトリックスチャート_**メニューでモニタリング対象から収集されたマトリックスデータを次のようにチャートで照会することができます。時間と指標を指定することは必須事項です。

![メトリクスチャートsc](/img/metrics-chart-sv.png)

### 上段オプション

![number 1](/img/number-01.png)領域の**_メトリクスチャート_**の上段オプションを通じてチャートの時間範囲とモニタリング対象エージェントを指定することができます。

-   **_時間_**：時間の総範囲でX軸の開始と終了を指定できます。

-   **_インターバル_**：時間間隔でX軸データの間隔を指定できます。

-   **_時間マージ_**：データマージ方式の1つとして、インターバルで指定した時間内のデータをマージできます。

    > 例）**_平均_**は1時間以内のデータ平均値を意味します。

-   ![アイコン](/img/ic-pageview.svg) **_エージェント_**：照会するエージェントを指定できます。 エージェントを指定しないと、全体が照会されます。

### 指標一覧

![number 2](/img/number-02.png)領域は、オプションを照会する指標のリストです。まず、**_カテゴリ_**を選択してください。 選択した**_カテゴリ_**下位の指標を照会した後、希望する指標を選択してください。 **_カテゴリ_**と指標を選択すると、![number 1](/img/number-01.png)領域の上段メニューで指定した時間範囲のデータに基づいて![number 3](/img/number-03.png)領域でチャートウィジェットを照会することができます。

:::note 

**データマージ**

データマージは、**_時間マージ_**と**_オブジェクトマージ_**を提供します。 

-   **_時間マージ_**は、ソースデータからフィールド値が同じデータを一定の間隔でマージします。 
-   **_オブジェクトマージ_**は、異なるフィールド値を持つデータの中からタグが一致する場合、そのデータをマージします。

:::

### チャートウィジェット

![number 3](/img/number-03.png)領域のチャートウィジェットの左上で指標名を確認できます。チャートウィジェットの右上で次のオプションを確認できます。

![メトリクスチャートウィジェット上段](/img/metrics-chart-widget-upper-server.png)

-   時間移動：![left-arrow](/img/ic-caret-left.svg)![left-arrow](/img/ic-caret-right.svg)左矢印または右矢印ボタンで選択した時間範囲だけ-1、+1ずつ移動できます。 

    > 例えば、時間範囲が2月13日00:00~2月14日00:00の場合、![left-arrow](/img/ic-caret-left.svg)左矢印を選択すると、2月12日00:00~2月13日00:00データが照会できます。

-   インターバル/時間のマージ：![number 1](/img/number-01.png)上段メニューで指定したインターバルと時間のマージを修正できます。

-   モニタリング対象：![モニタリング対象アイコン](/img/ic-filter-bl.svg)アイコンを選択してモニタリング対象を指定できます。選択しない場合は、全体を対象に照会します。

-   時間比較：! [時間比較アイコン](/img/ic-hourgalss.svg)アイコンを選択すると、同じ指標の以前の時間帯の推移を比較できます。

-   スナップショット：![スナップショットアイコン](/img/ic-camera.svg)アイコンを選択して、ウィジェットのオプションを除いたチャートのスナップショットを撮ることができます。

-   CSV：![CSVダウンロードアイコン](/img/ico-csv.svg)アイコンを選択してチャートを描くデータをCSVファイルとしてダウンロードできます。

-   詳細を見る：![詳細アイコン](/img/ico-detail.svg)アイコンを選択して詳細照会が可能です。モニタリング対象が多い場合に有用であり、モニタリング対象の指標推移を個別に確認できます。

:::note

メトリクスチャートウィジェットの上段にオプションが見えない場合は、![もっと見るアイコン](/img/ico-more.svg)アイコンを選択してください。

:::
