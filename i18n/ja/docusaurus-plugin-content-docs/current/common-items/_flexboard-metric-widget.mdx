## メトリクスとは

プロジェクトで収集中の数字のパフォーマンス指標を**メトリクス**と呼びます。メトリックは「カテゴリ」に分類され、カテゴリ別タグを使用してデータを探索できます。

-   **カテゴリ**（Category）：関連する指標をまとめる単位
-   **タグ**（Tags）：収集対象を区分できる固有情報が含まれたデータ
-   **フィールド**（Fields）：エージェントから収集した指標

メトリクスの詳細については、[次の文書](metrics-intro)を参照してください。

## メトリクスウィジェットの種類

**_すべてのメトリクス_**で追加できるウィジェットタイプは、カテゴリに対するタグとフィールド情報を一覧化したテーブル形式のウィジェット、単一フィールドに対する推移グラフに分けます。

-   **テーブルウィジェット**：選択したメトリックカテゴリに関するデータを表形式で提供します。

      ![テーブルウィジェット](/img/metric-widget-table.png)

-   **シリーズウィジェット**：選択したメトリックカテゴリの選択フィールドに関するデータをシリーズチャート形式として提供します。

      ![シリーズウィジェット](/img/metric-widget-series.png)
