## ![](/img/number-02.png)コンテンツフィルター領域

#### Contentフィルター

ログのコンテンツの中から、そのキーワードを含むログのみをフィルタリングします。 

![](https://img.whatap.io/media/images/livetail_008_contents_filter.png)

-   入力ウィンドウにフィルターを入力し、![アイコン](/img/ic-search.svg)**_検索_**ボタンをクリックします。クライアントからログデータをフィルタリングし、[number-s3](/img/number-03.png)を参照することができます。 
-   キーワードを含むContentの文字列がハイライトされます。
-   １つまたは複数のキーワードでフィルタリングできます。

<!-- whatap is good~whatap is the best -->

#### Contentフィルター条件

複数キーワードでフィルタリングをする場合は、次のように記述します。

| 入力文字列                 | 説明                                       | 結果                |
| --------------------- | ---------------------------------------- | ----------------- |
| `a b c`               | 各キーワードをスペースで区切りします。                      | a, b, c           |
| `"Whatap is good."`   | 分かち書きをキーワードに含めたい場合`''`または`""`で囲みます。      | Whatap is good.   |
| `"Whatap\\ is good."` | ""で囲まれたキーワードから`\`を含む場合、`\\`で入力する必要があります。 | Whatap\\ is good. |

#### ハイライト

目的のキーワードを簡単に識別できるようにハイライト機能を提供します。

![ライブテールハイライト](/img/lt-highlight.png)

-   ![アイコン](/img/ic-highlight.svg) **_ハイライト_**ボタンをクリックしてキーワードを入力します。 
-   入力ウィンドウの左側の**_色_**を選択すると、選択できるカラーメニューが表示されます。 
-   基本的にログレベルによるハイライト(<span class='logwr'>WARN</span>, <span class='loger'>ERROR</span>, <span class='logft'>FATAL</span>)が適用されています。   
-   設定した内容は**プロジェクト単位**で保存されます。
