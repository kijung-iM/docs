## テーブルスペースサイズ

ホーム画面 > プロジェクト選択 > **_統計/レポート_** > **_テーブルスペースのサイズ_**

各テーブルのスペースごとの最大容量と使用量を時間の経過とともに照会することができます。グラフチャートの形で提供し、1日から最大3週間前まで推移情報を確認できます。

<ProdImg img='db-stat-table-space.png' desc='' />

1.  **_時間_**で照会したい時間を設定してください。

2.  インスタンスで推移情報を照会するインスタンス名を選択してください。

3.  ![検索アイコン](/img/ico-btn-search.png)ボタンを選択してください。

:::note

照会時間は、最大3週間まで選択できます。任意の日付と時刻を照会するには、時計の部分を選択してください。日付と時刻のテキスト領域をクリックすると、日付と時刻を選択するオプションが表示されます。

:::
