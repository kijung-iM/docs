## エージェントダウンロード（最新バージョン）

1.  [WhaTapモニタリングサービス](https://service.whatap.io)に移動してログインしてください。

2.  エージェントを更新するプロジェクトを選択してください。

3.  左側のメニューで**_管理_** > **_エージェントアップデート_**を選択してください。

4.  **_最新バージョン_** または、 **_最新バージョンですべてのエージェントにダウンロード_**ボタンをクリックしてください。

    ![エージェントアップデート](/img/java-update-agent.png)

エージェントをダウンロードしてから、アプリケーションサーバーを再起動してください。アップデートを自動適用します。
