## Flexボードの編集

**_Flexボード_**メニューの**_ダッシュボード一覧_**で![編集アイコン](/img/ico-edit.svg)ボタンを選択してください。画面右側にFlexボード管理ウィンドウが表示されます。

:::note

**Flex 보드 편집 기능 권한 안내**

-   ホーム画面 > **_Flexボード_**

    개인 계정 대시보드로 권한에 따른 영향은 없으나 읽기 전용으로 공유된 대시보드의 경우 수정할 수 없습니다. 

-   ホーム画面 > プロジェクト選択 > **_ダッシュボード_** > **_Flexボード_**

    프로젝트 **수정** 권한 또는 **프로젝트 플렉스보드 편집** 권한, **Site Admin** 권한을 가진 사용자가 수정할 수 있습니다.

-   メンバー権限の詳細については、[次の文書](../project/project-structure#member-auth)を参考にしてください。

:::

![Flexボード管理](/img/flexboard-manage.png)

設定できる属性は次のとおりです。

-   **_名前の変更_**：ダッシュボード名を修正できます。

-   **_プロジェクト_**：ダッシュボードのウィジェットからデータを読み込むプロジェクトを選択できます。

    :::note

    このオプションは、ホーム画面 > **_Flexボード_**メニューでのみ設定できます。

    :::

-   **_レイアウト_**

    -   ウィジェットを選択し、ドラッグして場所を移動できます。

    -   ウィジェットの右下をマウスで選択し、ドラッグしてウィジェットのサイズを調整できます。

    -   ウィジェットを削除するにはウィジェット右上の![削除アイコン](/img/ico-trash.svg)ボタンを選択してください。

    -   **_オーバーラップ_**トグルボタンをアクティブにすると、ウィジェットの配置が重なった状態で配置できます。

    :::note

    ホーム画面 > **_Flexボード_**メニューに入った場合、各ウィジェットごとにデータを読み込みするプロジェクトを選択できます。ウィジェット右上の![追加アイコン](/img/ico-add.svg)ボタンをクリックし、目的のプロジェクトを選択してください。プロジェクトを検索して選択でき、複数選択することもできます。選択を完了した後、**_適用_**ボタンを選択してください。

    :::

-   **_ウィジェット.json_**：ダッシュボードのウィジェット設定をJSON形式で読み込むことができます。![コピーアイコン](/img/ico-copy.svg)ボタンをクリックすると、JSONの内容をコピーします。

-   **_オプション.json_**：ダッシュボードに設定されたオプションをJSON形式で読み込むことができます。 ![コピーアイコン](/img/ico-copy.svg)ボタンをクリックすると、JSONの内容をコピーします。

## Flexボードの削除

**_Flexボード_** > **_ダッシュボード一覧_**から削除しようとする項目の右側にある![削除アイコン](/img/ico-trash.svg)ボタンをクリックしてください。確認メッセージが表示されたら、**_削除_**ボタンを選択してください。

:::note

-   ダッシュボード所有者がダッシュボードを削除する場合、ダッシュボードを共有しているすべてのユーザーアカウントもダッシュボードを削除します。
-   ダッシュボードを共有したユーザーがダッシュボードを削除した場合、そのユーザーの**_ダッシュボード一覧_**から削除されます。元のダッシュボードは影響を受けません。

:::
