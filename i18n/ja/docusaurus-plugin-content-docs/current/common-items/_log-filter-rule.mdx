フィルター入力時に使用できる文法の一覧です。**_ライブテール_**と**_ログ検索_**は同じ文法を使います。

| 文法            | 例              | 説明                         |
| ------------- | -------------- | -------------------------- |
| include(+)    | `+ote-gateway` | ote-gatewayを含むもののみをフィルタリング |
| exclude(-)    | `-ote-gateway` | ote-gateway以外のフィルタリングのみ    |
| startWith(\*) | `ERROR*`       | ERRORで始まるものだけをフィルタリング      |
| endWith(\*)   | `*ERROR`       | ERRORで終わるものだけをフィルタリング      |
