フィルター入力時に使用できる文法の一覧です。**_ライブテール_**と**_ログ検索_**は同じ文法を使います。

| 文法            | 例              | 説明                       |
| ------------- | -------------- | ------------------------ |
| include(+)    | `+ote-gateway` | ote-gatewayを含むものをフィルタリング |
| exclude(-)    | `-ote-gateway` | ote-gatewayを除くものをフィルタリング |
| startWith(\*) | `ERROR*`       | ERRORで始まるものをフィルタリング      |
| endWith(\*)   | `*ERROR`       | ERRORで終わるものをフィルタリング      |
