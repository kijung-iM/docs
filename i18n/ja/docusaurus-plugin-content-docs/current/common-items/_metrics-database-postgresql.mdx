**_メトリクスチャート_**メニューで検索できるメトリクス指標です。

<!-- 추후 업데이트 필요 관련 지표 추가 및 필드 값으로 교체 -->

<div class='metric'>

| Metric                | Description                                                            |
| --------------------- | ---------------------------------------------------------------------- |
| Active Sessions       | クエリを実行中のセッション数                                                         |
| Deadlock Count        | 検出されたdeadlockの数                                                        |
| Commit Count          | コミットされたトランザクションの数                                                      |
| Rollback Count        | ロールバックされたトランザクションの数                                                    |
| DML Count             | DML（insert、 update、 delete）の実行回数                                       |
| BLKS Read             | 読み込んだディスクブロック数（Physical read）                                          |
| BLKS Hit              | ディスクブロックキャッシュヒット数（ディスクブロックが既にバッファキャッシュにあるため読み込みを必要としない回数。Logical read） |
| BLKS Hit Ratio        | blks hit(キャッシュヒット)와 blks read(ディスク読み込み回数)の比率                           |
| Check Point           | checkpoint実行された回数                                                      |
| Total Sessions        | すべてのセッション数                                                             |
| Total Runtime         | active sessionの実行時間の合計                                                 |
| Long Running Sessions | 実行時間の長いセッションの数                                                         |
| Lock Wait Sessions    | lock 을 wait 하는 세션 수                                                    |
| Table Hit Ratio       | テーブルのblks hit(キャッシュヒット)와 blks read(ディスク読み込み回数)の比率                      |
| Index Hit Ratio       | インデックスのblks hit(キャッシュヒット)와 blks read(ディスク読み込み回数)の比率                    |
| Vacuum Running Count  | vacuumが実行された回数                                                         |

</div>
