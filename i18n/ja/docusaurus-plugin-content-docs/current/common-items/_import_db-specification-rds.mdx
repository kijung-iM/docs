-   **DB接続のみでモニタリング開始**

    データベース接続さえ成功すれば、すぐにモニタリングサービスを利用できます。

-   **リアルタイムDBモニタリング**

    リアルタイムでDB性能指標と実行中のSQLを収集し、問題点の分析と性能の最適化に必要なデータを共有し、データベースの意思決定を可能にします。Active Session、Lock Treeなど障害の主な懸念事項のダッシュボードを提供し、通知内容をリアルタイムで検知することができます。特定のSQLのPlan情報を確認し、SQLチューニングの手がかりを提供します。

-   **特定時点分析**

    指標の短期推移、長期推移、指標比較機能を提供することで、過去の時点での問題状況を分析できます。SQL、Lock Tree、Parallel Query、テーブルスペースの増減などの履歴情報を確認できます。 データベースパラメータの変更履歴を追跡します。
