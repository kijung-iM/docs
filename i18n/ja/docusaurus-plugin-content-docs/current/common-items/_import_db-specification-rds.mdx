-   **DB接続のみでモニタリング開始**

    データベース接続さえ成功すれば、すぐにモニタリングサービスを利用できます。

-   **リアルタイムDBモニタリング**

    リアルタイムでDB性能指標と実行中のSQLを収集し、問題点の分析と性能最適化に必要なデータを共有し、データベースの意思決定を可能にします。Active Session、Lock Treeなど障害の主な関心事であるダッシュボードを提供し、通知発生の内容をリアルタイムで検知することができます。特定SQLのPlan情報を確認し、SQLチューニングの手がかりを提供します。

-   **特定時点分析**

    指標の短期推移、長期推移、指標比較機能を提供することで、過ぎ去った時点の問題状況を分析することができます。SQL、Lock Tree、Parallel Query、テーブルスペース増減などの履歴情報を確認することができます。 データベースパラメータの変更履歴を追跡します。
