CPU、メモリ、ディスク使用量の収集設定を追加しない場合、Javaのjmxライブラリを介して収集されます。

-   **linux_proc_stat_enabled** <span class='type'>Boolean</span><br/>
      デフォルト値`false`<br/>
      CPU、メモリ、ディスク使用量をLinuxの_proc_ディレクトリから収集するかどうかを設定します。

-   **oshi_enabled** <span class='type'>Boolean</span><br/>
       デフォルト値`false`<br/>
        CPU、メモリ、ディスク使用量を測定する時、oshiライブラリを使用するかどうかを設定します。
    :::note
    whatap.agent.2.1.0バージョン以降でサポートします。
    :::

-   **sigar_enabled** <span class='type'>Boolean</span><br/>
        CPU、メモリ、ディスク使用量を測定する時、sigarライブラリを使用するかどうかを設定します。
    :::note
    whatap.agent.2.1.0バージョン以下の場合、デフォルト値は`true`であり、whatap.agent.2.1.0バージョン以上のデフォル値は、`false`です。
    :::
