WhaTapエージェントのインストールが完了しましたら、[WhaTapモニタリングサービス](https://service.whatap.io)にログインしてください。WhaTapモニタリングサービスは、円滑なサービス提供と安定した運用、異常検知モニタリングおよび把握のための情報を提供します。

DBパフォーマンス管理は、データベースのパフォーマンスをリアルタイムでモニタリングし、問題点を診断および分析します。これを通じて、問題となるSQLをチューニングまたは適用して問題点が解決されたかをリアルタイムでモニタリングします。

WhaTapデータベースモニタリングサービスは、これらの一連のタスクを循環管理できるように設計しました。データベースの動作をモニタリング、診断、分析し、可用性とパフォーマンスを一定のレベルに保つのに役立つように構成されています。データベースに何らかの問題が発生した場合、担当者にアラームを送信して状況を認知し、収集されたデータを通じてデータベースの状況をより精密に分析してRoot Causeを明確にするのに役立つサービスです。

WhaTapデータベースモニタリング サービスは、次の主要な機能を提供します。

-   **マルチモニタリング**

    Mulit-Serverモニタリング／Multi-instanceモニタリング／Multi-Databaseモニタリング

-   **リアルタイムモニタリング**

    セッション情報／リソース情報／各指標の現況／Session Detail

-   **トレンド分析**

    Performance Trend分析／Trend Compare／Long Term Trend／SQL連携

-   **SQL 分析**

    Top-N Query分析／SQL負荷量分析／SQL実行回数の分析／SQL Plan情報

-   **Lock Info**

    Lock情報／Lock Tree構造／(HOLDER/WAITER)

-   **分析**

    項目別：日別、週別、月別照会／日付別およびインスタンス間の比較分析

-   **管理**

    Agentの実行方式／DEVOPSサポート／通知イベントの受信設定／通知の伝達方式の設定／ユーザー別、チーム別の通知設定

-   **イベント**

    イベント履歴の照会／イベント統計／イベント設定

-   **レポート**

    日次レポート／週次DBレポート／月次DBレポート
