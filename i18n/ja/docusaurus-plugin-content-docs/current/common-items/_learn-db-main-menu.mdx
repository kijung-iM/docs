WhaTapエージェントのインストールが完了しましたら、[WhaTapモニタリングサービス](https://service.whatap.io)にログインしてください。WhaTapモニタリングサービスは、円滑なサービス提供と安定した運用、異常検知モニタリングおよび把握のための情報を提供します。

DBパフォーマンス管理は、データベースのパフォーマンスをリアルタイムでモニタリングし、問題を診断および分析します。問題のあるSQLをチューニングまたは適用し、問題が解決されたかをリアルタイムでモニタリングします。

WhaTapデータベースモニタリングサービスは、これらの一連のタスクをローリングベースで管理するように設計されています。データベースの動作をモニタリング、診断、分析して、一貫性のあるレベルの可用性とパフォーマンスを維持するように設計されています。データベースに何らかの問題が発生した場合、担当者にアラームを送信して状況を確認し、収集されたデータからデータベースの状況をより正確に分析してRoot Causeを特定するのに役立つサービスです。

WhaTapデータベースモニタリング サービスは、次の主要な機能を提供します。

-   **マルチモニタリング**

    Mulit-Serverモニタリング／Multi-instanceモニタリング／Multi-Databaseモニタリング

-   **リアルタイムモニタリング**

    セッション情報／リソース情報／各指標の現況／Session Detail

-   **トレンド分析**

    Performance Trend分析／Trend Compare／Long Term Trend／SQL連携

-   **SQL 分析**

    Top-N Query分析／SQL負荷分析／SQL実行数の分析／SQL Plan情報

-   **Lock Info**

    Lock情報／Lock Tree構造／(HOLDER/WAITER)

-   **分析**

    項目別：日次、週次、月次ビュー／日別およびインスタンス別の比較分析

-   **管理**

    Agentの実行方法／DEVOPSサポート／通知イベントの受信設定／通知配信方法の設定／ユーザー別、チーム別の通知設定

-   **イベント**

    イベント履歴の照会／イベント統計／イベント設定

-   **レポート**

    日次レポート／週次DBレポート／月次DBレポート
