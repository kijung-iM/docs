-   **大容量規模でも効率的な構造**

    多数のDBサーバー管理時にも単一化された性能データ収集および効率的なデータ保存構造を通じて最適な分析環境を提供します。

-   **クラウドDBモニタリング可能**

    Queryベースのモニタリングデータ収集構造で、DBサーバーの種類に制限はありません。AWS、GCP、Azure、OCI環境のDBもモニタリングでき、クラウドリソース指標と統合モニタリングにも対応しています。

-   **統合モニタリング**

    アプリケーション、サーバー、データベースモニタリング製品をすべてインストールすると、サービス異常現象把握時に連携したビューを通じて障害発生時に迅速な原因分析が可能になります。

-   **WhaTap SaaS**

    様々なデータベース製品を選択でき、月間サブスクリプション方式で使用期間を自由に決めることができます。（インストール型も提供可能）
