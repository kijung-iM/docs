-   **whatap.enabled** <span class='type'>Boolean</span>

    デフォルト値 `true`

    機能のすべてを有効にします。ただし、`false`である場合、サーバーと最小限に通信を維持するために情報は送信されます。

-   **transaction_enabled** <span class='type'>Boolean</span>

    デフォルト値 `true`

    トランザクション追跡機能をアクティブにします。`whatap.enabled`オプションの値が`false`の場合は無視してください。

-   **counter_enabled** <span class='type'>Boolean</span>

    デフォルト値 `true`

    パフォーマンス カウンター(トランザクション、リソースなど) 追跡を有効にします。`whatap.enabled`オプションの値が`false`の場合は無視してください。

-   **stat_enabled** <span class='type'>Boolean</span>

    デフォルト値 `true`

    統計情報の追跡機能を活性化します。5分単位で収集するトランザクション、SQL、HTTPCALL、UserAgent、Client IPなどの統計データなどが該当します。`whatap.enabled`オプションの値が`false`なら無視します。

-   **license** <span class='type'>String</span>

    エージェントのインストール時にサーバーから与えられたアクセスキーを入力します。 アクセスキーには、エージェントが属するプロジェクトとセキュリティ通信のためのパスワードキーが含まれます。

-   **encrypt_level** <span class='type'>encrypt_level [1, 2, 3]</span>

    デフォルト値`2`

    WhaTapエージェントは、サーバーにデータを送信する際、データ属性に基づいて選択的に暗号化します。高いセキュリティを維持しながらも、パフォーマンス上の利点を持っています。データ タイプに関係なく、一括した暗号化ポリシーを適用するには、次のオプションを参照してください。

    -   `1`：暗号化送信機能を無効
    -   `2`：SQLパラメータ、Plain Textなどの機密属性の暗号化された転送
    -   `3`：すべての項目の暗号化送信

-   **realtime_user_thinktime_max** <span class='type'>MiliSeconds</span>

    デフォルト値`300000`

    リアルタイムブラウザのユーザー数を測定する場合、同一ユーザーとして認められる呼び出し間隔の最大値を設定します。

-   **time_sync_interval_ms** <span class='type'>MiliSeconds</span>

    デフォルト値`300000`

    エージェントとサーバー間の同期化する時間間隔を設定します。同期化しない場合は`0`に設定してください。
