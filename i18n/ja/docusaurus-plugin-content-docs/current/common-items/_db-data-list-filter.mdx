## データ一覧のフィルタリング

テーブル内のデータを検索語として入力し、フィルタリングできます。

1.  テーブル右上の![フィルタリングアイコン](/img/ico-filter.svg)ボタンをクリックしてください。 

2.  カラム項目と条件を選択し、検索語を入力してください。 

3.  **_保存_**ボタンを選択してください。

保存した内容は![フィルタリングアイコン](/img/ico-filter.svg)ボタンの横に表示されます。 

:::note

-   保存した条件を選択してフィルタリングを適用または解除できます。 
-   追加のフィルタリング条件を作成して重複適用できます。 
-   フィルタリング条件を削除するには![削除アイコン](/img/ico-remove-white.svg)ボタンをクリックしてください。

:::
