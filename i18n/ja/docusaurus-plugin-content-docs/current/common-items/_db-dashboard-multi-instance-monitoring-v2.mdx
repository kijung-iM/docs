## マルチインスタンスモニタリング

**_インスタンスモニタリング_**メニューは、1つのインスタンスだけをモニタリングすることに対し、**_マルチインスタンスモニタリング_**は、複数のインスタンスの指標を1つの画面で確認して比較することができます。 様々な重要インスタンス指標をグループ化した値を通じてインスタンスの現況を把握することができます。 グラフチャートにより、インスタンスの指標値の推移とどのインスタンスの指標値が高いかを簡単に確認できます。 最も長く持続するアクティブセッション情報を提供します。 詳細については、[次の文書](multi-instance-monitoring)を参照してください。
