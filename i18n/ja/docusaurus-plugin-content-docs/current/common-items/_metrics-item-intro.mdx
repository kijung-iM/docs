WhaTapモニタリングサービスの初期画面でプロジェクトを選択し、**_プロジェクトメニュー_**下位の**_メトリクス_**メニューを選択してください。

メトリクスは、モニタリング対象の性能情報を表現する時系列データです。プロジェクトを作成してアイテムをインストールすると、メトリクスデータが収集されます。収集後、次のような**_メトリクス_**メニューの機能を通じて統合分析機能を利用することができます。

-   **_メトリクスキューブ_**
-   **_メトリクスチャート_**
-   **_メトリクス異常検知_**

:::note 

**メトリクスカテゴリ**

アイテム別のメトリクスは、カテゴリ名で区別できます。

| アイテム                   | カテゴリ接頭語  |
| ---------------------- | -------- |
| Amazon CloudWatch      | aws\_    |
| Amazon ECS             | ecs\_    |
| Naver Cloud Monitoring | ncloud\_ |
| Oracle Cloud Monitor   | oci\_    |
| Azure Monitor          | azure\_  |

:::
