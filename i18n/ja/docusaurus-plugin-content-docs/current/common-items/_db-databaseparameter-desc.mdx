ホーム画面 > プロジェクト選択 > **_分析_** > **_データベースパラメータ_**

データベース設定値照会機能を提供し、過去日付の当時のデータベース設定値も照会できます。比較機能を利用して、データベースのパラメータ値を日付別に比較できます。デフォルトでは、現在の時間のデータを表示できます。 

データベースにパラメータ値が原因で問題が発生した場合、データベースパラメータ比較機能を通じて迅速に原因を把握できます。

<ProdImg img='db-databaseparameter.png' desc='데이터베이스 파라미터'/>

1.  **_基準日付_**と**_比較日付_**を設定してください。

2.  **_インスタンス_**項目から希望のインスタンスを選択してください。 

3.  ![検索アイコン](/img/ico-btn-search.png)ボタンをクリックしてください。 

現在の時間と選択した過去の日付との変更を一目で確認したい場合は、**_変更事項を表示_**ボタンをクリックしてください。
