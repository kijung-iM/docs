**_トポロジー_**メニューの基本的な使い方について説明します。

![アプリケーショントポロジーの基本動作](/img/topology-control.png)

トポロジー画面では、ユーザーの利便性を次のように基本的な機能を提供します。

-   自動更新：規定の30秒間隔の自動更新により、トポロジーの変更を自動的に更新します。 
    :::note
    更新間隔を変更するには、![設定アイコン](/img/ico-setting.svg)を選択し、希望の時時刻を入力します。5秒から5分まで設定することができます。ブラウザを更新すると最新情報にアップデートします。
    :::

-   ズームイン/ズームアウト：マウスホイールを操作してトポロジを拡大または縮小できます。ノード数が多い場合は、必要に応じてトポロジを表現できます。ズームイン/ズームアウトのレベルは、内部設定で保持されます。

-   ノードドラッグ：ノードを左クリックしてドラッグし、目的の場所に固定できます。

-   プロジェクトノードをクリック
    -   **_ダッシュボード_** > **_アプリケーショントポロジ_**メニューでノードをクリックすると、当該プロジェクトのアプリケーショントポロジ(**_インスタンストポロジ_**)に移動します。
    -   ホーム画面 > **_統合アプリケーショングループトポロジ_**メニューでノードをクリックすると、そのプロジェクトのトポロジ情報を取得し、現在のトポロジで表現します。

-   ハイライトフィルタリング：ノードおよび隣接ノードのリンクをフィルタリングしてハイライトできます。
