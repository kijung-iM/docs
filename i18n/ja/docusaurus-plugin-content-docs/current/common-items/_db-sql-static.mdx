## SQL統計

ホーム画面 > プロジェクト選択 > **_統計／レポート_** > **_SQL統計_**

データベースごとの統計データを提供します。SQL別の時系列チャートで発生時間とパフォーマンスを確認することができます。 

**_SQL統計_**メニューの詳細については、次の動画を参照してください。

{@youtube: lt_R3SeTbOc}

:::note
5秒ごとに収集したセッションの情報と、当時実行中のSQL Textに基づいて提供される統計ページです。このページは、収集方法によって誤差が生じる可能性があります。
:::
