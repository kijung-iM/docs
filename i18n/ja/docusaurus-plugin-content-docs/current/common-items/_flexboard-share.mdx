Flexボードを他のアカウントのユーザーと共有または再利用できます。

## メンバーとFlexボードを共有する

ユーザーが作成したFlexボードを他ユーザーと共有して同じダッシュボードを見ることができます。これにより、タスクを共有できます。共有したダッシュボードを修正した場合、共有されたすべてのユーザーは同じダッシュボードを確認できます。この機能はホーム画面 > **_Flexボード_**メニューでのみ利用できます。

1.  **_Flexボード_**メニューの**_ダッシュボード一覧_**で共有しようとするダッシュボードの![共有アイコン](/img/ico-flexboard-share.svg)ボタンを選択してください。

  ![Flexボード共有](/img/flexboard-share-01.png)

1.  **_ダッシュボード共有_**ウインドウが表示されたら**_+アカウント_**ボタンをクリックしてください。

  ![Flexボード共有](/img/flexboard-share-02.png)

1.  ダッシュボードを共有するユーザーアカウントを選択してください。

  ![Flexボード共有](/img/flexboard-share-06.png)

```
*共有対象を追加するには、***+ アカウント***ボタンをクリックしてください。ユーザーアカウントを追加選択できます。
*ダッシュボードを共有できるユーザーアカウントは、選択したプロジェクトに所属するユーザーアカウントです。
*ダッシュボードのすべてのプロジェクトに対して**修正権限**がない場合、`読み取り専用`タグが表示されます。
```

1.  **_ダッシュボードを共有する_**ボタンを選択してください。**_読み取り専用_**チェックボックスを選択すると、共有されたユーザーはダッシュボードを修正できません。

  ![Flexボード共有](/img/flexboard-share-03.png)

ダッシュボードを共有した項目は、**_ダッシュボード一覧_**で`共有`タグが表示されていることを確認できます。共有タグにマウスをオーバーすると、ダッシュボードを共有したメンバーに関する情報を確認できます。

| ダッシュボードを共有された場合                           | ダッシュボードを共有した場合（ダッシュボードの所有者）               |
| ----------------------------------------- | ----------------------------------------- |
| ![Flexボード共有](/img/flexboard-share-04.png) | ![Flexボード共有](/img/flexboard-share-05.png) |

:::note 

**ダッシュボードの共有条件**

-   ダッシュボード共有機能はホーム画面 > **_Flexボード_**メニューでのみ利用できます。

-   ダッシュボードを共有しようとするメンバーは、ダッシュボードのすべてのプロジェクトに対する**修正権限**を持っている必要があります。 

-   すべてのプロジェクトに対する**修正権限**がない状態でダッシュボードを共有すると、**_読み取り専用_**モードで共有します。権限の詳細については、[次の文書](../project/project-structure)を参照してください。

-   共有されたダッシュボードを他ユーザーに再び共有する場合、そのダッシュボードの属性は**_読み取り専用_**モードでない必要があります。

-   共有条件に該当する場合にのみ、ダッシュボード共有ボタンが表示されます。

    ![Flexボード共有](/img/flexboard-sharable-list.png)

:::

:::note

-   ダッシュボード所有者がダッシュボードを削除する場合は、ダッシュボードを共有したすべてのユーザーアカウントからもそのダッシュボードを削除します。
-   ダッシュボードを共有されたユーザーがダッシュボードを削除すると、そのユーザーの**_ダッシュボード一覧_**から削除されます。元のダッシュボードは影響を受けません。

:::
