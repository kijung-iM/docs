## チャートおよびアクティブセッション領域の活用

画面上の![レイアウトアイコン](/img/ico-layout-set.svg)ボタンをクレックしてください。***レイアウト設定***ウィンドウで***1x3***に設定し、***保存***ボタンを選択してください。***時間***は***最近24時間***として設定してください。次のように、1日間のデータベースのコア指標を詳細に確認できます。

<ProdImg img='db-count-trend-1x3.png' desc='카운트 추이' />

-   ![数字1](/img/number-01.png)**_時間_**で希望する時間間隔を設定してください。

    -   緑色のアイコンをクリックすると、検索時間を選択できます。
    -   検索時間を選択した状態で ![左方向アイコン](/img/left-arrow.svg)または ![右方向アイコン](/img/right-arrow.svg)ボタンを選択すると、設定した時間間隔で検索時間を変更できます。
    -   日付と時間部分を選択することで、目的の時間を細かく設定できます。 設定を完了した後 **_確認_** ボタンを選択してください。

-   ![数字2](/img/number-02.png)**_インスタンス_**：プロジェクトと接続したインスタンスを選択することができます。

-   指標チャートの特定時点をクリックすると(![数字3](/img/number-03.png))、該当地点を赤い線で表示し、当時収集されたアクティブセッションを確認することができます。 

-   チャートの指標を変更するには右上の![設定アイコン](/img/ico-setting.svg)ボタンをクリックしてください。 

-   アクティブセッションデータは5秒単位で収集しています。画面の下の![数字4](/img/number-04.png)5秒単位セルを選択してデータを照会することができます。

-   1分単位で時間を移動するには![数字5](/img/number-05.png)![以前の時間アイコン](/img/ico-left-time.svg)または![以降の時間アイコン](/img/ico-right-time.svg)ボタンを選択してください。

-   ![数字5](/img/number-05.png)![コラムアイコン](/img/ico-column.svg)：テーブルのカラムヘッダー項目を表示または非表示にすることができます。

-   ![数字5](/img/number-05.png)![フィルターアイコン](/img/ico-filter.svg)：テーブルのカラムヘッダー項目を基準に一覧をフィルタリングすることができます。ボタンをクリックすると、各ヘッダーカラムで**_含む_**、**_含まない_**、***一致***、**_一致しない_**条件を設定できます。

-   ![数字5](/img/number-05.png)![ダウンロードアイコン](/img/ico-download.svg)：テーブルの内容をCSVファイルとしてダウンロードできます。
