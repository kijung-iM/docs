## エージェントのアップデート

ホーム画面 > プロジェクト選択 > **_管理_** > **_エージェントアップデート_**

![エージェントアップデート](/img/db-update-agent.png)

エージェントを個別にするには、各エージェント項目の右側にある**_最新バージョン_**ボタンを選択してください。すべてのエージェントを一度に更新するには、**_最新バージョンですべてのエージェントにダウンロード_**ボタンを選択してください。

エージェントが位置するサーバーに自動的にダウンロードします。ダウンロードが完了したら、エージェントを再起動してください。最新バージョンのエージェントとして自動適用します。
