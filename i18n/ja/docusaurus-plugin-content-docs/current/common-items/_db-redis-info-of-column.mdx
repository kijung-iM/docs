## カラム情報の案内{#info-of-column}

**_アクティブセッション_**テーブルのカラム項目の説明です。詳細については、[次の文書](https://redis.io/commands/client-list/)を参照してください。

| 項目        | 説明                                                                    |
| --------- | --------------------------------------------------------------------- |
| id        | 固有の64bitクライアントID                                                      |
| addr      | クライアントのIPアドレスとポート番号                                                   |
| age       | 総接続時間                                                                 |
| cmd       | 最後に実行されたコマンド                                                          |
| db        | 現在のデータベースID                                                           |
| events    | 'r`：クライアントソケットを読み取める（eventloop）<br/>`w\`：クライアントソケットが書き込める(event loop) |
| fd        | クライアントソケットファイルのディスクリプタ（filedescriptor）番号                              |
| flags     | クライアントの種類、Nは正常なクライアントを意味します。（normal／master／replica／pubsub）            |
| idle      | 接続がアイドル状態の時間（秒）です。                                                    |
| multi     | MULTI/EXECコンテキストのコマンド数                                                |
| obl       | 出力バッファのサイズ                                                            |
| oll       | 出力リストのサイズ（バッファがいっぱいの場合、応答がこのリストに待機される）                                |
| omem      | クライアントが出力バッファに使用するメモリ量                                                |
| psub      | subscriptionsパターンのマッチング数                                              |
| qbuf      | クエリバッファのサイズ（0はクエリ依存ではない）                                              |
| qbuf-free | クエリバッファ空き容量（0はバッファがいっぱいで空き容量がないことを意味）                                 |
| sub       | subscriptionsチャンネル数                                                   |
| user      | クライアントの認証済みユーザー名                                                      |
