ホーム面 > プロジェクト選択 > **_分析_** > **_キューブ_**

WhaTapモニタリングサービスの初期画面でプロジェクトを選択し、**_プロジェクトメニュー_**下位の**_分析_** > **_キューブ_**を選択してください。ここで、国別アクセス、ヒートマップ、トランザクション、TOPトランザクション、TPS、応答時間、リソース使用量に関する情報を提供します。

## キューブとは？

WhaTapは5分単位で作成されたパフォーマンス統計を**キューブ**と呼びます。キューブ分析は、キューブに保存された5分単位のパフォーマンスデータを利用する分析機能です。

![キューブapm](/img/cube-apm.png)

## ![](/img/number-01.png)キューブのセレクトパネル

左側![number 1](/img/number-01.png)領域のキューブセレクトパネルで、特定の時間に基づいてキューブを選択します。 キューブは5分単位で保存されるため、特定の時間を選択する必要があります。 

-   **パネル関心メトリクス**

    -   **_TPS_**

    -   **_応答時間_**

    -   **_Tx Error件数_**

    -   **_アクティブTX件数_**

    -   **_同時接続ユーザー_**

-   **キューブ表示色**

    キューブは、選択されたインジケーターの数値に応じて異なる色で表示されます。

    -   <span class='highfg'>藍色</span>：<span class='highfg'>高い数値</span>

    -   <span class='lowfg'> 水色</span>：<span class='lowfg'>low</span>

## ![](/img/number-02.png)アプリケーション選択

上段![number 2](/img/number-02.png) 領域のアプリケーションセレクトオプションを使用して、特定のアプリケーションのデータを選択できます。 右側の![down-arrow](/img/down-arrow.svg)ボタンをクリックすると、次のように選択可能なアプリケーション全体を確認できます。 

![キューブ選択アプリケーション](/img/cube-select-app.png)

-   **_全体選択_**：すべてのアプリケーションを表示します。 デフォルト値は**_全体選択_**と**_アクティブアプリケーション_**が選択されています。 

-   **_アクティブアプリケーション_**：アプリケーション間でアクティブなアプリケーションを指定および表示します。 

-   **_選択したものだけを表示_**：アプリケーションからユーザーが指定したアプリケーションのみを表示します。 

-   **_検索_**：アプリケーションが多数の場合は、入力ウィンドウからそのアプリケーションを検索できます。

-   <font color='5B7DEC'>数字</font>/<font color='979797'>数字</font>: <font color='5B7DEC'>지指定されたアプリケーション数</font>/<font color='979797'>全体のアプリケーション数</font>を表示します。

## ![](/img/number-03.png)キューブデータ

![number3](/img/number-03.png)領域で該当するキューブデータを**_ヒートマップ_**、**_国_**、**_TPS_**、**_応答時間_**、**_CPU_**、**_同時接続ユーザー_**、**_上位TX_**、**_ヒープメモリ_**などのチャートから確認することができます。

-   **_ヒートマップ_**

    キューブ時間の経過に伴うトランザクション分布を示します。トレースの詳細分析が可能です。 

    ![right-arrow](/img/right-arrow.svg)ボタンを選択すると、キューブ時間中に**_トップスタック_**と**_ユニークスタック_**を参照できます。

    :::note

    スタック分析機能を使用できるアプリケーションは**Java**と**Python**です。 

    :::

-   **_国/地域_**

    クライアントIPに基づいて国をマッピングし、どの国からトランザクションが来ているかを示します。マップの色は、トランザクションの量を相対的に表示します。

-   **_TPS_**

    Tansaction Per Second(**_TPS_**)は、毎秒処理されるトランザクション数を意味します。

-   **_インスタンスの応答時間_**

    トランザクションの平均応答時間です。

-   **_CPU_**

    プロジェクトのアプリケーション(エージェント)をIPでソートし、使用率の高いサーバーのCPUをチャートで表示します。

-   **_同時接続ユーザー_**

    リアルタイムブラウザのユーザー数を表示します。 

-   **_トップTX_**

    呼び出し件数の多い5つのトランザクションが表示されます。 トランザクション統計は、より詳細なデータを見つけることができます。

-   **_ヒープメモリ_**

    アプリケーション(インスタンス) のヒープメモリ使用量が高い順にインスタンスを表示します。

:::note

-   チャート右上の![個別アイコン](/img/ic-line.svg)アイコンを選択すると、統合された折れ線チャートを提供します。
-   チャート右上の![個別アイコン](/img/ic-multiline-chart.svg)アイコンを選択すると、個々の折れ線チャートを提供します。
-   チャート右上の ![バーチャートアイコン](/img/ic-dashboard-bar-chart.svg)アイコンを選択すると、バーチャートが表示します。 

:::
