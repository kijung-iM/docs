WhaTapモニタリングサービスは、円滑なサービス提供と安定的な運営、異常の監視および把握のための情報を提供します。対象APIに接続して、すべての指標とクラウドイベントをリアルタイムで収集します。収集したデータをダッシュボードを通じてリアルタイムで確認し、トラブル原因に対する直接分析ができます。

ホーム画面でプロジェクトを選択してください。**_プロジェクトメニュー_**下位に次のように構成されています。

-   **項目別ダッシュボードメニュー**

    視覚化されたチャートでプロジェクト全体の現況を一目で把握できます。また、モニタリング中の全体資源規模を確認し、対象資源をフィルタリングすることができます。現在のサービスとシステムの正確な状態を直感的に把握できます。

-   **Flexボード**

    基本的に提供されるアイテム別ダッシュボードの他に、ユーザーが希望するさまざまなダッシュボードを構成できます。ウィジェットを追加し、サブチャートを活用できます。

-   **メトリクス**

    メトリクスデータを活用した統合分析機能を提供します。各カテゴリーのメトリクスフィールドデータ推移分析機能を提供する**_メトリクスキューブ_**、単一メトリクス照会機能を提供する**_メトリクスチャート_**などを提供します。

    -   **_メトリクスキューブ_**
    -   **_メトリクスチャート_**
    -   **_メトリクス異常検知_**

-   **警告通知**

    メトリクベースのイベント設定と照会機能を提供します。 

    -   **_イベント設定_**
    -   **_イベント受信設定_**
    -   **_イベント記録_**
