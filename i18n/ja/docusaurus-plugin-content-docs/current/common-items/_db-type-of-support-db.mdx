## サポート可能なデータベースの種類

WhaTapデータベースモニタリングサービス(DBX)では、パフォーマンス分析のために現在進行中のStat指標、セッションデータを収集し、収集されたデータを保存します。特定期間の全体的な推移を分析または比較することができます。また、特定の時点で進行中だったセッションを把握し、遅いQueryを分析することもできます。収集されたSQLを再加工してSQLの統計データ分析が可能です。

DBXでは、次のデータベースをサポートします。**Java8**以降**JDK**を設置する必要があります。
