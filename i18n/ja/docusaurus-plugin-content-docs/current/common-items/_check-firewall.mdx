## ファイアウォール設定の確認

WhaTapのデータ収集サーバーに対するTCPアウトバウンドポリシーを設定しないと、モニタリング情報を送信できません。 **ファイアウォール遮断**を**解除**してください。

`telnet`コマンドを使用して、"Connected to `IP`"を確認してください。

```sh
$ telnet 52.193.60.176 6600
Trying 52.193.60.176...
Connected to 52.193.60.176.
Escape character is '^]'.
```

:::note

収集サーバーの情報は[WhaTapモニタリングサービス](https://service.whatap.io/)から当該プロジェクトを選択してから**_管理_** > **_エージェントインストール_** メニューから確認できます。

:::
