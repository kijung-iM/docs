## 使用設定の案内

**_ファイルログ通知_**メニューを利用するには、XOSエージェントに次のオプションを追加してください。キーワードを設定または除外してログ発生履歴を照会することができます。

```ini title='xos.conf'
file={fullpath} # ログファイルのパスおよびファイル名
filekey={通知発生キーワード}
fileexcl={通知除外キーワード}
```

:::note

-   ログファイルの読み取り権限が必要です。
-   複数のファイルを設定するには、`file1`, `filekey1`, `fileexcl1` / `file2`, `filekey2`, `fileexcl2`のような形式でオプションを追加してください。
-   1つのファイルから複数のキーワードを設定するには、`filekey=test1|test2`と同じ形式で縦棒（|）を区切り文字として利用してください。
-   `filekey`と`fileexcl`を一緒に適用する場合、AND条件に該当します。 たとえば、`filekey=error`, `fileexcl=row`でオプションを追加した場合、ログファイルにerror、rowを含む文字列は**_ファイルログ通知_**で表示されません。

:::
