### エージェント名（ONAME）重複使用不可

WhaTapDBXモニタリングはエージェントの識別のためにエージェントのIPアドレスとポート情報を利用します。ユーザー環境のモニタリングするデータベースサーバーが同一IP、同一ポートを使用する場合、WhaTapサーバーで該当サーバーインスタンスを区分できません。

-   サーバーインスタンスが動的に拡張される環境でない場合は、[次の文書](agent-dbx-settings#naming)を参照してデータベース識別のための名称を直接設定して迂回することができます。

-   モニタリングするデータベースのinternal addressまたはネットワーク仮想化により重複したIPを利用する場合は、[次の文書](agent-dbx-settings#naming)を参照して、別途のデータベース名パターンを活用してください。
