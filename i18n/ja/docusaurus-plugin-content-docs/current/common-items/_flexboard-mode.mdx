Flexボードの画面モードによって提供する機能について説明します。

## 画面モード

**_Flexボード_**は、次の画面モードを提供します。画面モードによって設定できる機能が異なります。

-   **_修正モード_**：ウィジェットテンプレートの一覧からFlexボードのレイアウトにウィジェットを配置したり、ウィジェットを修正、削除、位置を調整することができます。

-   **_管理者_**：ウィジェットの詳細なプロパティをJSON形式で管理できます。レイアウトに配置されたウィジェットの右上の![コードアイコン](/img/ico-custom-code.svg)ボタンをクリックします。

-   **_ビューモード_**：ウィジェット配置、サイズ調節、詳細オプションなどの設定を完了した後、ダッシュボードなどの各ウィジェットのデータをリアルタイムでモニタリングできます。

**_Flexボード_**には、次の共通の機能を使用できます。

-   データフィルタリング
-   リアルタイムデータの検索範囲を設定
-   印刷モード
-   Flexボード管理
-   データマージオプション

## データフィルタリング

Flexボードは数百種類もの異なる種類のデータを処理します。データフィルタリング機能を使用して、モニタリング対象の概要を含むFlexボードを設定します。

![Flexボード-フィルター](/img/flexboard-filter.png)

1.  画面の上の![指示線1](/img/number-01.png)![フィルターアイコン](/img/ico-filter.svg)**_フィルター_**をクリックしてください。 

2.  画面左に![指示線2](/img/number-02.png)**_フィルター_**ウィンドウが表示されます。

    -   **_エージェント_**オプションを選択した場合は、エージェントの種類またはアプリケーション一覧に基づいてフィルタリング対象を選択できます。フィルタリング対象をすべて選択するには、**_全体選択_**を選択します。必要に応じて、**_検索_**機能を使用して選択できます。

    -   **_メトリクス_**オプションを選択した場合は、**_カテゴリ_**オプションでプロジェクトとエージェントタイプに応じてフィルタリング対象を選択できます。**_カテゴリ_**オプションは、**_+ And_**を選択することで追加できます。**_含む_**、**_除外_**オプションを選択すると、文字列でフィルタリング対象を自動的に選択します。

3.  フィルタリング対象の選択が完了すると、**_適用_**ボタンを選択します。 

ユーザーが設定したフィルタリング設定を保存して使用するには、**_フィルター保存_**ボタンを選択します。フィルター名を入力し、**_保存_**を選択します。![指示線3](/img/number-03.png)**_保存されたフィルター_**は**_フィルター_**ウィンドウの下部に表示します。

フィルタリング条件は、エージェントタイプとメトリプスタイプに分けられます。

| タイプ       | エージェント                                                                         | メトリックス                        |
| --------- | ------------------------------------------------------------------------------ | ----------------------------- |
| 選択オプション   | WhaTapプロジェクトのモニタリング対象<br/>(アプリケーション、サーバ、データベース、<br/>コンテナ及びKubernetes PODとNODE) | エージェントを含むすべてのメトリックタグ          |
| 適用範囲      | すべてのウィジェット                                                                     | メトリックデータを使用するすべてのウィジェット       |
| フィルタリング条件 | チェックボックス                                                                       | **_一致_**、**_含む_**、**_除外_** 方式 |

:::note

-   エージェントタイプは、モニタリングのデフォルト対象のみを選択オプションとして提供します。

-   メトリックタイプは、プロジェクトで収集されるすべてのメトリック データのタグを選択オプションとして提供します。 メトリックタイプは、クラウドモニタリングなどの非エージェント形式でデータを収集し、フィルタリングを適用する場合に便利です。

-   メトリクスの詳細については、[次の文書](metrics-intro)を参照してください。

:::

## リアルタイムデータの検索範囲を設定

**_Flexボード_**メニュー画面の右上にあるUI要素を使用して、ウィジェットにデータを出力する時間を設定できます。

![リアルタイムデータ照会](/img/flexboard-time-set.png)

![一時停止アイコン](/img/ico-pause.svg)ボタンをクリックすると、リアルタイムデータ参照を一時停止し、次のように参照時間を設定できます。

![リアルタイムデータ参照](/img/flexboard-time-set-pause.png)

-   右端の緑色の背景にある時間ボタンを選択します。さまざまな時間設定から選択できるオプションが表示されます。目的の時間を設定します。

-   ![左アイコン](/img/left-arrow.svg)または![右アイコン](/img/right-arrow.svg)ボタンをクリックすると、設定された時間範囲を調整します。

-   日付と時間テキスト領域をクリックして、日付と時間を選択するオプションを表示します。

-   検索時間の設定をすべて完了すると![再生アイコン](/img/ico-play-time.svg)ボタンをクリックします。リアルタイムのデータ検索を開始します。

## 印刷

Flexボード画面をPDFとして保存および印刷することができます。Flexボード画面右上の![印刷アイコン](/img/ico-print.svg)ボタンをクリックします。画面を印刷モードに切り替えます。次のオプションを設定し、**_PDFダウンロード_**を選択します。

-   **_フィルターオプション表示_**：画面にフィルターオプションをオンまたはオフに切り替えることができます。
-   **_画面を横方向／縦方向に回転_**：画面を横向きまたは縦向きに回転することができます。

印刷モードを終了するには右上の![終了アイコン](/img/ico-close.svg)**_印刷モード終了_**を選択します。

## Flexボード管理

Flexボードのレイアウト変更、ウィジェットの位置変更、サイズ調整、名前変更などの管理機能を提供します。Flexボード画面右上の![設定アイコン](/img/ico-setting.svg)ボタンを選択します。**_Flexボード管理_**ウィンドウが表示されます。 

-   **_名前変更_**：Flextボードの名前を変更することができます。変更する名前を入力します。

-   **_レイアウト_**：ウィジェットの配置を変更、位置やサイズを調節できます。配置したウィジェットを削除するには![削除アイコン](/img/ico-trash.svg)ボタンをクリックします。**_オーバーラップ_**切り替えボタンを選択してオプションを有効にすると、ウィジェットを重ねて配置できます。

-   **_他のユーザーの修正を制限_**：他のユーザーがFlexボードを修正できないように設定します。修正権限のあるメンバーのみ修正または削除、ウィジェットを編集できます。ボタンをクリックすると、**_他のユーザーの修正を制限：on_**に変更します。

-   **_ウィジェットJSON_**：Flexボードに含まれるウィジェットをJSON形式で編集および管理することができます。

-   **_オプションJSON_**：Flexボードの設定をJSON形式で編集および管理することができます。プロジェクト情報、Flexボードフィルター、時間選択などの設定を編集できます。

すべての設定を完了した後、画面の右下の**_保存_**ボタンをクリックします。

:::note

-   **_ウィジェットJSON_**、**_オプションJSON_**機能はダッシュボードの管理者権限を持つユーザーのみが使用できます。

-   **_Flexボード管理_**機能は**_ダッシュボード_** > **_Flexボード_**メニューから**_ボード_**一覧の![編集アイコン](/img/ico-edit.svg)ボタンをクリックしてアクセスすることができます。

:::

## ダッシュボードをお気に入りとして登録

![Flexボード-お気に入り](/img/flexboard-favorite.png)

ユーザーが作成したFlexボードは、**_ダッシュボード_**メニューでお気に入りとして登録できます。**_ダッシュボード_** > **_Flexボード_**メニューから**_ボード_**一覧の![お気に入りアイコン](/img/ico-favorite.svg)ボタンを選択します。お気に入りに設定された項目は![お気に入り設定アイコン](/img/ico-favorite-check.svg)に変更されます。
