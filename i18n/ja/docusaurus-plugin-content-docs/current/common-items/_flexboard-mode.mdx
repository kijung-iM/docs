Flexボードの画面モードに応じて提供する機能を説明します。

## 画面モード

**_Flexボード_**は、次のような画面モードを提供します。画面モードによって設定できる機能が異なります。

-   **_修正モード_**：ウィジェットテンプレートの一覧からFlexボードのレイアウトにウィジェットを配置したり、ウィジェットを修正、削除、位置調整することができます。

-   **_管理者_**：ウィジェットの詳細プロパティをJSON形式で管理できます。レイアウトに配置したウィジェットの右上の![コードアイコン](/img/ico-custom-code.svg)ボタンを選択してください。

-   **_ビューモード_**：ウィジェット配置、サイズ調節、詳細オプションなどの設定を完了した後、ダッシュボードのように各ウィジェットのデータをリアルタイムでモニタリングすることができます。

**_Flexボード_**では、次の共通機能を使用することができます。

-   データフィルタリング
-   リアルタイムデータの照会範囲設定
-   印刷モード
-   Flexボード管理
-   データマージオプション

## データフィルタリング

Flexボードは数百種類のデータを扱います。データフィルタリング機能を利用してモニタリングしようとする対象を要約し、Flexボードを構成してください。

![Flexボード-フィルター](/img/flexboard-filter.png)

1.  画面の上の![指示線1](/img/number-01.png)![フィルターアイコン](/img/ico-filter.svg)**_フィルター_**を選択してください。 

2.  画面左に![指示線2](/img/number-02.png)**_フィルター_**ウィンドウが表示されます。

    -   **_エージェント_**オプションを選択した場合は、エージェントの種類またはアプリケーション一覧に基づいてフィルタリング対象を選択することができます。フィルタリング対象をすべて選択するには、**_全体選択_**を選択してください。必要な場合は**_検索_**機能を利用して選択することができます。

    -   **_メトリクス_**オプションを選択した場合は、**_カテゴリ_**オプションでプロジェクト、エージェントタイプに応じてフィルタリング対象を選択することができます。**_カテゴリ_**オプションは**_+ And_**を選択して追加することができます。**_含む_**、**_除外_**オプションを選択すると、文字列でフィルタリング対象を自動選択します。

3.  フィルタリング対象の選択が完了したら、**_適用_**ボタンをクリックしてください。 

ユーザーが設定したフィルタリング設定を保存して利用するには、**_フィルター保存_**ボタンを選択してください。フィルター名を入力し、**_保存_**を選択してください。![指示線3](/img/number-03.png)**_保存されたフィルター_**は**_フィルター_**ウィンドウの下部に表示されます。

フィルタリング条件は、エージェントタイプとメトリプスタイプに区分します。

| タイプ       | エージェント                                                                         | メトリックス                        |
| --------- | ------------------------------------------------------------------------------ | ----------------------------- |
| 選択オプション   | WhaTapプロジェクトのモニタリング対象<br/>（アプリケーション、サーバ、データベース、<br/>コンテナ及びKubernetes PODとNODE） | エージェントを含むすべてのメトリックタグ          |
| 適用範囲      | すべてのウィジェット                                                                     | メトリックデータを使用するすべてのウィジェット       |
| フィルタリング条件 | チェックボックス                                                                       | **_一致_**、**_含む_**、**_除外_** 方式 |

:::note

-   エージェントタイプは、モニタリングのデフォルト対象のみを選択オプションとして提供します。
-   メトリックタイプは、プロジェクトで収集中のすべてのメトリック データのタグを選択オプションとして提供します。 メトリックタイプは、クラウドモニタリングのようなエージェントではない形でデータを収集した後、フィルタリングを適用する際に便利です。
-   メトリクスの詳細については、[次の文書](metrics-intro)を参照してください。
:::

## リアルタイムデータの照会範囲設定

**_Flexボード_**メニュー画面の右上にあるUI要素を利用してウィジェットにデータを出力する時間を設定することができます。

![リアルタイムデータ照会](/img/flexboard-time-set.png)

![一時停止アイコン](/img/ico-pause.svg)ボタンをクリックすると、リアルタイムデータ照会を一時停止し、次のように照会時間を設定できます。

![リアルタイムデータ照会](/img/flexboard-time-set-pause.png)

-   一番右の緑色の背景内の時間ボタンを選択してください。さまざまな時間設定を選択できるオプションが表示されます。目的の時間を設定してください。

-   ![左アイコン](/img/left-arrow.svg)または![右アイコン](/img/right-arrow.svg)ボタンをクリックすると、設定された時間範囲だけ調節できます。

-   日付と時間テキスト領域をクリックすると、日付と時間を選択するオプションが表示されます。

-   照会時間の設定をすべて完了したら![再生アイコン](/img/ico-play-time.svg)ボタンを選択してください。リアルタイムデータの照会を開始します。

## 印刷

Flexボード画面をPDFとして保存し印刷することができます。Flexボード画面右上の![印刷アイコン](/img/ico-print.svg)ボタンをクリックしてください。画面を印刷モードに切り替えます。次のオプションを設定し、**_PDFダウンロード_**を選択してください。

-   **_フィルターオプション表示_**：画面にフィルターオプションをオンまたはオフにすることができます。
-   **_画面を横／縦方向に回転_**：画面を横または縦に回転することができます。

印刷モードを終了するには右上の![終了アイコン](/img/ico-close.svg)**_印刷モード終了_**を選択してください。

## Flexボード管理

Flexボードのレイアウトを変更したり、ウィジェットの位置変更、サイズ調整、名前変更などの管理機能を提供します。Flexボード画面右上の![設定アイコン](/img/ico-setting.svg)ボタンを選択してください。**_Flexボード管理_**ウィンドウが表示されます。 

-   **_名前変更_**：Flextボードの名前を変更することができます。変更したい名前を入力してください。

-   **_レイアウト_**：ウィジェットの配置を変更したり、位置やサイズを調節することができます。配置したウィジェットを削除するには![削除アイコン](/img/ico-trash.svg)ボタンをクリックしてください。**_オーバーラップ_**トグルボタンを選択してオプションをアクティブにすると、ウィジェットが重なり合うように配置できます。

-   **_他ユーザーの修正を制限_**：他のユーザーが該当Flexボードを修正できないように設定します。修正権限のあるメンバーのみが修正または削除、ウィジェットを編集できます。ボタンをクリックすると、**_他ユーザーの修正を制限：on_**に変更します。

-   **_ウィジェットJSON_**：Flexボードに含まれているウィジェットをJSON形式で編集して管理することができます。

-   **_オプションJSON_**：Flexボードの設定をJSON形式で編集して管理することができます。プロジェクト情報、Flexボードフィルター、時間選択などの設定値を編集できます。

すべての設定を完了したら、画面右下の**_保存_**ボタンをクリックしてください。

:::note

-   **_ウィジェットJSON_**、**_オプションJSON_**機能はダッシュボード管理者権限のユーザーにのみ提供します。
-   **_Flexボード管理_**機能は**_ダッシュボード_** > **_Flexボード_**メニューから**_ボード_**一覧の![編集アイコン](/img/ico-edit.svg)ボタンをクリックしてアクセスすることもできます。
:::

## ダッシュボードをお気に入りとして登録

![Flexボード-お気に入り](/img/flexboard-favorite.png)

ユーザーが作成したFlexボードは、**_ダッシュボード_**メニューの下位にお気に入りのメニューとして登録できます。**_ダッシュボード_** > **_Flexボード_**メニューから**_ボード_**一覧の![お気に入りアイコン](/img/ico-favorite.svg)ボタンを選択してください。お気に入りとして設定された項目は![お気に入り設定アイコン](/img/ico-favorite-check.svg)に変更されます。
