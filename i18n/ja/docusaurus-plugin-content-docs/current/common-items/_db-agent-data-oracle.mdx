-   **stat** <span class='type'>String</span>

    Oracleモニタリング情報から収集および処理されるstat指標をコンマで区切って設定します。次のデフォルト値を使用します。

    ```ini title='whatap.conf'
    stat=logons current,user commits,session logical reads,CPU used by this session,DB time,enqueue waits,physical reads,physical writes,execute count,db block changes,opened cursors current,parse count (total),parse count (hard),sorts (memory)
    ```

-   **event** <span class='type'>String</span>

    Oracleモニタリング情報から収集および処理されるevent指標をコンマ(,)で区切って設定します。次のデフォルト値を使用します。

    ```ini title='whatap.conf'
    event=db file sequential read,db file scattered read,enq: TX - row lock contention
    ```

-   **exclude_background** <span class='type'>String</span>

    デフォルト値`3`

    Oracleモニタリング情報のうち、セッション数およびセッション一覧を収集する場合、backgroundセッションを除外するかどうかを設定します。

    -   `0`：除外しない
    -   `1`：セッション一覧の収集時、backgroundセッションを除外
    -   `2`：セッション個数を処理時、backgroundセッションを除外
    -   `3`：セッション一覧と個数を処理時、backgroundセッションを除外
