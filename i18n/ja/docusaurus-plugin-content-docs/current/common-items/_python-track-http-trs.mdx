-   **profile_http_header_enabled** <span class='type'>Boolean</span>

    デフォルト値`false`

    トレースにHTTPヘッダーの情報を出力します。ヘッダー収集の有無にかかわらず、ヘッダーをトレースに露出するかどうかに対する設定です。 

-   **profile_http_parameter_enabled** <span class='type'>Boolean</span>

    デフォルト値`false`

    HTTPパラメータのトレーシングを活性化します。ただ、パラメータは別途のセキュリティ キーを入力して照会できます。セキュリティキーは、エージェントインストールパスの_paramkey.txt_ファイルに6桁で指定します。_paramkey.txt_が存在しない場合は自動生成され、randomキーが設定されます。

-   **trace_normalize_enabled** <span class='type'>Boolean</span>

    デフォルト値 `true`

    トランザクションURLをパーシングして正規化する機能を活性化します。

    :::note

    値を`false`に変更すると、パスパラメータのパーシングが無効になります。 この場合、統計データの意味が弱まるため、デバッグ用途にのみしばらく使用することをお勧めします。

    :::

-   **trace_auto_normalize_enabled(\*)** <span class='type'>Boolean</span>

    デフォルト値 `true`

    トランザクションURLを正規化するときにパターン値をアノテーションから抽出して自動的にパーシングする機能を活性化します。

-   **trace_normalize_urls** <span class='type'>String</span>

    トランザクションURLをパーシングして正規化します。呼び出しURLパターンをパーシングしてパスパラメータを削除します。  

    :::note

    例)**/a/{v}/b**と宣言すれば、**/a/123/b** → **/a/{v}/b**に置き換えます。複数登録するときはコンマ(,)を使用します。**/a/\*/b**で使用して、特定の形式のパスパラメータを削除して収集できます。

    :::

-   **web_static_content_extensions(\*)** <span class='type'>String</span>

    デフォルト値`js, htm, html, gif, png, jpg, css, swf, ico`

    スタティックコンテンツを判断する拡張子を指定します。ここに設定された拡張子を持つトランザクションは、トレース追跡とカウントが除外されます。

-   **trace_ignore_url_prefix** <span class='type'>String</span>

    トランザクション収集を除外するURL prefixを指定します。指定した値と前の部分が一致するURLは、トランザクションパフォーマンスを収集しません。
