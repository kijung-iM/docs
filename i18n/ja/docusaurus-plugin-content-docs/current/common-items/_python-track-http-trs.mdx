-   **profile_http_header_enabled**<span class='type'>Boolean</span><br/>
      デフォルト値 `false`<br/>
      トレースにHTTPヘッダー情報を出力します。ヘッダー収集の有無にかかわらず、ヘッダーをトレースに表示するかに関する設定です。 

-   **profile_http_parameter_enabled**<span class='type'>Boolean</span><br/>
      デフォルト値 `false`<br/>
      httpパラメータのトレーシングを活性化します。ただ、パラメータは別途のセキュリティ キーを入力して照会できます。セキュリティキーは、エージェントインストールパスの_paramkey.txt_ファイルに6桁で指定します。_paramkey.txt_が存在しない場合は自動生成され、randomキーが設定されます。

-   **trace_normalize_enabled**<span class='type'>Boolean</span><br/>
        デフォルト値 `true`<br/>
        トランザクションURLをパーシングして正規化する機能を活性化します。
    :::note
    値を`false`に変更すると、パスパラメータの構文解析が非活性化されます。この場合、統計データの意味が弱まるため、デバッグ用に限ってしばらく使用することをお勧めします。
    :::

-   **trace_auto_normalize_enabled(\*)**<span class='type'>Boolean</span><br/>
      デフォルト値 `true`<br/>
      トランザクションURLを正規化するとき、パターン値をアノテーションから抽出して自動的に構文解析する機能を活性化します。

-   **trace_normalize_urls**<span class='type'>String</span><br/>
        トランザクションURLを構文解析して正規化します。呼び出しURLパターンを構文解析してパスパラメータを削除します。<br/>
    :::note
    例）**/a/{v}/b**と宣言すると**/a/123/b** → **/a/{v}/b**に置き換えます。複数登録するときはコンマ（,）を使います。**/a/\*/b**のように使用して、特定形式のパスパラメータを削除して収集できます。
    :::

-   **web_static_content_extensions(\*)**<span class='type'>String</span><br/>
      デフォルト値 `js, htm, html, gif, png, jpg, css, swf, ico`<br/>
      静的コンテンツを判断する拡張子を指定します。ここに設定された拡張子を持つトランザクションは、トレース追跡とカウントが除外されます。

-   **trace_ignore_url_prefix**<span class='type'>String</span><br/>
      トランザクション収集の例外url prefixを指定します。指定した値と前の部分が一致するurlは、トランザクション性能を収集しません。2つ以上の値を指定するには、コンマ（,）を区切り文字として利用してください。
