WhaTapモニタリングサービスは、円滑なサービス提供と安定的な運営、異常の監視および把握のための情報を提供します。対象APIに接続して、すべての指標とクラウドイベントをリアルタイムで収集します。収集したデータをダッシュボードを通じてリアルタイムで確認し、トラブル原因に対する直接分析ができます。

ホーム画面でプロジェクトを選択してください。**_プロジェクトメニュー_**下位に次のように構成されています。 

-   **Flexボード**<br/>
    ユーザーが定義型統合ダッシュボードを提供します。視覚化されたチャートでプロジェクト全体の現況を一目で把握できます。また、モニタリング中の全体リソースの規模を確認し、対象リソースをフィルタリングすることができます。

-   **メトリックス**<br/>
    メトリックスデータを活用した統合分析機能を提供します。各カテゴリーのメトリックスフィールドデータ推移分析機能を提供する**_メトリックスキューブ_**、単一メトリックス照会機能を提供する**_メトリックスチャート_**などを提供します。
    -   **_メトリクスキューブ_**
    -   **_メトリクスチャート_**
    -   **_メトリクス異常検出_**

-   **警告通知**<br/>
    メトリックベースのイベント設定と照会機能を提供します。 
    -   **_イベント設定_**
    -   **_イベント受信設定_**
    -   **_イベント記録_**
