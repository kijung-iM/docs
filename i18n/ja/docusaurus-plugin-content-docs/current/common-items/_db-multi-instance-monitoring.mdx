![マルチインスタンスモニタリング](/img/db-multi-instance-monitoring.png)

**_マルチインスタンスモニタリング_**メニューでは、多数のデータベースインスタンスのコア指標を統合して確認する機能を提供します。各インスタンスの間でロードバランシングの現状を比較できる機能も提供します。

詳細については、次の動画を通じて参照してください。

{@youtube: JWoW9v32-OU}

## 状態およびロードバランス領域の活用

特定の指標を比較してモニタリングするには、**_状態_**と**_ロードバランス_**各チャートの右上の![設定アイコン](/img/ico-setting.svg)ボタンを選択してください。比較指標を変更できます。

## アクティブセッション領域の活用

マルチインスタンスモニタリング中に特定のインスタンスで実行中の**_アクティブセッション_**、**_ロックツリー_**、**_プロセス情報_**を確認するには、インスタンス名ボタンをクリックしてください。
