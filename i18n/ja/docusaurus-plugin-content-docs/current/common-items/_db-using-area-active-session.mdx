## アクティブセッション領域を活用する

화면 아래에 위치한 **_액티브 세션_** 영역을 통해 실시간 수행 중인 세션 정보를 확인할 수 있습니다. 다음은 테이블 오른쪽 위에 버튼에 대한 기능 안내입니다.

<Xclude product='postgresql'>

-   ![停止アイコン](/img/ico-pause.svg)：リアルタイムで収集するデータの変化を止めることができます。再起動するには![スタートアイコン](/img/ico-play-grey.svg)ボタンをクリックしてください。

</Xclude>

-   ![更新アイコン](/img/ico-refresh.svg)：テーブルのリストを更新できます。

-   ![カラムアイコン](/img/ico-column.svg)：テーブル内のカラムヘッダー項目を表示または非表示にすることができます。

<InDoc product='postgresql'>

-   ![필터 아이콘](/img/ico-filter.svg): 테이블의 컬럼 헤더 항목을 기준으로 목록을 필터링할 수 있습니다.

-   ![세션 킬 아이콘](/img/ico-kill-session.svg): 현재 진행 중인 세션을 중지할 수 있습니다.

</InDoc>

<Xclude product='postgresql'>

-   ![フィルターアイコン](/img/ico-filter.svg)：テーブルのカラムヘッダー項目を基準に一覧をフィルタリングできます。ボタンをクリックすると、各ヘッダーカラムで**_含む_**、**_含まない_**、**_一致_**、**_一致しない_**条件を設定できます。

    :::note

    フィルター条件のオプションを入力するテキストボックスが見えない場合は、カラムの幅を調整してください。

    :::

-   ![ダウンロードアイコン](/img/ico-download.svg)：テーブルの内容をCSVファイルとしてダウンロードできます。

</Xclude>
