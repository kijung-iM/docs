-   **profile_error_httpc_time_max** <span class='type'>Int</span>

    デフォルト値`10000`

    HTTPCの実行時間が指定した値を超えると、TOO SLOWエラーとして処理されます。`0`に設定するとエラーとして処理しません。

-   **profile_httpc_resource_enabled** <span class='type'>Boolean</span>

    デフォルト値`false`

    トレースからHTTP Call ステップを収集する際に、そのステップで使用したCPUとメモリ使用量を追跡します。

-   **profile_position_httpc** <span class='type'>Boolean</span>

    デフォルト値`false`

    HTTPCが実行する時点のStackTraceを記録します。

-   **trace_httpc_normalize_enabled** <span class='type'>Boolean</span>

    デフォルト値 `true`

    トランザクション内のHTTPC URLを解析および正規化する機能を有効にします。

-   **trace_httpc_normalize_urls** <span class='type'>String</span>

    正規化するHTTPC URLパターンを設定します。呼び出しURLパターンを解析して、pathパラメータを削除します。

    :::note

    例）**/a/{v}/b**と宣言すると、**a/123/b** → **a/{v}/b**に置き換えます。複数登録するときは、コンマ（,）を区切り文字として使用します。 置換パターンの整理後、補完が必要です。

    :::

-   **hook_httpc_patterns** <span class='type'>String</span>

    HTTP outbound呼び出しを行う`full packageクラス名`.`メソッド`を設定します。HTTP Callを実行するメソッド(method)のフルパス(full path)を登録して使用します。登録後のクラス再定義(class redefine)を行うか、エージェントを再起動する必要があります。

    ```ini title='whatap.conf'
    hook_httpc_patterns=io.home.test.baseapp.app.post.service.HookHttpcService.*
    ```

    :::note

    -   クラスを複数登録するには、コンマ(,)を区切り文字として使用してください。 

    -   ワイルドカードとしてはアスタリスク(\*)のみ使用でき、正規式は使用できません。

    :::
