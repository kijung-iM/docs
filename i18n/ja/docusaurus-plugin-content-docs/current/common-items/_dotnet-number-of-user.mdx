-   **trace_user_cookie_keys** <span class='type'>String</span><br/>
    設定したクッキーの値をユニークなユーザー情報として選択します。 複数を登録する場合は、コンマ（,）を区切り文字として使用します。 設定した順で検索します。

-   **trace_user_using_ip** <span class='type'>Boolean</span><br/>デフォルト値`false`<br/>
    リアルタイムユーザーは、IPアドレスに基づいて集計します。 IPアドレスではなくクッキーに基づいてユーザーを割り当てたい場合、値を`false`に変更します。
