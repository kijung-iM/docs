ホーム画面 > プロジェクト選択 > **_ダッシュボード_** > **_インスタンスモニタリング_**

![インスタンスモニタリング](/img/db-instance-monitoring.png)

**_インスタンスモニタリング_**メニューではデータベースサーバーの核心指標とアクティブセッションに関する詳細な情報、ロックツリー(Lock Tree)、PQツリー(PQ tree)、プロセス(Process)情報をリアルタイムでモニタリングできます。また、問題になる可能性のある警告を確認することもできます。

## 指標チャート領域の活用

リアルタイムで収集するデータベースサーバーの情報をチャートで確認できます。各指標チャート右上の![設定アイコン](/img/ico-setting.svg)ボタンを選択してください。チャートを選択できる**_カウント_**ウィンドウが表示されます。

![カウントウィンドウ](/img/count-window.png)

## レイアウト設定

指標チャートのレイアウトを設定するには、画面右上の![レイアウト設定アイコン](/img/ico-layout-set.svg)ボタンをクリックしてください。9つ（3x3）の指標チャートがデフォルトレイアウトに設定されており、ページに入るチャートの数を変更できます。

![レイアウト設定](https://img.whatap.io/media/user_guide_database/chart_layout.png)

## アクティブセッション領域の活用

画面の下に位置する***アクティブセッション***領域を通じてリアルタイムで実行中のセッションとクエリ履歴を確認することができます。***query***項目を選択すると、クエリ（***query***）のFullText及び***Plan***情報を確認することができます。

## イベント領域の活用

インスタンスのモニタリング中にイベントが発生した場合は、右側のイベント領域にその履歴を表示します。当該イベントの履歴を選択すると、その時点の**_分析_** > **_カウント推移_**メニューへ移動します。**_カウントの推移_**と**_アクティブセッション_**に対する履歴を確認することができます。
