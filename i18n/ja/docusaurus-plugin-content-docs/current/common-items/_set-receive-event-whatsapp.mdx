### WhatsApp 알림 수신 추가하기

WhatsApp을 통해 알림을 수신할 수 있습니다.

<ImgLang img='set-event-whatsapp-notification.png' desc='WhatsApp'/>

1.  画面の右上のプロフィールアイコンを選択してください。 

2.  ポップアップメニューが表示されたら、**_アカウント情報管理_**ボタンをクリックしてください。 

3.  **_사용자 전화번호_** 섹션에서 **_WhatsApp_** 버튼을 선택하세요. 

4.  **_전화번호_**에 SMS를 수신할 전화번호를 입력하세요.

5.  **_인증번호 전송_** 버튼을 선택하세요.

6.  WhatsApp 애플리케이션으로 전송된 인증번호 6자리를 입력하세요.

7.  **_인증하기_** 버튼을 선택하세요.

:::note

등록한 전화번호를 변경하려면 **_번호 변경하기_** 버튼을 선택한 다음 위의 과정을 반복하세요.

:::
