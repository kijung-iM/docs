ホーム画面 > プロジェクト選択 > **_警告通知_** > **_イベント記録_**

警告通知が発生した履歴を確認できます。過去１年間の履歴が表示できます。各項目を設定してから！[表示アイコン](/img/ico-btn-search.png)ボタンを選択します。

![イベント記録](/img/event-history.png)

:::note
選択したプロジェクトによっては、画面イメージが異なる場合があります。
:::

-   **_時刻選択_**

    -   右側の緑色のボタンを選択して参照時間を選択することができます。
    -   ![左アイコン](/img/left-arrow.svg)または ![右アイコン](/img/right-arrow.svg)ボタンを選択して、選択した表示時間だけ間隔を移動することができます。
    -   詳細な時刻を選択するには、日付または時刻の領域を選択してください。詳細な時刻を設定し、**_確認_**ボタンを選択します。

-   **_フィルター_**：タイトルの内容に基づいてイベント履歴をフィルタリングできます。

-   **_すべて選択_**：プロジェクトに含まれるエージェントを選択できます。

-   **_CSV_**：閲覧したイベント履歴結果をcsvファイルとして保存することができます。 **_CSV_** ボタンをクリックして **_最大CSV行数_**を入力し **_ダウンロード_** ボタンを選択します。

-   **_カラム選択_**：表示されたイベント履歴結果の列項目を追加することができます。

-   **_イベント設定_**：**_警告通知_** > **_イベント設定_**メニューに移動します。
