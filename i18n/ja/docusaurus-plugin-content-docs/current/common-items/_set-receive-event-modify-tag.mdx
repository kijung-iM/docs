### イベント受信タグを修正・削除する

1.  **_ユーザー別イベント受信設定_**セクションのユーザー一覧で**_+ タグ追加_**または**_+_**を選択してください。

2.  **_イベント受信タグ_**ポップアップウィンドウが表示されたら**_タグ一覧_**から修正または削除する項目の![編集アイコン](/img/ico-edit.svg)ボタンを選択してください。

3.  タグ名を修正するか、色を変更してから**_適用_**ボタンを選択してください。

    タグを削除するには![削除アイコン](/img/ico-trash-red.svg) **_タグ削除_**ボタンを選択してください。

:::note

イベントに適用中の**_イベント受信タグ_**項目は削除できません。

:::
