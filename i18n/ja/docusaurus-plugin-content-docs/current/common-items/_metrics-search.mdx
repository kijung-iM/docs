ホーム画面 > プロジェクト選択 > **_サイトマップ_** > **_メトリクス照会_**

**_マトリックス照会_**メニューでタグベースで特定のメトリクスを照会することができます。 

![メトリクス照会sc](/img/metrics-search-java.png)

## ![](/img/number-01.png)時間とカテゴリの選択

![number 1](/img/number-01.png)領域でメトリクスが収集された**_時間_**と**_最大個数_**および**_カテゴリ_**を指定することができます。 **_時間_**と**_カテゴリ_**は、必ず指定してください。

-   **_時間_**：メトリクスが収集された時間を指定して照会することができます。 デフォルト値は`1時間`です。 基本オプションとして提供する照会時間以外に、ユーザーが直接**_時間_**タブを選択して日付と時間を指定することができます。

-   **_最大数_**：![number 3](/img/number-03.png)一覧に照会するメトリクスの最大数を指定できます。 `10`、`50`、`100`、`200`、`300`、`1000`、`2000`、`3000`個まで設定できます。

-   **_カテゴリ_**：関連指標の分類単位です。 **_カテゴリ_**タブを選択し、ご希望の**_カテゴリ_**を指定することができます。

-   ![更新アイコン](/img/ic-refresh-w-bg.svg) **_更新_**：**_更新_**アイコンを選択すると**_カテゴリ_**、**_タグ_**および**_フィールド_**オプションを再読み込むことができます。

## ![](/img/number-02.png)タグとフィールドの選択

![number 2](/img/number-02.png)領域で**_タグ_**と**_フィールド_**を選択します。 ユーザーが個別に指定しない場合、デフォルト設定は完全選択です。 

-   **_タグ_**：収集された対象を区分できる固有情報データです。

-   **_フィールド_**：モニタリング対象から収集された指標です。

-   **_フィルター_**：![フィルタリングアイコン](/img/ico-filter.svg) **_タグ値でフィルタリング_**ボタンを選択し、タグ値を設定してフィルタリングできます。

    > 例）`oname`の値を`demo-8101`に設定してフィルタリングしたデータを照会することができます。

-   ![検索アイコン](/img/ic-search-on-press.svg) **_検索_**：条件を設定して**_検索_**アイコンを選択すると、![number 3](/img/number-03.png)領域でそのメトリクスの原本データを照会できます。 

-   ![CSVダウンロードアイコン](/img/ic-csv-download-ms.svg) **_CSVダウンロード_**：**_CSVダウンロード_** ボタンを選択して、該当メトリクスの原本データをCSVファイルとしてダウンロードできます。

## ![](/img/number-03.png)メトリクステーブル

収集されるメトリクスを事前に特定できないため、収集中のすべてのメトリクスの原本データを確認することが重要です。  上記の条件領域でご希望の条件を設定した後![number 3](/img/number-03.png)領域で、そのメトリクスの原本データをテーブル形式で照会できます。 ユーザーが**_タグ_**と**_フィールド_**の各条件を指定することによって、テーブルのカラムが変更されます。

:::note

-   メトリクスの照会時、**_時間_**と**_カテゴリ_**は必ず指定してください。 

-   メトリクス照会時、**_タグ_**と**_フィールド_**指定は選択事項です。 

:::
