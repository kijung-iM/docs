Flexボードに配置したウィジェット管理を通じて、ダッシュボードをユーザーがカスタマイズすることができます。

## ウィジェット追加

作成済みのFlexボードにウィジェットを追加できます。

1.  **_Flexボード_**メニューの**_ボード_**一覧からウィジェットを追加するダッシュボードを選択してください。

2.  画面左上の**_修正モード_**を選択してください。

3.  左側の**_ウィジェットテンプレート_**一覧から希望するウィジェットを選択してください。メトリクスウィジェットを追加するには、**_すべてのメトリクス_**を選択してから、目的のウィジェットをクリックしてください。

    -   ホーム画面の**_Flexボードメニュー_**に入った場合、ウィジェットを追加した場合は**_+プロジェクト選択_**ボタンをクリックして収集対象プロジェクトを選択してください。
    -   追加するウィジェットがある場合は、同じプロセスを繰り返してください。

すべてのプロセスが完了したら、画面上の**_ビューモード_**を選択してレイアウトを確認し、[一覧へアイコン](/img/ico-exit.svg)(**_一覧へ_**)ボタンをクリックしてください。

## ウィジェット属性の変更

**_Flexボード_**メニューの**_ボード_**一覧からウィジェット属性を変更するボードを選択してください。**_修正モード_**にアクセスします。

### ウィジェット名の変更

名前を変更するウィジェットの名前の右側の![編集アイコン](/img/ico-edit.svg)ボタンをクリックしてください。ウィジェットの**_タイトル_**、**_フォントサイズ_**、**_色_**を設定し、**_保存_**ボタンを選択してください。

:::tip

-   保存時、**_全体ウィジェットに適用_**トグルボタンの機能をアクティブにすると、フォントサイズと色が他のウィジェットにも適用されます。
-   ウィジェットのタイトルをロック設定するには![設定アイコン](/img/ico-invisible.svg)ボタンを選択し、***保存***ボタンをクリックしてください。

:::

### 時間設定

時間を設定しようとするウィジェットの右上の![時間設定アイコン](/img/ico-time-set.svg)ボタンを選択してください。**_時間選択_**ウィンドウが表示されます。

-   ダッシュボード画面に設定した時間を適用するには、**_ダッシュボード設定に従う_**をチェックしてください。ダッシュボードの時間設定は右上に位置します。

-   ユーザーが希望する時間設定を適用するには、**_ダッシュボード設定に従う_**のチェックを解除してください。![左矢印アイコン](/img/left-arrow.svg)または![右矢印アイコン](/img/right-arrow.svg)ボタンを利用して希望する時間を設定してください。

時間設定が完了したら![終了アイコン](/img/ico-close.svg)ボタンをタップしてください。

:::note
ウィジェットのデータタイプによっては、時間設定機能をサポートしていない場合があります。
:::

### 出力データオプションの設定

チャート型ウィジェットの出力データオプションを設定して、ユーザーが希望するデータを選択できます。ウィジェットの右上にある![もっと見るアイコン](/img/ico-more.svg)ボタンを選択してください。

-   プロジェクト、エージェント、アプリケーションなどの種類を選択してデータをフィルタリングできます。

-   個別、マージデータを選択して、チャートに表示されるグラフのタイプを変更できます。

-   ![もっと見るアイコン](/img/ico-more.svg)>![設定アイコン](/img/ico-setting.svg)を選択してください。**_チャート設定_**ウィンドウで詳細オプションを変更することができます。設定できるオプションは次のとおりです。

    -   **_チャートタイプ_**：集計する現在の数値をチャートに表示できるオプションです。
    -   **_y軸チャートの最大値_**：y軸にチャート最大値を**_自動_**または**_固定_**で選択することが。
    -   **_水平線設定_**：**_追加_**ボタンをクリックしてしきい値または範囲を水平線として表示することができます。複数追加して適用できます。水平線を追加した後は、**_適用_**ボタンをクリックしてこそチャートに反映されます。
    -   **_チャートの最大値表示_**：チャートに表示されるグラフの最大値を表示します。
    -   **_보조 차트_**: 위젯의 주요 차트에 보조 차트를 추가 설정할 수 있습니다. 보조 차트의 종류를 선택한 다음 레이아웃, 크기, 데이터 병합 여부를 선택하세요. 다른 위젯에도 공통 적용하려면 **_전체 위젯에 적용_** 버튼을 선택하세요. 보조 차트는 데이터를 다양한 방법으로 보고 싶을 때 유용합니다.

    オプション設定をすべて完了したら、**_チャート設定_**ウィンドウ左上の![閉じるアイコン](/img/ico-close.svg)ボタンをクリックするか、**_チャート設定_**ウィンドウ領域外をクリックしてください。

:::note
ウィジェットのデータタイプによって選択できるオプションは異なる可能性があります。
:::

### メトリクスウィジェットデータの変更{#changedata-metric-widget}

メトリックウィジェットのデータ照会条件を変更できます。 

1.  **_Flexボード_** > **_修正モード_**で変更するウィジェットの右上の![もっと見るアイコン](/img/ico-more.svg)ボタンを選択してください。

  ![メトリクスウィジェットデータの変更](/img/metric-widget-data.png)

1.  ![編集アイコン](/img/ico-edit-metric.svg)ボタンをクリックしてください。データ設定の編集ウィンドウが表示されます。

  ![メトリクスウィジェットデータの編集](/img/metric-edit-window.png)

1.  次を参照して設定を変更し、[適用アイコン](/img/left-arrow.svg)**_ウィジェットに適用_**ボタンを選択してください。

    -   ウィジェットの作成時にデフォルトで生成されるメトリックオプションがタブに表示されます。
    -   メトリックスオプションには、**_カテゴリ_**と**_フィールド_**、**_タグ_**があります。
    -   **_カテゴリ_**項目は現在、変更機能を制限しています。
    -   **_フィールド_**、**_タグ_**項目は複数選択することができます。選択したタグオプションとしてチャートデータをグループ化します。
    -   ![検索アイコン](/img/ico-btn-search.png)ボタンをクリックして原本データを照会することができます。
    -   **_オブジェクト併合_**および**_時間併合_**項目の詳細については、[データ併合オプション](#datamerge-option)を参照してください。  

### データ併合オプション{#datamerge-option}

Flexボードの**_すべてのマトリックス_**一覧に配置されたマトリックスウィジェットは、データ併合オプションを提供します。

1.  **_修正モード_**で**_すべてのマトリックス_**ボタンを選択し、**_ウィジェットテンプレート_**一覧を**_すべてのマトリックス_**一覧として変更してください。

2.  メトリックウィジェットをレイアウトに配置してください。

3.  메트릭스 위젯 오른쪽 위에 ![더보기 아이콘](/img/ico-more.svg) 버튼을 선택하세요.

4.  タグ（例：エージェント名）オプションを選択すると、データマージオプションが表示されます。ウィジェットのサイズによって、メトリクスオプションの位置が異なる場合があります。

![メトリクスウィジェット-データマージオプション](/img/flexboard-datamerge.png)

データマージは、**_オブジェクトマージ_**と**_時間マージ_**方法を提供します。

-   **_オブジェクトマージ_**<br/>
    左側で選択したタグ（例：エージェント名）を基準にチャートデータをグループ化するときに使用します。この時、異なるフィールド（例：CPU）値を持つデータの中でタグが一致する場合、そのデータをマージする方法です。

-   **_時間マージ_**<br/>
    照会時間が長い場合、主に使用します。ソースデータからフィールド値が同じデータ間で、時間範囲に応じて一定の時間間隔（5分または1時間）でデータをマージします。このマージ方法は、時間マージと呼ばれます。

-   **AWS CloudWatchメトリクス**<br/>
    AWSモニタリング指標の場合、統計方式がフィールド名の後にsuffix形式として追加されます。データマージのデフォルト値は、このsuffixを参照しています。

    ![AWS ColudWatchメトリクス](/img/blog_12.png)

## ウィジェットコピー

1.  ウィジェットの右上にある![もっと見るアイコン](/img/ico-more.svg)ボタンをクリックしてください。 
2.  ![コピーアイコン](/img/ico-copy.svg)ボタンをクリックしてください。
3.  **_コピー_**ウィンドウが表示されたら**_+ ウィジェット追加_**を選択してください。
4.  一覧にあるウィジェットの名前を変更してください。
5.  **_適用_**ボタンを選択してください。

**_修正モード_**画面でコピーしたウィジェットを確認することができます。

## ウィジェット削除

1.  ウィジェットの右上にある![もっと見るアイコン](/img/ico-more.svg)ボタンをクリックしてください。 
2.  ![削除アイコン](/img/ico-remove.svg)ボタンを選択してください。
3.  確認メッセージウィンドウが表示されたら、**_削除_**ボタンを選択してください。削除を取り消すには**_キャンセル_**ボタンをクリックしてください。
