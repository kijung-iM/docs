## Error: Not found PHP ini directory

<!-- .ini files -> .ini 파일의 항목? -->

<!-- Scan this dir for additional .ini files 표기 문제 -->

<!-- 내용 정리 및 수정 필요 -->

PHP環境の中で`Scan this dir for additional .ini files`項目の値を確認できなかった場合に発生します。PHPコンパイルインストールオプション`--with-config-file-scan-dir=PATH`が設定されていない場合、その環境情報は存在しません。 

次の説明書を参考にして[PHPコマンド（CLI）の位置](#check-php-cli-path)を正確に確認した上で、[PHP拡張モジュールおよびwhatap-phpサービス選択インストール](#extension-modul-manual-setting)項目を進めてください。
