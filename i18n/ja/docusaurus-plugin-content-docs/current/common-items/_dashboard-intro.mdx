WhaTapエージェントのインストールが完了したら、[WhaTapモニタリングサービス](https://service.whatap.io)にログインしてください。初期画面は**_プロジェクト一覧_**です。**_プロジェクト一覧_**でエージェントをインストールしたサーバーを確認することができます。エージェントをインストールした1つのサーバーはプロジェクトであり、モニタリング単位です。プロジェクト一覧のプロジェクトを選択すると、 **_ダッシュボード_**に入ることができます。

![エージェントとダッシュボード](/img/agent-dashboard.png)

## ダッシュボードとは？

WhaTapモニタリングサービスでダッシュボードはプロジェクトの全体現況を一目で把握できる機能です。モニタリング中のリソース全体の規模を確認でき、リアルタイムダッシュボードでモニタリング対象リソースをフィルタリングできます。進行中のトランザクションと終了したトランザクション情報をリアルタイムで更新し、視覚化されたチャートを提供します。オペレーターは、サービスとシステムの正確な現在の状態を直感的に把握できます。また、素早い障害確認と潜在的な問題要素をすばやく識別できます。

ダッシュボードで把握できる情報は次のとおりです。

-   応答時間分布チャートにより、応答時間範囲を調整したり、エラーデータをフィルタリングしたりできます。
-   進行中のトランザクションを分析し、終了したトランザクションをリアルタイムでアップデートします。
-   TPS、応答時間、CPU、メモリ、リアルタイム使用などのリソース状況を把握できます。
-   最新のデータを自動更新し、リアルタイムのモニタリングサービスを提供します。
