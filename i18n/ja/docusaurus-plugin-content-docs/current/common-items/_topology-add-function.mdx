トポロジーメニューから使用できるアドオンについて説明します。

## ハイライトフィルター

特定ノードを選択し、関連関係を持つノードを強調して表示するための機能を提供します。大量ノード間の複雑な関連関係をトポロジーで簡単に確認できます。

![ハイライトフィルター](/img/topology-highlight.png)

-   特定のノードにマウスオーバーすると、当該ノードや関連ノード、リンク情報などをハイライトとして表示します。
-   キーワード入力により、ノードの選択を保持できます。トポロジー更新時もハイライト状態が維持されます。

## ノード選択機能の利用

画面右下の**_ノード選択(ダブルクリック)_**ボックスをダブルクリックしてください。**_ノード選択_**ボックスで右側のサブノードをクリックすると、ハイライトフィルターを適用できます。

![ノード選択](/img/topology-select-node.png)

本チャートにはプロジェクト、DB呼び出し、外部呼び出しノードをグルーピングしてダイアグラムとして提供し、**_ノード選択_**ボックスで右側の下位ノードをクリックすると、該当ノードの名称がハイライトフィルターに適用されます。

**_ノード選択_**ボックスをダブルクリックすると、次の機能が利用できます。

-   ズームイン/ズームアウト
-   ドラッグ
-   下位ノードのクリック時のハイライトフィルターの適用
-   上位ノードのクリック時の下位ノードの表示・非表示

**_ノード選択_**ボックスでハイライトフィルター機能を利用しない場合は、外郭の点線をダブルクリックするか**_ノード選択を非表示(ダブルクリック)_**をダブルクリックしてください。

:::note
メニューによって**_ノード選択_**機能の位置は異なる場合があります。**_アプリケーショントポロジー_** メニューでは**_ノード選択_**ボタンでメニューを提供します。
:::

## 可変フォント

チャートに表現する情報が多くなるほど、ズームアウトでチャートを縮小して見る可能性が高くなります。チャートを縮小する場合、ノードの位置を固定すればノード識別は可能ですが、文字のサイズが相対的に小さくなり呼び出し情報を識別するのが難しい場合があります。トポロジーチャートでは、フォントのサイズをノード間リンクの距離に比例して表示します。

![可変フォント](https://img.whatap.io/media/user_guide_application/topology/dynamic_font_size.png)

## コンテキストメニュー

トポロジーのノードで右クリックすると、コンテキストメニューが表示されます。コンテキストメニューには、アプリケーションモニタリングダッシュボードに移動するためのリンクなどのメニューを提供します。

![コンテキストメニュー](https://img.whatap.io/media/user_guide_application/topology/context_menu.png)

:::note
**_コンテナマップ_**、**_ノードリソース_**、**_コンテナリソース_**メニューはコンテナモニタリングに限って提供します。
:::

## ノード位置の保存と復元

トポロジー画面では、チャートのノード位置はマウスを左クリックした状態でドラッグして変更できます。変更したノードの位置は、別のメニューに入った後、トポロジー画面に戻ってきても維持されます。

-   **_位置保存_**：ユーザーが変更したノードの位置情報を保存できます。
-   **_位置復元_**：ユーザーが位置保存を実行した時点にノードの位置を戻します。
-   **_位置初期化_**：ノードの位置を初期化します。

:::note

-   ノードロケーションメニューは、ユーザーが入ったトポロジーメニューによっては対応されない場合があります。
-   変更したノードの位置情報は、ウェブブラウザーのローカルストレージに保存されます。他のユーザーの画面では、変更されたノードの位置は反映されません。
:::

## 別名の付与と除去

トポロジー画面右下の**_ノード選択_**ボックスを通じてノードの別称を設定できます。

1.  **_ノード選択(ダブルクリック)_**ボックスをダブルクリックしてください。

2.  別名を設定するノードに移動し、右クリックします。
    ![別名入力](/img/node-custom-name.png)

3.  別名入力ウィンドウが表示されたら、別名を入力してEnterキーを入力してください。

別名情報を保存します。チャートのデータを新たに照会するときに保存した別名を適用します。

保存した別名を削除するには、トポロジー画面メニューの**_別名初期化_**ボタンを選択してください。保存した別名情報を削除します。チャートデータを新たに照会する時点で画面に適用します。

:::note
ノードの別名情報は、ウェブブラウザのローカルストレージに保存されます。
:::

## インバウンド/アウトバウンドフィルター

**_インバウンド_**/**_アウトバウンド_**トグルボタンを利用して、トポロジー画面に外部モジュールおよび外部呼び出しノードが表示有無を変更できます。

-   インバウンド/アウトバウンドの表示状態
    ![インバウンド/アウトバウンドの表示状態](https://img.whatap.io/media/user_guide_application/topology/inbound_outbound_displayed.png)

-   インバウンド/アウトバウンドの非表示状態
    ![インバウンド/アウトバウンドの非表示状態](https://img.whatap.io/media/user_guide_application/topology/inbound_outbound_hidden.png)

## しきい値超過情報の強調

ノード間の呼び出しレスポンス時間およびエラー件数のしきい値設定時にしきい値を超えたノードおよび呼び出し関係を強調して表示します。トポロジー画面のメニューで**_レスポンス時間(ms)_**と**_エラー件数_** テキスト入力ボックスにご希望の設定値を入力してください。

![しきい値超過情報強調](https://img.whatap.io/media/user_guide_application/topology/threshold_emphasized.png)

:::note
**_アプリケーショントポロジー_**メニューでは、**_警告表示_**を選択した後、設定値を入力できます。
:::
