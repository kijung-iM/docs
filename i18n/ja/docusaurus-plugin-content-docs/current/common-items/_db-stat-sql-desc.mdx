## SQL統計

ホーム画面 > プロジェクト選択 > **_統計／レポート_** > **_SQL統計_**

データベースごとの統計データを提供します。提供する機能は次のとおりです。

-   SQL別の時系列チャートを通じて発生時間および性能を確認できます。 
-   複数のインスタンスの統計情報を合算して参照することができます。
-   使用目的別にデータを分類して照会することができます。

:::note

5秒ごとに収集したセッションの情報と、当時実行中のSQL Textに基づいて提供される統計ページです。このページは、収集方法によって誤差が生じる可能性があります。

:::
