-   **tx_caller_meter_enabled**<span class='type'>Boolean</span><br/>
    デフォルト値`false`<br/>
    トランザクション情報でトポロジ統計を生成します。`mtrace_enabled`オプションの値が`true`であれば動作します。

-   **sql_dbc_meter_enabled**<span class='type'>Boolean</span><br/>
    デフォルト値`false`<br/>データベースの接続情報でトポロジ統計を生成します

-   **httpc_host_meter_enabled**<span class='type'>Boolean</span><br/>
    デフォルト値`false`<br/>トポロジマップでhttpc outbound情報を表現します。

-   **actx_meter_enabled** <span class='type'>Boolean</span><br/>
    デフォルト値`false`<br/>トポロジマップでアクティブトランザクション状態を表現します。
