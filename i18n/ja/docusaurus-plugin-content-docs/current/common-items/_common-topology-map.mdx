-   **tx_caller_meter_enabled** <span class='type'>Boolean</span>

    デフォルト値`false`

    トランザクション情報でトポロジ統計を生成します。`mtrace_enabled`オプションの値が`true`の場合、機能します。

-   **sql_dbc_meter_enabled** <span class='type'>Boolean</span>

    デフォルト値`false`

    データベースの接続情報を使用してトポロジ統計を生成します.

-   **httpc_host_meter_enabled** <span class='type'>Boolean</span>

    デフォルト値`false`

    トポロジマップでhttpc outbound情報を表示します。

-   **actx_meter_enabled** <span class='type'>Boolean</span>

    デフォルト値`false`

    トポロジマップでアクティブトランザクションの状態を表現します。
