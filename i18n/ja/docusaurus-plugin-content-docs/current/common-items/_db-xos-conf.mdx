## xos.confファイルの設定

_xos_ディレクトリをデータベースサーバーにコピーしてから、_xos.conf_ファイルにエージェントサーバー情報を入力して追加オプションを設定してください。

-   **dbx_ip** <span class='type'>String</span>

    DBXエージェントをインストールしたサーバーのIPアドレスを入力してください。  
-   **dbx_port** <span class='type'>String</span>

    デフォルト値`3002`

    DBXエージェントと通信するポートを設定してください。DBXエージェントとはUDPで通信します。そのポートにファイアウォールを設定した場合は、UDP通信のためファイアウォールを解除してください。  
-   **cpu_limit** <span class='type'>Int</span>

    デフォルト値`0`

    limitの設定値より大きい値を持つプロセス情報を収集します。1/100%単位で1.23%の場合は`123`として設定してください。  
-   **mem_limit** <span class='type'>Int</span>

    デフォルト値`10240`

    limitの設定値より大きい値を持つプロセス情報を収集します。K bytes単位で10240は10M bytesです。

-   **file** <span class='type'>String</span>

    **_로그_** 메뉴(**_라이브 테일_**, **_로그 트렌드_**, **_로그 검색_**, **_로그 설정_**)를 이용하려면 `file` 옵션에 모니터링하려는 로그 파일명의 절대 경로로 입력하세요. 

    :::note

    -   이 옵션은 DBX 에이전트 1.6.7 버전 이상에서 지원합니다.

    -   **_로그_** 메뉴에 대한 자세한 내용은 [다음 문서](log/learn-main-menu)를 참조하세요.

    :::
