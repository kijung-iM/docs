## テーブルサイズの増減

ホーム画面 > プロジェクト選択 > **_統計／レポート_** > **_テーブルサイズ増減_**

<ProdImg img='db-database-tablesize.png' alt='테이블 사이즈 증감'/>

テーブルおよびインデックスの使用量を日別に収集し、期間による全体およびテーブル別使用量の増減情報を提供します。収集基準はエージェント設定パラメータの`table_min_row`(デフォルト値：1000件以上)、 `tables_hour `(デフォルト値：午前5時)の値に従います。収集基準を変更するには、**_管理_** > **_エージェント設定_**メニューに移動してください。

1.  **_基準日付_**を設定してください。今日の日付がデフォルト値として設定されます。

2.  データを比較する**_比較日付_**を設定してください。

3.  **_インスタンス_**一覧でデータベース設定値を照会するインスタンスを選択してください。

4.  ![検索アイコン](/img/ico-btn-search.png)ボタンを選択してください。

**_テーブル増減_**セクションでは**_件数_**、**_データ_**、**_インデックス_**別増減率と**_全体_**増減率を確認することができます。**_テーブル別の容量詳細_**セクションでは**_テーブル名_**別に詳細増減率を確認することができます。 

:::note

-   **_エージェント設定_**メニューの詳細については、[次の文書](agent-settings#agent-options)を参照してください。

-   収集したいテーブルの照会権限が必要です。次のコマンドを実行して権限を取得してください。

    ```sql

    grant select on [DB_NAME].[TABLE_NAME] to whatap;
    /* または */
    grant select on *.* to whatap;

    ```

-   照会した内容をcsv形式のファイルにダウンロードするには、**_CSVダウンロード_**ボタンをクリックしてください。

:::
