## データベースパラメータ

ホーム画面 > プロジェクト選択 > **_分析_** > **_データベースパラメータ_**

データベースのパラメータ値を日付別に比較できます。 デフォルトでは、現在の時間のデータを表示できます。 **_日付_**で過去の日付を選択してから![検索アイコン](/img/ico-btn-search.png) ボタンをクリックしてください。 現在の時間と選択した過去の日付との変更を確認するには、**_View Changes_**ボタンをクリックしてください。
