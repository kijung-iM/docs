## ダッシュボード作成

Flexボードを作成し、ウィジェットを配置して独自のダッシュボードを作成できます。

1.  **_Flexボード_**メニューから右上の**_+ダッシュボード作成_**ボタンを選択してください。

2.  **_ダッシュボード作成_**ウィンドウが表示されると**_名前_**入力テキストボックスにダッシュボード名を入力します。

3.  ウィジェットの配置方法を選択し、**_ダッシュボード作成_**ボタンをクリックしてください。

    -   **_固定レイアウトのFlexボード_**：ピクセルベースで自由にウィジェットを配置できます。

    -   **_反応型フレックスボード_**：ブラウザサイズを基準としてグリッド応答レイアウトを提供します。 **_反応型フレックスボード_**を選択してグリッドを設定し、**_ダッシュボード作成_**ボタンを選択してください。

4.  **_ウィジェットテンプレート_**選択画面でダッシュボードに配置するウィジェットを選択してください。 メトリクスウィジェットを追加するには**_ウィジェットテンプレート_**から右側の**_すべてのメトリクス_**ボタンを選択してください。

    :::note

    -   **_ウィジェットテンプレート_**：一般的なモニタリング状況で重要な指標を要約して事前定義されたウィジェット一覧を選択することができます。
    -   **_すべてのメトリクス_**：ユーザーのプロジェクトで収集中のすべてのメトリック データに基づいてウィジェットを作成するために使用します。 メトリクスウィジェットの詳細については、[次の文書](flexboard-metric-widget)を参照してください。

    :::

5.  配置したウィジェットの位置を移動または、サイズを変更します。

    -   配置したウィジェットの上部にマウスカーソルを移動してください。カーソルの形が十字形に変更されましたら、ウィジェットをマウスでクリックした状態でドラッグすると位置を移動することができます。

    -   配置したウィジェットの右下にマウスカーソルを移動してください。カーソルの形が矢印の形に変更されたら、ウィジェットをマウスでクリックした状態でドラッグすると任意のサイズに調整することができます。

6.  追加するウィジェットのすべての配置が終わりましたら、**_ビューモード_**を選択してレイアウトを確認してください。レイアウトを再度変更する場合は、**_修正モード_**を選択してウィジェットの配置を変更してください。

すべてのプロセスが終わりましたら、右上の![一覧へ行くアイコン](/img/ico-exit.svg)(**_一覧へ行く_**)ボタンをクリックしてください。**_Flexボード_**メニューの**ボード**一覧から生成したボードを確認することができます。生成したボードを選択して、新たに生成したFlexボードを確認することができます。

:::note

顧客の意見を反映してダッシュボードのテンプレートまたはウィジェットを追加する場合、必要なダッシュボードテンプレートまたはウィジェットがありましたら、[support@whatap.io](mailto:support@whatap.io)までお問い合わせください。

:::
