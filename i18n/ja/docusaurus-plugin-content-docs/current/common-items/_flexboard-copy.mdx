## Flexボードコピー

ユーザーが作成したFlexボードをコピーして、他のユーザーに渡すことができます。

**_Flexボード_**メニューの**_ボード_**一覧から![コピーアイコン](/img/ico-copy.svg)ボタンを選択してください。 **_自分に_**または**_他のユーザー_**オプションを選択します。

-   **_自分にコピー_**オプションを選択すると、**_ボード_**一覧に「\_copie d」prefixが付いた項目に追加されます。

-   **_他のユーザーにコピー_**オプションを選択した場合は、**_+アカウント_**をクリックします。他のユーザーアカウントのメールアドレスを入力し、**_ダッシュボードのコピー_**ボタンをクリックします。**_読み取り専用_**オプションを選択した場合、共有ユーザーはダッシュボードを変更できません。

:::note

**_プロジェクトメニュー_** > **_Flexボード_**メニューからコピーすると、プロジェクトメンバーとダッシュボードが共有されます。

:::

## Flexボードのエクスポート、インポート

jsonファイルとして保存されたFlexボードの設定を他のユーザーアカウントからインポートおよびコピーができます。ダッシュボード管理者権限の機能です。

1.  **_Flexボード_**メニューの**_ボード_**リストから![書き出しアイコン](/img/ico-export.svg)ボタンをクリックします。 

2.  ウェブブラウザからFlexボードの設定をJSONファイルとしてダウンロードします。

3.  画面の右上の![読み込みアイコン](/img/ico-import.svg)**_読み込み_**ボタンを選択します。

4.  JSONファイルを選択します。

:::note

この機能は、**プロジェクトを変更する権限**を所有するユーザーのみ使用できます。

:::
