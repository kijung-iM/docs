## Flexボードコピー

ユーザーが作成したFlexボードをコピーして他のユーザーに渡すことができます。

**_Flexボード_**メニューの**_ボード_**一覧から![コピーアイコン](/img/ico-copy.svg)ボタンを選択してください。 **_自分に_**または**_他人にコピー_**オプションを選択してください。

-   **_自分にコピー_**オプションを選択すると、**_ボード_**一覧に「\_copie d」prefixが付いた項目が追加されます。

-   ***他人にコピー***オプションを選択したら**_+アカウント_**をクリックしてください。他のユーザーアカウントのメールを入力し、**_ダッシュボードコピー_**ボタンをクリックしてください。**_読み取り専用_**オプションを選択すると、共有されたユーザーがダッシュボードを修正できません。

:::note
**_プロジェクトメニュー_** > **_Flexボード_**メニューでコピー機能はプロジェクトメンバーとダッシュボードを共有します。
:::

## Flexボードの書き出し、読み込み

jsonファイルとして保存したFlexボード設定を他のユーザーアカウントから読み込みしてコピーできます。ダッシュボード管理者権限の機能です。

1.  **_Flexボード_**メニューの**_ボード_**リストから![書き出しアイコン](/img/ico-export.svg)ボタンを選択してください。 

2.  ウェブブラウザからFlexボード設定をJSONファイルとしてダウンロードします。

3.  画面右上の![読み込みアイコン](/img/ico-import.svg)**_読み込み_**ボタンを選択してください。

4.  JSONファイルを選択してください。

:::note
この機能は、**プロジェクトの修正権限**を所有しているユーザーのみ利用できます。
:::
