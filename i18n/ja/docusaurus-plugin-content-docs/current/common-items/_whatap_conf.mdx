## whatap.confファイルを設定する

圧縮を解凍した`$WHATAP_HOME`パスに移動してください。_$WHATAP_HOME/whatap.conf_ファイルの内容にアクセスキーとデータ収集サーバーのアドレスが正常に含まれているか確認してください。値がなければ[WhaTapモニタリングサービス](https://service.whatap.io)画面に移動し、**_インストール案内_** セクションで**アクセスキー**値をコピーして貼り付けてください。

```ini title=$WHATAP_HOME/whatap.conf
license={アクセスキー}
whatap.server.host={収集サーバーのIP}
```

:::note

-   **_ダウンロード_**ボタンをクリックしてエージェントをダウンロードした場合は、_whatap.conf_に設定値が入っています。 

-   サーバーから直接ダウンロードした場合は、**アクセスキー**と**収集サーバーIP**を直接に入力してください。

-   _whantap.conf_ファイルの詳細については、[次の文書](set-agent)を参考にしてください。

:::
