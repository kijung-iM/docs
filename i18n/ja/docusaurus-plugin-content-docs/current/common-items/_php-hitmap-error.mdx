## 応答時間分布表にトランザクションが表示されない場合

<!-- 응답 시간 분포도(히트맵) -> 응답 시간 분포도 즉 히트맵 -->

CPU グラフとメモリ グラフは表示できますが、次の状況では応答時間分布 (ヒートマップ) を表示できません。 

-   エージェントは、収集サーバーに正常に接続されましたが、トレーサーは、PHP拡張モジュールに正常に適用されない場合、
-   セットアップ後にApache および PHP-FPM サービスを再起動していない場合

1.  PHP 拡張モジュールが適用されていることを確認します。

    ```bash title='PHP 확장 모듈 확인'
    $ sudo php -m

    [PHP Modules]
    bz2
    calendar
    Core
    ctype
    curl
    date
    …
    whatap               # WhaTapモジュールの読み込みを確認
    …
    [Zend Modules]
    ```

2.  ApacheおよびPHP-FRMサービスを再起動してください。

3.  PHP拡張モジュールが適用されない場合、正常にインストールされません。_whatap.so_または_whatap.ini_ファイルパスがPHP環境と一致していることを確認してください。<br/>
        次の説明書を参照して[PHPコマンド（CLI）パス](#check-php-cli-path)、[PHP拡張モジュールパス](#check-php-extension-module)、[PHP追加ini設定パス](#php-ini-path)の項目を確認してください。
