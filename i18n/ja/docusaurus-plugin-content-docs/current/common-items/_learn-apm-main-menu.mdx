WhaTapエージェントのインストールが完了しましたら、[WhaTapモニタリングサービス](https://service.whatap.io)にログインしてください。WhaTapモニタリングサービスは、円滑なサービス提供と安定した運用、異常検知モニタリングおよび把握のための情報を提供します。

アプリケーションモニタリングは、リアルタイムでトランザクションをモニタリングします。また、特定の時点を設定した分析機能および運用の利便性を高めるためのレポート機能、通知手段を提供します。

WhaTapモニタリングサービスは、次の主な機能を提供します。

-   **リアルタイムモニタリング**

      リアルタイムトランザクションの現状／応答時間の分布図（ヒットマップ） / サーバーの現状／リアルタイムユーザ（Realtime User）の現状／トランザクションの現状／リソースの現状

-   **トランザクション分析**

      トランザクショントレーシング／スタック分析／マルチトランザクショントレーズ（連携追跡）／応答時間分布図（ヒートマップ）／SQL追跡／HTTP Call追跡

-   **事後分析**

      Cube分析／スタック分析／Performance Trend分析／応答時間の分布図（ヒートマップ）／エラー統計

-   **統計**

      Transaction／Error／SQL／Remote HTTP Call／Client IP／ブラウザ通計

-   **管理**

      Agentの実行方式／開発および運営環境分離時のサポート／通知の設定／通知の伝達手段設定

-   **レポート**

      Daily Report／Daily Application Check／Weekly Report／Monthly Report
