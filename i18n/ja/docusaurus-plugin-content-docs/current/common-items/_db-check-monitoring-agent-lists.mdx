## モニタリングの確認

サーバーから正常にログがアップロードされたことを確認し、コンソールに正常登録しているかどうかを確認するには、[WhaTapモニタリングサービス](https://service.whatap.io)の初期画面に移動してください。確認したいプロジェクトを選択し、画面の左側にある**_管理_** > **_エージェント一覧_**メニューを選択してください。**_エージェント一覧_**でデータベースサーバーの名前を確認することができます。データベースサーバーの名前を別途設定するには、[次の文書](agent-dbx-settings#naming)を参照してください。
