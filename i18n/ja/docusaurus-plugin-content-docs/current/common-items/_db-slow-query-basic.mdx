## 基本画面案内

<ProdImg img='db-slow-query.png' desc='slow query'/>

-   ![指示線1](/img/number-01.png) **_時間選択_** / **_インスタンス_**

    進行中のセッションの時間を変更するか、個々のインスタンスを選択できます。

-   ![指示線2](/img/number-02.png)**_Slow Query_**

    Slow Query가 발생한 시점을 차트로 표시합니다. 마우스로 특정 영역을 드래그하면 ![지시선 3](/img/number-03.png) **_Slow Query List_**에서 상세 정보를 확인할 수 있습니다.

-   ![指示線3](/img/number-03.png) **_Slow Query List_**

    **_Slow Query_** チャートでドラッグした領域のSlow Query詳細情報を一覧に表示します。 **_SQL_**項目を選択すると、クエリ（**_query_**）のFull Textおよび**_Plan_**情報が確認できます。
