WhaTapのログモニタリングは、統合システムの構築に基づいてユーザーの利便性とアクセシビリティを高めました。WhaTapは独自の技術力をベースにしっかりとしたデータ収集を通じて、ユーザーが主に使用するライブテール、ログトレンド、ログ検索、イベント通知はもちろん、Parserの効率性をサポートします。 

**_ログ_**メニューは照会及び分析とオプション設定などの機能を提供します。**_イベント設定_**メニューはログ関連イベント通知を設定することができます。

-   ホーム画面 > プロジェクト選択 > **_ログ_**
    WhaTapモニタリングサービスの初期画面でプロジェクトを選択し、プロジェクトメニューの下位にある**_ログ_**メニューを選択してください。次の機能を活用すると、複雑なログにより簡単にアクセスでき、様々な条件を通じてリアルタイムで確認および分析が可能です。 
    -   **_ライブ テール_**
    -   **_ログトレンド_**
    -   **_ログ検索_**

<!-- * ***설정*** -->

-   ホーム画面 > プロジェクト選択 > **_警告通知_** > **_イベント設定_**\
    WhaTapモニタリングサービスの初期画面でプロジェクトを選択し、プロジェクトメニューの下位にある**_警告通知_**メニューを選択し、**_イベント設定_**メニューに入ってください。 イベント条件を設定し、メール、SMS、メッセンジャー、AppPushなど様々なパスから通知を受信することができます。
