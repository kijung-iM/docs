## 日間データベースのレポート

日間データベースレポートは、1日あたりのデータベースのパフォーマンスを数値とチャートで確認できるレポートです。レポート作成時に必要に応じて開始日、開始時間、終了時間を設定することができ
特定のインスタンスだけを別に選択して照会することができます。すべての選択時にすべてのインスタンスを一度に照会します。

### 性能推移チャート

[日間データベースレポート-パフォーマンス推移チャート](/img/db_daily_report_01.png)

データベースインスタンスごとに指標チャートを確認できます。各指標の最大値については円形で表示し、最大値が発生した時間を一緒に表示します。

:::note

-   チャートで表現される指標は、Config設定によって異なる場合があります。

-   各チャートの右上にある**_CSV_**ボタンを利用してデータをエクセルファイルとしてダウンロードすることができます。

:::

### SQL TOP 10

![日間データベースレポート-SQL TOP 10](https://img.whatap.io/media/images/db_daily_report_02.png)

照会期間中に全体または選択したインスタンスが実行したSQLクエリ統計を確認できます。クエリ実行時間の最大値に基づいて降順にソートし、上位10項目のみを照会します。

## 週次データベースレポート

週次データベースレポートは、1週次のデータベースパフォーマンスを数値とチャートで確認できるレポートです。開始日を直接設定することができ、照会日を基準として7日前から照会できます。

### 性能推移チャート

![週次データベースレポート-パフォーマンス推移チャート](/img/db_weekly_report_01.png)

データベースインスタンスごとに、週間指標を数値およびチャートで比較できます。照会期間中の最大値は円形で表示し、その数値が発生した時間を一緒に表示します。

:::note

-   チャートで表現される指標は、Config設定によって異なる場合があります。

-   各チャートの右上にある**_CSV_**ボタンを利用してデータをエクセルファイルとしてダウンロードすることができます。

:::

### SQL TOP 10

![週次データベースレポート-SQL TOP10](/img/db_weekly_report_02.png)

照会期間中に全体または選択したインスタンスが実行したSQLクエリ統計を確認できます。クエリ実行時間の最大値(elapse max)に基づいて降順にソートし、上位10項目のみを照会します。
