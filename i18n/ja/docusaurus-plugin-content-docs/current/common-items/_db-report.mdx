## 日間データベースのレポート

日間データベースレポートは、1日あたりのデータベースのパフォーマンスを数値とチャートで確認できるレポートです。レポート作成時に必要に応じて開始日、開始時間、終了時間を設定することができ
特定のインスタンスだけを別に選択して照会することができます。すべての選択時にすべてのインスタンスを一度に照会します。

### 性能推移チャート

[日間データベースレポート-パフォーマンス推移チャート](/img/db_daily_report_01.png)

データベースインスタンスごとに指標チャートを確認できます。各指標の最大値については円形で表示し、最大値が発生した時間を一緒に表示します。

:::note

-   チャートで表現される指標は、Config設定によって異なる場合があります。
    <!-- 담당자 문의 Config ?? -->
-   各チャートの右上にある**_CSV_**ボタンを利用してデータをエクセルファイルとしてダウンロードすることができます。
:::

### SQL TOP 10

![日間データベースレポート-SQLTOP10](https://img.whatap.io/media/images/db_daily_report_02.png)

조회 기간에 전체 또는 선택한 인스턴스가 수행한 SQL 쿼리 통계를 확인할 수 있습니다. 쿼리 수행 시간의 최대값을 기준으로 내림차순 정렬하여 상위 10개 항목만을 조회합니다.

## 週間データベースレポート

週間データベースレポートは、1週間のデータベースパフォーマンスを数値とチャートで確認できるレポートです。開始日を直接設定することができ、照会日を基準として7日前から照会できます。

### 性能推移チャート

![週間データベースレポート-パフォーマンス推移チャート](/img/db_weekly_report_01.png)

데이터베이스 인스턴스별로 주간 지표를 수치 및 차트로 비교해 볼 수 있습니다. 조회 기간에 최대값은 원형으로 표시하며 해당 수치가 발생한 시간을 함께 표시합니다.

:::note

-   チャートで表現される指標は、Config設定によって異なる場合があります。
    <!-- 담당자 문의 Config ?? -->
-   各チャートの右上にある**_CSV_**ボタンを利用してデータをエクセルファイルとしてダウンロードすることができます。
:::

### SQL TOP 10

![週間データベースレポート-SQL TOP10](/img/db_weekly_report_02.png)

조회 기간에 전체 또는 선택한 인스턴스가 수행한 SQL 쿼리 통계를 확인할 수 있습니다. 쿼리 수행시간의 최대값(elapse max)을 기준으로 내림차순 정렬하여 상위 10개 항목만을 조회합니다.
