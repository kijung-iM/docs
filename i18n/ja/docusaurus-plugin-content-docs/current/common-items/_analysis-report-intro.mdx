WhaTapモニタリングサービスの初期画面でプロジェクトを選択して、**_プロジェクトメニュー_**の下位メニューを選択してください。 

-   **_プロジェクトメニュー_** > **_分析_**
    **_分析_**メニューは**メトリクスチャート**、**MSA分析**、**ヒットマップ**、**スタック**、**キューブ**、**トランザクション検索**などを通じて分析を提供します。

-   **_プロジェクトメニュー_** > **_統計／報告書_**
    **_統計／レポート_**メニューは**週間および日間と月間報告書**、そして**統計**を提供します。
