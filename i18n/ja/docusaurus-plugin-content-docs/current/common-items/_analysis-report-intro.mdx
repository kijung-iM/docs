WhaTapモニタリングサービスの初期画面でプロジェクトを選択して、**_プロジェクトメニュー_**のサブメニューを選択します。 

-   **_プロジェクトメニュー_** > **_分析_**

    **_分析_**メニューは**メトリクスチャート**、**MSA分析**、**ヒートマップ**、**スタック**、**キューブ**、**トランザクション検索**などを通じて分析を提供します。

-   **_プロジェクトメニュー_** > **_統計/レポート_**

    **_統計/レポート_**メニューには、**週次・日次・月次レポート**と**統計**が表示されます。
