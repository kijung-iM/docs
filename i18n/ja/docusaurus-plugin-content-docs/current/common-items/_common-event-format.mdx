## 共通の通知テンプレート

共通テンプレートは、ほとんどの場合に適用できます。 さまざまな環境で同じフォーマットで通知を受けることができます。

-   Product Type：アプリケーション、DB、Kubernetes
-   Event Type：アプリケーション通知、データベース通知、メトリック通知
-   Event Channel : sms, mobile, 3rd party plugin , plugin

### イベントタイトル

```log title='이벤트 제목 포맷'
[Level][Platform][ProjectName][ApplicationName][EventTitle]
```

```log title='이벤트 제목 예시'
[Info][JAVA][アプリケーションプロジェクト][TC-0-1-8081][CRITICAL_HIGH_MEMORY]
```

:::note

-   Application Nameを設定していない場合は、イベントのタイトルから除外されます。
-   Platformは次のいずれかで表示します。
    -   JAVA
    -   NODEJS 
    -   PYTHON
    -   PHP
    -   DOTNET
    -   GO
    -   POSTGRESQL
    -   ORACLE
    -   MYSQL
    -   MSSQL
    -   BSM_JAVA
    -   CLOUDWATCH
    -   TIBERO
    -   KUBERNETES
    -   KUBE_NS
    -   URLCHECK
    -   URLCHECK_ADMIN
    -   CUBRID
    -   ALTIBASE
    -   CLUSTER
    -   REDIS
    -   MONGODB
    -   VR
    -   RUM
:::

### イベントメッセージ

イベントメッセージに含まれる情報です。 「optional」が 「false」の場合は、常にメッセージに含まれます。 「optional」が「true」の場合、そのデータを確認できれば表示します。

```log title='이벤트 메시지 예시'
Project Name : アプリケーションプロジェクト
Project Code : 3
Application Name : TC-0-1-8081
Event Message : RECOVERED: Memory is too high. less than 10%
Event ON Time : 2022-04-12 18:53:24 +0900
Event OFF Time : 2022-04-12 18:53:24 +0900
Alert Type : APPLICATION_MEMORY
Metric Name : memory
Metric Value : 20
Metric Threshold : 10
Stateful : true
```

以下は、イベントメッセージのコンポーネントです。

| En                                                | Ko              | サポートされている通知タイプ   | 説明                                           |
| ------------------------------------------------- | --------------- | ---------------- | -------------------------------------------- |
| Project Name                                      | プロジェクト名         | 全体               | -                                            |
| Project Code                                      | プロジェクトCode      | 全体               | -                                            |
| Application Name                                  | アプリケーション名       | 全体（optional）     | oname                                        |
| Event Message                                     | イベントメッセージ       | 全体               | -                                            |
| Alert Type                                        | イベントタイプ         | 全体               | 下記のAlertTypeの種類表を参考                          |
| Event ON Time                                     | イベント発生時間        | 全体               | 2022-04-13 10:40:49 +0900で+0900はGMTを意味します。   |
| Event OFF Time                                    | イベント解除時間        | 全体（optional）     | 2022-04-13 10:40:49 +0900で+0900はGMTを意味します。   |
| Metric Name                                       | メトリクス名          | 全体（optional）     | イベント条件判断で使用するメトリクスの名前。                       |
| Metric Value                                      | メトリック値          | 全体（optional）     | メトリック値がメトリックしきい値を超えると、イベント発生条件が満たされている場合です。  |

| Query Interval                                    | クエリ間隔           | 複合メトリックの通知       | -                                            |
| Silent Time                                       | 無音時間            | 複合メトリックの通知       | -                                            |
| Query                                             | URL             | Exception通知      | Exceptionを発生させたリクエストのURL                     |
| TXID                                              | トランザクションID      | Exception通知      | -                                            |
| Class                                             | エラークラス名         | Exception通知      | -                                            |
| Log Message                                       | ログメッセージ         | サーバー - ファイルログの通知 | -                                            |
| Log File                                          | ログファイルのパス       | サーバー - ファイルログの通知 | -                                            |
| IP                                                | IP              | サーバー通知の全体        | -                                            |
| CPU                                               | CPU             | サーバー通知の全体        | イベント発生当時のSnapshot                            |
| CPU_load1                                         | CPU_load1       | サーバー通知の全体        | イベント発生当時のSnapshot                            |
| CPU_loadPerCore                                   | CPU_loadPerCore | サーバー通知の全体        | イベント発生当時のSnapshot                            |

| Disk Name | Used Percent | Free Size | IO Percent | ディスクパフォーマンス     | サーバー通知の全体        | イベント発生当時のSnapshot                            |
