## カラム情報の案内{#column-info}

<Tabs groupId="db-columns">

<TabItem value="activesession" label="Active Session">

| 項目                | 説明                                                                                                       |
| ----------------- | -------------------------------------------------------------------------------------------------------- |
| `pid`             | プロセスID                                                                                                   |
| `db`              | データベース名                                                                                                  |
| `user`            | SQL実行アカウント                                                                                               |
| `application`     | クライアントアプリケーション名                                                                                          |
| `client_host`     | 接続されたクライアントのホスト名(`client_addr`の逆方向DNS照会で報告される)                                                           |
| `client_addr`     | 클라이언트의 IP 주소                                                                                             |
| `client_port`     | 클라이언트의 포트 번호(유닉스 소켓을 사용하는 경우 -1)                                                                         |
| `query_hash`      | クエリのハッシュ値（WhaTap内部管理値）                                                                                   |
| `query_start`     | 現在活性状態のクエリが開始された時間(活性状態ではない場合、最後のクエリが開始された時間)                                                            |
| `state`           | 백엔드 프로세스의 현재 상태(active/idle 등)                                                                           |
| `state_change`    | state가 변경된 최근 시간                                                                                         |
| `xact_start`      | 当該プロセスの現在のトランザクションが開始された時間(活性トランザクションがない場合は`null`)                                                       |
| `backend_start`   | 해당 백엔드 프로세스가 시작된 시간                                                                                      |
| `backend_xmin`    | 해당 백엔드 프로세스가 처리 중인 `xid`의 최소값                                                                            |
| `backend_type`    | 해당 백엔드 프로세스의 유형, `autovacuum worker`, `logical replication launcher`, `client backend`, `checkpointer` 등 |
| `backend_xid`     | 해당 백엔드가 사용 중인 `xid`                                                                                      |
| `wait_event_type` | 해당 백엔드 프로세스가 대기 중인 이벤트 `type`                                                                            |
| `wait_event`      | 해당 백엔드 프로세스가 현재 대기 중인 경우 대기 이벤트 이름                                                                       |

</TabItem>

<TabItem value="locktree" label="Lock tree">

| 項目             | 説明                                            |
| -------------- | --------------------------------------------- |
| `holder type`  | `holder` 세션의 `type` (transaction id, tuple 등) |
| `lock mode`    | `holder_mode` (exclusive lock, shared lock 등) |
| `waiter type`  | `waiter` 세션의 `type` (transaction id, tuple 등) |
| `lock request` | `waiter_mode` (exclusive lock, shared lock 등) |

</TabItem>

<TabItem value="process-info" label="Process Information">

다음 항목은 XOS 에이전트를 설치한 경우 수집되는 지표입니다.

| 項目         | 説明                                         |
| ---------- | ------------------------------------------ |
| `cputime`  | CPU使用時間                                    |
| `cpuusage` | CPU使用率                                     |
| `elapse`   | CPU 사용 경과 시간                               |
| `vsize`    | 가상 메모리 사이즈(Kb)                             |
| `rss`      | Resident Set Size(RSS)、プロセスに関連する物理的ページ数    |
| `state`    | 프로세스 상태                                    |
| `ioread`   | ブロックの読み取りに費やされた実際時間(milliseconds)          |
| `iowrite`  | ブロックの書き込みに費やされた実際時間(milliseconds)          |
| `pss`      | 프로세스 고유 메모리 사용량 + 하나의 프로세스가 차지하는 공유 메모리 비율 |
| `uid`      | user id                                    |
| `cmd`      | 수행 중인 명령어                                  |
| `longcmd`  | cmd의 전체 경로                                 |

</TabItem>
</Tabs>
