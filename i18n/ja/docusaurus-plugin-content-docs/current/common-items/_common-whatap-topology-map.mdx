-   **whatap.tx_caller_meter_enabled** <span class='type'>Boolean</span>

    デフォルト値`false`

    トランザクション情報でトポロジ統計を生成します。`mtrace_enabled`オプションの値が`true`であれば動作します。

-   **whatap.sql_dbc_meter_enabled** <span class='type'>Boolean</span>

    デフォルト値`false`

    データベースの接続情報でトポロジ統計を生成します.

-   **whatap.httpc_host_meter_enabled** <span class='type'>Boolean</span>

    デフォルト値`false`

    トポロジマップでhttpc outbound情報を表現します。
