WhaTapエージェントのインストールが完了したら、[WhaTapモニタリングサービス](https://service.whatap.io)にログインしてください。初期画面は**_プロジェクト一覧_**です。**_プロジェクト一覧_**でユーザーが作成したプロジェクトを確認することができます。**_プロジェクト一覧_**で作成したプロジェクトを選択すると、**_[インスタンス一覧](instance-list)_**に入ることができます。

WhaTapデータベースモニタリングサービスでダッシュボードはプロジェクトの全体現況を一目で把握できる機能です。モニタリング中のリソース全体の規模を確認でき、リアルタイムダッシュボードでモニタリング対象のリソースをフィルタリングできます。

-   主なパフォーマンス指標とステータスを表示できるように構成されています。パフォーマンス指標は5秒間隔で更新されます。
-   現在進行中のアクティブセッション（Query含む）、ロック情報、プロセス情報を確認することができます。Public Cloud、Private Cloudの両方を適用するためにクエリベースのモニタリングで製品を構成しました。
-   1つのインスタンス内で複数のDBをフィルタリングできます。
-   収集したデータに基づいて、特定の1日あたりの全体的な推移を分析できる機能を提供します。特定の時点で進行中だったセッションを把握し、遅かったQueryを分析することができます。
