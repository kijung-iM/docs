### ファイアウォールを確認する方法

`telnet`コマンドを実行し、接続関連情報をが正常に表示します。(`telnet[サーバーIP][サーバーポート]`)

```bash title='예시'
$ telnet 52.193.60.176 6600
Trying 52.193.60.176...
Connected to 52.193.60.176.
Escape character is '^]'.
```

:::note
WhaTapの収集サーバー情報は、**_プロジェクトメニュー_** > **_管理_** > **_エージェントのインストール_**メニューから確認できます。
:::
