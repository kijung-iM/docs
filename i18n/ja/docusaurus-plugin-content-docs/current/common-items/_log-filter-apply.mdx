フィルターを適用すると、入力した条件に合うログをフィルタリングします。複数のフィルターを入力できます。フィルターのタグが同じ場合、OR(`||`)で、 そうでない場合はAND(`&&`)として適用されます。 

入力ウィンドウに値を直接入力したり**_フィルター_**入力ウィンドウをクリックしてフィルターを指定できます。入力ウィンドウに値を入力するときは、分かち書きでキーワードを区別します。キーワード内の分かち書きを含む場合_''_ または _""_符号で囲んで入力してください。符号で囲んだキーワードに_\\_を含む場合_\\\\_で入力してください。![検索アイコン](/img/ic-search-on-press.svg) **_検索_**ボタンをタップするとフィルターが適用されたデータを![number-s3](/img/number-03.png)領域で照会することができます。
