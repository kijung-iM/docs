## 基本画面ガイド

<ProdImg img='db-instance-list.png' desc='인스턴트 목록' />

-   ![数字1](/img/number-01.png)領域は、プロジェクトに設定されているインスタンスの状態を表示します。表示されたステータス情報は、インスタンスの合計数と一致し、テーブルの**_状態_**カラム情報と一致します。

-   **_状態_**カラムで正常でない状態アイコンをクリックすると**_イベント_**ウィンドウが表示されます。当該インスタンスで発生したイベントの履歴を確認できます。

-   各インスタンスを選択すると、**_インスタンスモニタリング_**メニューが表示されます。選択したインスタンスの詳細をリアルタイムでモニタリングできます。 

    :::note

    **_インスタンスモニタリング_**の詳細については、[次の文書](instance-monitoring)を参照してください。

    :::

-   主要なパフォーマンス指標をグラフチャート形式で提供します。最も高い数値をチャートに表示します。チャートにカーソルを合わせると、その場所の数値が表示されます。パフォーマンス指標の詳細については、[次の文書](metrics-data-list)を参照してください。

<Xclude product='redis'>

-   **_インスタンスの一覧_**で**_サーバー名_**コラムに表示された <span class='mark master'>M</span>, <span class='mark instance'>S</span>, <span class='mark master'>C</span>はそれぞれMaster、Slave、Clusterを意味します。Replicationで構成されたDBの場合に表示します。

</Xclude>

-   画面右上の![通知アイコン](/img/ico-noti.svg)ボタンをクリックすると、リアルタイムの通知が確認できます。通知設定の詳細については、[次の文書](set-notice)を参照してください。

    :::note

    画面の一番上にある固定領域の詳細については、[次の文書](../main-ui-intro)を参照してください。

    :::
