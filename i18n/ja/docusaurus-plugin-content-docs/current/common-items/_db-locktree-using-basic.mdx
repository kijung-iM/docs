<InDoc product='postgresql'>

![숫자 1](/img/number-01.png) **_시간_**, ![숫자 2](/img/number-02.png) **_인스턴스_**에서 조회하길 원하는 날짜와 시간, 인스턴스를 선택하세요. 설정한 시간에 락이 발생했다면 **_Lock Wait Wessions_**와 화면 아래 **_Lock Tree_** 테이블에 조회한 데이터를 표시합니다.

</InDoc>

-   グラフチャートで特定の時間をクリックすると、画面下のテーブルから選択した時間のロック情報を確認できます。画面下の詳細情報から秒単位ボタンをクリックすると、5秒単位でロック情報を照会することができます。

-   グラフチャートで特定の時間領域をドラッグすると、選択した領域を拡大してロック情報を確認できます。
