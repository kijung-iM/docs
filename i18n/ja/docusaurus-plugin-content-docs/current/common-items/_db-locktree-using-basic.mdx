ご希望の![数字1](/img/number-01.png)日付と時間を設定し、[数字2](/img/number-02.png)の一覧からインスタンスを選択してください。設定した時間にロックが発生した場合は、**_lock wait sessions_**と共に画面下のテーブルに照会したデータを表示します。

-   グラフチャートで特定の時間をクリックすると、画面下のテーブルから選択した時間のロック情報を確認できます。画面下の詳細情報から秒単位ボタンをクリックすると、5秒単位でロック情報を照会することができます。
-   グラフチャートで特定の時間領域をドラッグすると、選択した領域を拡大してロック情報を確認できます。
