### オプションを追加する

<ProdImg img='app-set-agent-in-service.png' desc='에이전트 설정' />

1.  **_エージェント一覧_**でオプションを追加しようとする![数字1](/img/number-01.png)エージェントを選択してください。

2.  ![数字2](/img/number-02.png) **_オプション作成_**から追加するオプション項目を選択してください。

    ![](/img/add-agent-option.png)

    -   **_検索_**で追加するオプションを見つけることができます。テキストを入力すると、一致するオプションをフィルタリングします。
    -   **_直接入力_**を選択すると、オプションキーと値を入力できます。

3.  選択したオプションキーの説明とデフォルト値を確認し、設定値を入力してください。

    ![](/img/add-agent-option-value.png)

    -   選択したオプションをキャンセルするには![削除アイコン](/img/ico-trash.svg)ボタンを選択してください。
    -   オプションを追加設定するには**_+ 追加する_**ボタンを選択して上記2のプロセスを繰り返してください。

4.  ご希望のすべてのオプションを追加した場合は、画面右上の**_適用_**ボタンを選択してください。

選択したオプションと設定値をエージェントに適用します。

:::note

-   修正中の内容を初期化するには、画面右上の![初期化アイコン](/img/ico-refresh.svg)ボタンを選択してください。
-   オプション値に何も入力していない状態で**_適用_**ボタンを選択すると、そのオプションが削除されます。
-   すでに追加されたオプションは、オプション一覧から選択できません。
-   アプリケーションの種類やエージェントのバージョンによって、適用できるオプションキーは異なる場合があります。
-   例として提供された画像は、アプリケーションの種類やエージェントによって異なる場合があります。

:::
