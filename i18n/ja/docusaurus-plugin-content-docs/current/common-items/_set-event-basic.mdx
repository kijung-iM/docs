## 基本オプション

以下は、各項目ごとに設定できる基本オプションです。

-   **_エージェントの無効化アラート_**

    設定した時間にエージェントからデータが受信できない場合は、アラート通知を送信します。トグルボタンを選択して、機能をオンまたは、オフにすることができます。

-   **_イベント状態が解決された場合の追加通知_**

    アプリケーションイベントの項目で発生したイベントの状態が解決されると追加通知の送信有無が選択できます。トグルボタンを選択して機能をオンまたは、オフにすることができます。

-   **_エージェント重複通知_**

    モニタリング中のエージェントのうち、同じIDを設定したエージェントが2つ以上ある場合、メトリクスを重複して収集することによって発生する警告通知です。エージェント名が重複して設定されていないかエージェント設定で関連オプションを確認してください。ブルー/グリーンのようなデプロイ状況でエージェントが重複して警告通知が発生した場合、この機能を利用することをお勧めします。同名のエージェントが同時に2つ以上起動した場合、選択した時間中は通知が発生しません。

-   **_条件_**

    ![指示線1](/img/number-01.png)条件は<span class='slow'>警告(Warning)</span>レベル、![指示線2](/img/number-02.png)条件は<span class='vslow'>危機(Critical)</span>レベルです。

-   ![指示線3](/img/number-03.png)**_続行_**

    選択した時間にイベント条件が続く場合に通知を送信します。

-   ![指示線4](/img/number-04.png)**_ミュート_**

    通知を送信した後、選択した時間に同じ通知を送信しません。

-   ![指示線5](/img/number-05.png)On / Off

    トグルボタンを選択して発生するイベントをオンまたは、オフにすることができます。
