## カウント推移チャートの活用

カウント推移関連メニューから次の機能を活用できます。

![カウント推移24時間](/img/analysis-count-24.png)

画面右上の![レイアウトアイコン](/img/ico-layout-set.svg)ボタンをクレックしてください。***レイアウト設定***ウィンドウで***1x3***に設定し、***保存***ボタンを選択してください。***時間***は***最近24時間***として設定してください。次のように、1日間のデータベースのコア指標を詳細に確認できます。

![カウント推移の時間別確認](/img/analysis-count-usage.png)

-   指標チャートの特定時点をクリックすると(![指示線1](/img/number-01.png))その地点を赤い線で表示し、当時収集されたアクティブセッションを確認することができます。 

-   アクティブセッションデータは5秒単位で収集しています。画面の下の![指示線2](/img/number-02.png)秒単位セルを選択し、5秒単位でデータを確認することができます。

-   1分単位で時間を移動するには![指示線3](/img/number-03.png)![以前の時間アイコン](/img/ico-left-time.svg)または![以降の時間アイコン](/img/ico-right-time.svg)ボタンを選択してください。

-   詳細情報データセルに表示する項目を追加または削除するには![指示線3](/img/number-03.png)![コラムアイコン](/img/ico-column-set.svg)ボタンを選択してください。**_コラム編集_**ウィンドウが表示されたら、希望する項目を追加または削除することができます。

-   ![指示線3](/img/number-03.png)![フィルターアイコン](/img/ico-filter.svg)ボタンをクリックすると、データセルヘッダーにフィルターを適用して希望するデータだけをソートできます。

-   画面の下に出力されたデータをcsvファイルとしてダウンロードできます。![指示線3](/img/number-03.png)![ダウンロードアイコン](/img/ico-download.svg)ボタンを選択してください。
