-   **addin_monitor_num** <span class='type'>Int</span>

    デフォルト値`0`

    パフォーマンスカウンターの拡張のために追加宣言するクラス数を設定します。`addin_monitor_num`に設定した値だけ`addin_monitor_x`を宣言します。

-   **addin_monitor_0** <span class='type'>Int</span>

    デフォルト値`0`

    ```ini title='whatap.conf'

    addin_monitor_num=2

    addin_monitor_0=a.a.Class1

    addin_monitor_1=a.a.Class2

    ```

-   **stat_domain_enabled** <span class='type'>Boolean</span>

    デフォルト値`false`

    クライアントに接続中のドメイン別トランザクション統計収集機能をアクティブにします。

-   **stat_domain_max_count** <span class='type'>Int</span>

    デフォルト値`7000`

    1つのJVMから5分間収集するドメイン別トランザクション統計の最大レコード数です。

-   **stat_mtrace_enabled** <span class='type'>Boolean</span>

    デフォルト値`false`

    マルチサーバトランザクションでCaller&Calleeの間のバージョンごとの依存性統計を収集します。

-   **mtrace_spec** <span class='type'>String</span>

    デフォルト値`v1`

    現在のインスタンスのアプリケーションバージョンを設定します。任意の文字列を設定できます。このデータは、呼び出し統計のために使用されます。

-   **stat_mtrace_max_count** <span class='type'>Int</span>

    デフォルト値`7000`

    マルチサーバトランザクションでCaller & Calleeの間のバージョン別依存性統計の最大レコード数です。
