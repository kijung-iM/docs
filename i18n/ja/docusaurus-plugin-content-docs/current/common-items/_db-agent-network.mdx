エージェントをインストールしたデータベースサーバーとWhaTap収集サーバーとの間でネットワーク通信のための設定です。

-   **license**<span class='type'>String</span><br/>
    エージェントのインストール時にサーバーから付与されたプロジェクトアクセスキーを設定します。プロジェクトアクセスキーには、エージェントが属するプロジェクトとセキュリティ通信のためのパスワードキーが含まれています。

-   **whatap.server.host** <span class='type'>String</span><br/>デフォルト値 `127.0.0.1,127.0.0.1`<br/>
    エージェントが収集したデータを送信するWhaTap収集サーバーのIPアドレスを設定してください。収集サーバーの二重化で2つ以上のIPを持っている場合は、コンマ(,)を区切り文字として使用してください。設定したIPアドレスには、収集サーバーproxyデーモンがリスニング状態でサービスされなければなりません。

-   **whatap.server.port**<span class='type'>String</span><br/>デフォルト値`6600`<br/>
    収集サーバーポートを設定します。ポートは1つだけ設定できるので、`whatap_server_host`に設定した収集サーバは同一ポートを使用する必要があります。

-   **dbms**<span class='type'>String</span><br/>
    モニタリングするデータベースのタイプを設定します。DBのタイプは次のいずれかを設定できます。 
    -   `oracle`
    -   `postgresql`
    -   `mysql`
    -   `mssql`

-   **db_ip**<span class='type'>String</span><br/>
    モニタリングするデータベースサーバーのIPアドレスを設定します。

-   **db_port**<span class='type'>Int</span><br/>
    データベースが通信に使用するポートを設定します。各データベースのデフォルトポートは次のとおりです。
    -   Oracle: `1521`
    -   PostgreSQL: `5432`
    -   EDB: `5444`
    -   MySQL: `3306`
    -   MS SQL: `1433`

-   **db**<span class='type'>String</span><br/>
    データベース接続に必要なDB名を設定します。 
    -   Oracleの場合、SIDを設定します。
    -   PostgreSQLとEDBの場合、設定しない場合はデフォルトでPostgreSQLのDBを使用します。
    -   MS SQLの場合、設定しなければデフォルト値としてmasterのDBを使用します。
    -   MySQLの場合、設定しなくてもいいです。
