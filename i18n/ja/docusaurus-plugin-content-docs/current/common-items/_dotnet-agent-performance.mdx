:::note

-   エージェントの性能オプションを有効にするには、設定後に、WhaTap .NET Windows サービスを再起動する必要があります。

-   エージェントのパフォーマンスを拡大すると、CPU、メモリ使用率が増加する可能性があります。
:::

-   **tx_max_count** <span class='type'>Int</span><br/>
    デフォルト値`5000`<br/>トランザクションの最大保存数です。 同時に処理するトランザクションが多い場合は、そのサイズをさらに大きくしてください。

-   **tx_default_capacity**<span class='type'>Int</span><br/>
    デフォルト値`101`<br/>トランザクションを保存するバッファの開始サイズです。

-   **tx_load_factor** <span class='type'>String(Float)</span><br/>
    デフォルト値`0.75`<br/>トランザクションを保存するバッファのサイズを徐々に増加させるための基準です。 101個から75％以上保存すると、バッファのサイズを2倍にします。
                              

-   **queue_log_enabled** <span class='type'>Boolean</span><br/>
    デフォルト値`false`<br/>queue関連ログを出力します。

-   **queue_tcp_enabled**<span class='type'>Boolean</span><br/>デフォルト値`true`<br/>
    TCPデータ送信処理に使用するChannelの代わりに、別のQueueが使用されます。 データ優先度付きキューを使用します。

-   **queue_tcp_sender_thread_count** <span class='type'>Int</span><br/>デフォルト値`2`<br/> 
    TCPデータ送信を処理するスレッド数を設定します。 スレッド数を増やすと、データ送信をより迅速に処理できます。 

-   **net_send_buffer_size** <span class='type'>Int</span><br/>デフォルト値`1024`<br/> 
    TCPデータ処理のChannel bufferを設定します。

-   **queue_udp_enabled** <span class='type'>Boolean</span><br/>デフォルト値`false`<br/> 
    UDPデータ受信処理にChannelの代わりに別のQueueを使用します。

-   **queue_udp_size** <span class='type'>Int</span><br/>デフォルト値`2048`<br/> 
    UDPデータ受信バッファを設定します。 チャンネル、キューバッファを設定します。 

-   **queue_udp_read_thread_count** <span class='type'>Int</span><br/>デフォルト値`3`<br/> 
    UDPデータ受信を処理するスレッドの数を設定します。 スレッド数を増やすと、UDPバッファからより迅速にデータを読むことができます。 

-   **queue_udp_overflowed_size** <span class='type'>Int</span><br/>デフォルト値 `4096`<br/> 
    UDPデータ受信にQueueを使用する場合は、overflowed Queueバッファのサイズを設定します。

-   **queue_udp_process_thread_count** <span class='type'>Int</span><br/>デフォルト値`1`<br/> 
    UDPデータを処理するスレッド数を設定します。 スレッド数を増やすと、UDPで読んだデータをより迅速に処理できます。

-   **queue_profile_enabled**<span class='type'>Boolean</span><br/>デフォルト値`false`<br/>
    トレース情報を処理する時、Channelの代わりに別のQueueを使用します。

-   **queue_profile_size** <span class='type'>Int</span><br/>デフォルト値`512`<br/> 
    トレース情報を処理するためのバッファを設定します。

-   **queue_profile_process_thread_count** <span class='type'>Int</span><br/>デフォルト値`1`<br/> 
    トレース情報を処理するスレッド数を設定します。 スレッド数を増やすと、より迅速に処理できます。

-   **queue_text_enabled** <span class='type'>Boolean</span><br/>デフォルト値`false`<br/> 
    テキストデータ処理は、Channelの代わりにQueueを使用します。

-   **queue_text_size** <span class='type'>Int</span><br/>デフォルト値`512`<br/> 
    テキストデータを処理するためのバッファを設定します。
