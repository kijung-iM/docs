## カウント推移

ホーム画面 > プロジェクト選択 > **_分析_** > **_カウント推移_**

![カウント推移](/img/analysis-count.png)

**_カウントの推移_**メニューでは、次の情報を確認できます。

-   一日のデータベース運用の推移

-   特定の時間帯のコア指標

-   **_アクティブセッション_**

    画面の下の**_アクティブセッション_**タブを選択してください。現在進行中のセッションの情報（Query含む）を確認します。10秒以上進行しているセッションの場合は<span class='slow'>黄色</span>、20秒以上のセッションの場合は<span class='vslow'>赤色</span>として表現します。Queryのプラン情報を一緒に確認することができます。

-   **_ロックツリー_**

    画面の下の**_ロックツリー_**タブを選択してください。ロックに関する情報を確認できます。holderとwaiterの構造をツリー形式で表現して提供します。10秒以上進行しているセッションの場合は<span class='slow'>黄色</span>、20秒以上のセッションの場合は<span class='vslow'>赤色</span>として表現します。

-   **_プロセス情報_**

    画面の下の**_プロセス情報_**タブを選択してください。XOSエージェントをインストールした場合、データベースサーバーのプロセスに関する情報を確認できます。
