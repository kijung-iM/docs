WhaTapエージェントは、エージェントごとに必要な設定を_whatap.conf_ファイルに書き込みます。エージェントは、環境変数を使用して構成ファイルを見つけてロードします。コンテナやPaaS環境でNode.jsサーバーが動作する場合、環境変数を使用してエージェント設定を構成します。ユーザーの利便性を高めるため、[WhaTapモニタリングサービス](https://service.whatap.io)は、エージェント設定機能を用意しています。
