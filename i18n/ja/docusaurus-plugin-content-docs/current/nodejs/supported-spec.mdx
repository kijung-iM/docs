## Node.jsのサポート環境

エージェントをインストールする前に、Node.jsアプリケーションサーバーが以下の仕様を満たしていることを確認してください。

-   **Node.jsバージョン**：7.10.0バージョン以降

-   **オペレーティング システム**: Windows、Mac (OSX)、Linux、Ubuntu、Centos など、Node.Jsを実行できるすべての環境をサポートします。

-   **データベース**：MySQL、Microsoft SQL、MongoDB、Redisなど

-   **アプリケーションサーバー**：Node.js基本モジュールを使用するすべてのフレームワーク（Express、Hapi、Mean、Koa、Restifyなど）

:::note

WhaTapサーバーにデータを送信するには、6600番ポートが開かれている必要があります。

:::
