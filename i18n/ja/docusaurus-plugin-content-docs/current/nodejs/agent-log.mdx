---

アプリケーションサーバーの稼働中は、モニタリング情報を収集します。収集された情報はログファイルに保存されます。

-   **log_rotation_enabled** <span class='type'>Boolean</span>

    デフォルト値 `true`

    エージェントログファイルを日付別に保存する機能を活性化します。ログファイル名は_whatap-`yyyymmdd`.log_形式で保存します。値が`false`であればログファイル名は_whatap.log_形式で保存します。

-   **log_keep_days** <span class='type'>Int</span>

    デフォルト値`7`

    ログファイルの保存期間を設定します。`log_rotation_enabled`のオプション値が`true`の場合にのみ動作します。

-   **log_datasource_lookup_enabled** <span class='type'>Boolean</span>

    デフォルト値 `true`

    InitialContext Lookup時DataSourceの場合、ログを記録する機能を活性化します。
