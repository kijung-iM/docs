エージェントによって収集されたデータから統計を収集する機能に関連するオプションです。

-   **mtrace_enabled** <span class='type'>Boolean</span>

    デフォルト値`false`

    マルチトランザクション追跡機能（Multi Transaction ID、以下MTID）の使用有無を設定します。MTIDを追跡すると、登録したすべてのアプリケーション間の呼び出しを確認できます。

-   **stat_domain_enabled** <span class='type'>Boolean</span>

    デフォルト値`false`

    クライアントに接続中のドメイン別トランザクション統計収集機能をアクティブにします。

-   **stat_mtrace_enabled** <span class='type'>Boolean</span>

    デフォルト値`false`

    MSAアーキテクチャにおいてサービス間の呼び出し統計を活性化します。値が`true`の場合、外部呼び出しをするときに自分の情報を転送します。
