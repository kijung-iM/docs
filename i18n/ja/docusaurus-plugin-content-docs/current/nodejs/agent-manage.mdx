## エージェントのアップデート

アプリケーションを中断し、`npm`を使用してアップデートします。アップデートを進めてから、アプリケーションを再起動してください。

```bash title='SH'
$ npm update whatap
```

:::note

特定のバージョンにダウンロードしたい場合には、次のようにバージョンを示します。

```bash title='SH'
$ npm install whatap@{version}
```

:::

## エージェント削除（中止）

1.  ソース（例、app.js）に挿入したエージェントコードを削除してください。

    ```javascript title='app.js'
    var whatap=require('whatap').NodeAgent; // 削除
    ```

2.  アプリケーションを中断し、`npm`を利用してモジュールを削除してください。

    ```bash title='SH'
    $ npm uninstall whatap
    ```
