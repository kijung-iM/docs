Focusは、WhaTapモニタリングで提供しない任意のデータを時系列でアップロードするプログラムです。 

運用チームが直接開発したチェックスクリプトやデータベースクエリを周期的または一回限りで動作して収集された値をアップロードできます。アップロードされた値は、当該プロジェクトの分析・メトリクスチャートメニューで確認できます。また、統合メトリクスダッシュボードに追加できます。

Focusを使用するには、WhaTapプロジェクトの中でServer、Applicatoon（Java、PHP、NodeJS、Python）、Database、Kubernetesプロジェクトのいずれかを選択した後に、当該プロジェクトのプロジェクトアクセスキー文字列と収集サーバーIPを実行オプションとして指定して使用することができます。
