X86環境で動作するWindows、Linux、FreeBSD、OSXおよびARMv5、6、7およびARM64 LinuxOSに適用できます。 

| 製品    | 環境              | サポート環境               |
| ----- | --------------- | -------------------- |
| Focus | Windows         | Windows 2008R2以降     |
| ^     | Linux           | Debian 7.0以降         |
| ^     | ^               | Ubuntu 12.04以降       |
| ^     | ^               | CentOS、Red Hat 6.0以降 |
| ^     | ^               | Amazon Linux 1.0以降   |
| ^     | ^               | SUSE 12.1以降          |
| ^     | FreeBSD         | FreeBSD 10以降         |
| ^     | OSX             | Lion 10.7以降          |
| ^     | Raspberry Pi OS | 8以降                  |

:::note
対応一覧に含まれていない環境の場合は、support@whatap.ioまで別途お問い合わせください。
:::
