## 分析

収集した情報をもとに、特定の1日間の全体的な推移を分析できる機能を提供しています。異なる日付に対する推移データを比較してデータベースの性能推移を比較します。特定の時点で進行中だったセッションを把握し、遅延が発生したクエリ（Query）を分析することができます。

1日で発生したLockの推移を分析することができます。Lockを発生させたセッション（Lock Holder）とLockを待機しているセッション（Lock Waitier）に対してツリー構造で確認することができます。このような機能により、特定時点で発生したLock情報であるHolderとWaiterの関係を確認してクエリ（Query）を分析することができます。

1日以上の長期トレンド、収集したデータのTop Nクエリを分析できる機能を提供します。

## 統計/レポート

収集したSQL統計を通じてアプリケーション、データベース、ユーザー、ホスト単位で統計を出すことができます。毎時間保存したテーブルスペース及びデータベースサイズの時間別変化量を確認でき、毎日保存したテーブルサイズの変化量を感知することができます。

時間単位の性能指標の推移、一日中発生したSQL Top 10に関する日間報告書を閲覧することができます。週間報告書では、1日単位の性能指標の推移や先週の性能指標を比較し、1週間のSQL Top 10を確認することができます。月間レポートでは、日単位の性能指標の推移、月間に発生したSQL Top 10を確認します。

## 機能およびメニューの説明

WhaTapモニタリングサービスの初期画面でプロジェクトを選択して、**_プロジェクトメニュー_**の下位メニューを選択してください。 

-   **_プロジェクトメニュー_** > **_分析_**メニューでは次の機能が利用できます。
    -   **_カウント推移_**
    -   **_カウント推移比較_**
    -   **_ロックツリー_**
    -   **_デッドロック_**
    -   **_データベースパラメータ_**
    -   **_メトリクスチャート_**
    -   **_メトリクス異常検出_**
    -   **_ログファイルビューア_**
    -   **_ログファイルビューア（cloud）_**

-   **_プロジェクトメニュー_** > **_統計/レポート_**

      **_統計/レポート_**メニューでは、**週間・日間・月間レポート**と**統計**を提供します。
