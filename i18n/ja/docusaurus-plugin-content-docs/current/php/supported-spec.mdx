## PHPエージェント動作環境

WhaTap PHPモニタリングエージェントをインストールする前に、サポート環境を確認してください。

-   **対応バージョン**

    -   PHP extension moduleとして動作
    -   PHP 5.2バージョン以降、7.0バージョン以降、8.0 ~ 8.2バージョン
    -   NTS / ZTS (Zend Thread Safe)

-   **OS**

    -   CentOS／Red Hat 6.x（64bit）以降
    -   Debian／Ubuntu 12.04(64bit) 以降
    -   Alpine Linux（64bit）
    -   FreeBSD 10.x（64bit）以降

-   **データベース**

    -   Oracle (OCI8)
    -   MySQL
    -   MSSQL (mssql, sqlsrv)
    -   PostgreSQL (psql)
    -   Redis (phpredis)
    -   CUBRID
    -   Tibero (odbc)

-   **対応ライブラリ**

    -   PDO
    -   MySQLi
    -   cURL
    -   oci8
    -   mssql
    -   sqlsrv
    -   psql
    -   phpredis
    -   CUBRID
    -   odbc

-   **アプリケーションサーバー**

    -   Apache
    -   PHP-FPM (Nginx)
