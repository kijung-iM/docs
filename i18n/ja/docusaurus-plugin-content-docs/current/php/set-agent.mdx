PHPエージェントは、必要な設定を_whatap.ini_ファイルに作成して変更することができます。ユーザーの利便性を高めるため、[WhaTapモニタリングサービス](https://service.whatap.io)でもエージェント設定機能を提供しています。

:::note

一般的に設定はランタイムにすぐreloadされます。ただし、該当設定のメカニズム上、ApacheまたはPHP-FPMが起動する時に適用されるオプションは、再起動しなければならない場合があります。

:::
