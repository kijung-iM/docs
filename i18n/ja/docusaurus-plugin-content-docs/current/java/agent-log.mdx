アプリケーションサーバーの稼働中は、モニタリング情報を収集します。収集された情報はログファイルに保存されます。

## パスおよび名前設定

エージェントログのパスと名前は、_whatap.conf_ファイルで設定できます。`log_root`のデフォルト値として_${WHATAP_HOME}/logs_パスを使用します。

```ini title='whatap.conf'
log_root=System.getProperty("whatap.home", ".") + "/logs"
log_name=whatap
```

## ログ管理オプション

_whatap.conf_ファイルにログ関連オプションを作成し、ログ内容を管理することができます。

```ini title='whatap.conf'
watchlog_enabled=true
log_rotation_enabled=true
log_oname_enabled=true
log_keep_days=7
```
