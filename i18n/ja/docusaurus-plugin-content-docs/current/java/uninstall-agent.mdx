## エージェント削除

アプリケーションサーバーでJVMオプションの`-javaagent`設定を削除し、サーバーを再起動してください。 以後、`$WHATAP_HOME`環境変数を削除してください。

## エージェントロールバック

1.  [WhaTapモニタリングサービス](https://service.whatap.io)でロールバックしたいプロジェクトを選択してください。

2.  左側のメニューで**_管理_** > **_エージェントアップデート_**を選択してください。

3.  **_他のバージョン_**ボタンを選択してください。

    ![管理 - エージェントロールバック1](/img/agent_rollback_1.png)

4.  ロールバックしたいバージョンを選択し**_選択されたバージョン_**または **_選択されたバージョンですべてのエージェントにダウンロード_**ボタンをクリックしてください。

    ![管理 - エージェントロールバック2](/img/agent_rollback_2.png)

5.  エージェントをインストールする際、修正したスクリプトから`WHATAP_JAR`環境変数をロールバックするバージョンに変更してください。

    ![管理 - エージェントロールバックJVM修正](/img/agent_rollback_3.png)
