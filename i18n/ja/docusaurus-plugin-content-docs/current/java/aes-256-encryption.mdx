id: aes-256-encryption
title: AES256暗号化
description: WhaTapエージェントから収集した情報を暗号化できます。
tags:
  - Java
  - AES256
---

WhaTapアプリケーション エージェントは、収集したデータを暗号化してサーバーに転送します。このため、XOR（Exclusive-OR）演算と高度暗号化標準（Advanced Encryption Standard、以下AES）暗号化を使用し、平文を128bit単位に分けて暗復号化を行います。256ビットまで拡張してセキュリティを強化できます。

基本的にJava Cryptography Extension（以下JCE）は128ビットに対応してします。AES256ビットを適用するには、JCEをアップデートする必要があります。

基本環境でAES256を適用すると、次のようにエラーが発生します。

```bash
Unsupported keysize or algorithm parameters.
##あるいは、
Illegal key size or default parameters.
```

1.  次のリンクからバージョンに合ったファイルをダウンロードしてください。

    :::note

    **Javaバージョン別のJCEダウンロードリンク**

    -   [JAVA 8](https://www.oracle.com/technetwork/java/javase/downloads/jce8-download-2133166.html)
    -   [JAVA 7](http://www.oracle.com/technetwork/java/javase/downloads/jce-7-download-432124.html)
    -   [JAVA 6](http://www.oracle.com/technetwork/java/javase/downloads/jce-6-download-429243.html)
    -   [JAVA 5](http://www.oracle.com/technetwork/java/javasebusiness/downloads/java-archive-downloads-java-plat-419418.html#jce_policy-1.5.0-oth-JPR)
    -   [JAVA 1.42](http://www.oracle.com/technetwork/java/javasebusiness/downloads/java-archive-downloads-java-plat-419418.html#7503-jce-1.4.2-oth-JPR)

    :::

2.  _`$JAVA_HOME`/jre/lib/security_パスにファイルを上書きしてください。

3.  CEを適用した場合は、_`${WHATAP_HOME}`/whatap.conf_ファイルに次の設定を追加してください。

    ```ini title='whatap.conf'
    cypher_level=256
    ```

4.  アプリケーションサーバーを再起動すると、ASE256の暗号化が適用されます。
