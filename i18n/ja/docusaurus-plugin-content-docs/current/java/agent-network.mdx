{@include: ../common-items/_common-network.mdx}

-   **net_send_queue1_size** <span class='type'>Int</span>

    デフォルト値`256`

    トレース情報とアクティブスタックを除く残りのデータ転送に使用するQueueのサイズを設定します。設定したサイズが10以下の場合は10に設定します。

-   **net_send_queue2_size** <span class='type'>Int</span>

    デフォルト値 `512`

    トレース情報とアクティブスタックデータの転送に使用するQueueのサイズを設定します。設定したサイズが10以下の場合は10に設定します。
