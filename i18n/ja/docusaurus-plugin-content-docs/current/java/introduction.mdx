Web Application Server（WAS）、バッチアプリケーションなど、JVM環境で動作するすべてのアプリケーションに適用できます。アプリケーションの障害分析のためにThread Dump（スレッドダンプ）を確保する手間を省いて、すでに確保されたスタックを通じて障害状況を再現しなくても問題の原因を正確に特定できます。
