エージェントをインストールした後の確認事項です。もしインストールに問題がある場合は、[次の文書](agent-troubleshooting)を参考にしてください。

## WAS再起動

アプリケーションサーバーを再起動してください。駆動ログに次のようなWhaTapキャラクターログが表示されたら、`JAVA_OPTS`設定が完了した状態です。
