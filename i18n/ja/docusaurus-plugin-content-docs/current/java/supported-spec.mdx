エージェントをインストールする前に、JVMアプリケーションサーバーが次の仕様を満たしていることを確認してください。

Web Application Server（WAS）だけでなく、デーモンやバッチ アプリケーションなど、JVMで動作するすべてのアプリケーションに適用できます。Java 1.5以下は[support@whatap.io](mailto:support@whatap.io)に問い合わせるか[次の文書]](add-jvm-opt/java-under-ver)を参照してください。

## JVM

| サポート範囲          | 環境   | OS             | JVMバージョン          |
| --------------- | ---- | -------------- | ----------------- |
| Fully support   | Java | JVMで起動するすべてのOS | Java6以降, Java17以下 |
| Limited support | Java | JVMで起動するすべてのOS | Java1.5以下         |

## Web Application Server (WAS)

Tomcat、JBoss、JEUS、WebLogic、WebSphereなどJava EE Specを準拠するすべてのApplication Server、[次のリンク](https://www.oracle.com/technetwork/java/javaee/overview/compatibility-jsp-136984.html)を参照ください。

## Framework / Library

-   Spring Boot Project
-   Netty, Akka HTTPおよびPlay Frameworkなどの非同期Framework
-   その他、Java EE Specに準拠するApplication Serverで動作するすべてのライブラリ

## DataSource

-   Generic JDBC (any JDBC compliant driver)
-   DB2 JDBC
-   Derby JDBC
-   H2 JDBC
-   jTDS JDBC
-   MariaDB JDBC
-   Microsoft SQL Server JDBC
-   MySQL mysql-connector-java
-   Oracle ojdbc14, ojdbc5, ojdbc6, ojdbc7, ojdbc8
-   Postgres JDBC
-   Tibero
-   Jedis Redis driver
