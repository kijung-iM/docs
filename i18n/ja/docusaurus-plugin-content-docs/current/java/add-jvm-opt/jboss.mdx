JBossを使用するアプリケーションサーバ環境でJVMオプションを追加する方法を確認してください。

## 環境変数の追加

JVMオプションに`-javaagent`および`-Djboss.modules.system.pkgs`オプションを追加してください。

### JBoss7.0以降、EAP6.0以降

設定位置：（JBossインストールパス）/bin/standalone.conf(domain.conf)

```bash title=standalone.sh
#!/bin/sh
########## WHATAP ############
WHATAP_HOME=/home/ec2-user/whatap
WHATAP_JAR=`ls ${WHATAP_HOME}/whatap.agent-*.jar | sort | tail -1`
JAVA_OPTS="${JAVA_OPTS} -javaagent:${WHATAP_JAR} -Djboss.modules.system.pkgs=whatap"
########## WHATAP ############
```

:::note

JBoss5.0バージョン以下は_(JBossインストールパス)/bin/run.conf_パスで設定してください。エラーが発生すると`-Djboss.platform.mbeanserver=true`オプションを追加してください。

:::
