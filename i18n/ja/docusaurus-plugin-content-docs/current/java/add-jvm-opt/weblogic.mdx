WebLogicを使用するアプリケーションサーバ環境でJVMオプションを追加する方法を確認してください。 

使用するオペレーティング システムを確認してから、設定を完了してください。

<Tabs>
<TabItem value="windows" label="Windows" default>

```batch title="{WebLogic 설치 경로}/user_projects/domains/{사용자도메인}/bin/startWebLogic.bat"
rem ########## WHATAP START ############
set WHATAP_HOME=C:\whatap
for /f %%f in ('dir /b /on "%WHATAP_HOME%\whatap.agent-X.Y.Z.jar"') do set last=%%f
set WHATAP_JAR=%last%
set WHATAP_OPTS=-javaagent:%WHATAP_HOME%\%WHATAP_JAR%

if "x%JAVA_OPTS%"=="x" goto setWhatap

set JAVA_OPTS_TMP=%JAVA_OPTS:"=%
if not "x%JAVA_OPTS_TMP:whatap=%"=="x%JAVA_OPTS_TMP%" goto endWhatap

:setWhatap
set JAVA_OPTS=%JAVA_OPTS% %WHATAP_OPTS%

:endWhatap
rem ########## WHATAP END ############
```

</TabItem>
<TabItem value="mac" label="MAC">

```bash title="{WebLogic 설치 경로}/user_projects/domains/{사용자도메인}/bin/startWebLogic.sh"
########## WHATAP START ############
WHATAP_HOME=/whatap
WHATAP_JAR=`ls ${WHATAP_HOME}/whatap.agent-*.jar | sort | tail -1`
JAVA_OPTS="${JAVA_OPTS} -javaagent:${WHATAP_JAR} "
########## WHATAP END ############
```

</TabItem>
