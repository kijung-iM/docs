Tomcatを使用するアプリケーションサーバ環境でJVMオプションを追加する方法を確認してください。使用するオペレーティングシステムに合わせて設定してください。

## 環境変数の追加

### **Linux**

_catalina.sh_ファイルの上段に`JAVA_OPTS`環境変数を追加してください。

```bash title='{Tomcat 설치 경로}/bin/catalina.sh' showLineNumbers
########## WHATAP START ############
WHATAP_HOME=/whatap
WHATAP_JAR=`ls ${WHATAP_HOME}/whatap.agent-*.jar | sort | tail -1`
JAVA_OPTS="${JAVA_OPTS} -javaagent:${WHATAP_JAR} "
########## WHATAP END ############
```

:::note
`ls ${WHATAP_HOME}/whatap.agent-*.jar|sort|tail-1`項目は、今後エージェントアップデートを進める場合、最新のWhaTapエージェントを適用するためです。
:::

### **Windows**

-   _setup.bat_ファイルを通じて実行する場合

    _catalina.bat_ファイルの上段に`JAVA_OPT`環境変数を追加してください。

    ```batch title='{Tomcat 설치 경로}/bin/catalina.bat' {3} showLineNumbers
    rem ########## WHATAP START ############
    set WHATAP_HOME=C:\whatap
    for /f %%f in ('dir /b /on "%WHATAP_HOME%\whatap.agent-*.jar"') do set last=%%f
    set WHATAP_JAR=%last%
    set WHATAP_OPTS=-javaagent:%WHATAP_HOME%\%WHATAP_JAR%

    if "x%JAVA_OPTS%"=="x" goto setWhatap

    set JAVA_OPTS_TMP=%JAVA_OPTS:"=%
    if not "x%JAVA_OPTS_TMP:whatap=%"=="x%JAVA_OPTS_TMP%" goto endWhatap

    :setWhatap
    set JAVA_OPTS=%JAVA_OPTS% %WHATAP_OPTS%

    :endWhatap
    rem ########## WHATAP END ############
    ```

-   TomcatをWindowsインストーラーでインストールした場合

    1.  **_開始_** > **_Apache Tomcat `X.Y.Z.`_** > **_Configure Tomcat_**を選択してください。

    2.  **_Java_**タブを選択してください。

    3.  **_Java Options_**テキストボックスに`-javaagent`オプションを追加してください。

    ```text
    -javaagent:{WhaTapインストールパス}/whatap.agent-X.Y.Z.jar
    ```
