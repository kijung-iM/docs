## 分析

収集された情報に基づいて、特定の日の全体的な傾向を分析する機能を提供します。異なる日の傾向データを比較して、データベースの性能の傾向を比較します。特定の時点で進行中のセッションを特定し、低速クエリ(Query)を分析できます。

日中に発生したLockの傾向を分析することができます。Lockの原因となったセッション(Lock Holder)とLockを待機しているセッション(Lock Waitier)をツリー構造で確認できます。この機能を使用すると、特定の時点で発生したLock情報のHolderとWaiterの関係を確認することでクエエイ(Query)を分析できます。

1日以上の長期トレンド、収集したデータのTop Nクエリを分析できる機能を提供します。

## 統計/レポート

収集したSQL統計を使用して、アプリケーション、データベース、ユーザー、ホスト単位で統計を生成できます。保存したテーブルスペース及びデータベースサイズの時間ごとの変化を確認し、毎日保存されるテーブルサイズの変化を検出できます。

時間単位の性能指標の傾向と１日を通して発生したSQL Top 10の日次レポートを表示できます。週次レポートでは、毎日の性能指標の傾向と前週の性能指標を比較でき、その週のSQL Top 10を確認できます。月次レポートでは、毎日の性能指標の傾向と1か月間に発生したSQLイベントの Top 10を特定します。

## 機能およびメニューの説明

WhaTapモニタリングサービスの初期画面でプロジェクトを選択して、**_プロジェクトメニュー_**のサブメニューを選択します。 

-   **_プロジェクトメニュー_** > **_分析_**メニューでは次の機能を使用できます。
    -   **_カウント推移_**
    -   **_カウント推移比較_**
    -   **_ロックツリー_**
    -   **_デッドロック_**
    -   **_データベースパラメータ_**
    -   **_メトリクスチャート_**
    -   **_メトリクス異常検出_**
    -   **_ログファイルビューア_**
    -   **_ログファイルビューア (cloud)_**

-   **_プロジェクトメニュー_** > **_統計/レポート_**

      **_統計/レポート_**メニューには、**週次・日次・月次レポート**と**統計**が表示されます。
