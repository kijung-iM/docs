## Golangエージェントの動作環境

-   **Golangバージョン**：1.14バージョン以降

-   **OS**

    -   CentOS/Redhat 6.xバージョン(64bit) 以降 
    -   Debian/Ubuntu 12.04バージョン(64bit) 以降
    -   Amazon linux (64bit)
    -   Alpine linux（64bit）

-   **ライブラリの互換性**

    -   net/http, gin, chi, fiber, gorilla, echo, fasthttp
    -   database/sql, gorm, redigo
    -   grpc, sarama, k8s.io/client-go

:::note

フレームワークまたはライブラリの適用方法および例は、[GitHubサイト](https://github.com/whatap/go-api-example)を参照してください。

:::
