## WhaTapデータベースエージェントのダウンロード

{@include: ../common-items/_db-install-agent-steps.mdx}

1.  圧縮して解凍したフォルダに入り、_whatap.conf_ファイルを確認してください。_whatap.conf_ファイルにプロジェクトのアクセスキー情報とWhaTapサーバー情報、DB接続情報を入力します。

```ini title='whatap.conf'
license=プロジェクトアクセスキー
whatap.server.host=13.124.11.223/13.209.172.35 // WhaTapサーバー情報

dbms=redis
db_ip={DBサーバーのIPアドレス}
db_port={DBサーバーのポート番号}
```

1.  次のディレクトリにJDBCドライバーをダウンロードします。 _解凍されたフォルダ/jdbc_

  データベースサーバーのオペレーティングシステムとバージョン用のJDBCドライバーをダウンロードしてください。https://www.cdata.com/drivers/redis/jdbc/
