WhaTapモニタリングサービスを利用するには、会員登録を行ってください。

:::tip

<InDoc product='quick-guide'>

-   WhaTapモニタリングを15日間無料で体験できます。支払い方式の詳細については、[次の文書](management/billing)を参照ください。 

</InDoc>
<Xclude product='quick-guide'>

-   WhaTapモニタリングを15日間無料で体験できます。支払い方式の詳細については、[次の文書](../management/billing)を参照ください。 

</Xclude>

-   会員登録してプロジェクト作成後、プロジェクトアクセスキーを発行した時点から無料トライアルが利用できます。
-   体験期間終了後にサービスを続けて利用したい場合は、決済情報を登録してプロジェクトを有料に切り替えてください。

:::

1.  [WhaTapホームページ](http://www.whatap.io/) へ移動します。
2.  右上の**_[無料で始める](https://www.whatap.io/ko/signup/)_**ボタンをクリックします。
3.  名前、メール、パスワード、会社名を入力します。
4.  **_アカウント作成_**ボタンをクリックします。

:::note

通知をSMSで受信するには、ログインして右上のプロファイルアイコン > **_アカウント情報管理_**の順に選択して **_ユーザーの電話番号_**セクションに移動し、電話番号を入力します。

:::
