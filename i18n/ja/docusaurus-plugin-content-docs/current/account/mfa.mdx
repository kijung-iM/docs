**マルチファクター認証(Multi-factor authentication, MFA)**とは、2回以上の認証手続きを経てユーザーを認証する方式です。 マルチファクター認証を使用して、アカウントとサービスに対するセキュリティを強化できます。

## マルチファクタ認証を設定する

WhaTapモニタリングサービスはマルチファクター認証方式で**_OTP_**方式と**_電子メールトークン_**方式を提供します。 OTP方式を推奨します。

:::note

**_OTP_**とは？ **One Time Password**の略語で固定されたパスワードではなく、ランダムに作成された使い捨てパスワードによる認証方式を意味します。

:::

次の手順に従って**_アカウント情報管理_**ページに移動し、マルチファクター認証を設定することができます。

1.  [WhaTapモニタリングサービス](https://service.whatap.io)にログインしてください。

2.  画面右上のプロファイル アイコンを選択してください。ドロップダウンメニューが表示されます。

3.  **_アカウント情報管理_**を選択してください。

4.  ページを下にスクロールして、**_マルチファクター認証_**セクションに移動してください。

    ![マルチファクター認証](/img/account-mfa-section.png)

**_[OTP](#otp)_**または**_[メールトークン](#email)_**の中から好きな認証方式を選択してください。

## OTP認証{#otp}

マルチファクター認証方式をOTPとして選択した場合は、モバイル端末にOTPアプリがインストールされてある必要があります。

-   Android：[Google OTP](https://play.google.com/store/apps/details?id=com.google.android.apps.authenticator2)
-   iOS：[Google Authenticator](https://apps.apple.com/app/google-authenticator/id388497605)

### OTPによるのマルチファクター認証設定

![OTP 認証選択](/img/account-mfa-otp.png)

1.  マルチファクター認証方式で**_OTP_**を選択し、**_OTPキー（再）作成_**を選択してください。

2.  **_QRコード_**または**_OTPキー_**を選択した後、モバイル機器のOTPアプリでキーを登録してください。

    -   QRコードを使用する際は**_Scan a QR code_**を選択し、モバイル機器でQRコードをスキャンしてください。 (便宜上、QRコード方式のご使用をお勧めします。)

    -   OTPキーを使用する際は、**_Enter a setup key_**を選択し、モバイル端末でユーザーアカウントとOTP キーを入力してください。

    ![Google OTPアプリ](/img/account-mfa-google.png)

3.  発行されたOTPトークンを**_アカウント情報管理_**ページの**_認証_**テキストボックスに入力してください。 

4.  **_OTPトークン検証_**を選択してください。

OTPベースのマルチファクタ設定を完了しました。 マルチファクター認証設定を確認するには、ログアウトした状態で再度ログインしてください。

### OTPによるマルチファクター認証を使用する

1.  [ログイン](https://service.whatap.io/account/login)ページからログインしてください。

2.  マルチファクター認証画面が表示されたら、モバイル端末のOTPアプリでOTPキーを確認してください。

3.  発行されたOTPキーを入力した後、**OTPトークン検証**を選択してください。

OTPトークン検証に成功すると、サービス画面にアクセスできます。

## メールトークン認証{#email}

### メールトークンによるのマルチファクター認証の設定

![メールトークン認証選択](/img/account-mfa-email.png)

1.  マルチファクター認証方式で**_Eメール_**を選択し、**_メールで認証トークンを転送_**を選択してください。

2.  メールでno-reply@whatap.ioから受信した認証トークン(Verification Code)を確認してください。 メールを受信できなかった場合は、迷惑メールボックスを確認してください。

3.  受信した認証トークンを**_アカウント情報管理_**ページの**_認証_**テキストボックスに入力してください。 

4.  **メールトークン検証**を選択してください。

メールベースのマルチファクター設定を完了しました。マルチファクター認証設定を確認するには、ログアウト後に再度ログインしてください。

### メールトークンによるマルチファクター認証を使用

1.  [ログイン](https://service.whatap.io/account/login)ページからログインしてください。

2.  マルチファクター認証画面が表示されたら、**メールで認証トークンを送信 **を選択してください。

3.  メールで受信した認証トークンを入力し、 **メールトークン検証 **を選択してください。

メール認証トークンの検証に成功すると、サービス画面にアクセスできます。

## MFAをクリアする

![マルチファクター認証取り消し](/img/account-mfa-disable.png)

マルチファクター認証をアクティブにした場合にのみ、マルチファクター認証設定を解除できます。 **_[アカウント情報管理](https://service.whatap.io/v2/account/management)_**ページの**_マルチファクター認証_**セクションに移動し、**_マルチファクター認証取り消し_**または**_OTPキー削除_**を選択してください。
