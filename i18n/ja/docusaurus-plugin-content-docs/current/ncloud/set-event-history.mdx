ホーム画面 > プロジェクト選択 > **_アラート通知_** > **_イベント記録_**

警告通知の履歴を確認できます。過去1年間の履歴を表示できます。各項目を設定し、![表示アイコン](/img/ico-btn-search.png)ボタンをクリックします。

![イベント履歴](/img/event-history-aws.png)

:::note

選択したプロジェクトによって、画面イメージが異なる場合があります。

:::

-   **_時間選択_**

    -   右側の緑色のボタンを選択して表示時間を選択することができます。
    -   ![左アイコン](/img/left-arrow.svg)または ![右アイコン](/img/right-arrow.svg)ボタンを選択すると、選択した表示時間だけ間隔を移動します。
    -   詳細な時刻を選択するには、日付または時間の領域を選択してください。詳細な時間を設定し、**_確認_**ボタンを選択します。

-   **_フィルター_**：タイトルの内容に基づいてイベント履歴をフィルタリングできます。

-   **_すべて選択_**：プロジェクトに含まれるエージェントを選択できます。

-   **_CSV_**：閲覧したイベント履歴結果をcsvファイルとして保存することができます。 **_CSV_** ボタンをクリックして **_最大CSV行数_**を入力し **_ダウンロード_** ボタンを選択します。

-   **_カラム選択_**：取得されたイベント履歴結果の列項目を追加することができます。

-   **_イベント設定_**：**_警告通知_** > **_イベント設定_**メニューに移動します。
