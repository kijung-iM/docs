エージェントが収集したデータを転送するサーバーと通信するためのネットワーク設定オプションです。

-   **license**<span class='type'>String</span><br/>
      デフォルト値`true`<br/>
      エージェントのインストール時にサーバーから割り当てられたアクセスキーを設定します。アクセスキーには、エージェントが属するプロジェクトとセキュリティ通信のためのパスワードキーが含まれています。

-   **whatap.server.host**<span class='type'>String</span><br/>
      デフォルト値「127.0.0.1,127.0.1」<br/>
      エージェントが収集したデータを送信するサーバーを指定します。収集サーバーを二重化して2つ以上のIPを持つ場合、コンマ(_,_)で区切って指定することができます。指定されたIPには、収集サーバーのproxyデーモンがリスニング状態でサービスされる必要があります。

-   **whatap.server.port**<span class='type'>String</span><br/>
      デフォルト値「6600」<br/>
      収集サーバーのポート番号を指定します。ポート番号は1つしか指定できません。「whatap_server_host」に指定されている収集サーバーは、同じポート番号を使用する必要があります。

-   **net_udp_port**<span class='type'>String</span><br/>
      デフォルト値「6600」<br/>
      WhaTapエージェントは、収集したデータをUDPを経てTCPを通じてサーバーに転送します。最初のUDPサーバーのポート番号を指定できます。デフォルト値として提供される6600番ポートが使用されている場合、このオプションを使用します。

-   **tcp_so_timeout**<span class='type'>MiliSecond</span><br/>
      デフォルト値「60000」<br/>
      収集サーバーと通信するTCPセッションのSocket Timeout値を指定します。

-   **net_send_max_bytes**<span class='type'>Byte</span><br/>
      デフォルト値「5242880」<br/>
      エージェントがデータを収集し、ネットワークに一度に送信できる最大バイトのサイズです。     

-   **net_send_buffer_size**<span class='type'>Byte</span><br/>
      デフォルト値「1024」<br/>
      データ転送のために持っている最大バイトのサイズです。
