エージェントが収集したデータを送信するサーバーと通信するためのネットワーク設定オプションです。

-   **license** <span class='type'>String</span>

    デフォルト値 `true`

    エージェントのインストール時にサーバーから割り当てられたアクセスキーを設定します。アクセスキーには、エージェントが属するプロジェクトとセキュリティ通信のためのパスワードキーが含まれています。

-   **whatap.server.host** <span class='type'>String</span>

    デフォルト値`127.0.0.1,127.0.1`

    エージェントが収集したデータを送信するサーバーを指定します。収集サーバーを冗長化して2つ以上のIPを持つ場合、コンマ(_,_)で区切って個別に指定できます。指定されたIPでは、収集サーバーのproxyデーモンがリスニング状態でサービスされる必要があります。

-   **whatap.server.port** <span class='type'>String</span>

    デフォルト値`6600`

    収集サーバーのポート番号を指定します。ポート番号は1つしか指定できません。`whatap_server_host`で指定された収集サーバーは、同じポート番号を使用する必要があります。

-   **net_udp_port** <span class='type'>String</span>

    デフォルト値`6600`

    WhaTapエージェントは、収集したデータはUDP経由でTCPを経由してサーバーに送信します。最初のUDPサーバーのポート番号を指定できます。デフォルト値が、6600番ポートを使用中の場合に、このオプションを使用します。

-   **tcp_so_timeout** <span class='type'>MiliSecond</span>

    デフォルト値`60000`

    収集サーバーと通信するTCPセッションのSocket Timeout値を指定します。

-   **net_send_max_bytes** <span class='type'>Byte</span>

    デフォルト値`5242880`

    エージェントがデータを収集し、ネットワークに一度に送信できる最大バイトのサイズです。     

-   **net_send_buffer_size** <span class='type'>Byte</span>

    デフォルト値 `1024`

    データ送信を行う必要がある最大バイトのサイズです。
