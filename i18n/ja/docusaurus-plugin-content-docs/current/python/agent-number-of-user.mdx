Pythonアプリケーションサーバーに接続したユーザーの数を集計します。

-   **trace_user_enabled(\*)**<span class='type'>Boolean</span><br/>
      デフォルト値`true`<br/>
      リアルタイムでのユーザーの追跡有無を決定します。この値を有効にすると、デフォルト値はIPになります。

-   **trace_user_using_ip(\*)**<span class='type'>Boolean</span><br/>
      デフォルト値`true`<br/>
      リアルタイムユーザーの区分をIPにする場合に設定します。このデータは、Real Time Userで使用できます。`user_header_ticket`オプションは排他的に設定を同時に適用することはできません。
