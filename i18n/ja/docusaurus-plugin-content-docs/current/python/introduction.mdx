WhaTap Pythonアプリケーションモニタリングは、Pythonベースのウェブアプリケーションサーバーモニタリングサービスを提供します。Web framework、Server、Batchを同時にモニタリングする韓国唯一の製品です。

主要ウェブフレームワークのアプリケーションに適用でき、DB Query、外部呼び出しを追跡してどこで呼び出しが遅れているかを検出することができます。httpまたはhttpsではない特定のトランザクションに対してデータ追跡をしたい場合、特定のメソッドのモニタリングが可能です。
