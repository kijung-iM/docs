:::note

-   WhaTapモニタリンサービスでは、モニタリング対象をオブジェクトまたはエージェントとして混在して使用できます。

:::

-   **object_name** <span class='type'>String</span>

    デフォルト値`{type}-{ip2}-{ip3}-{process}`

    アプリケーションを識別するエージェント名(ONAME) システムです。ONAMEに基づいてOIDが生成されます。 

    | 名称         | 説明                                    |
    | ---------- | ------------------------------------- |
    | {type}     | whatap.app_nameに設定された値を使用します。         |
    | {ip#}      | IPを分割するときは、#番目の桁を使用します。               |
    | {process}  | whatap.app_process_nameに設定された値を使用します。 |
    | {hostname} | サーバーのホスト名を使用します。                      |

-   **app_name** <span class='type'>String</span>

    アプリケーションを識別するエージェント名(ONAME) 体系で使用されるアプリケーション名です。`object_name`の`{type}`に対応する値です。

-   **app_process_name** <span class='type'>String</span>

    アプリケーションを識別するエージェント名(ONAME)体系で使用されるアプリケーションプロセス名です。 

    アプリケーションサーバーのCPU、Heap Memoryなどを収集する対象プロセスを設定します。`object_name`の`{process}`に対応する値です。
