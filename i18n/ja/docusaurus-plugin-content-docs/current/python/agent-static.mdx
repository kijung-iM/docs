エージェントによって収集されたデータから統計を収集する機能に関連するオプションです。

-   **stat_domain_enabled**<span class='type'>Boolean</span><br/>
      デフォルト値「false」<br/>
      ドメイン別のトランザクション統計を収集します。

-   **stat_domain_max_count**<span class='type'>Int</span><br/>
      デフォルト値「7000」<br/>
      5分間アプリケーションで収集するドメイン別のトランザクション統計の最大レコード数です。

-   **stat_mtrace_enabled**<span class='type'>Boolean</span><br/>
      デフォルト値「false」<br/>
      マルチサーバートランザクションにおいてCallser&Callee間のバージョンごとの依存性統計を収集します。

-   **stat_mtrace_max_count**<span class='type'>Int</span><br/>
      基本値 「7000」<br/>
      マルチサーバートランザクションにおいてCallser&Callee間のバージョン別の依存性統計の最大レコード数です。  

-   **stat_login_enabled**<span class='type'>Boolean</span><br/>
      デフォルト値「false」<br/>
      ログイン別のトランザクション統計を収集します。

-   **stat_login_max_count**<span class='type'>Int</span><br/>
      デフォルト値「7000」<br/>
      5分間アプリケーションで収集するログイン別のトランザクション統計の最大レコード数です。

-   **stat_referer_enabled**<span class='type'>Boolean</span><br/>
      デフォルト値「false」<br/>
      Referer別のトランザクション統計を収集します。

-   **stat_referer_max_count**<span class='type'>Int</span><br/>
      デフォルト値「7000」<br/>
      5分間アプリケーションで収集するReferer別のトランザクション統計の最大レコード数です。

-   **stat_tx_max_count**<span class='type'>Int</span><br/>
      デフォルト値「5000」<br/>
      トランザクション統計情報の数を制限します。 5分間収集され、サーバーに送信される統計情報で最大レコード数を制限します。

-   **stat_httpc_max_count**<span class='type'>Int</span><br/>
      デフォルト値「5000」<br/>
      Http Call統計情報の個数を制限します。5分間収集され、サーバーに送信される統計情報で最大レコード数を制限します。

-   **stat_error_max_count**<span class='type'>Int</span><br/>
      デフォルト値「1000」<br/>
      Error統計情報の個数を制限します。5分間収集され、サーバーに送信される統計情報で最大レコード数を制限します。

-   **stat_useragent_max_count**<span class='type'>Int</span><br/>
      デフォルト値「500」<br/>
      User Agent統計情報の個数を制限します。5分間収集され、サーバーに送信される統計情報で最大レコード数を制限します。
