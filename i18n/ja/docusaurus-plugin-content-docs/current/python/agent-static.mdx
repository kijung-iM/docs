エージェントによって収集されたデータから統計を収集する機能に関連するオプションです。

-   **stat_domain_enabled** <span class='type'>Boolean</span>

    デフォルト値`false`

    ドメインごとにトランザクション統計を収集します。

-   **stat_domain_max_count** <span class='type'>Int</span>

    デフォルト値`7000`

    5分間にアプリケーションによって収集されるドメインごとのトランザクション統計の最大レコード数です。

-   **stat_mtrace_enabled** <span class='type'>Boolean</span>

    デフォルト値`false`

    マルチサーバートランザクションにおいてCaller&Callee間でバージョンごとの依存関係の統計を収集します。

-   **stat_mtrace_max_count** <span class='type'>Int</span>

    デフォルト値`7000`

    マルチサーバトランザクションでCaller&Calleeの間のバージョンごとの依存関係の統計における最大レコード数です。

-   **stat_login_enabled** <span class='type'>Boolean</span>

    デフォルト値`false`

    ログインごとのトランザクション統計を収集します。

-   **stat_login_max_count** <span class='type'>Int</span>

    デフォルト値`7000`

    5分間にアプリケーションで収集するログインごとのトランザクション統計の最大レコード数です。

-   **stat_referer_enabled** <span class='type'>Boolean</span>

    デフォルト値`false`

    Referer別のトランザクション統計を収集します。

-   **stat_referer_max_count** <span class='type'>Int</span>

    デフォルト値`7000`

    5分間にアプリケーションで収集するRefererごとのトランザクション統計の最大レコード数です。

-   **stat_tx_max_count** <span class='type'>Int</span>

    デフォルト値`5000`

    トランザクション統計情報の数を制限します。 5分間に収集され、サーバーに送信される統計情報で最大レコード数を制限します。

-   **stat_httpc_max_count** <span class='type'>Int</span>

    デフォルト値`5000`

    Http Call統計情報の数を制限します。5分間に収集され、サーバーに送信される統計情報で最大レコード数を制限します。

-   **stat_error_max_count** <span class='type'>Int</span>

    デフォルト値 `1000`

    Error統計情報の数を制限します。5分間に収集され、サーバーに送信される統計情報で最大レコード数を制限します。

-   **stat_useragent_max_count** <span class='type'>Int</span>

    デフォルト値`500`

    User Agent統計情報の数を制限します。5分間に収集され、サーバーに送信される統計情報で最大レコード数を制限します。
