WhaTapのログモニタリングサービスのユーザーからのよくある質問を確認してください。 

* * *

<div class ='faq'>

### ログ原文復元

| Q. ログを収集した後、原文を復元できますか？ |
| ----------------------- |

> A. ログ原文復元機能を提供していません。 現在のログ収集は、セキュリティ目的ではなく分析目的で提供されています。
>
> :::note
>
> **インストール刑**の場合、ログ原文復元ツールを別途提供します。
>
> :::

<br/>

### ログのバックアップと復旧

| Q. ログをバックアップして復旧できますか？ |
| ---------------------- |

> A. 収集されたログは日付単位で保存されます。 過去日付のファイルをバックアップした場合は、そのフォルダに復旧できます。 

<br/>

### ログデータの削除周期

| Q. ログデータの削除サイクルはどうなりますか？ |
| ------------------------ |

> A. **_ログ設定_**メニューから設定した**_データ保持期間_**と同じです。 ただし、記録されたログの保存期間は時間単位であるため、設定した**_データ保持期間_**に追加で1時間までデータが残っている場合があります。
>
> :::note
>
> **_データ保持期間_**を設定していない場合、デフォルト値は1日です。 **_データ保持期間_**設定の詳細については、[次の文書](../log/log-setting#log-data-setting)を参照してください。
>
> :::

<br/>

### エージェントのログ転送周期

| Q. エージェントからのログ転送サイクルはどうなりますか？ |
| ----------------------------- |

> A. エージェントのログバッファ容量(64 KB)に到達したり、転送周期(2秒)に達した場合は、ログを転送します。 ネットワーク伝送時に_zip_形式で圧縮して転送し、収集サーバーから受信時に圧縮を解凍して保存します。

<br/>

### 他社ログソリューションとの連携

| Q. 他のログソリューションと連携できますか？ |
| ----------------------- |

> A. 現在、他社ログソリューションとの連携機能は提供しておりません。
