#### 2023年5月2日

-   <Status>New</Status> StackExchcange Redisの最新ライブラリを追加適用(2.x)

-   <Status>New</Status> Oracle.ManagedDataAccessの最新ライブラリを追加適用(19.19.0バージョンまで)

-   <Status>New</Status>特定のアプリケーションプールのみを選別して追跡する

    -   **application_pool_filter_enabled** <span class='type'>Boolean</span>

        デフォルト値`false`

        `application_pool_filter`オプションに登録したアプリケーションのみを追跡するには、`true`に設定してください。

    -   **application_pool_filter** <span class='type'>String</span>

        追跡したいアプリケーションの名前を入力してください。 コンマ(,)区切り文字として複数設定することができます。

        ```ini title='whatap.conf'
        application_pool_filter="DefaultAppPool, userSiteAppName"
        ```

-   <Status>Fixed</Status> <b>ASP.NET Web API</b>を追跡できないバグを修正

-   <Status>Fixed</Status>アクティブトランザクションを追跡できないバグを修正

-   <Status>Fixed</Status>トランザクションIDが時々重複するバグを修正

-   <Status>Fixed</Status>複数のスレッドが同じログファイルに同時にアクセスしてログが無視される現象を修正

-   <Status>Fixed</Status> 사용자 수가 측정되지 않는 버그 수정

    [.NET 설정하기 - 사용자 수](../../dotnet/agent-number-of-user) 참조

-   <Status>Fixed</Status> <code>trace_httpc_normalize_urls</code> 옵션이 동작하지 않는 버그 수정

    [.NET 설정하기 - HTTPC, API Call](../../dotnet/agent-httpcapicall) 참조

-   <Status>Fixed</Status> <code>profile_http_header_enabled</code> 옵션이 동작하지 않는 버그 수정

    [.NET 설정하기 - 트랜잭션](../../dotnet/agent-transaction) 참조
