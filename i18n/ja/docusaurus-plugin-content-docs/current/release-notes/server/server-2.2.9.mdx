リリーズ：2022-12-27

### Windowsモニタリングエージェントにおける02時頃の自動再起動エラーを修正

リリーズ：2022-12-14

<Status>Fixed</Status> 2.2.8バージョンのコードエラーにより、Windowsモニタリングエージェントが自動再起動するエラーがあります。午前2時頃に停止され、Agent NoDataの誤検出が発生する可能性があります。 2.2.8バージョンを使用している場合は、アップデートをお勧めします。
