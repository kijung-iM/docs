リリーズ：2022-11-29

:::caution

Windowsモニタリングエージェントは、自動再起動エラーの発生により午前2時頃に停止し、エージェントのNoData の誤検知が発生する可能性があります。2.2.8バージョンを使用している場合、2.2.9バージョン以降へのアップデートをお勧めします。インストールファイルを再ダウンロードすることでサーバーから再実行することができます。 エージェントの削除は不要です。

:::

## Windowsプロセスモニタリングの所有者名の処理を改善

-   Windowsプロセスモニタリングの所有者名の検索APIが改善され、キャッシュ処理による呼び出しの負荷が軽減されました。

## Linux CPU IRQの収集エラーを改善

-   Linux CPU IRQの収集時、irq計算時のコードエラーによる収集するエラーを修正しました。
