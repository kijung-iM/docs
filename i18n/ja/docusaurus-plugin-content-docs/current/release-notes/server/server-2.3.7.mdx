## デプロイ履歴

リリーズ：2023年6月8日

<p><Status>Fixed</Status> <code>hostguid.enabled=true</code>オプションを適用する場合<b>AWS EC2</b>を含む仮想マシンの<code>Guid</code>値を<code>oid</code>(エージェント固有番号)生成時に含めるように修正</p>

-   IPアドレスが再利用されても<code>oid</code>が変わるように変更し、解約された<code>oid</code>を再利用する際に<code>Guid</code>値を使用して、デフォルト通知の適用が欠落する可能性のあるエラーを防止します。
