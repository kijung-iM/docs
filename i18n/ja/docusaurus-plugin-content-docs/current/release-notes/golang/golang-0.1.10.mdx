## ベータ - アップデート

### ウェブトランザクションに含まれないSQL、HTTP Call情報の収集

既存のウェブトランザクションの場合、contextにWhaTap情報を含み（`race.Start()`呼び出しが必要）固有トランザクションID（`Txid`）が発行された場合に限ってSQL、HTTP Call情報を収集しました。 

固有のトランザクションIDがない場合、収集された情報は統計情報で照会できます。
