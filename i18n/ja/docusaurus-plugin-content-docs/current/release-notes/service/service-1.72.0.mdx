リリーズ：2022-05-11

## `共通`

**_ログモニタリング_**

-   トランザクションプロファイルのログタブの追加
-   ログの構文解析機能の追加 
-   ログ照会現況メニューの追加

**_Flexボード_**

-   ユーザー作成ダッシュボードを組織単位で管理
-   アクティブトランザクションウィジェットチャートのクリック時、詳細照会機能の追加 
-   ダッシュボード管理およびプロジェクト選択機能の使用性改善
-   適用したフィルターオプションの選択状態が表示されない問題を修正、ダッシュボード上段に追加したフィルターに関する情報表示
-   ウィジェット編集機能の使用性改善

**_請求・決済履歴_** - CSVダウンロード機能の追加 

**_メトリクスイベント設定_** - イベント発生条件および対象フィルタリング条件入力の使用性改善

**_組織管理_**

-   メニュー移動およびプロジェクト削除時に選択した組織情報が初期化される現象を修正
-   Kubernetesネームスペースプロジェクト組織及びグループ移動時、一覧に反映されない現象を修正 

## アプリケーション

**_トランザクションプロファイル_** - ログ照会機能の追加

**_トランザクションマップ_** - フィルター機能の修正 

**_アクティブトランザクション_** - チャート初期の読み込み遅延現象の修正

**_統計 - ドメイン照会_** - 空きデータの照会時に読み込み状態が続く現象を修正

## サーバー

**_プロセスイベント設定_** - 持続時間の設定時、0分~999分と入力値を制限

## データベース

**_カウンター推移_** - タイムオプション変更を連続するとチャートデータが隠れる現象を修正

## コンテナ

**_ネームスペース管理_**

-   Kubernetesマスタープロジェクトの **_管理 - ネームスペース管理_** メニューの追加
-   ネームスペースプロジェクト追加機能
-   Node、コンテナ、Podステータスの照会

**_ネームスペース現況_**

-   プロジェクト一覧でクラスタープロジェクトをの時クリック時に移動していたKubernetesマスターグループビューをマスタープロジェクト内の **_リソース - ネームスペースステータス_** メニューに移動
-   左側のプロジェクトリストでは、既存のKubernetesマスターグループの代わりに、ユーザーが指定したプロジェクトグループに表示内容を修正 

**_コンテナマップ_**

-   Container、Podカテゴリのメトリクス詳細照会ウィジェットの追加
-   チャート可読性の改善

## URLモニタリング

**_無料化_** - URL登録及びプロジェクト作成の無制限利用

**_Flexボード_** - URL一覧ウィジェットの追加
