**_イベント設定_** - メトリクスイベントにイベントルールテスト機能を追加

**_レポート_** - レポートの再作成機能を追加

**_Flexボード_** - シリーズチャートメトリクスオプションにプロジェクトオプションを追加

**_ログ検索機_** - スクローリングによる次ページ照会エラーの修正

**_ アプリケーション固定トポロジ / アプリケーショントポロジー_** - エージェントフィルター選択エラーの修正

## データベース

**_Mongo/Mysql/PostgreSQL_** - 「統計-データベースサイズ」メニューの追加

**_Cubrid_**

-   「統計-データベースサイズ」画面の追加
-   「統計-テーブルサイズ増減」画面の追加

**_SGAサイズ増減_** - 「SGAサイズ」とメニュー名を変更

**_Mongo/Redis_** - 「マルチインスタンスモニタリング」のカテゴリ選択のメトリクスプレビューエラーを修正
