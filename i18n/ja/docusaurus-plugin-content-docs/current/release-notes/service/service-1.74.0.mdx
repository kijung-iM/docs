## 「共通」

**_Flexボード_**

-   反応型フレックスボード（グリッド型でウィジェット配置）機能を追加
-   サーバープロセスCPU、Memory TopNウィジェットの追加
-   サーバーリソースPlusテンプレートのウィジェット構成の変更、テンプレートを利用したユーザー作成ダッシュボードのフィルタリング適用バグの修正
-   統合Flexボードを他のユーザーと共有する場合、最近入力したメール情報を表示 

**_エージェント設定_** - 新規エージェント設定メニューの追加

-   アプリケーション、コンテナプロジェクト - エージェント設定 - 新規エージェントの設定リンクの提供
-   設定値入力機能の使用性を改善
-   エージェントオプションの検索機能の強化
-   JSONを利用した書き出し、読み込み機能の追加
-   エージェント種類およびプロジェクト単位のエージェント設定（デフォルト値）編集機能の追加（プロジェクト修正権限がある場合）
-   すべてのエージェント適用機能の改善

**_プロジェクト一覧_** - プロジェクト別のカードビュースタイルの改善 

**_メンバー管理_** - メンバー招待時のメール検証機能の改善

**_MXQL実行機_** - テーブルデータ表現の修正

## アプリケーション

**_エージェントのインストール_** - Goモニタリングライブラリの使用案内リンクを修正

**_マルチトランザクション_** - 日付選択オプションの追加

## サーバー

**_整備計画_** - メンバー修正権限の適用、画面エラーの修正

**_サーバーエージェントの移転_** - 組織変更時のプロジェクト選択一覧の更新エラーを修正

## コンテナ

**_コンテナマップ_**

-   Podカテゴリ選択時にポッド別のコンテナ情報を追加、コンテナステータス（ConState）フィールド選択機能の追加
-   Kubernetesイベント照会機能の追加
-   メトリクス別説明案内ポップアップの追加
-   しきい値の設定時、チャート臨界色にウエイトを表現、フィールド値が0の場合は正常状態として表示を変更

**_マルチサービスヒートマップ_** - ウィジェット配置をソート
