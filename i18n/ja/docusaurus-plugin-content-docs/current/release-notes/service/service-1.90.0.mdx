## 2022-10-24

### コンテナマップ

<Status>Fixed</Status> ContainerカテゴリとPodカテゴリのヒートマップチャートに描かれた情報が異なる現像を修正

### 「共通」

<Status>Fixed</Status> <UI>アカウント情報</UI> - 既存のパスワードが9文字以下の場合、新規パスワードに変更されない現象を修正

## 2022-10-20

### ログ

-   <Status>Fixed</Status> 無料体験期間が画面に「NaN」として表示される現象を修正（NaN→0）
-   <Status>Fixed</Status> ログモニタリング非活性化プロジェクトに入ると、「ログモニタリング活性化案内モーダル」の内容のない現象を修正
-   <Status>Fixed</Status> <UI>ライブTail</UI> メニューで構文解析されたキーを照会するAPIを呼び出さない現象を修正

### URL

<Status>Fixed</Status> URL一覧 - 名前変更ボタンの誤字（cancle → cancel）を修正

## 2022-10-18

### チャート

-   <Status>Feature</Status> Out of Memory メトリクスチャートへの移動ロジックを追加 - 分析動線の生成のため、行をクリックするとメトリクスチャートに移動するよう修正
-   <Status>Fixed</Status> ヒートマップのダークテーマ適用時の画面処理を改善するよう修正

### リソース

-   <Status>Change</Status> <UI>リソース</UI> > <UI>マスターメタ情報</UI> > 「データなし」メッセージを <UI>マスターメタデータなし</UI>に修正

### コンテナマップ

-   <Status>Change</Status> 要約を見るウィンドウが開いた状態でステータス別の個数テキストがカットする現象を修正
-   <Status>Change</Status> ログ検索タブで（ログカテゴリが存在する場合）ログ対象のカテゴリを修正
-   <Status>Fixed</Status> カテゴリがPodのときコンテナが複数ある場合、一部コンテナ情報のみを表示するバグを修正

### アプリケーションの現状/性能推移

-   <Status>Fixed</Status> エージェント情報が誤って表示されるエラーを修正

### ログ

-   <Status>Feature</Status> <UI>ライブTail</UI>, <UI>検索機</UI>、<UI>エクスプローラー</UI>にテーブルカラムの順序変更機能を追加
-   <Status>Change</Status> ログ設定のデータ保持期間オプションを修正

    -   トライアルプロジェクト：最大3日までデータを維持
    -   有料プラン転換プロジェクト：最大40日までデータを維持
    -   _front.conf_ ファイルに 「ui_log_retention_limit」オプションを追加すると、その値までデータを維持するように修正

### Kubernetes

-   <Status>Change</Status> クラスタープロジェクトでヘッダーメニューを通じてネームスペース移動時に、既存メニューを維持した状態で移動できるように修正
-   <Status>Change</Status> 性能要約ページウィジェットデータを修正

### 「共通」

-   <Status>Change</Status> 照会および分析権限をオフできないように非活性化処理を修正
-   <Status>Fixed</Status>通知受信設定 - サードパーティーSlackのredirect urlが正常に接続されないバグを修正
-   <Status>Fixed</Status> <UI>統合レポート</UI> メニューの最初進入時、時間選択値が正常に初期化されないバグを修正
