リリーズ：2022-09-14

## 「共通」

**_Flexボード_**

-   ダッシュボードのPDF書き出し機能を追加 
-   ウィジェット間のグループ化機能を追加（ウィジェットテンプレート > その他 > ウィジェットグループ化）
-   お気に入りのダッシュボード間のメニュー移動時の時間照会範囲を更新

**_統合ダッシュボード_** - トランザクションプロファイルの連続照会ができなかった問題を修正

**_時間セレクタ_** - 時間移動時の問題を修正

## インテグレーション

**_インテグレーションプロジェクト_**

-   クラウドサービスをはじめとする様々なモニタリング対象を単一プロジェクトで統合的に管理するための新規製品の追加
-   インテグレーションアイテム別のダッシュボードと収集メトリクス活用分析メニュー、イベント設定機能を提供
-   インテグレーション項目
    -   Amazon CloudWatch
    -   Amazon ECS
    -   Naver Cloud Monitoring
    -   Oracle Cloud Monitor
    -   Azure Monitor

## アプリケーション

**_マルチトランザクション追跡_** - プロジェクト一覧の階層化表示エラーを修正

**_トランザクションプロファイル_** - ダイアグラムフォントサイズの調整問題を修正

## コンテナ

**_コンテナマップ_**

-   Container画面のActiveTX演算規則の改善
-   詳細画面 > イベントNormal、Warningタイプのフィルタリング機能を追加
-   詳細画面 > カード上段のステータス別の数量部分表示スタイルの改善

**_Out of Memory Killedコンテナ_** - 照会時間の範囲選択オプションを削除、テーブルソート機能を修正 

## ログ

**_ログ設定_** - 現在使用量の照会機能を追加 

## サイト管理者

**_決済管理_** > **_プロジェクト有料転換_** - 製品種類、プラットフォームカラムをマージ

**_決済管理_** > **_請求管理_**、**_アカウント状態_** - 「open」から 「正常」とカラム名を変更

**_決済管理_** > **_請求書_** - 「Sub Total」から「使用額」とカラム名を変更、顧客表示（設定）、顧客表示（状態）、割引情報未表示カラムの非表示・表示メッセージを修正
