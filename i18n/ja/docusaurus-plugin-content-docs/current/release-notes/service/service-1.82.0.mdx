## `共通`

**_Flexボード_** - アクティブトランザクションのイコライザーチャートのクリック時、アクティブトランザクションが表示される機能を追加

**_エージェント設定_** - オプション追加で移動ボタンを追加

**_ログモニタリング_** - ログ一覧スタイルの修正

**_イベント照会_** - メトリクス通知などの他のタイプの通知での処理履歴機能を追加

**_レポート_** - アプリケーションユーザー定義レポートから不要な選択項目を削除

**_Flexボード_** - テーブルウィジェットフィルターの未適用エラーを修正

**_メトリクス照会_** - メタのないカテゴリの選択時、メトリクスエクスプローラーのショートカットリンクが動作しないエラーを修正

**_ログエクスプローラー_** - スクロールを下げるとデータの整合性が崩れるエラーを修正

**_ログ検索機_** - 検索結果がない場合、以前に検索されたデータが表示されるエラーを修正

**_メンバー一覧_** - メンバーの検索時、検索ウィンドウに入力された値が残るエラーを修正

**_プロジェクト設定_** - プロジェクト名を変更した後、他のメニューに移動すると変更前の名前が表示されるエラーを修正

**_プロジェクトリスト_** - 組織の選択時、プロジェクトカード一覧と組織のプロジェクトが一致しないエラーを修正

**_エージェント設定_** - エージェントのクリック時、エージェント一覧 ツリーが閉じるエラーを修正

## アプリケーション

**_トランザクション追跡_** - ツリービュー機能の追加・補完

**_ヒートマップ_** - ヒートマップでx軸移動ボタンのクリック時の時間変更エラーを修正

## コンテナ

**_コンテナマップ_** - 以前のコンテナログを照会できるタブを追加

**_コンテナマップ_** - 発生したイベントに合うタブにフォーカスが合わせられるよう機能を追加

**_コンテナマップ_** - 正常（青）のしきい値の区域別の3段階区分をなくし、単一色に修正

**_コンテナマップ_** - Podカテゴリで最新のログメッセージタブが見えないエラーを修正

**_ノード詳細_** - コンテナ一覧のカラムとデータを変更

**_コンテナ一覧_** - MemWsByRequestカラムデータエラーの修正
