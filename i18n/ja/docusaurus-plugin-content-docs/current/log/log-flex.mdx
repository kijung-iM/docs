**アクセスログ**は、サーバーに接続するクライアントの接続情報を残すログファイルです。アクセスログに含まれる内容は次のとおりです。 

-   リモートホストのIPアドレス（呼び出し元）
-   リクエスト時間
-   HTTPステータスコード
-   HTTPヘッダーを除く転送バイト数
-   リクエストの処理にかかった時間（ms）
-   リファラー（Referrer）
