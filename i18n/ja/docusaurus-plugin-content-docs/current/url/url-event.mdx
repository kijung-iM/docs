ホーム画面 > プロジェクト選択 > **_警告通知_** > **_イベント設定_**

URLモニタリングは、イベント設定を通じてメール形式の警告通知を受けることができます。モニタリング中のプロジェクト画面では、リアルタイムで発生するイベントについても通知を提供します。 登録されたURLを対象に次のように個別イベント設定ができます。 

1.  イベントリスト画面で編集したいURLの ![修正アイコン](/img/ico-edit.svg) **_修正_** アイコンを選択してください。

  ![イベント一覧画面](/img/url-event.png)

1.  該当URLに対する**_イベント設定の修正_**ウィンドウを確認してください。 

  ![urlイベント設定ウィンドウ](/img/url-event-mod.png)

  **_ステータス_**、 **_経過時間_** などのイベントルールは、モニタリングメトリクスと条件式を組み合わせて設定します。 

```
* **ステータスイベント**<br/>
    ステータスイベントは、選択した ***ステータス*** コードのオプションによってイベントが発生します。<br/>
    ![ステータスイベント](https://img.whatap.io/media/user_guide_url/alert/alert_status.png)

* **経過時間イベント**<br/>
    経過時間イベントは、リアルタイム収集中の ***経過時間*** 値が、ユーザーが入力したしきい値と条件文に該当する場合にイベントが発生します。<br/>
    ![経過時間イベント](https://img.whatap.io/media/user_guide_url/alert/alert_elapsed.png)
```

:::note 

**通知メッセージとは**

該当ルールでイベントが発生した場合に表示するタイトルです。 個々のイベント ルールに対して通知メッセージを設定できます。 設定を適用しない場合、基本的に表示される通知メッセージは **_ステータスコード_**、 **_経過時間_**、 **_URL：詳細理由_** となります。

:::
