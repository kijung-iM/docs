WhaTapのURLモニタリングは、エンドユーザーの立場からのウェブサイトの接続が正常かどうかを判断するためのサービスを提供します。 ユーザーの立場からウェブサイトへのアクセスの正常有無をブラウザーなどを通じて点検する運用業務を行います。WhaTapのモニタリングエージェントが該当サイトへのアクセスを行い、接続の正常有無を提供します。 サイトへのアクセスの実行は、WhaTapのチェックエージェントがインストールされているサーバーから接続を試みることになります。 接続テストの結果は、http呼び出しに対するレスポンスコードとレスポンス時間を提供します。 

ユーザーは、WhaTapのURLモニタリングサービスを個数制限なしにすべての機能を無料で使用できます。 モニタリングが必要なウェブサイトを登録し、ウェブサイトの点検を開始してください。 

## 主な特長

-   **グローバルサービス向けのURLチェック**

    北米、ヨーロッパ、アジア、韓国等のサービス拠点を選択してモニタリングを行うことができます。

-   **フリーサービス**

    WhaTapのURLモニタリングサービスのすべての機能を無料かつ時間制限なしで提供します。

-   **サイトアベイラビリティチェック**

    エンドユーザーのサイト接続の異常有無をHTTPステータスコードを通じてリアルタイムで確認します。

-   **サイト性能モニタリング**

    北米、ヨーロッパ、アジア、韓国等に位置するサーバーから、サイトにアクセスするまでにかかる時間を確認します。

-   **URL毎のアラート設定**

    URL毎にeメール、モバイルプッシュ、サードパーティープラグインを利用したアラート通知が設定できます。

    :::note

    URLモニタリングサービスは、SMS通知を提供していません。

    :::

-   **統合ダッシュボード**

    サーバー、アプリケーション、データベースなど、WhaTapの他のサービスとともにダッシュボードを構成できます。
