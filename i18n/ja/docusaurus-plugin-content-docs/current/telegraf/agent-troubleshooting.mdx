whatap-telegrafサービスが実行中でないか、エラーが発生した場合は再起動します。

-   **Red Hat/CentOS**  
    ```bash title=SH
    $ service whatap-telegraf restart
    ```
-   **Debian/Ubuntu**  
    ```bash title=SH
    $ sudo service whatap-telegraf restart
    ```
-   **FreeBSD**  
    ```bash title=SH
    $ service whatap_telegraf restart
    ```
