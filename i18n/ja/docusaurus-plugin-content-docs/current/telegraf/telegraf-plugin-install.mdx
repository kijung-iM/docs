## whatap output plugin

Telegrafのwhatap output plugin(tcp)を活用して、Telegrafで収集したメトリクスをWhaTap収集サーバーに送信することができます。 収集されたデータは、WhaTapのプロジェクト内部で確認できます。

:::note

Telegrafオープンソース内でwhatap pluginのレビューが進行中です。 正式リリース前は、whatap pluginを含むTelegrafを交換して使用してください。

:::

次の一覧にあるオペレーティングシステムは、ダウンロードをインストールできます。 他のオペレーティングシステムは、[コンパイルインストール](#コンパイル-インストール)で進めてください。

-   Red Hat & CentOS
-   Ubuntu & Debian
-   FreeBSD
-   windows

Telegrafreleaseバージョンを基準にwhatap pluginを追加して事前に作成したインストールファイルです。Telegrafが既にインストールされている場合は、実行ファイルだけを交換してwhatap pluginを活性化することができます。

## コンパイルインストール

### 事前準備事項

-   **Golang SDK構成**

-   **go depインストール**

-   **ソースファイル**

    _${GOPATH}/src/github.com/influxdata/telegraf/_ ディレクトリの下に圧縮ファイルを解凍してください。

    :::note

    Telegraf releaseバージョンにwhatap pluginを含むソースファイルです。 

    :::

    -   https://repo.whatap.io/telegraf/telegraf-release-1.12_whatap.tar.gz
    -   https://repo.whatap.io/telegraf/telegraf-release-1.13.2_whatap.tar.gz
    -   https://repo.whatap.io/telegraf/telegraf-release-1.14.5_whatap.tar.gz
    -   https://repo.whatap.io/telegraf/telegraf-release-1.15.4_whatap.tar.gz
    -   https://repo.whatap.io/telegraf/telegraf-release-1.16.0_whatap.tar.gz

### インストール手順

:::note

ビルドに関する詳細は [Telegraf github](https://github.com/influxdata/telegraf)ページを参考にしてください。

:::

_${GOPATH}/src/github.com/influxdata/telegraf_ 以下のMakefileでビルドを進めてください。

1.  依存ファイルのインストールを進めてください。 依存ファイルのインストールはgo depを使用するため、go depを先にインストールする必要があります。

    ```bash title=SH
    # make deps
    ```

2.  実行ファイルをビルドしてください。 _${GOPATH}/bin_ ディレクトリに実行ファイル（telegraf）が生成されます。

    ```bash title=SH
    # make go-install
    ```

3.  インストールパッケージが必要な場合は、`make package`コマンドで作成できます。 クロスコンパイルを進めて、オペレーティングシステムごとにパッケージファイルを作成してください。

    ```bash title=SH
    # make package
    ```
