Telegrafは、データベース、システム、IoTセンサーからメトリックおよびイベントを収集・送信するためのプラグインベースのサーバーエージェントです。TelegrafはGoで作成されており、外部依存性なく単一バイナリとしてコンパイルされ、必要最低限のスペースを使います。

-   **全種類のデータ収集・送信**
    -   **データベース**\
          MongoDB、MySQL、Redisなどのデータソースに接続し、メトリックを収集・送信します。 
    -   **システム**\
          現代的なクラウドプラットフォーム、コンテナ、オーケストレータースタックからメトリックを収集します。 
    -   **IoTセンサー**\
          IoTセンサーおよびデバイスから重要なステータス保存データ（圧力レベル、温度レベルなど）を収集します。
-   **Telegrafプラグイン**\
      Telegrafは、メトリックを収集、処理、集計、および作成するプラグインベースのエージェントです。 入力、出力、アグリゲーター、およびプロセッサを含む4つのカテゴリのプラグインに対応しています。

TelegrafにおけるWhaTapのoutput pluginを活用し、WhaTapのTelegrafエージェントをインストールしてTelegrafで収集するメトリックをWhaTap収集サーバーに送信することができます。
