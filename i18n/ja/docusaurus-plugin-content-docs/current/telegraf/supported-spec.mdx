WhaTapのTelegrafエージェントをインストールする前に、動作環境を確認してください。 

-   **OS**

    -   Linux Red Hat 6.x x64以降
    -   Ubuntu 12.x x64以降
    -   FreeBSD 10.x x64以降

-   **Telegraf**

    -   Telegraf 1.3以降 

    :::note

    socket_writerouput pluginに対応するバージョンに限って可能です。

    :::
