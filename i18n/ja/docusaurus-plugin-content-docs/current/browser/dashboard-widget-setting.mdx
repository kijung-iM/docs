ダッシュボード メニューに配置したウィジェットをユーザーが希望するオプションに設定できます。

:::note
웹 브라우저의 화면 크기에 따라 위젯 오른쪽 위에 아이콘의 모양이 다를 수 있습니다. ![더보기 아이콘](/img/ico-more.svg) 버튼을 선택하면 추가 아이콘을 확인할 수 있습니다.
:::

## グラフチャート基準データの変更

![위젯](/img/widget-sample.png)

위젯의 오른쪽 위에 ![지시선 1](/img/number-01.png) 메트릭스 버튼을 선택하세요. **_메트릭스_** 팝업창이 나타납니다. 원하는 항목을 선택해 그래프 차트의 데이터를 변경할 수 있습니다.

:::note

-   **_メトリクス_** ポップアップウィンドウで選択できる項目は、ウィジェットによって異なる場合があります。
-   위젯의 데이터에 따라 메트릭스 버튼을 지원하지 않을 수 있습니다.
:::

## フルスクリーン表示

전체 화면으로 보고 싶은 위젯의 오른쪽 위에 ![상세 보기 아이콘](/img/ico-detail-mode.svg) 버튼을 선택하세요. 웹 브라우저 상에서 위젯의 그래프 차트를 전체 화면 크기로 확인할 수 있습니다.

![ウィジェットフルスクリーン](/img/widget-fullscreen-mode.png)

-   画面右上の ![指示線 1](/img/number-01.png)オプションボタンを選択して、データを様々な方法で照会することができます。
-   화면 오른쪽 ![지시선 2](/img/number-02.png) 영역은 메트릭스 버튼을 통해 선택할 수 있는 데이터 유형입니다. [다음 문서](#change-data)를 참조하세요.

:::note
ウィジェットのデータによっては、選択できるオプションが異なる場合があります。
:::

### データ照会時間の変更

현재 진행 중인 데이터의 흐름을 실시간으로 확인할 수 있고, 지나간 시간의 데이터를 확인할 수도 있습니다. ![시간 선택 아이콘](/img/ico-time-set.svg)을 선택하세요. **_시간 선택_** 팝업창이 나타납니다.

![時間選択](/img/widget-select-time.png)

-   グラフチャートのデータ照会時間を変更するには **_ダッシュボード設定に従う_**のチェックボックスを解除してください。
-   グラフチャートをリアルタイムでモニタリングするには ![リアルタイムアイコン](/img/ico-play-time.svg)ボタンを選択してください。
-   照会時間は ![一時停止アイコン](/img/ico-pause.svg)ボタンを選択して変更できます。
-   ![왼쪽 방향 아이콘](/img/left-arrow.svg) 또는 ![오른쪽 방향 아이콘](/img/right-arrow.svg) 버튼을 선택해 시간을 이동할 수 있습니다. 이동 간격은 오른쪽에 설정된 ![지시선 1](/img/number-01.png) 시간 간격입니다.
-   ![指示線 1](/img/number-01.png)時間間隔ボタンをクリックして時間間隔を選択できます。
-   日付と時刻を直接選択するには、日付と時刻領域をクリックしてください。

### 以前時間と比較する

지난간 시간의 데이터를 현재 시간과 비교해 볼 수 있습니다. ![이전 시간과 비교 아이콘](/img/ico-timing.svg) 버튼을 선택하세요. **_이전 시간과 비교_** 팝업창이 나타납니다.

![以前時間と比較](/img/widget-compare-time.png)

초, 분, 시간, 일 단위로 시간을 비교할 수 있고, 원하는 날짜를 선택해 비교할 수도 있습니다. 원하는 옵션을 선택한 다음 **_적용_** 버튼을 클릭해야 그래프 차트에 적용할 수 있습니다. 이전 시간 데이터와 비교를 취소하려면 **_미설정_**을 클릭한 다음 적용 버튼을 선택하세요.

### 基準データの変更{#change-data}

![메트릭스](/img/widget-metric.png)

1.  오른쪽 위에 위치한 옵션 중 위젯의 지표에 해당하는 텍스트를 선택하세요. (보통 세번째에 위치합니다.)

2.  メトリクスポップアップウィンドウが表示されます。

3.  項目を選択してください。

グラフチャートのデータを選択した項目に変更します。

### 上位Nデータを確認する

-   **_상위 5_** 버튼을 클릭하면 최대 상위 10 항목까지 선택할 수 있습니다. 선택한 수치만큼 높은 순으로 그래프 차트에 데이터를 적용할 수 있습니다.

    ![上位N](/img/widget-n-data.png)

-   ![상위 N 아이콘](/img/ico-topn-graph.svg) 버튼을 선택해 상위 N 데이터 항목을 그래프 차트에 오버레이 형식으로 표시할 수 있습니다. 그래프 차트에서 상위 N 데이터 항목을 사라지게 하려면 ![그래프 아이콘](/img/ico-normal-graph.svg) 버튼을 선택하세요.

### MXQLでデータを照会する

MXQL을 이용해 데이터를 조회하려면 ![메트릭스 편집 아이콘](/img/ico-edit-metric.svg) 버튼을 선택하세요. MXQL 편집창이 나타납니다.

:::note
MXQLについての詳細は [次の文書](../mxql/mxql-overview)を参考にしてください。
:::

### 前日のデータ表示

![前日のデータ表示](/img/widget-before-data.png)

-   전일 데이터와 비교해 그래프 차트를 확인하려면 ![전일 데이터 비교 아이콘](/img/ico-before-data.svg) 버튼을 선택하세요. ![지시선 1](/img/number-01.png) 화면 왼쪽 위에 전일 데이터의 값을 표시할 수 있습니다.
-   화면 오른쪽 위에 ![지시선 2](/img/number-02.png) **평균** 버튼을 클릭해 그래프 차트의 데이터 유형을 변경할 수 있습니다. 선택할 수 있는 항목은 다음과 같습니다. 기본값은 **_평균_**입니다.
    -   **_最近値_**
    -   **_最大値_**
    -   **_平均_**
    -   **_合計_**

### テーブルカラムの変更

데이터 테이블 위젯의 경우 전체 화면에서 컬럼을 변경할 수 있습니다. 오른쪽 위에 ![컬럼 아이콘](/img/ico-column.svg) 버튼을 선택하세요. **_컬럼 편집_** 창이 나타나면 테이블에 표시할 컬럼을 숨기거나 추가할 수 있습니다. 설정을 완료한 다음 **_저장_** 버튼을 선택해야 데이터 테이블 차트에 적용할 수 있습니다.

![カラム編集](/img/widget-column.png)

## チャート設定

y軸の最大値を設定し、水平線を追加してしきい値を設定するには、ウィジェットの右上の ![設定アイコン](/img/ico-setting.svg)ボタンを選択してください。

![チャート設定](/img/widget-chart-set.png)

-   **_y축 차트 최대값_**: **_자동_** 또는 **_고정_**을 선택해 y축의 최대값을 설정할 수 있습니다. **_고정_**을 선택한 경우 수치를 입력할 수 있습니다. 수치를 입력했다면 **_적용_** 버튼을 선택하세요.

-   **_水平線設定_**：グラフチャートに水平線を追加してしきい値を設定し、効率よくモニタリングできます。

    -   **_+ 추가_** 버튼을 선택한 다음 **_라벨_**과 **_임계치_** 항목에 수치를 입력하고, 색상을 선택하세요. **_임계치_** 항목에서는 **_범위_**를 선택해 수평선 구역을 설정할 수 있습니다. **_적용_** 버튼을 선택해 설정을 완료하세요.

    -   水平線を削除するには！[削除アイコン](/img/ico-trash-red.svg)ボタンを選択してください。

-   **_차트 최대값 표시_**: 그래프 차트에 최대값을 표시할 수 있습니다. 가장 높은 수치의 값을 차트에 텍스트로 표시합니다.
