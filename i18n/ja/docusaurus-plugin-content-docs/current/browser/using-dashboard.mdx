ダッシュボードで照会時間を変更し、フィルターを適用して全体ウィジェットで出力されるデータ値を変更できます。

## データ照会時間の変更

現在進行中のデータの流れをリアルタイムで確認でき、過ぎ去った時間のデータを確認することもできます。

![real time](/img/browser-time-selector.png)

右側に設定された時間だけリアルタイムでデータを照会します。 緑色のボタンを選択すると、様々な時間を選択できます。

![一時停止アイコン](/img/ico-pause.svg)ボタンをクリックすると、リアルタイムデータ照会を一時停止し、次のように照会時間を設定できます。

![past time](/img/browser-time-selector-past.png)

-   一番右側の緑色のボタンを選択してください。 様々な時間設定を選択できるオプションが表示されます。 ご希望の時間間隔を選択してください。

-   ![左方向アイコン](/img/left-arrow.svg)または![右方向アイコン](/img/right-arrow.svg)ボタンを選択して時間を移動できます。 移動間隔は、右側に設定された時間間隔です。

-   日付と時刻を直接選択するには、日付と時刻領域をクリックしてください。

-   再びリアルタイムでモニタリングするには、![リアルタイムアイコン](/img/ico-play-time.svg)ボタンを選択してください。

## ページグループフィルター

ページグループごとのフィルタリングオプションを適用できます。 ダッシュボード画面左上の**_ページ·グループ_**フィルターボタンを選択してフィルターオプションを設定できます。 

-   **_一致_**：選択したページグループと一致するデータを照会できます。

    ![Page group filter](/img/browser-dashboard-filter.png)

<!---->

-   **_含む_**：特定の文字列を含むページグループのデータを照会できます。

    ![Page group filter](/img/browser-dashboard-filter-include.png)

-   **_除外_**：特定の文字列を含むページグループを除いてデータを照会できます。

    ![Page group filter](/img/browser-dashboard-filter-exclude.png)

-   **_フィルター追加_**：複数のフィルターを適用するには**_+フィルター追加_**ボタンを選択し、ご希望の条件を入力してください。 **AND**条件でフィルターを追加できます。

    ![Page group filter](/img/browser-dashboard-filter-and.png)
