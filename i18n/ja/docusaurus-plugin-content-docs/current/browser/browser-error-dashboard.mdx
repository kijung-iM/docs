ホーム画面 > プロジェクト選択 > **_ダッシュボード_** > **_ブラウザーエラー_**

![Browser error dashboard](/img/browser-error-dashboard.png)

**_ブラウザーエラー_**ダッシュボードは、ブラウザーアプリケーションのエラーをモニタリングし、視覚化してリアルタイムで表示するダッシュボードです。 ダッシュボードを通じて迅速にエラーを把握し、原因を見つけて修正できます。 ページ別、エラータイプ別、AJAXエラー件数、ブラウザー別、エラーメッセージ別エラー件数など多様な情報を提供します。

{@include: _dashboard-note.mdx}

## ページごとのエラー数

![Error count by pages](/img/browser-error-by-page.png)

各ページで発生したすべてのエラー件数をページごとに表示します。 チャートの上にマウスオーバーすると、ページ グループのエラー件数をツールチップ形式で確認できます。 水平バーチャートは、照会時間中に発生したページごとのエラー数の合計であり、上位8つまで表示します。

## エラーの種類別のエラー数

![Error count by types](/img/browser-error-by-types.png)

コンソールエラーとソースエラーを表示します。 水平バーチャートは、照会時間中に発生したエラータイプ別エラー数の合計であり、上位8つまで表示します。

## AJAXエラー件数

![AJAX error count](/img/browser-error-ajax-by-error.png)

ブラウザーアプリケーションで発生するAJAXエラー件数に対し、fetchまたはxhrタイプに分類して表示します。 AJAXエラーは、ステータスコードが400以上だったり、リクエスト自体ができなくてキャンセルされた場合を意味します。

## ブラウザ別エラー件数

![Error count by browsers](/img/browser-erorr-by-browsers.png)

ブラウザごとに発生したエラー件数を円グラフで表示します。 ブラウザがどのようなエラーに頻繁に遭遇するか、簡単に把握することができます。 また、特定のブラウザで発生するエラーが他のブラウザよりも多い場合、開発者はそのブラウザに対する最適化作業を進めることができます。

## エラーメッセージ別のエラー数

![Error count by messages](/img/browser-error-by-messages.png)

コンソールエラーを除いたソースエラーおよびAJAXエラーに関するメッセージを表示します。 水平バーチャートは、照会時間中に発生したエラーメッセージごとのエラー数の合計であり、上位8つまで表示します。

## エラーメッセージテーブル

![Error message table](/img/browser-error-table.png)

エラーメッセージに基づいたエラー数とブラウザーごとに発生したエラー数をテーブル一覧として表示します。 どのブラウザーでどのエラーが最も多く発生するかを把握し、問題を解決するために利用できます。 エラー発生の主な原因が何かを把握するのにも役立ちます。 新しいエラーが発生した場合は、このテーブルを参考にして迅速に対処できます。
