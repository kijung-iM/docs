## プロジェクトとは

WhaTapのモニタリングサービスにおいて、プロジェクトはモニタリング単位を意味します。プロジェクト単位でモニタリング単位が区分されるとともに、テナント権限も管理します。

第一に、プロジェクトは図のようにモニタリング要素別に区分します。アプリケーション、サーバー、コンテナなど、モニタリングエージェントごとの基準に従います。

<ImgLang img='project-01.png' desc='プロジェクト作成'/>

第二に、プロジェクトはモニタリング単位によって区別されます。JavaアプリケーションでもFront、Gateway、APIといった用途別にプロジェクトを区分することができます。1つのプロジェクト内に用途の区別なくアプリケーションを登録することもできます。

<ImgLang img='demo-project.png' desc='Demo project'/>

:::note
プロジェクトの階層構造の詳細については、[次の文書](../project/project-structure)を参考にしてください。
:::

## プロジェクト管理

ホーム画面 > プロジェクト選択 > **_管理_** > **_プロジェクト管理_**

プロジェクト概要の確認と管理が可能で、次の機能を提供します。そのプロジェクトの修正権限が必要です。

<ImgLang img='project-manage.png' desc='プロジェクト管理'/>

-   **_プロジェクト名_**：プロジェクト名を修正できます。

-   **_プロジェクト説明_**：プロジェクトに関する説明を入力することができます。

-   **_プロジェクトコード_**：プロジェクトを識別する数字型コードです。

-   **_アプリケーション時間帯_**：時間帯を変更することができます。

-   **_APIトークン_**：**_再発行_**ボタンを選択してAPIトークンを再発行することができます。

修正した項目がある場合は、**_保存_**ボタンを選択してください。


## プロジェクト削除

次の2つの方法でプロジェクトを削除できます。

-   ホーム画面の全体プロジェクト一覧から削除しようとするプロジェクト項目の右上の![設定アイコン](/img/ico-setting.svg)ボタンを選択してください。ドロップダウンメニューで**_プロジェクト削除_**を選択してください。

-   ホーム画面 > プロジェクト選択 > **_管理_** > **_プロジェクト管理_**メニューに移動してください。画面右下の**_プロジェクト削除_**を選択してください。

:::caution
ユーザー一覧情報とこれまで収集したデータはすべて削除されます。削除したデータは再復旧できません。
:::

## プロジェクトメンバーを確認する{#member-list}

ホーム画面 > プロジェクト選択 > **_管理_** > **_プロジェクトメンバー_**

選択したプロジェクトに所属するメンバーの一覧を確認できます。

![プロジェクトメンバー一覧](/img/project-member-list.png)

-   メンバー名の右側の<code class='history'>ログイン：...</code>タグは、最近接続した記録です。
-   決済権限を持つメンバー名の右側には<code class='pay'>決済</code>タグを表示します。
-   ユーザー自身のアカウント名の右側には<code class='my'>マイアカウント</code>タグを表示します。
-   WhaTapモニタリングサービスに登録していないアカウントを招待すると、招待対象が登録されるまで名前を**_招待待機中です_**と表示します。招待メールを再送信するには**_再招待する_**ボタンを選択してください。

## プロジェクトメンバーを招待する

1.  ホーム画面でメンバーを招待するプロジェクトを選択してください。

2.  画面左側のメニューで**_管理_** > **_プロジェクトメンバー_**を選択してください。

3.  画面右上の**_メンバー招待_**ボタンを選択してください。

4.  **_ユーザー権限_**項目で招待するユーザーの権限を設定した後、**_適用_**ボタンを選択してください。

5.  **_メール_**項目で招待するユーザーのメールを入力し、**_追加_**ボタンを選択してください。他のユーザーをさらに追加することができます。

6.  招待しようとするユーザーをすべて追加したら、**_完了_**ボタンを選択してください。

:::note

-   メンバー権限の詳細については、[次の文書](../project/project-structure#project-auth)を参考にしてください。
-   WhaTapモニタリングサービスに登録していないアカウントには、「新規アカウント」タグを表示し、会員登録メールを送信します。
-   招待を完了する前にキャンセルしたい招待メンバーがいる場合は、**_招待メンバー一覧_**で! [削除アイコン](/img/ico-trash-red.svg)ボタンを選択してください。
-   形式が正しくないメールアドレスを入力する場合は、招待できません。
-   すでに招待したメンバーは追加で招待することはできません。上位の組織やグループのメンバーは自動的に追加されます。

:::

### WhaTapのサポートを受ける

WhaTapモニタリングサービスの利用中のトラブルに対する支援が必要な場合、WhaTap支援チームをメンバーとして招待して支援を受けることができます。 

1.  **_プロジェクトメンバー_**メニューの画面右上の**_WhaTap支援チームを招待する_**ボタンを選択してください。

2.  メンバー招待ウィンドウが表示されたら、**_プロジェクト_**項目でプロジェクトを選択してください。

3.  **_完了_**ボタンを選択してください。

:::note

WhaTap支援チームのアカウントは自動入力されます。プロジェクトの種類によっては、支援チームのアカウントは1つ以上ある場合があります。「database」タイプのプロジェクトには、dbx@whatap.io アカウントが含まれます。

:::

## プロジェクトメンバーの権限を修正する

1.  ホーム画面でプロジェクトを選択してください。

2.  画面左側のメニューで**_管理_** > **_プロジェクトメンバー_**を選択してください。

3.  **_プロジェクトメンバー_**一覧で権限を修正するメンバーの右側の![編集アイコン](/img/ico-edit.svg)ボタンを選択してください。 

4.  **_メンバー権限修正_**ウィンドウが表示されたら、**_メンバー権限_**を設定してください。そのプロジェクトで許可する権限を設定できます。

5.  すべての設定が完了したら、**_適用_**ボタンを選択してください。

:::note

メンバー権限の詳細については、[次の文書](../project/project-structure#project-auth)を参考にしてください。

:::

## プロジェクトメンバーを削除する

1.  ホーム画面でプロジェクトを選択してください。

2.  画面左側のメニューで**_管理_** > **_プロジェクトメンバー_**を選択してください。

3.  **_プロジェクトメンバー_**一覧で削除するメンバーの右側の![削除アイコン](/img/ico-trash-red.svg)ボタンを選択してください。

4.  メッセージウィンドウが表示されたら、**_削除_**ボタンを選択してください。

:::note

-   プロジェクトメンバーの管理権限がない場合、他のメンバーを削除できません。
-   プロジェクトメンバーが1人の場合、他のメンバーを削除できません。
-   上位グループから継承されたメンバーは削除できません。
-   ユーザー自身のアカウントは、権限に関係なくプロジェクトから削除できます。
-   決済権限を持つメンバーは、決済権限を譲渡するまでは削除できません。

:::
