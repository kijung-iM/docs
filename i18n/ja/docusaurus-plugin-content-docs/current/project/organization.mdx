組織は、複数のグループをまとめる上位グループです。ほとんどの状況では、プロジェクトとグループだけで十分ですが、運用・管理サービスを専門とする企業であれば、複数の会社がグループとして存在します。グループやプロジェクトの数が非常に多いかもしれません。この場合、グループという概念だけでは十分ではありません。グループの統合管理できる更なる概念が必要です。組織という概念を活用してみてください。

専門的な運用・管理サービス企業は、複数のサブグループを上位の1つの組織にまとめて管理できます。それぞれのグループメンバーに権限を与えてみてください。権限を与えられたメンバーは独立してグループを管理できます。

## 組織作成

1.  画面右上のプロファイル アイコンを選択してください。ドロップダウンメニューが表示されます。

2.  **_組織作成_**を選択してください。

3.  **_組織名_**と**_ロゴ_**、**_説明_**を入力して「次へ」ボタンを選択してください。

4.  グループ一覧で組織に追加するグループを選択し、**_完了_**ボタンをクリックしてください。

**_組織が正常に作成されました。_**'というメッセージウィンドウが表示されたら、組織作成に成功したのです。

:::note

-   **_組織名_**は必須入力です。 **_ロゴ_**と**_説明_**は、入力しなくても組織を作成することができます。
-   修正権限のあるグループのみを組織に追加できます。
-   修正権限がなければ、そのグループは選択できません。
-   グループを選択しなくても、組織を作成できます。
-   権限の詳細については、[次の文書](project-structure#project-auth)を参考にしてください。
-   修正権限なしでそのグループに招待されると、グループに対する修正権限はありません。
-   上位グループから削除したり、他のグループに移動したりすることはできません。
-   조직 **_로고_** 이미지의 권장 사이즈는 가로 102 픽셀, 세로 102 픽셀 입니다. 
-   조직 **_로고_** 이미지는 3MB 이하 용량의 _png_, _jpg_, _jpeg_ 파일의 확장자만 가능합니다.

:::

## 組織に属するグループやプロジェクトを確認する

ホーム画面の左上の**_非組織_**を選択してください。**_組織一覧_**が表示されたら、希望する組織を選択してください。**_全体プロジェクト_**メニューの下位の組織に属するグループを確認することができます。**_プロジェクト一覧_**では、組織に属する全体グループのプロジェクト一覧を確認することができます。

## 組織管理{#manage}

![組織管理](/img/project-organize-manage.png)

1.  ホーム画面左上の非組織を選択してください。

2.  **_組織一覧_**が表示されたら、管理したい組織名にマウスオーバーしてください。 

3.  ![設定アイコン](/img/ico-setting.svg)ボタンを選択してください。

**_組織管理_**メニューに移動します。

![組織管理](/img/project-organize-manage-group.png)

-   ![指示線1](/img/number-01.png) 組織名領域

    ユーザーが属している組織の一覧を確認するには、![指示線 1](/img/number-01.png)の組織名領域を選択してください。**_組織一覧_**が表示されたら、他の組織を選択するか、**_非組織一覧に戻る_**を選択してプロジェクト一覧に戻ることができます。

-   ![指示線2](/img/number-02.png)**_組織一覧_**

    -   自分が属しているすべての組織がパネル一覧として表示されます。組織を変更するには、![指示線 4] （/img/number-04.png）**_移動する_**を選択してください。
    -   組織名とロゴ、説明を修正するには、![編集アイコン](/img/ico-edit.svg)ボタンをクリックしてください。
    -   組織を削除するには、![削除アイコン](/img/ico-trash.svg)ボタンをクリックしてください。属していたグループとプロジェクトは、組織に属していないグループに移動します。
    -   修正権限がない場合は、修正・削除ボタンが表示されません。

-   ![指示線3](/img/number-03.png)**_グループ一覧_**

    現在選択されている組織に属するすべてのグループが一覧として表示されます。![右方向アイコン](/img/right-arrow.svg)ボタンをクリックすると、選択したグループの管理画面に移動できます。

-   ![指示線5](/img/number-05.png)組織修正

    現在選択している組織の基本情報を修正するには、[編集アイコン](/img/ico-edit.svg)ボタンを選択してください。**_組織修正_**ウィンドウが表示されたら、組織名、ロゴ、説明を修正することができます。修正を完了した後、**_保存_**ボタンを選択してください。

## 組織メンバーを招待する

1.  **_組織管理_**メニューで**_メンバー一覧_**を選択してください。

2.  画面右上の**_メンバー招待_**ボタンを選択してください。

3.  **_組織_**項目で下位グループやプロジェクト単位を選択してメンバーを追加することもできます。

4.  **_ユーザー権限_**項目で招待するユーザーの権限を設定した後、**_適用_**ボタンを選択してください。

5.  **_メール_**項目で招待するユーザーのメールを入力し、**_追加_**ボタンを選択してください。他のユーザーをさらに追加することができます。

6.  招待しようとするユーザーをすべて追加したら、**_完了_**ボタンを選択してください。

:::note

招待を完了する前にキャンセルしたい招待メンバーがいる場合は、**_招待メンバー一覧_**で! [削除アイコン](/img/ico-trash-red.svg)ボタンを選択してください。

:::

## 組織メンバーを確認する{#member-list}

![組織メンバー確認](/img/organization-member.png)

**_組織管理_**メニューで**_メンバー一覧_**を選択してください。組織ツリー構造でグループまたはプロジェクトを選択できます。選択したグループ別、プロジェクト別に属するメンバーを確認できます。選択したグループ、プロジェクトごとにメンバーを追加・削除したり権限を修正したりすることができます。

### 組織メンバーの権限を修正する

1.  **_組織管理_**メニューで**_メンバー一覧_**を選択してください。

2.  **_組織メンバー_**一覧で権限を修正するメンバーの右側の![編集アイコン](/img/ico-edit.svg)ボタンを選択してください。 

3.  **_メンバー権限の修正_**ウィンドウが表示されたら、**_メンバー権限_**と**_組織のグループ別のユーザー権限_**を設定してください。

    -   **_メンバー権限_**：該当組織と組織に属するグループが許可する権限を設定できます。
    -   **_組織のグループ別のユーザー権限_**：選択したメンバーの権限を組織に属するグループ別に設定することができます。

4.  すべての設定が完了したら、**_適用_**ボタンを選択してください。

:::note

-   組織ツリー構造でグループまたはプロジェクトを選択し、グループ別、プロジェクト別に属するメンバーの権限を修正できます。選択したグループまたはプロジェクトによって適用できる権限の範囲は異なる場合があります。
-   メンバー権限の詳細については、[次の文書](project-structure#project-auth)を参考にしてください。

:::

## 組織メンバーを削除する

1.  **_組織管理_**メニューで**_メンバー一覧_**を選択してください。

2.  **_組織メンバー_**一覧で削除するメンバーの右側の![削除アイコン](/img/ico-trash-red.svg)ボタンを選択してください。 

3.  メッセージウィンドウが表示されたら、**_削除_**ボタンを選択してください。

:::note

-   下位のグループおよびプロジェクトでもメンバーを削除するには、メッセージウィンドウでチェックボックスをクリックし、**削除**ボタンを選択してください。
-   組織ツリー構造でグループまたはプロジェクトを選択し、グループ別、プロジェクト別に属するメンバーを削除できます。

:::
