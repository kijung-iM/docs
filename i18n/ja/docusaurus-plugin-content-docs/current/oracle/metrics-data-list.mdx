データベースメトリクスは、データベースの作業量とリソース使用量を包括的に収集します。 WhaTapモニタリングサービスは、継続的にメトリクスを収集してデータベースのボトルネックを確認し、性能問題の根本原因を識別するのに役立つデータを提供します。

メトリクス情報に関するより詳しい内容は、[次のリンク](https://docs.oracle.com/)を参照してください。

<div class='metric'>

| Metric                                   | Description                                                                 |
| ---------------------------------------- | --------------------------------------------------------------------------- |
| `cpu_used_by_this_session`               | ユーザー呼び出しが開始された時点から終了するまでのセッションで使用したCPU時間（10ミリ秒単位）                           |
| `db_time`                                | データベースのユーザーレベルの呼び出しを実行するのにかかる経過時間（microseconds）                             |
| `active_sessions`                        | クエリを実行中のセッション数                                                              |
| `cpu`                                    | CPU使用率                                                                      |
| `cpu_cores`                              | CPUコア数                                                                      |
| `db_block_changes`                       | SGAのブロックで変更されたブロック数                                                         |
| `db_file_scattered_read_time_waited`     | db file scattered readイベント待ちの総時間                                            |
| `db_file_scattered_read_total_timeouts`  | db file scattered readイベントの総タイムアウト回数                                        |
| `db_file_scattered_read_total_waits`     | db file scattered readイベント待ちの総回数                                            |
| `db_file_sequential_read_time_waited`    | db file sequential readイベント待ちの総時間                                           |
| `db_file_sequential_read_total_timeouts` | db file sequential readイベントの総タイムアウト回数                                       |
| `db_file_sequential_read_total_waits`    | db file sequential readイベント待ちの総回数                                           |
| `enqueue_waits`                          | 待機列の読み込みが遅れて、待機列の変換・読み込み中に発生した総待機回数                                         |
| `excute_count`                           | SQL文（statement）を実行した総呼び出し数                                                  |
| `fatal_count`                            | 致命的なエラーが発生した回数                                                              |
| `iowait_cpu`                             | CPUが入出力を待機するのに使用した時間の割合                                                     |
| `lock_wait_sessions`                     | 現在rowlockを獲得するために待機中の数                                                      |
| `logons_current`                         | 現在の総ログオン数                                                                   |
| `long_running_sessions`                  | 長時間実行中のセッション数                                                               |
| `メモリ`                                    | メモリ使用率                                                                      |
| `opened_cursors_current`                 | 現在開いているcursorの数                                                             |
| `parse_count_(hard)`                     | 構文分析呼出の総数（実際の構文分析）                                                          |
| `parse_count_(total)`                    | 構文分析呼び出しの総数（hard、soft、describe）                                             |
| `physical_reads`                         | ディスクで読み込んだデータブロック数                                                          |
| `physical_writes`                        | データをディスクに記録したブロック数、`physical writes direct`と`physical writes from cache`の合計 |
| `session_logical_reads`                  | メモリから読み込んだデータブロック数、`db block gets`と`consistent gets`の合計                     |
| `sorts_(memory)`                         | メモリで実行されたsort作業の数                                                           |
| `sys_cpu`                                | session에 의해 사용된 cpu time의 합                                                 |
| `total_elapse_time`                      | データベースインスタンスが最後に再起動して以来、すべてのリクエストの累積合計経過時間です。                               |
| `total_sessions`                         | すべてのセッション数                                                                  |
| `txn_sessions`                           | transaction이 있는 세션 수                                                        |
| `user_commits`                           | トランザクションあたりに実行されたユーザーコミット数                                                  |
| `user_cpu`                               | user cpu使用量(%)                                                              |
| `wait_sessions`                          | イベントを待機しているセッション数                                                           |
| `warning_count`                          | すべての警告数                                                                     |

</div>
