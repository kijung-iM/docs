**WhaTapモニタリングサービス**は、様々なプラットフォームに対応する統合モニタリングを提供します。 数多くのデジタルソースからデータを収集し、ユーザーがシステムとシステムの改善策を把握できるよう支援します。 

-   ユーザーは、**WhaTapとともに**様々なデータを統合することができます。 WhaTapエージェントと拡張ツールを活用してデータを収集し、一つのUIでユーザーだけのデジタル生態系を構築することができます。 
-   ユーザーは、**WhaTapとともに**データを分析することができます。 収集したデータに基づいて、ユーザー定義型のダッシュボードおよび各種チャートまたは強力なクエリ言語を活用してトラブルの根本原因を見つけ出し、システムを最適化できます。
-   ユーザーは、**WhaTapとともに**システムトラブルに迅速に対応することができます。 トラブルの兆候を先に感知し、トラブルが発生する前にユーザーに通知することができます。

### はじめに

[始まり](./) メニューでは、WhaTapモニタリングサービスを紹介し、プラットフォーム経験と機能の理解に役立つようサービスの流れを案内します。 

-   [始まり](./) メニューを選択するとホーム画面に移動します。 

-   [サポート環境](Support-env) 文書では、WhaTapモニタリングサービスの製品別動作環境の概要を確認できます。 

-   [ホーム画面（UI）の案内) 案内](main-ui-intro) 文書では、サービス初期画面の利用方法を案内します。 

-   WhaTapモニタリングサービスに関する情報は、[WhaTapモニタリングサービス案内](whatap-overview) 文書を参考にしてください。

-   WhaTapモニタリングサービスの利用が初めてなら、[クイックスタート](quick-guide) 文書を参考にしてください。

-   와탭 모니터링 서비스를 효율적으로 이용하고 싶다면 대시보드 분석 및 활용 사례를 안내하는 [학습하기](learning-guides) 문서를 참조하세요. 

-   와탭이 제공하는 소프트웨어 웨어 기반의 프록시 구성에 대해 알고 싶다면 [소프트웨어 프록시](software-proxy) 문서를 참조하세요.

### 製品固有のドキュメントに移動

**製品別文書のショートカット**メニューは、ユーザーが選択した製品に対してインストールから設定および活用ガイドラインまで統合案内文書を提供します。 

-   에이전트 설치 전 **지원 환경**을 확인하세요. 

-   와탭 모니터링 서비스를 이용하기 위한 에이전트 **설치하기** 문서를 참조하세요.

-   설치 점검 사항을 확인하고 설치 중 문제가 발생한다면 **설치 문제 해결** 문서를 참조하세요. 

-   에이전트 설정 옵션과 업데이트부터 삭제 등의 **관리** 방법을 알아보세요. 

-   서비스 화면의 **주요 메뉴 알아보기** 문서에서 **_대시보드_**, **_Flex 보드_**, **_보고서_**, **_경고 알림_** 설정 방법 등을 알아보세요.

各製品別文書は、[Docsホーム画面](./) 左側の基本メニューもしくは固定された上段の**製品**メニューで確認できます。

### 管理機能

[管理機能](manage) メニューは、WhaTapモニタリングサービスのアカウント、プロジェクト、グループなどを管理する方法と決済方法を案内します。 

-   보안이 강화된 와탭의 멀티팩터 인증 방식을 확인하세요. 

-   WhaTapモニタリングサービスのモニタリング単位であるプロジェクトの階層構造と管理方法について確認してください。 

-   와탭 모니터링 서비스의 정비 계획 관리 방법에 대해 알아보세요. 

-   WhaTapモニタリングサービスの無料利用期間である15日が過ぎた場合、決済情報を確認し、有料サービス転換について確認してください。

### リファレンス文書

[参照文書](reference) メニューは、WhaTap Docsで使用する主要概念および用語に関する資料集を紹介します。 メトリクス定義、WhaTapメトリクス照会のためのクエリ言語であるMXQLの文法およびコマンド案内、Open API活用文書および用語辞典を提供します。 또한 상품별 문서를 PDF 파일로 제공합니다. PDF 파일은 매달 초 지난 업데이트를 반영해 제공합니다. FAQ 문서는 지속적으로 업데이트해 제공할 예정입니다.
