新規作成したサーバーモニタリングプロジェクトを選択すると、**_エージェントのインストール_**画面が表示されます。 プロジェクト**_管理_** >**_エージェントのインストール_**メニューから行うことができます。

![](/img/server-install-windows.png)

1.  **エージェントファイル**をダウンロードしてください。**_管理_** > **_エージェントインストール_**  画面上部の![number-s](/img/number-01.png) OSタブでサーバーOSを選択してください。 ![number-s](/img/number-02.png) **_インストールファイルダウンロード_** タブで _whatap_infra.exe_ または、 _whatap_infra.zip_ ボタンを選択して、エージェントのインストールファイルをダウンロードしてください。  

    :::note

    -   セキュリティ設定により_.exe_形式のファイルがダウンロードできないユーザーのために、_.zip_形式のファイルも提供します。
    -   サーバーのセキュリティのために、ブラウザー経由で直接インストールするよりも、ダウンロードしたインストールファイルを実行することをお勧めします。
    :::

2.  モニタリング対象サーバーに接続して**エージェントファイル**をアップロードしてください。

3.  ダウンロードした**インストール ファイル**をダブルクリックして実行してください。  

4.  ![number-s](/img/number-03.png) **_インストールファイルの実行_** タブに移動して、発行されたプロジェクトアクセスキーとサーバーIPを確認してください。

5.  画面の指示に従って、***License Key***項目にプロジェクトアクセスキーを入力し、***WhaTap IP***項目にWhaTapサーバーのIPアドレスを入力してください。

    ![](https://img.whatap.io/media/user_guide_server/management/1230.png)  

6.  **_Next_** ボタンをクリックします。

7.  次の画面で **_Install_** ボタンをクリックします。 インストールを続行します。

8.  **_Finish_** ボタンをクリックして、インストールを終了してください。
