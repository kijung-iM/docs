WhaTapエージェントのインストールが完了しましたら、[WhaTapモニタリングサービス](https://service.whatap.io)にログインしてください。WhaTapモニタリングサービスは、円滑なサービス提供と安定した運用、異常検知モニタリングおよび把握のための情報を提供します。

<!-- 주요 메뉴 안내 개요 위치 -->

サーバーのモニタリングは、リアルタイム収集なのでトラブル原因の検出に容易です。運用の利便性を高めるためにレポート機能を強化し、好みの通知手段を選択できます。

WhaTapサーバーモニタリング サービスは、次の主要な機能を提供します。

-   **リアルタイム統合モニタリング**

    -   300台以下のサーバーリソースのリアルタイム使用量情報
    -   サーバー全体の要約指標を提供
    -   CPU、Memory、Disk I/O、IOPS使用量を基準に上位5つのサーバーのリストを提供
    -   CPU、Memoryの最大占有率を基準に上位5つのプロセスのリストを提供

-   **性能分析**

    -   収集サーバー全体の状況情報
    -   1分に10回以上収集されたCPU、メモリ、Disk、Networkの性能情報
    -   最大5つのサーバー間のメトリックスチャートによる性能の比較情報
    -   プロセスグループ化適用後のマルチプロセスリソース使用量の合計値情報

-   **イベント通知**

    -   メール、SMS、SNS(Telegram、Slack) 送信
    -   イベント継続期間およびしきい値設定によるイベント発生および過去のデータベースのイベントシミュレーション
    -   無音設定時間登録及びキーワードの繰り返しによる同一イベント通知の抑制機能
    -   個別通知をまとめてPolicyで提供し、特定サーバーに自動的に適用する機能
    -   再起動の有無、通信障害の持続時間、リソース使用量によるサーバーイベント設定の通知
    -   プロセス個数の増減、CPU使用量、メモリ使用量によるプロセスイベント設定の通知
    -   ファイルログキーワードによるイベント設定およびイベントリスク設定の通知

-   **統計/レポート**

    -   サーバー日間・週間レポート
    -   サーバー月間レポート

-   **管理機能**

    -   権限管理
    -   エージェント削除
    -   メンテナンス計画の事前入力
