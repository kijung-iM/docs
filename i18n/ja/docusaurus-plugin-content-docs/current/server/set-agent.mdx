WhaTapエージェントは、エージェントごとに必要な設定を_whatap.conf_ファイルにに書き込みます。 エージェントは、環境変数を使用して設定ファイルを見つけて読み込みます。 _whatap.conf_ファイルを使用して設定できるオプションは次のとおりです。

<!-- 
와탭 에이전트는 에이전트 별 필요한 설정을 *whatap.conf* 파일에 작성합니다. 에이전트는 환경변수를 통해 설정 파일의 위치를 파악하고 로딩합니다. 사용자의 편의를 위해 ***서버 설정*** 메뉴에서도 에이전트 설정 기능을 제공합니다. *whatap.conf* 파일 및 ***서버 설정*** 메뉴를 통해 설정할 수 있는 옵션은 다음과 같습니다.

## 서비스 화면에서 에이전트 설정하기

홈 화면 > 프로젝트 선택 > ***서버 설정*** > ***서버 설정***

:::note

* 이 기능은 **수정** 권한을 가진 멤버만 이용할 수 있습니다. **수정** 권한이 없는 멤버는 설정 내용을 조회만 할 수 있습니다.

* 옵션 값으로 설정할 수 있는 형식은 다음과 같습니다.

  * Boolean 형식의 값은 `true` 또는 `false`를 선택하세요.
  * 숫자 형식의 값은 숫자만 입력할 수 있습니다.

* 수정할 수 없는 옵션은 선택할 수 없습니다. (예, `license`)

* 추가 또는 수정, 삭제한 옵션에 따라 에이전트를 재시작해야할 수 있습니다. 

* 애플리케이션 종류 및 에이전트의 버전에 따라 적용할 수 있는 옵션 키는 다를 수 있습니다.

:::
