## オペレーティングシステム別のサポート環境

WhaTapのサーバーモニタリングエージェントをインストールする前に、サポート環境を確認してください。X86ベースで動作する **Linux**、**FreeBSD**、**Windows**に適用されます。

<!-- * Rocky Linux 8 이상 -->

-   **Linux**

    -   CentOS 5以降
    -   RedHat Server 5.0以降
    -   Rocky Linux 8.6以降
    -   SUSE 10.1以降
    -   Amazon Linuxすべてのバージョン 
    -   Ubuntu 12.04以降
    -   Debian 6.0以降

-   **Unix**

    -   AIX 6.1以降
    -   HP-UX 11.23以降
    -   SunOS 5.8以降
    -   FreeBSD 10以降

-   **Windows**

    -   Windows 2008 R2以降
    -   Windows 7 SP1バージョン以降

:::note

使用環境がサポート一覧に含まれていない場合は、support@whatap.io までお問い合わせください。

:::
