WhaTapウェブでは、サーバーで指定した任意のテキストファイルの内容を確認できます。 WhaTapサーバーモニタリング構成ファイルに、Web上で検索するテキストファイルを次のように追加します。

-   **Linux Shell**
    ```bash title=SH
    echo "webview.allow.`date +%s%N`=参照するファイルのフルパス" | sudo tee -a /usr/whatap/infra/conf/whatap.conf
    ```

-   **Windows Powershell**
    ```bash title=SH
    Add-Content -Path "C:\Program Files\WhatapInfra\whatap.conf" -Value "webview.allow.$(Get-Date -uformat %s)=参照するファイルのフルパス"
    ```
