ホーム画面 > プロジェクト選択 > **_レポート_**

WhaTapモニタリングサービスの初期画面でプロジェクトを選択し、プロジェクトメニューの下位の **_レポート_**を選択してください。 レポート画面で ![down-arrow](/img/down-arrow.svg) ボタンをクリックすると、日間、週間、月間に分類された様々な分析レポートを確認できます。

## **_日次サーバーレポート_**

**_日間サーバーレポート_**は、1日あたりのサーバーリソース使用量を確認できるレポートです。 開始日、開始時間、終了時間を直接設定して照会することができます。

### 概要

![](https://img.whatap.io/media/images/server_daily_report_01.png)

プロジェクト名とホスト数、発生した通知件数を要約して確認できます。 ホスト件数と通知件数の場合、前日と比べてどれだけ変動があったかを同時に確認できます。

-   **ホスト件数** ：照会期間中のサーバー総数です。
-   **通知件数**：照会期間中に発生した通知の総数です。

### サーバー一覧

![](https://img.whatap.io/media/images/server_daily_report_02.png)

サーバーごとに日間の主要メトリクスを確認できます。 CPU最大値（CPU Max）とメモリ最大値（MEM Max）の場合、その数値が発生した時間を同時に表示します。

アラートの数は、7日前、1日前、および表示日で比較できます。

:::note

CPU、メモリの平均、最大値が90%以上の場合は<span class='loger'>赤</span>として表示されます。

:::

### アラート履歴

![](https://img.whatap.io/media/images/server_daily_report_03.png)

照会した時間中に発生した通知記録について、発生時間、レベル、発生したサーバー名、該当通知が発生した原因を確認することができます。
各通知の詳細情報は**_警告通知_** >**_イベント記録_**メニューで照会することができます。
該当プロジェクトについて通知記録がない場合、「該当期間にデータが存在しません」という案内メッセージが表示されます。

:::note

通知のレベルに応じて <span class='loger'>Critical</span>は <span class='loger'>赤</span>、<span class='wer'>Warning</span>は <span class='wer'>オレンジ</span>、 <span class='mstop'>Info</span>は <span class='mstop'>グレー</span>として表示されます。

:::

## **_週次サーバーレポート_**

**_週次サーバーレポート_**は、一週次のサーバーのCPU、メモリ使用量を確認できるレポートです。
開始日を直接設定することができ、照会日を基準として7日前から照会できます。
開始日から7日間のデータを照会します。

### 全体リソースの使用率一覧

#### CPU

![](https://img.whatap.io/media/images/server_weekly_report_01.png)

照会期間中、各サーバーのCPU使用率を日別照会します。
照会される項目としてはCPU平均、最大95%、最大値があり、各数値はパーセンテージで表示されます。

:::note 

**最大95%**

上位5%のデータを除く最大値を意味します。 サーバー再起動などの理由により瞬間的に高くなった数値を除くためです。

:::

#### メモリ

![](https://img.whatap.io/media/images/server_weekly_report_02.png)

照会期間中、各サーバーのメモリ使用量を日別照会します。
照会される項目には、メモリ使用平均値、最大値があり、各数値はパーセンテージで表示されます。

#### ディスク

![](https://img.whatap.io/media/images/server_weekly_report_03.png)

照会期間中、各サーバーにマウントされたファイルシステムごとのメトリクスです。
マウント位置と全体ディスク、使用率を照会し、各数値はパーセンテージで表示されます。

:::note

-   数値が９０％以上の場合、<span class='loger'>赤</span>で表示されます。 
-   グループ名には`okindName`が表示されます。`okindName`を設定していない場合は空白で表示されることがあります。

:::

## **_月間サーバーレポート_**

**_月間サーバーレポート_**は、月間サーバーのリソースの使用量と情報を確認するためのレポートです。
開始日を直接設定することができ、照会日を基準として30日前から照会できます。

### 概要

![](https://img.whatap.io/media/images/server_monthly_report_01.png)

1 か月間のサーバーの合計リソース使用量を集計して履歴を表示することができます。

-   **全体ホスト**：照会期間中のサーバー総数です。
-   **全コア**：照会期間中に使用された総コア数です。
-   **全体ディスク**：照会期間中にマウントされた全体ファイルシステムの合計サイズです。
-   **全体通知**：照会期間中に発生した通知の全体件数です。

### サーバー情報

![](https://img.whatap.io/media/images/server_monthly_report_02.png)

クエリ期間中に稼働していたサーバーの IP アドレス、OS 情報、CPU 情報、およびメモリ使用量を表示します。

### アラート内容

![](https://img.whatap.io/media/images/server_monthly_report_03.png)

クエリ期間中のすべてのサーバーの通知の合計数を確認したり、サーバーごとに発生したアラートを週ごとに表示したりできます。

### サーバー使用一覧

![](https://img.whatap.io/media/images/server_monthly_report_04.png)

サーバーごとのCPU平均、最大値、メモリ平均、最大値を日単位で一目で確認できます。

:::note

数値が９０％以上の場合、<span class='loger'>赤</span>で表示されます。 

:::
