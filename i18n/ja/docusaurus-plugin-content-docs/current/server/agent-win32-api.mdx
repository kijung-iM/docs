title: Win32 APIの性能メトリクス収集
description: Win32 APIを使用して性能メトリクスを収集する手順について説明します。
tags:
  - Server

---

CPU、MEMORY、DISK、TRAFFIC、PROCESSの性能情報を収集する場合、Win32APIを使用します。Windows性能カウンターがオフまたは、使用不可能な環境で性能を収集する場合に使用できます。

WhaTapサーバーモニタリング設定ファイルに「native_api.enabled=true」オプションを追加します。 パワーシェルで以下を適用してます。

```bash title='Windows Powershell'
Add-Content -Path "C:\Program Files\WhatapInfra\whatap.conf" -Value "native_api.enabled=true"
Restart-Service "Whatap Infra"
```
