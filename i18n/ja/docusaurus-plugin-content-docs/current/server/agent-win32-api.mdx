---

CPU、MEMORY、DISK、TRAFFIC、PROCESSの性能情報を収集する際にはWin32APIを使用します。 Windows性能カウンターがオフになっているか、使用不可能な環境で性能を収集する場合に使用できます。

WhaTapサーバーモニタリング設定ファイルに「native_api.enabled=true」のオプションを追加します。 パワーシェルで次のように適用してください。

```bash title='Windows Powershell'
Add-Content -Path "C:\Program Files\WhatapInfra\whatap.conf" -Value "native_api.enabled=true"
Restart-Service "Whatap Infra"
```
