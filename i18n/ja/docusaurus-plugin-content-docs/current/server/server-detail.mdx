ホーム画面 > プロジェクト選択 > **_サーバー一覧_** > **_サーバー詳細_**

WhaTapモニタリングサービスの初期画面でプロジェクトを選択し、**_サーバー一覧_**メニューの**_サーバー詳細_**メニューを選択してください。**_サーバー詳細_**は、サーバーのリソース現況と実行プロセスの詳細情報を提供します。

![サーバー詳細](/img/server-detail.png)

## ![](/img/number-01.png)上段オプション

上段![number 1](/img/number-01.png)の領域で**_データ更新時間_**および検索**_サーバー_**を指定することができます。**_データ更新時間_**および**_サーバー_**指定オプションと当該サーバーの**_概要情報_**を確認できます。

### データ更新時間

サーバー詳細ページのデータは5秒ごとに自動的に更新されます。 

<div style={{textAlign: 'center'}}>

![sc](/img/server-detail-data.png)

</div>

-   ![一時停止アイコン](/img/ic-s-pause.svg)**_一時停止_**：ユーザーはデータの自動更新を停止することができます。 
-   ![再生アイコン](/img/ic-play-grn.svg)**_再生_**：ユーザーは停止した自動更新を再開することができます。
-   ![更新アイコン](/img/ic-refresh2.svg)**_更新_**：ユーザーは現在の時間を基準にデータを更新することができます。

### サーバー選択

**_サーバー選択_**オプションを使用すると、詳細を表示するサーバーを指定できます。選択ウィンドウの一覧で参照するサーバーを選択し、次の![number 2](/img/number-02.png)の領域でサーバーのリソース状態や実行プロセスの詳細情報を確認できます。

## ![](/img/number-02.png)サーバー詳細

### サーバーリソースの状態

![number 2](/img/number-02.png)の領域では、 **_CPU Usage_**、**_Memory Usage_**、**_Network_**、**_Disk I/O_**チャートを使用して、サーバーのリソース状況を確認することができます。

-   ![csvアイコン](/img/ic-csv-download.svg)：**_CSV_**アイコンを選択すると、チャート履歴をCSVファイルとして保存することができます。

-   ![カメラアイコン](/img/ic-camera.svg)：**_カメラ_**アイコンを選択して、画像形式でチャート画面を保存することができます。

-   ![詳細アイコン](/img/ic-zoom-in.svg)：**_詳細_**アイコンを選択すると、CPU使用率、メモリ使用率、ネットワーク、ディスクのI/Oなどの詳細チャートが表示されます。

    -   CPU使用率

        **_CPU Usage_**, **_CPU Idle_**, **_CPU Nice_**, **_CPU I/O Wait_**, **_CPU Steal_**, **_CPU IRQ_**, **_CPU Soft IRQ_**, **_CPU Load_**

    -   メモリ使用率

        **_Memory Usage_**, **_Memory Available_**, **_Memory SReclaimable_**, **_Memory SUnreclaim_**, **_Memory Slab_**, **_Memory Swap Used_**,**_ Memory Page Faults_**

    -   ネットワーク

        **_Traffic In/Out_**. **_Packet In/Out_**, **_Error In/Out_**, **_Dropped In/Out_**

    -   ディスクI/O

        **_Disk I/O_**, **_IOPS Read/Write_**, **_Disk Bph Read/Write_**, **_Used Space_**, **_Queue Length_**, **_Inode Used_**, **_Free Space_**

### プロセス一覧

#### 配置

-   **_プロセス一覧_**上段で選択したメトリクスに従い、降順または昇順のソートを指定して参照することができます。 

-   **_プロセス一覧_**の上段で選択したメトリクスによって上位の**_10個_**、**_20個_**、**_30個_**、**_全体_**プロセスを指定して確認することができます。 

#### グループ化

プロセスグループは、1つのインスタンスで同じ動作を行う複数のプロセスをまとめたものです。**_グループ化_**ボタンを選択してプロセスグループを追加することができます。プロセスグループ化の詳細については、[次の文書](server-detail-process-group)を参照してください。

### サーバー情報パネル

**_サーバー情報_**パネルで確認できる内容は次のとおりです。**_ディスプレイ名_**と**_メモ_**の場合**_サーバー情報_**パネルのみ編集が可能です。

-   ![通知アイコン](/img/ic-alert.svg) **_サーバーアラートリスト_**

    **_サーバー情報パネル_**上部の![通知アイコン](/img/ic-alert.svg)**_サーバー通知一覧_**ボタンを選択すると、現在のサーバーに割り当てられた通知を確認することができます。

-   **_Display Name_**/**_ID_**

    選択したサーバーの**_表示名_**(ONAME別名)と**_サーバーID_**(OID)を確認できます。![編集アイコン](/img/ic-edit.svg)**_修正_**アイコンを選択すると**_表示名_**を変更することができます。 

    :::note

    ONAMEは、デフォルト値としてホスト名が設定されます。 

    :::

-   **_OS_**

    選択したサーバーの**_OS情報_**を確認することができます。

-   **_IP Address_**

    選択したサーバーの**_IPアドレス_**を確認することができます。

-   **_Agent_**

    選択したサーバーの**_エージェントのバージョン_**を確認することができます。

-   **_Memo_**

    作成した**_メモ_**を確認する![修正アイコン](/img/ic-edit.svg)**_編集_**アイコンを選択してメモを編集および追加します。
