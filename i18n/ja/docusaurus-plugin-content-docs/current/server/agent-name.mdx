WhaTapは、モニタリング情報収集対象のインフラサーバーの識別のために、基本的にはサーバーから収集した情報を活用します。 デフォルト値は、サーバーのホスト名（hostname）です。

{@include: ../common-items/_agent-name-warning.mdx}

:::note

ユーザーが設定したエージェント名を、[WhaTapのモニタリングサービス](https://service.whatap.io)で確認するには、サーバーを再起動してください。

:::

ユーザーフレンドリーな名前を指定する場合、次のように設定ファイルに指定できます。

-   **Linux Shell**

    ```bash title=SH
    export WHATAP_SERVER_NAME=ユーザーフレンドリーな名前

    echo "oname=$WHATAP_SERVER_NAME" |sudo tee -a /usr/whatap/infra/conf/whatap.conf
    ```

-   **Windows Power Shell** 

    ```bash title=SH
    # 管理者権限が必要です。
    $WHATAP_SERVER_NAME="ユーザーフレンドリーな名前"

    Add-Content "C:\Program Files\WhatapInfra\whatap.conf" "oname=$WHATAP_SERVER_NAME"
    ```
