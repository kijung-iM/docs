WhaTapサーバーモニタリングサービスは、1分以内にインストールが可能で実務者にとって簡単に適用することができ、モニタリングが容易です。 CPU、Disk I/Oなど、瞬間的に重要な間隔を超える可能性のあるメトリクスに対しては5秒単位で収集することで、平均化検出方式の制限を克服します。また、20 秒間隔でのプロセスモニタリングも提供します。 
統合モニタリングの複数のダッシュボードを提供することで、システム全体の現状把握が容易になります。 クラウド環境モニタリングに適したメトリクスを提供し、スケールアウト環境にも適用できます。 

WhaTapのサーバーモニタリングサービスを利用すると、オペレーターを必要とせずサーバーを安定して運用することができます。

## サーバーモニタリングの特長

-   **あらゆる規模のサーバーモニタリング**<br/>
      統合モニタリングのための様々なタイプのダッシュボードが用意されており、システム全体をリアルタイムで確認することができます。 

-   **サーバーインベントリ管理**<br/>
      管理対象サーバーの一覧は、ユーザーが表示したい項目を選択可能なテーブル形式で提供し、小中規模のサーバーの一覧管理だけでなくリアルタイムモニタリングも可能です。

-   **リアルタイム統合通知**<br/>
      リソースの推移をユーザーがモニタリングすることで異常なメトリクスを検出するためのアクティブな応答からユーザーを開放するために、リアルタイムの統合通知履歴が提供され、通知履歴に基づいて障害対応が可能になります。 サーバー、プロセス、キーワード検出方式の通知を提供します。

-   **オペレータ向けのWhaTapレポート**<br/>
      大量のリソースを確実に運用する必要があるオペレーターにとって、レポート用のデータを手動で作成する手間を省くために、さまざまな種類のレポートを提供します。 ユーザーが必要とするレポートがある場合は、フィードバックに基づいてレポートタイプを追加して提供します。

-   **ウェブとモバイルでいつどこでもモニタリング**<br/>
      [サービス型ウェブアプリケーション](https://service.whatap.io)及びモバイルアプリ（android、ios）により、世界中どこでもリアルタイムでモニタリングと通知の受信が可能です。

-   **クラウド性能の脆弱性を検出**<br/>
      クラウド環境アプリケーションで発生する可能性のある脆弱性の自動検出と通知を提供します。

-   **適切なシステム規模の判断が可能**<br/>
      毎年必要なサーバーの規模は異なります。 クラウドを利用している場合、毎週、毎月ごとにリアルタイムで変化する場合があります。これはモニタリングを通じて予測し、適正なレベルの規模を算定します。

-   **プロセス性能の自動収集**
    すべてのプロセスのCPU、メモリ、I/O使用量を自動的に20秒単位で収集します。

-   **マルチチャネルおよびパーソナライズされた通知**<br/>
        電子メール、SMS、モバイルプッシュ、3rd Party Pluginを使用してパーソナライズされた通知サービスを提供します。
    :::note
    WhaTapモバイル専用Appは、**Public SaaS**に限り、使用できます。
    :::

-   **クラウド、物理サーバー、ハイブリッド環境**<br/>
      物理サーバーだけでなく、クラウド環境のモニタリングに適したメトリックスを提供し、スケールアウト環境にも適用できます。
