WhaTapサーバーモニタリングサービスは、1分以内にインストールが可能で実務者にとって簡単に適用することができ、モニタリングが容易です。 CPU、Disk I/Oなど、瞬間的に重要な間隔を超える可能性のあるメトリクスに対しては5秒単位で収集することで、平均化検出方式の制限を克服します。また、20 秒間隔でのプロセスモニタリングも提供します。 
統合モニタリングの複数のダッシュボードを提供することで、システム全体の現状把握が容易になります。 クラウド環境モニタリングに適したメトリクスを提供し、スケールアウト環境にも適用できます。 

WhaTapのサーバーモニタリングサービスを利用すると、オペレーターを必要とせずサーバーを安定して運用することができます。 

{@video: https://whatap-landing-resource.s3.ap-northeast-2.amazonaws.com/prd/application/monitoring_step_1.mp4}

## 主な特長

-   **多様な規模のサーバーモニタリング**

      統合モニタリング向けの様々な形のダッシュボードを提供し、システム全体の現況がリアルタイムで俯瞰できます。 

-   **サーバーインベントリ管理**

      管理対象サーバーの一覧は、ユーザーが表示したい項目を選択可能なテーブル形式で提供し、小中規模のサーバーの一覧管理だけでなくリアルタイムモニタリングも可能です。

-   **リアルタイム統合アラート**

      リソースの推移をユーザーがモニタリングすることで異常なメトリクスを検出するためのアクティブな応答からユーザーを開放するために、リアルタイムの統合通知履歴が提供され、通知履歴に基づいてトラブル対応が可能になります。サーバー、プロセス、キーワード検出方式の通知を提供します。

-   **システム運用担当向けの運用レポート**

      大量のリソースを確実に運用する必要があるオペレーターにとって、レポート用のデータを手動で作成する手間を省くために、さまざまな種類のレポートを提供します。 ユーザーが必要とするレポートがある場合は、フィードバックに基づいてレポートタイプを追加して提供します。

-   **クラウド性能脆弱性の検知**

      クラウド環境下のアプリケーションから発生し得る脆弱性を自動検知し、アラート通知します。

-   **適切なシステム規模の判断**

      毎年必要なサーバーの規模は異なります。 クラウドを利用している場合、毎週、毎月ごとにリアルタイムで変化する場合があります。これはモニタリングを通じて予測し、適正なレベルの規模を算定します。

-   **プロセス性能の自動収集**

      すべてのプロセスのCPU、メモリ、I/O使用量を20秒間隔で自動収集します。

-   **マルチチャネルおよびパーソナライズされた通知**

      eメール、モバイルプッシュ、サードパーティプラグインを通じたアラート受信ができます。

-   **クラウド、物理サーバー、ハイブリッド環境**\
      物理サーバーだけでなく、クラウド環境のモニタリングに適したメトリックスを提供し、スケールアウト環境にも適用できます。
