ホーム画面 > プロジェクト選択 > **_アラート通知_** > **_イベント記録_**

警告通知が発生した履歴を確認できます。 発生したすべてのイベント履歴一覧の照会が可能です。 

![サーバーイベント記録sc](/img/event-history-server.png)

-   ![number1](/img/number-01.png) **上段オプション**

    -   **_時間選択_**

        -   右側に位置する緑色のボタンを選択して照会時間を選択することができます。基本設定は12時間です。

        -   ![左アイコン](/img/left-arrow.svg)または ![右アイコン](/img/right-arrow.svg)ボタンを選択すると、選択した表示時間だけ間隔を移動します。

        -   詳細な時刻を選択するには、日付または時間の領域を選択してください。詳細な時間を設定し、**_確認_**ボタンを選択します。

    -   **_入力ウィンドウ_**  

        入力ウィンドウでサーバー名またはイベント名で検索できます。

    -   **_進行中イベントのみ表示_**

        イベント記録の一覧で **_進行中のイベント_**を照会できます。

    -   **_イベント解消時刻表示_**

        イベント記録一覧に**_イベント解消時刻_**カラムを追加できます。 

        :::note

        進行中のイベントは、**_イベント解消時刻_**を照会しません。

        :::

    -   **_CSV_**

        照会したイベント記録結果をcsvファイルとして保存することができます。**_最大CSV行数_**を入力し **_ダウンロード_** ボタンを選択してください。

-   ![number2](/img/number-02.png)**イベント記録一覧**

    -   **_深刻度_**

        危険度は <span class='vslow'>Fatal</span>（危険）, <span class='slow'>Warning</span>（警告）の色として表示されます。

    -   **_イベント名_**  

        イベントポリシーで設定した値に基づいて発生したイベント名です。

    -   **_イベント発生時刻_**  

        イベント発生時間です。

    -   **_+処理履歴_**  

        イベント処理履歴を追加できます。

    -   **_サーバー_**  

        サーバー名です。 

    -   **_スナップショット_**  

        発生したイベントに対する **_CPU_**、**_Memory Used_**、**_ディスク_**、**_ネットワーク_**に関するスナップショット情報です。 

### イベント詳細情報

イベント記録リストから該当 **_イベント名_**を選択すると、イベント発生の前後5分間の **_詳細_** 情報に移動します。

![サーバーイベント詳細](/img/server-event-detail.png)

-   ![number1](/img/number-01.png) **チャートウィジェット**

    通知発生の前後5分間の **_CPU_**、**_Memory Usage_**、**_Disk I/O_**、**_Network_** チャートを表示します。イベントに該当する部分の **_危険度_**を色として表示しています。

-   ![number2](/img/number-02.png) **プロセス**

    イベントに該当するプロセスの上位10個の情報を照会できます。 

-   ![number3](/img/number-03.png) **処理履歴詳細情報**

    イベントの処理履歴を照会することができます。**_処理履歴の追加_**ボタンを選択すると、履歴を作成して保存できます。
