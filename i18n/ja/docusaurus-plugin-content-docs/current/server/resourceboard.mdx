ホーム画面 > プロジェクト選択 > **_ダッシュボード_** > **_リソースボード_**

WhaTapモニタリングサービスの初期画面でプロジェクトを選択し **_プロジェクトメニュー_** 下位の **_ダッシュボード_** > **_リソースボード_**を選択してください。

![](/img/server-resourceboard.png)

**_リソースボード_**で、1つのプロジェクトに登録されているすべてのサーバーをモニタリングできます。 

プロジェクト内のすべてのサーバーの要約情報とリアルタイムリソース使用量の変化が確認できる **_CPU Resource Map_**を提供します。 全体リソースの規模、CPU、メモリ、プロセス使用率の **_Top 5_**を表示します。**_リソースボード_**を通じてトラブル状況を直ちに認知し、対応することができます。

プロジェクトに登録されている全体のサーバーの要約メトリクスを提供します。 10秒単位で更新されます。

-   **Servers**<br/>
      全体サーバーの数とトラブルのあるサーバーの数を提供します。 領域を選択すると **_サーバー一覧_** ページに移動します。
    -   例外状態：<span class='vslow'>危険</span>、<span class='slow'>警告</span>
    -   正常サーバー：<span class='snormal'>普通</span>, <span class='k8s-normal'>非活性</span>
-   **Total Cores**<br/>
      全体のサーバーのコア（vCPU）の合計です。
-   **Avg CPU**<br/>
      全体のサーバーのCPU平均使用量（%）です。 以下のゲージは、平均値、最大値、最小値を示します。
-   **Avg Memory**<br/>
      サーバー全体のメモリ平均使用量（%）です。 以下のゲージは、平均値、最大値、最小値を示します。
-   **Avg Disk**<br/>
      全体のサーバーのディスクデバイスの平均使用量（%）です。 以下のゲージは、平均値、最大値、最小値を示します。
-   **OS**<br/>
      プロジェクト内のlinuxとWindowsオペレーティングシステムの数です。

## ![](/img/number-03.png) CPU Resource Map

全体のサーバーのCPU使用量を示す分布図です。10分間のデータを表示し、10秒周期で更新されます。 セル色の濃さは、その領域にサーバーのリソース使用量が多く分布していることを意味します。

-   **詳細情報提供**<br/>
      セル領域をドラッグすると、詳細情報を提供します。
      ![](https://img.whatap.io/media/images/Screenshot_2021-03-17_server_14861_-\_Server_Monitoring18.png)
    -   サーバー一覧は、範囲内で収集されたデータの最大値に基づいてソートされます。
    -   基本5つのサーバーが選択されており、選択されたサーバーの使用量の推移をマルチラインチャートで提供します。
    -   **_更新_** ボタンを手動でクリックすると、現在のデータに更新できます。 

:::note

**マルチラインチャート**

1つのチャートに複数の線が見えるチャートです。

:::

## ![](/img/number-04.png)リソースイコライザー

**_CPU_**、**_Memory_**、**_Disk I/O_**、**_Disk IOPS_**の項目に対し、上位5つのサーバーの一覧をリアルタイムで表示します。

例えば、Disk I/O（%）メトリクスは、5秒間のディスク使用率を示しています。 Disk I/O（%）が80%を超えると、システムの性能に影響を与える可能性があります。 警告となるデフォルト値は 90%です。 Disk I/O（%）が100%なら、ディスクが休まずに働いているという意味です。

-   イコライザーチャート領域を選択すると、対象**_サーバー詳細_**ページに移動します。 
-   ![right-arrow](/img/right-arrow.svg)ボタンを選択すると **_リソースイコライザー_** ページに移動します。 サーバー全体のリアルタイム使用量が確認できます。

## ![](/img/number-05.png)Process Top 5

CPU、メモリの最大占有率を基準に上位5つのプロセス一覧を提供します。 ![right-arrow](/img/right-arrow.svg)ボタンを選択すると、次のようにサーバー全体に対する **_プロセス一覧_**が確認できます。
