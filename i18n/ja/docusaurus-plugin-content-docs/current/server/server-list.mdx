ホーム画面 > プロジェクト選択 > **_サーバー一覧_** > **_サーバー一覧_**

WhaTapモニタリングサービスの初期画面でプロジェクトを選択し、プロジェクトメニューの下位の **_サーバー一覧_** > **_サーバー一覧_** メニューを選択してください。 

**_サーバー一覧_**は、収集サーバーの全体の現状を示します。サーバーの管理機能およびサーバー間の比較機能を提供します。

![](/img/server-list.png)

## ![](/img/number-01.png)サーバーステータス

![](https://img.whatap.io/media/images/Screenshot_2021-03-17_server_14861_-_Server_Monitoring24.png)

収集サーバーのステータスごとの現状を示します。 ステータスは1分おきに更新します。

-   ![非活性アイコン](/img/ico-disable-agent.svg)**非活性**：エージェントは、そのサーバーのデータ収集が不可能な状態です。

-   ![危険アイコン](/img/ic-warning-critical.svg)**危険**：収集サーバーで危険イベントが発生した状態です。

-   ![警告アイコン](/img/ic-warning-filled.svg)**警告**：収集サーバーで警告イベントが発生した状態です。

-   ![正常アイコン](/img/ic-check-circle-filled.svg)**正常**：エージェントが収集サーバーで正常に動作している状態です。

-   ![アイコン](/img/ic-s-pause.svg)**一時停止**：エージェントがデータ収集を一時的に停止した状態です。

## ![](/img/number-02.png)データ更新

サーバー一覧ページのデータは60秒ごとに自動的に更新されます。

![sc](/img/server-list-data-rn.png)

</div>

-   ![一時停止アイコン](/img/ic-s-pause.svg)**_一時停止_**：ユーザーはデータの自動更新を停止できます。 
-   ![再生アイコン](/img/ic-play-grn.svg)**_再生_**：ユーザーは停止した自動更新を再起動できます。
-   ![更新アイコン](/img/ic-refresh2.svg)**_更新_**：ユーザーは現在の時間を基準にデータを更新できます。

## ![](/img/number-03.png)サーバーステータス

エージェントが割り当てられたサーバー全体のステータスを示します。一覧でエージェントを選択すると、**_サーバー詳細_**ページに移動します。**_サーバー詳細_**メニューの詳細については、[次の文書](server-detail)を参考にしてください。

### 検索

![number-s](/img/number-03.png)領域上右の **_検索ウィンドウ_**を活用してサーバー名（ONAME）、サーバーID（OID）、タグの中で入力値が含まれているサーバー一覧をフィルタリングすることができます。

### カラム

-   ![カラムアイコン](/img/ic-column.svg) **_カラム選択_**

    ![number-s](/img/number-03.png)領域上段メニューの ![カラムアイコン](/img/ic-column.svg)**_カラム選択_** ボタンをクリックすると、次のようにテーブルに表示するカラムを選択できます。

    ![](https://img.whatap.io/media/images/Screenshot_2021-03-17_server_14861_-_Server_Monitoring27.png)

    :::note

    選択した**_カラム_**はブラウザクッキー値として保存されます。また、ページの更新後もステータスが維持されます。 クッキーの削除またはその他の理由で設定間エラーが発生する場合もあります。その時は、全体選択状態に初期化されます。

    :::

-   **カラム幅設定**

    カラムタイトルのエッジをドラッグしてカラム幅を設定することができます。

-   **カラム配置**

    カラムのタイトルをクリックすると、昇順または降順の並べ替えが可能です。

    :::note

    **カラム幅** および **カラムソート** 設定値は、ブラウザークッキーとして保存され、ページの更新後もステータスが維持されます。クッキーの削除またはその他の理由で設定にエラーが発生した場合は、未設定状態で初期化します。

    :::

### フィルター

![number-s](/img/number-03.png)領域上段メニューの ![フィルターアイコン](/img/ico-filter.svg)**_フィルター_** ボタンをクリックすると、タグを通じてサーバーをフィルタリングできます。 

タグは、エージェントをインストールするときに自動的に割り当てられます。 ユーザー入力値を追加できます。 タグを通じて収集サーバーを識別するか、サーバーのステータスをフィルタリングします。

![](/img/server-list-filter.png)

-   ![number-s](/img/number-04.png)領域でタグを選択して、サーバーのステータスをフィルタリングできます。 

-   エージェントごとのタグを入力して、新しいタグ機能を活性化できます。![number-s](/img/number-05.png)領域下段の入力ウィンドウを確認してください。

### モニタリング管理

エージェントコマンドを通じてサーバーを管理する機能です。 ![サーバ管理アイコン](/img/ico-setting.svg)**_管理_** ボタンをクリックし、一覧から管理機能を適用するサーバーを選択してください。 右側に **_一時停止_**, **_再起動_**, **_解約する_** ボタンが活性化されます。

-   ![一時停止アイコン](/img/ic-s-pause.svg)**_一時停止_**：そのエージェントのデータ収集を一時停止します。サーバー一覧で一時停止として表示されます。

-   ![再起動アイコン](/img/ic-caret-right.svg)**_再起動_**：エージェントを再起動します。

-   ![中断アイコン](/img/ic-delete-critical.svg)**_中断する_**：エージェントの動作を中断します。 サーバー一覧から除外され、サーバーのステータスを確認できません。

### エージェント移送

![移転アイコン](/img/ic-swap.svg)**_エージェント移転_** ボタンを選択して、エージェントを他のプロジェクトに移転できます。

![](https://img.whatap.io/media/images/Screenshot_2021-03-17_server_14861_-_Server_Monitoring34.png)

1.  右画面で、移送する対象プロジェクトを選択します。

2.  左画面で、移送するエージェントを選択してください。

3.  ![right-arrow](/img/right-arrow.svg)**_右矢印_** ボタンをクリックして対象プロジェクトに移転します。

4.  **_保存_** ボタンをクリックすると移転が完了します。

:::note

-   実行直後10分以内に再び同一エージェントを移送する場合、競合現象を引き起こす可能性があります。 
-   収集されたデータは移転されません。 過去のデータ照会のため、既存のプロジェクトはしばらく削除しないでください。

:::

### サーバー間の比較

サーバー間の比較チャートを次のように提供します。 

![](/img/server-list-compare.png)

1.  ![チャート比較アイコン](/img/ic-multiline-chart2.svg)**_比較する_** ボタンを選択してください。

2.  ![number-s](/img/number-06.png)のようにサーバーリストの左側に**_比較する_** カラムが追加されたことを確認してください。

3.  **_비교하기_** 컬럼에서 비교를 원하는 서버를 선택하세요.

4.  비교 기능이 활성화되면 ![number-s](/img/ic-number-07.svg) 영역에서 비교 차트를 조회할 수 있습니다.

:::note

-   比較サーバーは5つまで選択できます。

-   サーバーごとに固有色が割り当てられます。固有色は ![number-s](/img/ic-number-07.svg)**_比較チャート_**でサーバーの区別に使用されます。

-   **_期間照会_** ボタンをクリックすると、過去のデータを照会できます。

:::
