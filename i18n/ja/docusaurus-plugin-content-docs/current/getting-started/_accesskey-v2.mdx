## アクセスキーの確認{#check-accesskey}

**アクセスキー**は、WhaTabサービスを有効にするための固有IDです。  

インストール案内セクションで**_プロジェクトアクセスキーを発行する_**ボタンを選択してください。 **アクセスキー**を自動的に発行した後、次のステップを進みます。

<InDoc product='kubernetes'>

:::caution

-   **アクセスキー**が有効ではない場合、サービスの利用が制限されます。 画面右上の![チャネル問い合わせアイコン](/img/ico-help.svg) **_チャンネル問い合わせ_**により更新または延長できます。

-   **アクセスキー**は、他のプロジェクトに重複使用することができません。**ネームスペースプロジェクト**にも別の**アクセスキー**が必要です。

:::

</InDoc>

:::note

プロジェクトを作成した後は、自動的に**_エージェントインストール_**ページに移動します。 **_エージェントインストール_**ページに移動しない場合は、画面左側のメニューから**_管理_** >**_エージェントインストール_**を選択してください。
