## プロジェクト作成{#create-project}

エージェントをインストールする前に、まずプロジェクトを作成してください。

1.  [WhaTapモニタリングサービス](https://service.whatap.io)に移動し、ログインします。

2.  プロジェクトを作成するには、画面左側のメニューの **_+ <TR sid='project_add_btn'/>_** ボタンを選択してください。

3.  **_製品カタログ_**画面でプロジェクトにインストールする製品を選択してください。

4.  **_プロジェクト名_**、**_データサーバー地域_**、**_タイムゾーン_**などの項目を順番に設定してください。

    <ProdImg img='create-project-v2.png' desc='프로젝트 생성'/>

5.  すべての設定を完了した後は **_プロジェクト作成_** ボタンを選択してください。

:::note

-   **_データサーバー地域_**銀リージョン（クラウドサービスを提供するためにインストールしたデータセンターのグループ）を意味します。 特定のリージョンを選択すると、そのリージョンに属するデータセンターにユーザーのデータを保存します。
-   **_タイムゾーン_**は通知、レポートを生成する基準時間です。
-   複数のプロジェクトをグループ化して管理するには **_プロジェクトグループ_**からグループを選択するか、グループを追加してください。 グループの詳細については、[次の文書](../project/group)を参考にしてください。
-   組織を選択した状態でプロジェクトを追加する場合は、**_組織下位グループ_**を必須に設定する必要があります。

:::
