エージェントをインストールするには、まずプロジェクトを作成する必要があります。

1.  [WhaTapモニタリングサービス](https://service.whatap.io)に移動し、ログインしてください。
2.  モニタリングプロジェクトを作成するには、左側のメニューで**_+ Project_**ボタンを選択してください。
3.  プロジェクト名を入力し、利用するプラットフォームを選択してください。
4.  近くのデータサーバー地域を選択してください。
5.  **_タイムゾーン_**を選択してください。<br/>デフォルト値は（GMT +9:00）Seoul、Tokyo、Yakutskです。
6.  **_保存_**ボタンを選択してください。

:::note

-   WhaTapのモニタリングプロジェクトは、モニタリング単位を意味します。**プロジェクト**に関する詳細は[次の文書](../project)で確認してください。
-   データサーバー地域は、リージョンを意味します。
-   クラウドでリージョンとは、クラウドサービスを提供するために設定したデータセンターです。
-   特定のリージョンを選択すると、そのリージョンに属するデータセンターにユーザーデータを保存します。
:::
