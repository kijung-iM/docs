## PostgreSQLモニタリングV2の新機能{#new-feature}

チャートおよび収集機能を強化したPostgreSQLモニタリングV2をアップデートしました。最新の機能を利用するには、エージェントをアップデートしてください。 

:::note

PostgreSQLモニタリング V2を利用するためのエージェントの最小バージョンは 1.6.7 です。エージェントアップデートの詳細については、[次の文書](agent-manage)を参考にしてください。

:::

-   **DB性能指標の情報提供**

    ![](/img/new-feature-01.png)

    指標に対するヘルプとチューニングに関するコツを追加し、ユーザーが数多くの指標を簡単に理解できるように推奨指標化、カテゴリー化して利便性を提供します。

-   **ダッシュボード機能の改善**

    ![](/img/new-feature-02.png)

    ダッシュボードでは、1つのチャートに複数の指標を追加できる機能を追加しました。また、プリセット機能を追加してユーザーが望む指標を素早く確認することができます。

-   **UI改善**

    インスタンスモニタリングフィルター機能の改善

    ![](/img/new-feature-03.png)

    セッションキル機能UIの改善

    ![](/img/new-feature-04.png)

    インスタンスモニタリングのフィルター機能を改善し、セッションキル機能と関連したユーザーインターフェース(User Iterface)を改善しました。

-   **Flexボードテンプレートおよびウィジェットの追加提供**

    <ProdImg img='new-feature-05-postgresql.png' desc=''/>

    DB指標の多数をメトリクスに切り替えて、様々なテンプレートとウィジェットを追加で提供します。

-   **ログ**

    ![](/img/new-feature-06.png)

    -   **_ライブテール_**と**\*ログトレンド**、**_ログ検索_**、ログ通知機能を通じて**DBログ**を検索して推移を確認することができ、ログ関連の通知を受けることができます。

    -   리눅스의 설치형 DB 서버의 로그 및 **AWS RDS**의 **CloudWatch** 로그, 이벤트를 한 메뉴에서 모니터링할 수 있습니다.
