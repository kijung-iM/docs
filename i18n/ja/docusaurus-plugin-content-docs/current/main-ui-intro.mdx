[WhaTapモニタリングサービス](https://service.whatap.io)にログインすると表示される初期画面の要素を紹介します。 初期画面の基本メニューは**_プロジェクト一覧_**です。

<ImgLang img='main-ui.png' desc='初期画面' />

## ![](/img/number-01.png)固定メニュー領域

-   **メニュー**：左側には、ユーザーが入力したメニューパスが表示します。

-   ![現在のページのテーマアイコン](/img/ico-theme.svg) **テーマ**：現在のページのテーマを変更できます。 デフォルト値は**_Light_**モードです。

    -   **_Light_**：基本的に適用されたテーマです。
    -   **_Dark_**：黒い背景のテーマを適用します。
    -   **_Presentation_**：ビームプロジェクトの使用時に最適化された色を適用します。
    -   **_Add Custom Theme_**：ユーザーが目的の色を設定できます。

-   ![WhaTap Docsアイコン](/img/ic-guide.svg) **WhaTap Docs**

    WhaTap Docsの技術文書ページに移動します。 WhaTap Docsの詳細については、[次の文書](welcome-to-whatapdocs)を参考にしてください。

-   ![顧客サポートアイコン](/img/ico-help.svg) **顧客サポート** / **設定・情報**

    オンラインの顧客サポートを受けることができます。 **_チャットをはじめる_**ボタンを選択して技術および費用/決済などに関するお問い合わせができます。 すでに相談した内容は**_相談一覧_**で確認できます。 有線でお問い合わせをするには![有線接続アイコン](/img/ico-tel.svg)を選択してください。

    お問い合わせ内容の言語設定、お知らせ、広告受信などを設定するには、![設定・情報アイコン](/img/ico-setting.svg)を選択してください。

-   **プロフィールアイコン**：一番右側に位置するアイコンを選択してください。 **_アカウント情報管理_**、**_組織作成_**、**_組織管理_**、ユーザー言語を選択できる機能を提供します。 **_ログアウト_**を選択すると、利用中のサービス画面でログアウトします。

## ![](/img/number-02.png)プロジェクト検索領域

-   **_プロジェクト名 / PCODE_**(テキスト入力欄)：プロジェクト名またはプロジェクトコードを入力して、目的のプロジェクトを検索できます。

-   ![お気に入りアイコン](/img/ico-favorite.svg) **_お気に入り_**：お気に入り設定したプロジェクトが一覧として表示されます。 プロジェクトのお気に入り設定をするには、プロジェクト一覧に表示されたプロジェクトカードビューの左上の![お気に入りアイコン](/img/ico-favorite.svg)を選択してください。 お気に入り設定を解除するには、もう一度![お気に入りアイコン](/img/ico-favorite-set.svg)を選択してください。

-   ![拡張カードビューアイコン](/img/ico-extend-card-view.svg) **_拡張カードビュー_**：プロジェクト一覧をカード形式で確認できます。 プロジェクトの基本情報を一緒に表示します。

-   ![縮小カードビューアイコン](/img/ico-reduction-view.svg) **_縮小カードビュー_**：プロジェクト一覧を**_拡張カードビュー_**より縮小されたカード形式で一覧を確認できます。

-   ![リストビューアイコン](/img/ico-list-view.svg) **_リストビュー_**：プロジェクト一覧を製品カタログ別に分類し、表形式で確認できます。

-   **_15 / page_**：ページあたりの一覧に表示されるプロジェクト数を設定できます。 1ページに最大100個まで表示されるように設定できます。  別のページに移動するには、ページ番号を選択してください。

## ![](/img/number-03.png)組織およびグループ、プロジェクト領域

-   **_非組織_**：組織を選択したり、**_組織_** > **_管理_**メニューに移動することができます。 組織管理の詳細については、[次の文書](project/organization)を参考にしてください。

-   **_+グループ_**：グループを新たに追加できます。 グループ追加の詳細については、[次の文書](project/group)を参考にしてください。

-   **_+プロジェクト_**：プロジェクトを追加できます。 プロジェクト追加の詳細については、[次の文書](project/project-manage#create-project)を参考にしてください。

-   ![検索アイコン](/img/ico-search.svg) **_検索_**：グループまたはプロジェクト、エージェント単位で検索できます。 プロジェクト検索の詳細については、[次の文書](#search-project)を参考にしてください。

-   ![フィルターアイコン](/img/ico-filter.svg)：フィルタリングを設定して、製品カテゴリ別にプロジェクト一覧を確認できます。 設定された値に従って、**_全体プロジェクト_**の下に一覧を表示します。

    <ImgLang img='main-project-filter.png' desc='プロジェクト一覧フィルタリング' />

-   ![上方向アイコン](/img/up-arrow.svg)または![下方向アイコン](/img/down-arrow.svg)ボタンを選択すると、プロジェクト一覧の領域を非表示または表示できます。

-   **_全体プロジェクト_**：全体プロジェクトの一覧を確認できます。 特定のプロジェクトを選択して進入した状態でこのメニューを選択すると、初期のプロジェクト一覧の画面に移動できます。

-   領域の下段にはプロジェクト一覧を確認できます。 グループは![フォルダアイコン](/img/ico-group.svg)の形で表示します。 グループ外プロジェクトは、製品の種類別アイコンとして表示します。

## ![](/img/number-04.png)統合メニュー領域

-   **_メニュー_**：マウスをカーソルしたままドラッグすると、メニュー領域を上下に動かしてサイズを変更することができます。

-   **_サイトマップ_**：ユーザーが入力したメニューパスで選択できるすべてのメニュー一覧を表示して移動します。

    :::note

    進入したメニューのパスまたはプロジェクトによって、**_サイトマップ_**ウィンドウで選択できるメニューは異なる場合があります。

    :::

-   **_プロジェクト一覧_**：全体プロジェクトの一覧を確認できます。

-   **_Flexボード_**：ユーザー定義型の統合ダッシュボードです。 アプリケーション、サーバー、データベース、コンテナなどのプロジェクトから収集したメトリクスでリアルタイムのダッシュボードを作成できます。

    <details>
    <summary>製品別文書</summary>

    -   [アプリケーションモニタリング > Flexボード](java/flex-board)
    -   [サーバーモニタリング > Flexボード](server/flex-board)
    -   [データベースモニタリング > Flexボード](mysql/flex-board)
    -   [Kubernetesモニタリング > Flexボード](kubernetes/flex-board)
    -   [Amazon ECS > Flexボード](amazon-ecs/flexboard)
    -   [Amazon CloudWatch > Flexボード](amazon-cloudwatch/flexboard)
    -   [Azure Monitor > Flexボード](azure/flexboard)
    -   [Naver Cloud Monitoring > Flexボード](ncloud/flexboard)
    -   [Oracle Cloud Monitor > Flexボード](oracle-cloud/flexboard)

    </details>

-   **_統合ダッシュボード_**：ユーザーが権限を保有しているすべてのプロジェクトのデータを照会できるダッシュボードを作成できます。 詳細については、[次の文書](navigation/int-dashboard)を参考にしてください。

-   **_統合メトリクスボード_**：ユーザーが権限を保有しているすべてのプロジェクトのメトリクスデータを照会できるダッシュボードを作成できます。 詳細については、[次の文書](navigation/int-metrics-board)を参考にしてください。

-   **_統合レポート_**：複数のプロジェクトに関するレポートを簡単に作成できます。 詳細については、[次の文書](report/integrated-report)を参考にしてください。

-   **_統合APPグループトポロジー_**：**_トポロジー_**機能に対応する複数のプロジェクトを一括選択し、単一トポロジーとして確認することができます。 詳細については、[次の文書](java/topology-type#int-topology)を参考にしてください。

-   **_リアルタイム通知リスト_**：ユーザーが権限を保有しているすべてのプロジェクトの通知をリアルタイムで確認し、通知状況をひと目で把握することができます。 詳細については、[次の文書](notification/rt-notification)を参考にしてください。

-   **_管理_**：全体グループとメンバーを管理できます。 詳細については、[次の文書](project/integrated-manage)を参考にしてください。

## プロジェクト検索{#search-project}

1.  画面左上の![検索アイコン](/img/ico-search.svg)ボタンをクリックすると、プロジェクト**_検索_**ウィンドウが表示されます。

    <ImgLang img='search-project.png' desc='プロジェクト検索ウィンドウ' />

2.  検索するプロジェクトまたはグループ名を入力すると、一致するプロジェクトの一覧が検索ウィンドウの下に表示されます。 

3.  目的のプロジェクトを選択します。

適切なプロジェクト画面に移動します。

:::note

**_検索_**ウィンドウでエージェントを基準にプロジェクトを検索するには、**_Group/Project_**をクリックし、ドロップダウン一覧から**_Agent_**を選択します。

:::
