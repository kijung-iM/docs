WhaTap AmazonECSモニタリングはMETA APIおよびCgroupディレクトリを通じてDockerコンテナ別資源使用量をリアルタイムで収集します。 Amazon ECS API連動WhaTap Taskを通じてAmazon ECS Service、Deployment、Container Instanceの状態を照会および収集します。

### 主な特長

-   Amazon ECSクラスターでエンドユーザーアプリケーションを運用する際に必要な可視化を提供します。 
-   Task内部のDockerコンテナ別リソース（CPU、Memory、BLKIO、Network）使用量をリアルタイムで収集します。 
-   運用時のTaskが、設計時に想定した限界負荷の範囲内で作動しているかが確認できます。 
-   アプリケーションの負荷がピークの際、リソースが過不足状態に陥ったら、その状況をリアルタイムに視覚化およびアラートして、トラブルに迅速に対応できるようにします。 
-   週次/月次レポートを通じてリソース使用量の推移を把握し、運用計画に反映することができます。
