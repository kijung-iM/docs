WhaTapモニタリングサービスの初期画面でプロジェクトを選択し、**_プロジェクトメニュー_**の下位にある**_Amazon ECS_**メニューからアイテム別ダッシュボードにアクセスしてください。

## Amazon ECS

| ダッシュボードメニュー             | Launch Type  | 説明                                                                                 |
| ----------------------- | ------------ | ---------------------------------------------------------------------------------- |
| **_TASK_**              | Fargate, EC2 | **_Task_**に属するコンテナ別性能**_CPU_**, **_Memory_**, **_Traffic_**, **_BLKIO_**照会         |
| **_Service_**           | Fargate, EC2 | **_Service_**別**_running_**, **_pending_**, **_desired task_**および関連メトリック           |
| **_ContainerInstance_** | EC2          | **_ContainerInstance_**別**_running_**, **_pending_**, **_desired task_**および関連メトリック |

### Task

ECS **_Task_**別の主要指数を視覚化します。

-   **チャート上の表示項**
    -   **_Total Task CPU Usage_**（プロジェクトの総CPU使用量）
    -   **_Total Task Memory_**(**_WorkingSet_**) **_Usage_**（プロジェクトの総メモリ使用量）
    -   **_Task CPU Usage_**(**_millicore_**)（Task別CPU使用量）
    -   **_Task Memory_**(**_WorkingSet_**) **_Usage_**(**_byte_**)(Task別メモリ使用量)
    -   **_Task Block I/O Read Byte_**（Task別Block I/O読み込み使用量）
    -   **_Task Block I/O Write Byte_**（Task星Block I/Oの書き込み使用量）
    -   **_Task CPU Usage_** (%)（Task別制限に対するCPU使用量）
    -   **_Task Memory Usage_** (%)（Task別制限に対するメモリ使用量）
    -   **_Task CPU Limit 설정_** (**_millis_**) (Task別CPUの制限使用量)
    -   **_Task Memory Limit設定_** (**_byte_**)（Task別メモリ制限使用量）
    -   **_Task Network Read Byte_**（Task別ネットワークの受信量）
    -   **_Task Network Transmit Byte_**（Task別ネットワークの送信使量）

### Service

ECS **_Service_**別の主要指数を視覚化します。
![ECS Service](https://img.whatap.io/media/images/aws_ecs_whatap_monitoring_service.png)

-   **チャート上の表示項**
    -   **_DesiredCount_**（希望する数）
    -   **_RunningCount_**（現在の数)
    -   **_PendingCount_**（未解決の数）
    -   **Cpu** （Service TaskのCPU使用量の合算）
    -   **_WorkingSet_**（Service Taskのメモリ使用量の合算） 

### ContainerInstance

ECS **_ContainerInstance_**, **_Service_**主要指数を視覚化します。

-   **チャート上の表示項**
    -   **_RunningCount Per Service_**（サービス別タスクの現在のTask）
    -   **_PendingCount Per Service_**(サービス別未決済)
    -   **_RunningTasksCount Per Node_**（ノード別現在数）
    -   **_PendingTasksCount Per Node_** (ノード別未決数)
    -   **_TrafficIn_**（ノード別ネットワークの受信bps）
    -   **_TrafficOut_**（ノード別ネットワークの送信bps）
    -   **_PacketIn_**（ノード別ネットワークの受信パケット）
    -   **_PacketOut_**（ノード別ネットワークの送信パケット）
    -   **_ReadBps_**（ノードディスク別の読みbps）
    -   **_WriteBps_**（ノードディスク別の書き込みbps）
    -   **_ReadIops_**（ノードディスク別の読み取り回数）
    -   **_WriteIops_**（ノードディスク別の書き込み回数）
    -   **_TotalSpace_**（ノードディスク別のサイズ）
    -   **_UsedSpace_**（ノードディスク別の使用量(byte)）
    -   **_UsedPercent_**（ノードディスク別の使用量％）
