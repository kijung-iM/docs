-   ホーム画面 > プロジェクト選択 > **_ダッシュボード_**

    WhaTapモニタリングサービスの初期画面でプロジェクトを選択し、プロジェクトメニューの下位にある**_ダッシュボード_**メニューを選択してください。 次の機能を活用すると、複雑なログにより簡単にアクセスでき、様々な条件とともにリアルタイムで確認および分析が可能です。 
    -   **_ライブ テール_**
    -   **_ログトレンド_**
    -   **_ログ検索_**

<!-- * ***설정*** -->

-   ホーム画面 > プロジェクト選択 > **_警告通知_** > **_イベント設定_**

    WhaTapモニタリングサービスの初期画面でプロジェクトを選択し、プロジェクトメニューの下位にある**_警告通知_**メニューを選択し、**_イベント設定_**メニューに入ってください。 イベント条件を設定し、メール、SMS、メッセンジャー、AppPushなど様々なパスから通知を受信することができます。
