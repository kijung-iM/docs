WhaTapの.NETアプリケーションモニタリングは、.NETフレームワーク4.5バージョンに対応しており、.NETによるウェブアプリケーションサーバモニタリングサービスを提供します。

.NETフレームワークアプリケーションをモニタリングできます。IIS、Console、サービス、ウィンドウアプリケーションをモニタリングします。DB Query、外部呼び出しを追跡して、どこで呼び出しが遅れているかを見つけることができます。

## 動作方式

ドットネットプログラムの場合、コンパイル後に中間言語（Microsoft Intermediate Language、MSIL）形式のexeまたはdlファイルに変換されます。これを共通言語ランタイム（Common Language Runtime、CLR）にあるJIT（Just-In-Time）コンパイラがネイティブコードに翻訳して動作します。WhaTapは、ドットネットフレームワークで提供するトレーディングAPIを通じて実行中のアプリケーションをモニタリングします。

![トレーシングアーキテクチャ](https://img.whatap.io/media/agent_dotnet/install/profiling-architecture.png)

ドットネットモニタリングサービスを使用するには、モニタリング対象アプリケーションにモニタリングエージェントをインストールする必要があります。エージェントは、ウィンドウにインストール可能なインストーラーとして提供します。

インストールを完了すると、WindowsサービスにWhaTapエージェントが自動登録されます。
