## エージェント削除

**スタートメニューから削除**

1.  ウィンドウで**_スタート_** ![Windows Start](/img/btn-windows.svg) > **_すべてのアプリ_**を選択し、表示された一覧から**WhaTap .NET**アプリを検索します。

2.  アプリを長押しするか、マウスの右ボタンをクリックしてから**_削除_**を選択してください。

**コントロールパネルから削除**

1.  ウィンドウのタスクバーから検索するときは、**コントロールパネル**と入力し、結果から選択してください。

2.  **_プログラム_** > **_プログラム・機能_**を選択してください。

3.  **_WhaTap .NET_**プログラムを選択し、右クリックして**_除去_**または**_除去・変更_**を選択してください。

画面の案内に従ってエージェントを削除してください。

## エージェントのアップデート

エージェントをアップデートするため、既存のバージョンを削除します。その後、新しいバージョンのエージェントをダウンロードして、インストールプロセスを再実行してください。

:::note

.NETエージェントインストールの詳細については、[次の文書](install-agent#install-agent)を参考にしてください。

:::
