WhaTapモニタリングサービスを使用するには、[会員登録](https://www.whatap.io/ko/signup/)後、プロジェクトを作成し、ターゲットサーバーにエージェントをインストールする必要があります。

## エージェントダウンロード

**アクセスキー**を発行すると、**_エージェントファイルダウンロード_**セクションでダウンロードボタンが活性化されます。**_EXEダウンロード_**または**_ZIPダウンロード_**ボタンを選択してエージェントインストールファイルをダウンロードしてください。

:::note

-   セキュリティ設定により.exe形式のファイルがダウンロードできないユーザーのため、.zip形式のファイルも提供します。

-   サーバーのセキュリティのために、ブラウザー経由で直接インストールするよりは、ダウンロードしたインストールファイルを実行することをお勧めします。

:::

## エージェントインストール{#install-agent}

1.  ダウンロードしたインストールファイルをダブルクリックして実行してください。

2.  画面の案内に従って、**_License Key_**項目には**アクセスキー**を、**_WhaTap IP_**項目には**WhaTapサーバーIP**を入力してください。<br/>WhaTapモニタリングサービスのプロジェクト作成画面で、**_アクセスキー_**と**_WhaTapサーバーIP_**の値をコピーすることができます。

    ![.NETインストール - 01](https://img.whatap.io/media/agent_dotnet/install/agent_install_02.png)

3.  **_Next_** ボタンをクリックします。

4.  次の画面で **_Install_** ボタンをクリックします。 インストールを続行します。

5.  **_Finish_** ボタンをクリックして、インストールを終了してください。

<details>
<summary>.NETエージェントインストール後の構成ファイル</summary>

モニタリング情報を収集するためのトレーサー、収集した情報をサーバーに転送するためのエージェントで構成します。

-   _WhaTap.ClrProfiler.dll_：.NET CLR Profilerです。.NETトレーシングAPIと接続されます。

-   _WhaTap.Trace.dll_：プロファイラーから転送するデータを処理し、エージェントに転送します。

-   _whatap_dotnet.exe_：エージェント、トレーサーから転送された情報を収集サーバーに転送するプログラムです。

-   _whatap.conf_：エージェントの環境設定ファイルです。

</details>

## モニタリングの開始

アプリケーションサーバーを再起動すると、エージェントが情報収集を始めます。
