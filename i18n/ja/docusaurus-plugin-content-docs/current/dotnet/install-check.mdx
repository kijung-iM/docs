## サービス動作の確認

**_WhaTap.NET_**サービスが正常登録されており、動作(**_実行中_**)状態であることを確認してください。

管理者権限で`services.msc`コマンドを実行するか、コントロールパネルでサービスアイコンをダブルクリックしてServices管理者を開きます。

![ウィンドウService管理者](https://img.whatap.io/media/agent_dotnet/install/whatap-windows-service.png)

**WhaTap .NET**サービスは、収集したモニタリングデータをWhaTapサーバーに送信します。関連ログは、_C:\\Program Files\\WhaTap .NET\\logs_パスで確認してください。
