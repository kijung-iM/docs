## 詳細分析

**_分析_** > \***_ヒートマップ_**メニューを選択してリアルタイムヒートマップをドラッグすると、下に**_TXトレース_**一覧が表示されます。選択領域のトランザクション情報を**_TXトレース_**一覧で確認することができます。一覧で項目を選択すると、当該トランザクションの**_統計_**情報ウィンドウが確認できます。 WhaTapは、トランザクションの性能を分析するために、名前やクライアント情報などの属性、トランザクションの処理性能、そして各区間別の詳細な実行履歴を収集して表示します。 **_統計_**ウィンドウを通じて詳細な分析要素を次のように見ましょう。 

:::note
ヒートマップの詳細については、[次の文書](trs-view#details)を参照してください。 
:::
