エージェントが収集したデータを使用してトポロジを作成するオプションです。

-   **tx_caller_meter_enabled** <span class='type'>Boolean</span><br/>デフォルト値 `false`<br/>
    トランザクション情報でトポロジー統計を生成します。

-   **sql_dbc_meter_enabled** <span class='type'>Boolean</span><br/>デフォルト値 `false`<br/>
    Database接続情報でトポロジー統計を生成します。

-   **httpc_host_meter_enabled** <span class='type'>Boolean</span><br/>デフォルト値 `false`<br/>
    トポロジーマップでhttpc outbound情報を表現します。
