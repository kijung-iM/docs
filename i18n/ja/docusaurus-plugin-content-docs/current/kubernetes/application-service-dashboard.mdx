ホーム画面 > プロジェクト選択 > **_ダッシュボード_** > **_アプリケーションサービスダッシュボード_**

コンテナ内部で実行中のアプリケーションの状態をリアルタイムで確認できるダッシュボードです。各ウィジェットの正常なデータ出力のためには、Kubernetes環境にノードエージェントをインストールし、コンテナ内部のアプリケーションにもWhaTapエージェントをインストールする必要があります。 

:::note
Java、Node.js、Pythonで開発したアプリケーションに限ってWhaTapエージェントインストールに対応しています。
:::
