ホーム画面 > プロジェクト選択 > **_リソース_** > **_ネームスペース現況_**

WhaTap Kubernetesクラスタープロジェクトおよび下位ネームスペースプロジェクトの現状を確認できます。ネームスペースプロジェクトではアクセスできないメニューです。 

**_ネームスペース現況_**メニューは、プロジェクト一覧画面と似ていますが、次のような違いがあります。

-   **_プロジェクト一覧_**

    ネームスペースプロジェクトを作成した後、そのプロジェクトを他のグループに移動することはできますが、ネームスペースをひと目で把握するのは困難です。

-   **_ネームスペース現況_**

    -   クラスタープロジェクトと同じグループに属していないネームスペースも確認できます。
    -   ひと目で関連プロジェクトをすべて確認できます。

クラスタープロジェクトの場合、Kubernetesプラットフォームを表す<span class='normal'>青色</span>のタグが表示されます。ネームスペースプロジェクトには、**_Namspace_**タグが表示されます。ウィジェット左上のタイトルはプロジェクト名で、ネームスペース名は一番下の**_Namspace_**タグの横に表示されます。

:::note
WhaTap Kubernetesサービスの**クラスタープロジェクト**は、**マスタープロジェクト**の新しい名前です。
:::
