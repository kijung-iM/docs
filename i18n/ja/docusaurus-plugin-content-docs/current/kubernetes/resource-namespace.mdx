ホーム画面 > プロジェクト選択 > **_リソース_** > **_ネームスペース状況_**

WhaTap Kubernetesクラスタープロジェクトおよび補助ネームスペースプロジェクトの状況を確認できます。ネームスペースプロジェクトではアクセスできないメニューです。 

**_ネームスペース状況_**メニューは、プロジェクト一覧画面と似ていますが、次のように違いがあります。

-   **_プロジェクト一覧_**

    ネームスペースプロジェクトを作成した後、プロジェクトを別のグループに移動できますが、ネームスペースを一目で確認するのことは困難です。

-   **_ネームスペース状況_**

    -   クラスタープロジェクトと同じグループに属していないネームスペースも確認できます。
    -   ひと目で関連プロジェクトをすべて確認できます。

クラスタープロジェクトの場合、Kubernetesプラットフォームを表す<span class='normal'>青色</span>のタグが表示されます。ネームスペースプロジェクトには、**_Namspace_**タグが表示されます。ウィジェット左上のタイトルはプロジェクト名で、ネームスペース名は一番下の**_Namspace_**タグの横に表示されます。
