![Out Of Memory Killedコンテナ](https://img.whatap.io/media/images/whatap_kubernetes_analysis_oom_container.png)

**Out Of Memory(OOM)**が発生したコンテナを迅速に検索し、その時点でメモリ使用量の推移(突然または緩やか)を確認できます。これにより、適切なPOD数とメモリのしきい値について客観的に判断できます。

:::note

コンテナのメモリ使用量がメモリ制限を超えた場合、または、コンテナが実行されているノードがメモリ不足になった場合に、Out Of Memoryが発生します。コンテナがコード137で終了した場合、Out Of MemoryによってSIGKILLを受信します。

:::
