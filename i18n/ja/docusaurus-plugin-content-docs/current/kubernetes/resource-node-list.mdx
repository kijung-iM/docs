## ノードリスト

ホーム画面 > プロジェクト選択 > **_リソース_** > **_ノード一覧_**

現在のプロジェクトのノードの現況情報をテーブル形で確認できます。

![ノード一覧](https://img.whatap.io/media/images/whatap_kubernetes_resource_node_list.png)

提供する主な情報は次のとおりです。

| 分類     | 主要情報                                                  |
| ------ | ----------------------------------------------------- |
| CPU    | コア数<br/>使用率<br/>Sys/User使用率                           |
| メモリ    | 容量<br/>使用率<br/>Swap使用率                                |
| ディスク   | 使用率<br/>I/O使用率<br/>inode使用量                           |
| その他    | Private IP<br/>Public IP<br/>稼働時間<br/>エージェントバージョンなど   |

次の機能を追加で提供します。

-   ![比較するアイコン](/img/ico-compare.svg) **_比較する_**：複数のノードを選択し、チャートベースで使用量を比較できます。
-   ![CSVダウンロードアイコン](/img/ico-csv.svg) **_CSVダウンロード_**：画面に表示された一覧データをCSVファイルとしてダウンロードできます。

サーバー一覧ページのデータは、60秒ごとに自動更新されます。ユーザーは、![一時停止アイコン](/img/ico-pause.svg)ボタンをクリックしてデータの自動更新を中止したり![再開アイコン](/img/ico-play-time.svg)ボタンをクリックして自動更新を再起動できます。

## ノード詳細情報

ホーム画面 > プロジェクト選択 > **_リソース_** > **_ノード詳細_**

特定のノードの詳細を確認できます。

![ノード詳細](https://img.whatap.io/media/images/whatap_kubernetes_resource_node_detail.png)

ノード詳細画面は、上段の4つのチャートとノード内のコンテナ詳細情報で構成されています。

-   上段チャート
    -   **_CPU Usage_**
    -   **_Memory Usage_**
    -   **_Network_**
    -   **_Disk Usage_**

-   コンテナ情報

    | 項目               | 説明                                                |
    | ---------------- | ------------------------------------------------- |
    | イメージ             | コンテナイメージ情報                                        |
    | Pod              | Pod名                                              |
    | コンテナ             | Container ID                                      |
    | コマンド             | コンテナ実行コマンド                                        |
    | CpuByLimit       | CPU Limit基準CPUの全体使用率                              |
    | MemoryWsByLimit  | コンテナメモリのLimit基準working set使用量                     |
    | CpuThrottledCnt  | コンテナCPU Throttled件数                               |
    | CpuThrottledTime | コンテナCPU Throttled時間                               |
    | MemLimit         | コンテナメモリLimitサイズ                                   |
    | MemRequest       | コンテナメモリRequestサイズ                                 |
    | MemWs            | コンテナメモリworking set                                |
    | IoReadBytes      | コンテナ全体ブロックデバイスの1秒当たり読み込んだバイトの合計                   |
    | IoReadIops       | コンテナ全体ブロックデバイスの1秒当たり読み込んだ件数の合計                    |
    | IoWriteBytes     | コンテナ全体ブロックデバイスの1秒当たり書き込んだバイトの合計                   |
    | IoWriteIops      | コンテナ全体ブロックデバイスの1秒当たり書き込んだ件数の合計                    |
    | NetRxBytes       | コンテナ全体ブロックデバイスの1秒当たり読み取りバイトの合計                    |
    | NetRxDropped     | コンテナネットワーク受信dropped件数                             |
    | NetRxError       | コンテナネットワーク受信エラー件数                                 |
    | NetRxIops        | コンテナネットワーク受信件数                                    |
    | NetTxByes        | コンテナネットワーク送信データサイズ                                |
    | NetTxDropped     | コンテナネットワーク送信dropped件数                             |
    | NetTxError       | コンテナネットワーク送信エラー件数                                 |
    | NetTxIops        | コンテナネットワーク送信件数                                    |
