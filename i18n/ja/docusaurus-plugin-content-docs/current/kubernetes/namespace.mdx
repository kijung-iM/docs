ホーム画面 > プロジェクト選択 > **_役人_** > **_ネームスペース管理_**

クラスタープロジェクトでは、**_ネームスペース管理_**メニューを通じてネームスペースプロジェクトを管理することができます。ネームスペースプロジェクトでは、そのメニューに進入することはできません。

:::note
클러스터 프로젝트의 **_대시보드_** > **_컨테이너 맵_** 메뉴 화면 위에는 하위 네임스페이스로 이동할 수 있는 리스트 박스를 제공합다.

![ネームスペース移動](https://img.whatap.io/media/images/whatap_kubernetes_container_map_namespace_project_list.png)
:::

### ネームスペースプロジェクトの作成

**_ネームスペース一覧_**でプロジェクトを作成する項目の**_+ プロジェクト追加_**ボタンを選択してください。**_プロジェクト追加_**ウィンドウが表示されたら**_確認_**ボタンを選択してください。

### ネームスペースプロジェクトの削除

1.  **_ネームスペース一覧_**で削除しようとするプロジェクト項目の一番右側の![設定アイコン](/img/ico-setting.svg)ボタンを選択してください。

2.  **_プロジェクト管理_**メニューに移動します。

3.  画面右上の**_プロジェクト削除_**ボタンを選択してください。

:::caution
ネームスペースプロジェクトを削除する場合、ユーザー一覧情報とこれまで収集したデータはすべて削除されます。削除したデータは再復旧できませんので、注意してください。
:::
