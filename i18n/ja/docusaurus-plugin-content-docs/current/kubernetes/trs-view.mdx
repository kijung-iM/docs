ホーム画面 > プロジェクト選択 > **_アプリケーション_** > **_ヒートマップ_**

コンテナ内のアプリケーションのヒートマップ情報を確認できます。ヒートマップは、トランザクション実行件数マップを含むレスポンス時間分布図を表示します。検索条件は、時間選択、分類およびアプリケーションに分けられます。

| 分類          | 説明                                              |
| ----------- | ----------------------------------------------- |
| エージェント      | コンテナ内部にインストールされているアプリケーションエージェント名               |
| エージェント種類    | エージェントがグループ化されたエージェントの種類(`OKIND`)               |
| エージェントサーバー  | エージェントが実行されているKubernetesノード名                    |

![ヒートマップトランザクション](https://img.whatap.io/media/images/whatap_kubernetes_application_hitmap_transaction.png)

ヒートマップの特定区分をドラッグすると、トレース情報が確認できます。トレースデータは、URLベースの検索が可能であり、アクティブスタックが存在するトレースは、マルチトランザクションで行うトレース情報をフィルタリングして表示できます。
