## エージェントのインストール確認

エージェントが正常にインストールされているかを確認するには、**_ダッシュボード_** > **_アプリケーションサービスダッシュボード_**メニューに移動してください。

ダウンロードしたファイルを直接インストールしても**_ダッシュボード_**メニューでエージェントを確認できない場合は、次の事項を確認してください。

-   コンテナで`ps -ef | grep whatap`コマンドを実行し、エージェントオプションを適用したか確認してください。
-   コンテナの_/whatap/logs_パスの内容を確認してください。エージェントログは_logs/{whatapの設定ファイル名}-yyyymmdd.log_形式のファイル名として出力されます。
