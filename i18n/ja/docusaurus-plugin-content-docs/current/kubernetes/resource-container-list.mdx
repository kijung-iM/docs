## コンテナリスト

ホーム画面 > プロジェクト選択 > **_リソース_** > **_コンテナ一覧_**

現在のプロジェクトのコンテナ現況情報をテーブル形で確認できます。

![コンテナ一覧](https://img.whatap.io/media/images/whatap_kubernetes_resource_container_list.png)

主な機能は次のとおりです。

-   コンテナ一覧情報の自動更新（基本60秒、強制更新可能、自動更新中止可能）
-   Node（エージェント）別コンテナ一覧を照会可能
-   コンテナ、Podベースの検索可能

次は表示される情報です。

| 項目            | 説明                                                |
| ------------- | ------------------------------------------------- |
| ContainerId   | Container ID                                      |
| ContainerName | Container Name                                    |
| Namespace     | コンテナが属するネームスペース                                   |
| ReplicaSet    | コンテナが属するレプリカセット                                   |
| CpuTotUsage   | コンテナCPU使用量                                        |
| CpuByRequest  | コンテナCPU要請対比使用率                                    |
| CpuRequest    | コンテナCPU要請                                         |
| CpuByLimit    | CPU Limit基準CPUの全体使用率                              |
| CpuLimit      | コンテナCPU Limit割当量                                  |
| NetRxBytes    | コンテナ全体ブロックデバイスの1秒当たり読み取りバイトの合計                    |
| NetTxByes     | コンテナネットワーク送信データサイズ                                |

## コンテナ詳細

特定のコンテナの詳細を確認できます。また、複数選択を通じてコンテナの状態を互いに比較できます。

![コンテナ詳細](https://img.whatap.io/media/images/whatap_kubernetes_resource_container_detail.png)
