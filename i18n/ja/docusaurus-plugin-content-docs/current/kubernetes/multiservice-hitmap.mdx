ホーム画面 > プロジェクト選択 > **_アプリケーション_** > **_マルチサービスヒートマップ_**

![マルチサービスヒートマップ](https://img.whatap.io/media/images/whatap_kubernetes_application_multiservice_hitmap.png)

複数のサービスを対象にヒートマップチャートを確認し、サービス別の状態を比較分析できます。

検索条件は次のとおりです。

| 項目         | 説明                                                                             |
| ---------- | ------------------------------------------------------------------------------ |
| 開始日        | 検索日                                                                            |
| 開始時間       | 3日、7日、1か月の検索時には使用できません。                                                        |
| 期間         | <ul><li>5分/10分/30分</li><li>1時間/3時間/24時間</li><li>3日/7日/1か月</li></ul>            |
| 区分         | エージェント種類                                                                       |
| アプリケーション   | アプリケーション                                                                       |
