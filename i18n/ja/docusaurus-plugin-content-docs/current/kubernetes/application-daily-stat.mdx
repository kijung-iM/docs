ホーム画面 > プロジェクト選択 > **_アプリケーション_** > **_日付別アプリケーション現状_**

![日付別アプリケーション現状](https://img.whatap.io/media/images/whatap_kubernetes_application_daily_app_stat.png)

日付別アプリケーションの現状を確認できます。検索条件は開始日、区分、アプリケーションであり、区分条件は次のとおりです。

| 分類          | 説明                                              |
| ----------- | ----------------------------------------------- |
| エージェント      | コンテナ内部にインストールされているアプリケーションエージェント名               |
| エージェント種類    | エージェントがグループ化されたエージェントの種類（`OKIND`）               |
| エージェントサーバー  | エージェントが実行中のKubernetesノード名                       |

提供するチャートは次のとおりです。

-   **_1時間当たりのトランザクション数_**
-   **_1時間当たりのユーザー数_**
-   **_1時間当たりのHTTP呼び出し件数_**
-   **_1時間当たりのSQL件数_**
-   **_1時間当たりのSQLパッチ件数_**
-   **_応答時間_**
-   **_1秒当たりのトランザクション_**
-   **_アクティブトランザクション_**
-   **_5分間の合計エラー件数_**
-   **_エラー率_**
