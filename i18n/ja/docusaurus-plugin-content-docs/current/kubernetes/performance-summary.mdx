ホーム画面 > プロジェクト選択 > **_リソース_** > **_パフォーマンス要約_**

最近の時間別チャートを通じてコンテナ/Pod数量、Node数量などクKubernetes主要指標にどのような変化があったかを確認することができます。

![パフォーマンス要約](https://img.whatap.io/media/images/whatap_kubernetes_resource_performance_summary.png)

Node数量ウィジェット右上の![もっと見るアイコン](/img/ico-more.svg) > ![ショートカットアイコン](/img/ico-shortcut.svg)ボタンを順番に選択すると**_リソース_** > **_Node一覧_**メニューに移動します。 

ウィジェットとして提供するNode情報は次のとおりです。

-   直近30分チャート
-   **_Node CPU (%)_**、**_Node Memory (%)_**など

Node CPUウィジェットとメモリウィジェットの右上の![もっと見るアイコン](/img/ico-more.svg) > ![ショートカットアイコン](/img/ico-shortcut.svg)ボタンを順番に選択すると**_分析_** > **_メトリクスチャート_**メニューに移動します。このメニューを使用すると、より詳細な条件でチャートを確認できます。

ウィジェットとして提供するコンテナ情報は次のとおりです。

-   直近10分チャート
-   **_Container Status_**、**_Container CPU (%)_**、**_Container Memory (%)_**など

ウィジェットとして提供するアプリケーション情報は次のとおりです。

-   直近10分チャート
-   **_Active TX_**、**_TPS_**、**_Response Time_**など

:::note
アプリケーション情報は、アプリケーションエージェントのインストール対象が存在するときに表示します。
:::
