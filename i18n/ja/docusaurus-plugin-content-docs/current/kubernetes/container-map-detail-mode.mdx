**_コンテナマップ_**メニューで選択した対象に対する詳細情報を、**_詳細を見る_**画面で確認できます。

1.  **_コンテナマップ_**メニュー画面で特定の対象（個別、集合）を選択してください。 

2.  画面右側に**要約を見る**画面が表示されます。 

  ![コンテナマップ - 要約を見る](/img/kube-container-map-detail-button.png)

1.  右上の![指示線1](/img/number-01.png)![詳細表示アイコン](/img/ico-detail-mode.svg)ボタンを選択すると、より詳しい情報が確認できます。

コンテナマップ**詳細を見る**画面は、次のように構成されています。

![詳細を見る画面](/img/container-map-detail-mode-desc.png)

-   ![指示線2](/img/number-02.png)対象カード（コンテナまたはPod）

-   ![指示線3](/img/number-03.png)対象の一覧

-   ![指示線4](/img/number-04.png)メニュー別画面
    -   **_トレース_** > **_ダッシュボード_**、**_トランザクション一覧_**
    -   **_メトリクス_**
    -   **_ログ_** > **_最近のログメッセージ_**、**_ログ検索_**
    -   **_イベント_** > **_Kubernates_**、**_WhaTap_**
