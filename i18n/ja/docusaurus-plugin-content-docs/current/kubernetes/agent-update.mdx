エージェントのアップデートが必要な場合は、次のコマンドを実行してください。

<Tabs groupId='k8stype'>

<TabItem value='kubernetes' label='KUBERNETES' default>

-   **マスターエージェント**

    ```bash
    kubectl set image deployment/whatap-master-agent whatap-master-agent=whatap/kube_mon:{エージェントバージョン} -n whatap-monitoring
    ```

-   **ノードエージェント**

    ```bash
    kubectl set image daemonset/whatap-node-agent whatap-node-agent=whatap/kube_mon:{エージェントバージョン} -n whatap-monitoring
    ```

</TabItem>

<TabItem value='eksfragate' label='EKS Fargate'>

**マスターエージェント**

```
kubectl set image deployment/whatap-master-agent whatap-master-agent=whatap/kube_mon:{エージェントバージョン}
```

</TabItem>

</Tabs>

:::note
WhaTap Kubernetesエージェントの最新バージョンは、[hub.docker.com](https://hub.docker.com/r/whatap/kube_mon/tags)で確認できます。
:::
