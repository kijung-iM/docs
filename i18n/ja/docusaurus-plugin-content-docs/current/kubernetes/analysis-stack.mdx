![스택](https://img.whatap.io/media/images/whatap_kubernetes_analysis_stack.png)

WhaTapは10秒（デフォルト値）間隔で収集したThread Stackを活用してメソッドレベルの性能遅延区間を分析します。この機能により、各コンテナの***トップスタック***、***ユニークスタック***、***アクティブスタック***を確認することができます。

:::note

***トップスタック***、***ユニークスタック***、***アクティブスタック***の詳細については、次の文書を参考にしてください。

-   [Java](../java/analysis-apm)
-   [Python](../python/analysis-apm)

:::
