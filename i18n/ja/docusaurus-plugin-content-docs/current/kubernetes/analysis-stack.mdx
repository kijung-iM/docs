ホーム画面 > プロジェクト選択 > **_分析_** > **_スタック_**

![スタック](https://img.whatap.io/media/images/whatap_kubernetes_analysis_stack.png)

WhaTapは10秒（デフォルト値）間隔で収集したThread Stackを活用してメソッドレベルの性能遅延区間を分析します。この機能により、各コンテナの**_トップスタック_**、**_ユニークスタック_**、**_アクティブスタック_**を確認することができます。

:::note

**_トップスタック_**、**_ユニークスタック_**、**_アクティブスタック_**の詳細については、次の文書を参考にしてください。

-   [Java](../java/analysis-apm)
-   [Python](../python/analysis-apm)

:::
