ホーム画面 > **_リアルタイム通知一覧_**

![リアルタイム通知一覧](/img/rt-notification.png)

ユーザーが権限を持つすべてのプロジェクトの通知をリアルタイムで確認でき、通知の状況を一目で確認することができます。タイムライン、通知タイトル、プロジェクト、アプリケーションを分類して確認することができます。

リアルタイムで通知リストを取得する間隔は20秒です。20秒前に通知リストを更新するには、画面右上の**_更新_**ボタンを選択してください。

## 通知検索

画面左上のテキスト入力欄に検索内容を入力し![検索アイコン](/img/ico-search.svg)ボタンを選択します。通知が検索できます。検索結果は、画面下のリストに表示されます。

## グループ、プロジェクト、製品別フィルタリング

通知リストは、グループ、プロジェクト、または製品の種類別にフィルタリングして確認することができます。画面左上の***プロジェクト***ボタンを選択してください。画面の右側に***プロジェクト***ウィンドウが表示されたら***グループ***、***プロジェクト***、***製品種類***別に項目を選択してください。

![通知フィルタリング](/img/rt-notification-filter.png)

-   選択した項目は、通知検索テキスト入力欄の右側に表示されます。複数選択ができます。
-   選択した項目をキャンセルするには、その項目をもう一度選択してください。
-   ***プロジェクト***ウィンドウの![閉じるアイコン](/img/ico-close.svg)ボタンを選択するか、***プロジェクト***ウィンドウ領域の外側をクリックするとフィルタリングした通知一覧が表示されます。
-   **_プロジェクト_**ウィンドウに多くのリストがある場合、**_検索_**を通じて探している項目を検索することができます。
-   必要なアイテムをフィルタリングしながら、通知コンテンツを検索することもできます。

## 通知一覧の時間を設定する

最近発生した通知リストを希望の時間帯別に確認できます。画面右上、**_更新_**ボタンの左側にある時間選択ボックスを選択してください。 **_最近1時間_**（デフォルト値）、**_最近 3時間_**、**_最近 6時間_**、**_最近 12時間_**、**_最近 1日_**を選択することができます。

選択した直近の時間登録に従って発生した通知がタイムラインで確認できます。最も多く発生した通知時間帯を確認するのは簡単です。

## 通知一覧のグループ化

![通知グループ化](/img/rt-notification-grouping.png)

通知リストは、***タイトル***、***プロジェクト***、***アプリケーション***別にグループ化してソートすることができます。グループ化を解除するには、***タイムライン***ボタンを選択します。

:::note
通知が発生した場合、サウンドをオンまたはオフにするには![音アイコン](/img/ico-noti-sound.svg)ボタンを選択します。
:::

## 通知リストから除外する

リアルタイム通知リストの一部を除外できます。 

1.  除外しようとする通知メッセージを検索またはフィルタリングしてリストに表示されます。 

2.  画面右上の**_イベント一括処理_**ボタンを選択してください。

3.  メッセージウィンドウが表示されたら、**_適用_**ボタンを選択してください。

:::note
作業を完了するのに30秒から1分ほどかかります。
:::

## スナップショットの確認

![스냅샷](/img/rt-notification-snapshot.png)

通知一覧で各項目別の一番右側に![スナップショットアイコン](/img/ico-detail.svg)ボタンが表示された場合、その通知に対するスナップショットを確認することができます。![スナップショットアイコン](/img/ico-detail.svg)ボタンを選択すると、画面右側に***Snapshot***ウィンドウが表示されます。

選択した通知に関する詳細は**_Snapshot_**ウィンドウで確認できます。

## 追加機能を活用する

通知一覧では、各項目別の一番右側に![詳細アイコン](/img/ico-more.svg)ボタンを選択して、次の追加機能にアクセスします。

-   **詳細**：該当通知の詳細とステータスが確認できるプロジェクトの関連メニューに移動します。

-   **_イベント記録_**：この通知に対するイベントメッセージが確認できるプロジェクトの**_イベント記録_**メニューに移動します。
